## 使用方式

解卦時若需要查看本卦或變卦的卦義，可先查本資料夾。

建議搭配：

```text
03_hexagrams/卦義
+
02_rules/用神系統
+
02_rules/六親
+
02_rules/動爻變化
+
02_rules/刑沖合
```

## 注意

六十四卦卦義主要作為卦象背景與趨勢提示，實際六爻斷卦仍以用神、旺衰、原神忌神、動變、月日、世應為主。
