# 八字索引

本索引用於快速進入八字資料系統。

## 八字資料位置

八字資料將集中存放於 `bazi/` 底下：

```text
bazi/
├── 00_index/                     八字內部索引
├── 01_foundation/                八字基礎概念
├── 02_ten_gods/                  十神系統
├── 03_stems_branches/            天干地支、藏干、刑沖合害
├── 04_pattern_structure/         格局、旺衰、用神喜忌
├── 05_luck_cycles/               大運、流年、流月
├── 06_question_types/            八字問事分類
├── 07_reading_templates/         八字解析模板
├── 08_case_archive/              八字案例庫
└── 99_sources/                   八字來源資料
```

## 八字適合處理

- 性格底層結構
- 感情模式與依附傾向
- 事業方向與適合職能
- 財運結構與理財傾向
- 健康體質與生活調整方向
- 大運流年趨勢
- 長期人生策略

## 使用方式

```text
如果是問長期性格、命盤結構、喜忌、大運流年、人生方向，優先使用八字系統。
如果是問單一事件成敗，優先使用六爻系統。
重大選擇可用八字看底層方向，再用六爻看當下事件。
```
