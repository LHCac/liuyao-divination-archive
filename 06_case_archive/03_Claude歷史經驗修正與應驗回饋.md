# Claude 歷史對話：經驗修正與應驗回饋

## 主題摘要

本檔整理歷史對話中對未來解卦最有價值的「修正經驗」與「應驗回饋」。這些內容應作為日後解卦時的校正依據，特別是排盤、六親裝卦、月日旺衰、暗動、假破、旬空與失物判斷。

## 一、排盤與六親裝卦錯誤修正

### 1. 同人卦宮位與六親修正

歷史對話中曾將「天火同人」誤用乾宮裝卦，後經使用者糾正，確認同人卦應以離宮資料安裝六親。

修正後重要資訊：

```text
同人卦：離宮
案例002 劉小姐：重新按離宮六親修正
```

經驗：

```text
If 卦宮不確定
Then 不可憑直覺裝卦，需回查 03_hexagrams/ 與納甲資料
```

### 2. 蒙卦宮位修正

屏屏案例中曾先以坎宮思路排盤，後續查資料確認：

```text
山水蒙：離宮四世卦
世爻在四爻，應爻在初爻
```

經驗：

```text
If 本卦為蒙卦
Then 需按離宮四世處理，不可只看下卦坎就判坎宮
```

## 二、日辰與月建修正

### 1. 日辰必須精準

案例001 原始輸入有「子亥日」不清楚，後修正為乙亥日。對話中也反覆提醒：日辰錯一個，月日對各爻的生克沖合完全不同。

經驗：

```text
If 使用者給的日辰模糊
Then 必須先要求確認，不可假設
```

### 2. 日令平相觀念

後續理論補強指出：

```text
爻生日令、爻剋日令，多作平相
不應直接以「日洩用神」判用神衰
```

經驗修正：

| 舊說法 | 修正後 |
|---|---|
| 日辰洩用神，用神稍弱 | 日洩多作平相，不直接扣旺衰 |
| 日辰洩世爻，世爻稍弱 | 日洩世爻多作平相 |
| 靜爻被日沖一律暗動 | 旺相被沖為暗動，衰敗被沖為日破 |

## 三、旺衰分層經驗

歷史對話後期建立了旺衰分層：

```text
動變組合
>
月建高層次：臨月、合月、月破
>
日辰高層次：臨日、合日、日沖
>
一般生克
>
日洩、日耗平相
```

重要原則：

```text
If 月破與日生同時出現
Then 月破高層次大於日生
```

```text
If 動爻化旺或假破成立
Then 動變趨勢可能蓋過日月短期沖破
```

## 四、假破經驗

### 案例021：Yu服飾店客戶來訪

原本判斷客戶來訪機率偏低，後用「假破」重審：

```text
應爻動化旺
變爻與月建形成合
動變組合 > 日月組合
```

後續應驗：

```text
客戶實際 12 號來
原本推 13 號，實際提前一天
```

經驗：

```text
If 動爻變爻逢月合或化旺
Then 不可直接以月破判死，需檢查假破
```

## 五、貪生忘克與通關經驗

對話後期補入「忌神貪生忘克」與「通關」：

```text
忌神原本克用神
但忌神若能生原神
原神再生用神
則忌神力量轉為助力
```

經驗：

```text
If 忌神動
Then 不要馬上判凶，先檢查：
1. 是否貪生忘克
2. 是否有通關爻
3. 通關爻是否旺相或暗動
```

## 六、失物卦應驗修正

### 案例025：美容師小P手機遺失

初判：

```text
兄弟旬空 = 沒有被人拿走
```

後續回饋：

```text
小P說神明指示可能被人拿走
```

修正：

```text
兄弟旬空不等於絕對沒被拿走
若兄弟旬空但臨玄武
仍要保留暗中被取走的可能
```

修正表：

| 情況 | 判斷 |
|---|---|
| 兄弟旬空，無玄武 | 較可排除遭竊 |
| 兄弟旬空，有玄武 | 遭竊跡象不明顯，但不可完全排除 |
| 兄弟旺相，有玄武 | 高度懷疑被偷或被暗中拿走 |

## 八、準確度提升建議

歷史對話中整理出的提高準確度方法：

```text
1. 每次卦後補應驗回饋
2. 日辰、月建一定確認清楚
3. 問題越具體越好
4. 背景資訊越多越能落地取象
5. 排盤與六親裝卦不可省略確認
6. 遇到多動爻，先抓用神主線
7. 不確定時先查 SOP，不直接跳結論
```
