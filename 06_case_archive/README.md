## 使用方式

每次解卦後，可將案例整理為：

```text
問題背景
↓
起卦資訊
↓
本卦、變卦、動爻
↓
用神與旺衰
↓
當時解讀
↓
後續驗證
↓
經驗修正
```

此資料夾屬於「經驗資料庫」，可用來校正後續判斷，但正式解卦仍以 `05_reading_templates/00_六爻解卦SOP.md` 與 `02_rules/` 為主。

## 歷史紀錄使用原則

```text
1. 歷史案例可作為參考經驗，但不可取代正式 SOP。
2. 有應驗回饋的案例優先參考。
3. 早期案例若有排盤或六親錯誤，需以後續修正版為準。
4. 同類型案例可用來做橫向比較，但仍要重新檢查本次卦象。
5. 自占與代占需分開使用，不可混用世應邏輯。
```
