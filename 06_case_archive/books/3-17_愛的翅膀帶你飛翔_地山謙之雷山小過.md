# 書籍案例 3-17｜愛的翅膀帶你飛翔：地山謙 → 雷山小過

## 基本資訊

來源：《六爻知先機》第三篇多層次斷卦訣，第284～289頁。

占卦日期：乙巳年己丑月己卯日。

空亡：申、酉。

本卦：地山謙。

變卦：雷山小過。

占問者：蕭先生，代占，父親。

占問：預測女兒這次期末考成績如何。

問卦類型：代占六親、考試功名。

特殊技法：代占六親、子孫為用神、旬空不等於真空、沖空、父母動沖剋用神、五爻尊位判名次。

## 背景摘要

蕭先生關心女兒期末考表現。此卦不是單純追求高分，而是父親用關愛陪伴孩子面對學業，觀察女兒能否在考試中穩定發揮並名列前茅。

## 排盤整理

坤宮：地山謙，世爻在五爻，應爻在二爻。

日辰：己卯日，卯木妻財。

月建：己丑月，丑土。

## 用神設定

代占女兒考試：

- 子孫爻五爻亥水：女兒，主要用神。
- 父母爻：學業成績、考題、考卷。
- 官鬼爻：功名、名次、入榜。
- 兄弟爻：同學、競爭者。
- 伏神卯木白虎妻財：名次排列與分數排序的關鍵因素。

## 解析重點

### 一、世爻與子孫同居五爻：父女連心

五爻亥水朱雀子孫同時為世爻與子孫用神。

代占六親時，父親占女兒，子孫為女兒。世爻又同居子孫位，表示父女連心、親情融洽，父親的關心不是壓迫，而是深切陪伴。

五爻為尊位，也為後續判斷名次前段提供重要依據。

### 二、旬空不一定是真空

六爻酉金兄弟旬空，但酉金得月建丑土相生，不作真空，仍具備力量。

旬空之爻若旺相或得月日生扶，不能直接判無力。

### 三、旬空逢日沖為沖空，仍具暗動之力

己卯日沖六爻酉金兄弟。

酉金雖旬空，但因得月生不是真空，逢日辰卯木沖，可作沖空，仍具暗動之力。

酉金勾陳兄弟暗動後，生五爻亥水朱雀子孫，表示同學、好友或競爭環境反而幫助女兒發揮。

### 四、父母動沖剋子孫：學業壓力重，但可轉化

父母爻代表學業成績、考題、考卷。父母動沖剋世爻或子孫用神，通常代表學業壓力太重、題目生疏或考題有阻礙。

本卦四爻丑土青龍父母發動，理論上剋五爻亥水子孫，表示考題具挑戰、學業壓力不輕。

但亥水子孫得六爻酉金兄弟暗動相生，因此壓力可被化解。

### 五、丑土父母動化午火官鬼回頭生：題目有深度，關係名次

四爻丑土青龍父母動化午火官鬼回頭生。

父母為成績，官鬼為功名。父母動化官鬼回頭生，表示此次考試題目設計較有深度，成績將關係到排名與名次。

午火官鬼又得日辰卯木生扶，功名力量強。

### 六、丑土貪生忘剋：壓力轉向成全同學與環境，不再直接傷女兒

四爻丑土父母發動，既可剋五爻亥水子孫，也可生三爻申金兄弟。

因申金兄弟可再生亥水子孫，且亥水又有酉金暗動生扶，形成壓力轉化的鏈條。

此處可作丑土貪生忘剋：父母爻不再直接剋子孫，而轉為生兄弟，兄弟再生子孫。

象意為：艱深考題對思維靈活、平日積累好的女兒與同學，反而能形成發揮機會。

### 七、伏神卯木白虎妻財：名次排列的關鍵

卯木白虎妻財伏於午火白虎官鬼之下，可作名次排序與分數排列的關鍵。

卯木出伏後：

- 剋辰土與丑土父母：代表女兒本次分數略低於平日水準。
- 生午火官鬼：功名、名次仍有力。
- 沖六爻酉金勾陳兄弟：部分同學或好友與名次無緣。

### 八、五爻尊位判名次前茅

五爻亥水子孫居尊位，且四爻變爻午火官鬼能透過官鬼名次力量與五爻子孫相互成局。

五爻尊位可作前段、前茅、高位、被看見。

因此雖然分數略低於平時，但女兒仍可在群體中脫穎而出，名列前茅。

## 結果驗證

期末成績揭曉後，女兒分數雖未及平日水準，但仍在群英中脫穎而出，名列前茅，並獲獎狀肯定。

## 可運用技法

| 技法 | 本案例用法 |
|---|---|
| 代占六親 | 父親占女兒考試，以子孫爻為用神 |
| 子孫為用神 | 子孫宜得生得合，不宜受剋受沖 |
| 旬空不等於真空 | 酉金旬空但得月建生，不作真空 |
| 沖空 | 非真空之旬空爻逢日沖，仍具暗動之力 |
| 父母動沖剋子孫 | 學業壓力、題目艱深、考題阻礙 |
| 丑土貪生忘剋 | 父母先轉生兄弟，再由兄弟生子孫 |
| 五爻尊位判名次 | 子孫居五爻，名列前茅、前段被看見 |
| 世爻與子孫同居 | 父女連心、親情融洽 |
| 伏神妻財 | 名次排序、分數略低於平常但功名仍成 |

## 決策建言摘要

代占子女考試時，不可直接以世爻為孩子，應先取子孫為用神。若子孫居五爻尊位，得暗動兄弟生扶，且官鬼功名有力，即使父母題目壓力較重、分數略降，仍可名列前茅。
