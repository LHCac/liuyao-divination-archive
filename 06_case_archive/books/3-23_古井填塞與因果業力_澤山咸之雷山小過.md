# 書籍案例 3-23｜古井填塞與因果業力：澤山咸 → 雷山小過

## 基本資訊

來源：《六爻知先機》第三篇多層次斷卦訣，第320～325頁。

占卦日期：庚子年乙酉月丁巳日。

空亡：子、丑。

本卦：澤山咸。

變卦：雷山小過。

占問者：林女士，自占。

占問：視網膜剝離，是否與六年前建宅古井填塞有關？

問卦類型：疾病占、風水辨爻法、逆向尋病因。

特殊技法：逆向尋病因、藥物副作用、貪生忘剋反效果、月破伏神可出、風水病因排除。

> 注意：疾病占只能作為象意參考，不可替代醫師診斷與治療。

## 背景摘要

林女士罹患視網膜剝離，長期身心煎熬。有人懷疑六年前新建住宅時填塞屋後古井，處理未圓滿而引發病痛。此卦不僅追溯疾病因由，也用風水辨爻法確認古井是否為真正病因。

## 用神設定

疾病占：

- 世爻三爻申金兄弟：林女士本人。
- 二爻午火勾陳官鬼：疾病，長期病症。
- 日辰巳火：亦為官鬼，官鬼能量加重。
- 四爻亥水白虎子孫：藥物、強效性藥品。
- 二爻伏神卯木妻財：飲食生活、副作用、生活代謝關鍵。
- 初爻辰土父母：古井、地基、風水低位。

家宅風水：

- 子孫：聚氣。
- 妻財：宅氣。
- 兄弟：破氣。
- 父母：房屋本體、地基、古井、文書。

## 解析重點

### 一、疾病占基本規則

疾病占中，世爻為本人，官鬼為疾病，子孫為藥物，妻財為飲食生活。

喜日辰或動爻剋官鬼，也喜日辰或動爻生世。

若日辰為官鬼，或官鬼動來剋世，病勢較重。子孫爻為藥物，旺相而動則藥對症；若子孫休囚、空破、受剋或被沖破，藥效不穩。

### 二、世爻騰蛇兄弟持世：重視養生，心中不安

三爻申金騰蛇兄弟持世。

兄弟剋妻財，裡象可取林女士注重養生，對食材與生活方式用心控制。騰蛇主不安、糾結、疑慮，也呼應她對古井填塞是否造成因果病因的懷疑。

### 三、官鬼與日辰同氣：疾病力量加重

二爻午火勾陳官鬼為長期病症。

丁巳日占，日辰巳火也是官鬼，日辰代表當下主宰力量，因此疾病能量加倍。

勾陳官鬼主長期、纏綿、拖延難解。

### 四、四爻白虎子孫日沖暗動：長年服用強效藥物

四爻亥水白虎子孫為藥物，靜而逢日辰巳火沖，成暗動。

暗動有逼迫之意，有生剋之力而無沖合之力。

亥水白虎子孫暗動後，剋二爻午火官鬼，表示藥物能控制疾病；同時生二爻伏神卯木妻財，代表藥物也會影響飲食生活、代謝或副作用層面。

白虎子孫可作強效藥、特殊藥物。

### 五、五爻兄弟動化退神：進退神皆有雙倍影響力

五爻酉金玄武兄弟發動，化申金玄武兄弟退神。

此案例提醒：卦爻化進神或退神，都隱含重複一次、雙倍影響力。退神不是完全消失，而是力量後退但仍持續作用。

酉金兄弟動生亥水子孫，沖伏神卯木妻財；變爻申金也生亥水子孫，剋伏神卯木妻財。

### 六、月破伏神卯木妻財可出

二爻伏神卯木妻財值月破。

月破之爻當月力弱，但出月則不破，因此仍可作可出的伏神。

卯木妻財出伏後，可生二爻午火官鬼，剋初爻辰土父母與上爻未土父母，沖五爻酉金兄弟。

### 七、逆向尋病因：由官鬼往回追

本案例最重要技法是逆向尋病因。

不是只看誰剋官鬼，而是看誰生官鬼。

能量鏈如下：

```text
官鬼午火（疾病）
← 得伏神卯木妻財之生
← 卯木妻財得四爻亥水子孫之生
← 亥水子孫為藥物
```

也就是：疾病之所以難除，是因為藥物透過副作用、飲食生活或代謝問題，間接讓疾病持續。

### 八、子孫不一定永遠吉：藥物可能是副作用來源

子孫在疾病占中通常為藥物，能剋官鬼為吉。

但本卦中，子孫亥水雖剋官鬼午火，也生卯木妻財；妻財卯木再生官鬼午火。

因此子孫既能治病，也可能因藥物副作用或不當使用，間接餵養疾病。

### 九、貪生忘剋的反效果

伏神卯木妻財受酉金兄弟沖，無日合解救，因此初期藥物不一定會立即引發副作用。

但變爻申金本可剋卯木妻財，卻先去生四爻亥水子孫，形成貪生忘剋。

亥水子孫又生卯木妻財，反而讓卯木妻財得生，進而生官鬼午火，引來副作用或病因延續。

這是貪生忘剋的反效果應用。

### 十、風水辨爻法：古井在初爻，但非病因主線

風水辨爻法中，古井位於初爻。

初爻辰土朱雀父母代表古井、地基、低位結構。但卦象顯示，疾病根源的主線在官鬼午火、伏神卯木妻財、四爻亥水子孫之間，並非初爻古井。

因此可排除「古井填塞」是主因的假設。

### 十一、結果驗證：藥物問題應驗

林女士後續查閱藥方說明，發現服用多年的高血壓藥物，醫師早有更換囑咐，但尚未執行，而病已先至。

卦象所示藥物副作用與病因關聯得到驗證。

## 結果判斷

視網膜剝離並非主要肇因於六年前古井填塞，而更可能與長期用藥、副作用或藥物調整未及時執行有關。

應回到醫療專業，檢查藥物使用、醫囑變更與相關副作用。

| 暗動 | 亥水子孫逢日沖，長期被迫服藥 |

| 月破伏神可出 | 卯木月破，出月不破，仍可出伏 |
| 逆向尋病因 | 官鬼 ← 妻財 ← 子孫，追出藥物副作用 |

| 貪生忘剋反效果 | 申金貪生亥水，反使卯木得生，引發副作用 |

| 受剋得生 | 官鬼受子孫剋，但得妻財生，病痛持續 |

## 決策建言摘要

疾病占不能只看到子孫剋官鬼就判藥到病除。若子孫又生妻財，妻財再生官鬼，則需警惕藥物副作用、生活代謝或不當使用反成病因。逆向尋病因可以從官鬼往回追，找出真正餵養疾病的來源。
