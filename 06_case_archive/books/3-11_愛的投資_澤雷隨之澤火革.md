# 書籍案例 3-11｜愛的投資：澤雷隨 → 澤火革

## 基本資訊

來源：《六爻知先機》第三篇多層次斷卦訣，第248～253頁。

占卦日期：戊戌年丁巳月甲寅日。

旬空：子、丑。

本卦：澤雷隨。

變卦：澤火革。

占問者：周女士，自占，女。

占問：友人經營工具機業，邀約投資入股，占問可否。

問卦類型：投資財運。

特殊技法：伏神出伏、占此應彼、變爻擴散、資金流向追蹤。

## 背景摘要

周女士離婚後獨立經營小企業，近年經濟不景氣，生意下滑，資金吃緊。朋友提議共同投資工具機公司，只要投入五百萬元，未來收益可觀。周女士動心，但因財務緊張而占問是否可投資。

## 排盤整理

兌宮：澤雷隨，兌宮一世卦，世爻在三爻，應爻在六爻。

## 用神設定

投資財運：

- 世爻三爻辰土妻財：周女士本人、資金狀態。
- 應爻六爻未土妻財：朋友、求財之地、表面投資方。
- 五爻酉金官鬼：疑似特別熟悉且有影響力的人，資金真正流向。
- 妻財辰土、未土：財利、利潤與投資資金。
- 午火子孫伏神：行銷技術、投資策略、說詞。
- 父母亥水、子水：文書契約。
- 兄弟寅木：耗財之星、鼓吹投資的人。

## 解析重點

### 一、世爻辰土勾陳妻財持世：財務不寬裕且陷入糾纏

三爻辰土勾陳妻財持世。

辰土受丁巳月火生，表面仍有資金基礎，但被甲寅日寅木兄弟剋制，表示周女士目前財務並不寬裕。

勾陳主長期、糾纏、踏實但黏膩，代表這筆財務決策不是單純投資，而與長期人情、舊關係或糾纏有關。

### 二、應爻未土玄武妻財：朋友靜而不動，並非真正主導者

六爻未土玄武妻財為應爻，可作朋友與表面投資方。

玄武持應，投資說法新潮但隱晦不明，有假象或資訊不透明之嫌。

更關鍵的是，應爻靜而不動，表示朋友沒有積極主導投資。此象提示朋友可能只是傳聲筒，而非真正受益者。

### 三、午火子孫伏神：飛剋伏本不可出，但得日辰生而出伏

四爻亥水騰蛇父母下伏午火子孫。

亥水剋午火，飛神剋伏神，本來伏神不可出。

但午火子孫得日辰寅木兄弟生扶，受剋之爻得生解救，因此成為可出伏神。

午火子孫出伏後，可生世爻辰土妻財、剋五爻酉金官鬼、合應爻未土妻財。

### 四、世爻動生合五爻官鬼：資金不走應爻而流向五爻

世爻辰土妻財發動，生合五爻酉金白虎官鬼。

這是全卦最大關鍵。占問表面是朋友邀約投資，理應關注應爻朋友與財利，但應爻靜而不動，世爻卻主動將金錢送向五爻官鬼。

五爻為尊位，白虎官鬼有壓力、傷耗、強勢之象。此爻代表特別熟悉且對周女士有影響力的人，或真正吸收資金者。

### 五、世爻動剋父母：無契約保障、心甘情願付出

世爻辰土發動，剋四爻亥水父母與初爻子水父母。

父母代表文書契約，父母被世財剋，表示周女士將金錢送出時，沒有正式契約保障，甚至是心甘情願、不附條件地付出。

### 六、占此應彼：表面問投資，實際問感情 / 熟人資金局

此卦占問投資，但應爻朋友靜而不動，世爻卻將財生合五爻官鬼。

此為占此應彼的警訊：卦象揭示真正核心不是工具機投資，而是周女士與某位熟悉之人的感情或信任糾葛。

### 七、變爻亥水父母擴散剋伏神午火

世爻辰土動化亥水父母。

變爻亥水對本位動爻辰土妻財無回頭生、剋、沖、合，因此變爻力量可擴散至旁爻與可出伏神。

亥水剋午火子孫，代表周女士內在有不想投資、想停止策略的念頭。但亥水在四爻先受辰土動爻所剋，表示這個不想投資的念頭曾被自己否定，投資意願反覆不定。

### 八、日辰臨二爻：寅木兄弟為鼓吹者

日辰甲寅日，寅木兄弟臨二爻。

日辰等同一個動爻，日辰之生剋沖合等同二爻意念。

寅木兄弟合變爻亥水父母，且生伏神午火子孫，表示有熟識、投緣、互信之人鼓吹周女士不要錯過這次機會。

### 九、朋友只是傳聲筒，真正受益者另有其人

日辰寅木兄弟生伏神午火子孫，午火子孫生合應爻未土妻財。

這表示所謂投資策略說詞注入到應爻朋友那裡，使朋友成為傳聲筒。

但資金完整流向為：

```text
日辰寅木兄弟
→ 生伏神午火子孫（投資策略說詞）
→ 生世爻辰土妻財（周女士資金）
→ 世爻發動生合五爻酉金官鬼（真正受益者）
```

因此資金並未真正投注到應爻工具機公司，而是最後歸向五爻官鬼。

## 結果判斷

此投資不宜參與。

表面是朋友邀約投資工具機公司，實則卦象顯示朋友不是主導者，只是傳聲筒。真正吸收資金者是五爻酉金官鬼所代表的熟悉之人。此局可能是騙局，投入資金恐一去不回。

## 可運用技法

| 技法 | 本案例用法 |
|---|---|
| 占此應彼 | 問投資，卦象卻揭示感情 / 熟人糾葛與資金流向 |
| 動爻是表徵，靜爻是裡象 | 世爻動揭示周女士主動送財給五爻官鬼 |
| 飛剋伏出伏條件 | 飛神亥水剋午火本不可出，但得日辰寅木生而可出 |
| 受剋得生解救 | 午火受亥水剋，得寅木生而解救 |
| 變爻擴散 | 變爻亥水對本位動爻無生剋沖合，力量擴散至伏神午火 |
| 日辰等同動爻 | 日辰寅木臨二爻，作鼓吹投資者意念 |
| 應爻靜不動 | 朋友不是真正主導者，只是傳聲筒 |
| 五爻尊位 | 五爻官鬼代表熟悉且有影響力的人、真正受益者 |
| 資金流向追蹤 | 伏神生世，世動生合官鬼，資金最後流向五爻 |
| 父母被剋 | 無契約保障、文書薄弱或心甘情願不立約 |

## 決策建言摘要

投資卦不可只看財爻旺不旺，必須追蹤資金流向。若應爻靜而世爻動生合其他高位爻，且父母文書被剋，需防假投資、無契約保障、傳聲筒與第三方受益。
