# 書籍案例 3-03｜錯付的真心：乾為天 → 天澤履

## 基本資訊

來源：《六爻知先機》第三篇多層次斷卦訣，第200～205頁。

占卦日期：乙巳年己卯月甲申日。

旬空：午、未。

本卦：乾為天，六沖卦。

變卦：天澤履。

占問：李先生與女友分手，問女友是否回心轉意。

問卦類型：男占姻緣感情。

## 排盤整理

乾宮：乾為天，乾宮純卦，世爻在六爻。

| 爻位 | 地支 | 六親 | 六神 | 動靜 |
|---|---|---|---|---|
| 六爻 | 戌土 | 父母 | 玄武 | 靜，世爻 |
| 五爻 | 申金 | 兄弟 | 白虎 | 靜 |
| 四爻 | 午火 | 官鬼 | 騰蛇 | 靜，旬空 |
| 三爻 | 辰土 | 父母 | 勾陳 | 動 → 丑土退神 |
| 二爻 | 寅木 | 妻財 | 朱雀 | 靜，受日沖暗動 |
| 初爻 | 子水 | 子孫 | 青龍 | 靜 |

## 用神設定

男占確認關係中的女友，以妻財為女友；世爻為李先生本人；官鬼為情敵；子孫為男命表達情愛的象徵。

## 解析重點

1. 六爻戌土玄武父母持世，玄武持世代表對感情老成持重、少了浪漫，也有暗中執著之象。世爻靜，表示被動守候。

2. 二爻寅木朱雀妻財為女友。朱雀妻財可取善辯、亮麗、善表達。寅木靜爻被甲申日沖起，成暗動。暗動只有生剋之力，沒有沖合之力。

3. 日辰申金臨五爻兄弟，五爻為尊位，案例以五爻申金兄弟取「當下當務之急的掙錢壓力」。申金沖寅木妻財，使女友暗動，象徵女友受掙錢、前途、物質壓力逼迫而心意轉變。

4. 寅木妻財暗動後可生午火官鬼，表示女友轉向情敵；又克三爻辰土父母與世爻戌土父母，代表破壞原有關係契約，對李先生形成無聲背棄。

5. 四爻午火騰蛇官鬼旬空，為情敵。騰蛇官鬼代表八面玲瓏、表裡不一；旬空則心性不定、善變。受寅木妻財暗動所生，表示情敵得到女友投奔。

6. 三爻辰土勾陳父母動化丑土退神。父母爻代表工作合約、長久文書、現實負擔。辰土動生五爻申金兄弟，顯示掙錢主因來自工作合約；辰土動克初爻子水子孫，象徵誤判損財或投資失利；辰土動沖世爻戌土，導致感情被沖開。

7. 受剋與受沖解救規則：初爻子水子孫受辰土與丑土剋，得日辰申金生而解，表示資金缺口可透過借貸補上。世爻戌土被辰土沖，受沖需日合解，無日合則傷害持續，感情無法挽回。

## 結果判斷

女友妻財暗動轉生官鬼，且克世爻，代表女友心意已轉。世爻受動爻沖而無日合解沖，感情漸行漸遠，無法挽回。

## 可運用技法

| 技法 | 用法 |
|---|---|
| 日辰臨爻位 | 日辰申金臨五爻，五爻兄弟象被環境放大 |
| 暗動 | 靜爻受日沖成暗動，可生剋但無沖合 |
| 受剋用生解 | 子孫受土剋，得申金生而暫解 |
| 受沖用日合解 | 世爻被辰戌沖，無日合則不解 |
| 男占確認對象取妻財 | 交往已久女友應以妻財為用，不只看應爻 |
| 退神 | 動化退神表示能量退縮，但仍可能造成連續傷害 |
| 子孫為男命情愛象徵 | 男命世爻若克子孫，真情表達受損 |

## 決策建言摘要

女友心意已轉，主要牽涉物質、前途與現實壓力。世爻被沖無解，執著只會加重傷害。
