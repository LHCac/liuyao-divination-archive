## 使用方式

```text
1. 先看問事類型，確認取用神。
2. 若是感情婚姻卦，可優先回查這五個案例。
3. 遇到多妻財、多官鬼、伏神、暗動、沖空、靜合、進神時，可查對應案例。
4. 不直接套用案例結論，應提煉其中規則後套入新卦。
```
