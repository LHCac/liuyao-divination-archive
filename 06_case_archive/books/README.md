## 使用方式

```text
1. 先看問事類型，確認取用神。
2. 若是感情婚姻卦，可優先回查感情案例。
3. 若是代占第三人稱、求財合作、方案選擇、職場轉職，可回查第二批案例。
4. 若遇到占此應彼、資金流向追蹤、動爻方向意念、日辰臨世應、飛神逢日沖出伏、收益與聲譽衝突，可回查第三批案例。
5. 若遇到考試功名、代占子女考試、售屋、官非官司、太歲入日辰、沖脫、風水破氣，可回查第四批案例。
6. 遇到多妻財、多官鬼、伏神、暗動、沖空、靜合、進神、年命爻、沖中逢合、合處逢沖、日辰合應沖世時，可查對應案例。
7. 不直接套用案例結論，應提煉其中規則後套入新卦。
```
