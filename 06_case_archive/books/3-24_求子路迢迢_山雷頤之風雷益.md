# 書籍案例 3-24｜求子路迢迢：山雷頤 → 風雷益

## 基本資訊

來源：《六爻知先機》第三篇多層次斷卦訣，第326～331頁。

占卦日期：壬寅年壬寅月辛卯日。

空亡：午、未。

本卦：山雷頤，游卦。

變卦：風雷益。

占問者：邱女士，自占。

占問：孕產求子，求助名醫皆失敗，不知有何障礙？

問卦類型：孕產求子、風水辨爻法。

特殊技法：元神概念、日辰為元神四時俱旺、孕產子孫規則、貪生忘剋、爻位判神格、風水辨爻。

> 注意：孕產與健康相關占卜只能作為象意參考，不可替代醫師診斷與治療。

## 背景摘要

邱女士與丈夫皆具高學歷與穩定工作，結婚多年未能懷孕。醫學檢查顯示兩人身體正常，兩次人工授孕仍失敗，因此懷疑是否有居家風水或其他玄學層面的阻礙。

日辰：辛卯日，卯木。

月建：壬寅月，寅木。

伏神巳火子孫伏於子水父母之下，得日辰卯木生，為可出伏神。

伏神酉金青龍官鬼雖逢日沖，但得飛神辰土生合，亦可出。

## 用神設定

孕產子息：

- 子孫爻：胎孕、胚胎、子息用神。
- 子孫爻元神：能生子孫者，即兄弟寅木、卯木。
- 父母爻：傳統習俗、宗教信仰、醫療文書，也可能剋子孫。
- 妻財爻：飲食、食補、食療、生活調理。
- 世爻辰土妻財：邱女士本人與食補養生。

## 解析重點

### 一、世爻妻財：食補養生、生活調理

三爻辰土青龍妻財持世，四爻戌土朱雀妻財同現。

妻財在孕產占中可代表飲食、食補、食療、生活方式。世爻妻財得日辰卯木合，表示邱女士為求好孕，日常嚴謹飲食，重視食補與生活調理。

### 二、父母爻為傳統習俗與宗教，爻位可判神格

父母爻在此案中除醫療文書外，也可取宗教信仰、傳統習俗、神佛與經文。

初爻子水父母可作地基、土地龍神、低階神靈。

五爻子水父母居尊位，可作菩薩、經文、較高層次的信仰力量。

爻位高低代表神格層次，是風水辨爻法的延伸應用。

### 三、五爻父母動化子孫：祈願求子，但父母先剋子孫

五爻子水父母發動，生二爻與六爻寅木兄弟，同時剋伏神巳火子孫。

父母在孕產占中若剋子孫，對胎孕不利。此象代表邱女士四處參拜祈願，將希望寄託於神佛，但在卦理上父母水仍對子孫火形成壓力。

五爻父母動化巳火子孫，變爻又生世爻辰土、戌土妻財，表示宗教祈願與求子意念最終仍回到她自身的生活調理與食補行為。

### 四、伏神巳火子孫得日辰元神生，受剋得解

伏神巳火子孫為胎孕用神，伏於五爻子水父母之下。

子水父母剋巳火子孫，本為不利。

但伏神巳火得日辰卯木兄弟相生。卯木能生巳火，因此卯木為子孫之元神。

日辰為六爻主宰，四時俱旺，因此日辰卯木元神能有效解救受剋的子孫用神。

### 五、元神概念：能生用神者即元神

孕產占中，能生子孫胎孕者，就是胎孕之元神。

此案中寅木兄弟、卯木兄弟皆可作元神。日辰卯木為元神，力量特別強。

元神旺，則子孫胎孕有得救與培育成功的條件。

### 六、伏神酉金官鬼可出：方法不被認同，但可形成轉化

三爻辰土妻財下伏酉金青龍官鬼。

酉金逢日辰卯木沖，但得飛神辰土生合，故可出。

酉金官鬼受日沖，象徵某些新潮食療、方法或策略不被普遍認同。

酉金又合辰土青龍妻財，可作選擇新潮食療或特殊方法。

### 七、貪生忘剋救元神

伏神酉金青龍官鬼本可剋寅木兄弟元神。

但酉金先去生子水父母，形成貪生忘剋，忘記剋寅木兄弟。

因此寅木元神得以保留，胚胎培育仍有成功機會。

### 八、用神與元神雙重得救，但成敗仍在著床與風水

巳火子孫得日辰卯木元神生，受剋得解。

寅木元神又因酉金官鬼貪生忘剋而免於受剋。

用神與元神都經過貪生忘剋與元神生扶而得救，表示胚胎雖有機會健康培育，但著床關鍵仍可能受到居家風水影響。

### 九、風水辨爻法：二爻與六爻是關鍵

本案風水辨爻：

| 爻位 | 風水位置 | 本案爻象 | 分析 |
|---|---|---|---|
| 初爻 | 地基 / 土地 | 子水白虎父母 | 剋制巳火子孫的位置 |
| 二爻 | 爐灶 | 寅木玄武兄弟 | 胎兒元神所在，受影響關鍵 |
| 五爻 | 尊位 / 菩薩 / 天頂 | 子水父母動 | 水爻，可作壁刀煞 |
| 六爻 | 水口 | 寅木騰蛇兄弟 | 胎兒元神所在 |

二爻常透露居家風水玄機。此案受剋的寅木元神位於二爻爐灶與六爻水口，剋制巳火子孫的子水父母位於初爻與五爻。

五爻水爻可作壁刀煞，提示風水布局需要調整。

## 結果驗證

邱女士壬寅年測卦，至癸卯年冬月，占師協助進行風水布局。後於甲辰年春天傳來佳音，亥月喜獲麟兒。

## 可運用技法

| 技法 | 本案例用法 |
|---|---|
| 孕產用神 | 子孫爻為胎孕用神，子孫得生吉、受剋凶 |
| 元神 | 能生用神者為元神；卯木兄弟為子孫元神 |
| 日辰為元神 | 日辰四時俱旺，能有效解救受剋用神 |
| 爻位判神格 | 初爻父母為土地龍神，五爻父母為菩薩經文 |
| 貪生忘剋 | 酉金先生子水父母，忘剋寅木元神 |
| 騰蛇深層象 | 積德定數、宿緣與宿世因緣 |
| 風水辨爻法 | 二爻爐灶、六爻水口、五爻水爻壁刀煞 |
| 伏神可出 | 受剋伏神得日辰元神生，或得飛神生合，可出 |
| 第二爻玄機 | 二爻動靜常透露居家風水關鍵 |

## 決策建言摘要

孕產求子卦以子孫為胎孕用神，能生子孫者為元神。若子孫受父母剋，但得日辰元神生，仍有得救可能；若元神又因忌神貪生忘剋而保留，則胚胎培育有希望。但若風水辨爻顯示二爻、六爻、水口或爐灶受損，則須同步調整居家環境。
