# Claude 自占經驗修正與應驗回饋

## 主題摘要

本檔整理 Claude「六爻-算自己」歷史對話中，對未來解卦最有價值的排盤修正、斷卦經驗、應驗回饋與心理提醒。此檔應作為後續自占時的重要校正資料。

## 一、排盤與納甲修正

### 1. 巽卦內外卦地支不可混用

自占001「風水渙」中，使用者原先把巽辛內卦地支誤用在上卦外卦位置。

修正：

```text
巽辛內卦：丑、卯、巳
巽辛外卦：未、酉、亥
風水渙上卦為巽，應使用外卦未、酉、亥
```

經驗：

```text
If 巽卦在上卦
Then 四五六爻用未、酉、亥，不可用丑、卯、巳
```

### 2. 陰卦不是全部逆排

歷史對話中釐清：

```text
坤、兌：逆排
離、巽：順排
```

經驗：

```text
If 陰卦為離或巽
Then 仍按順序升排，不可一律逆排
```

### 3. 六神由初爻往上排

曾發生把六神從六爻往下排的錯誤。

經驗：

```text
六神一律從初爻起，依日干定初爻六神，再往上排到六爻
```

### 4. 火水未濟世應位置修正

小P一週後態度卦中，使用者原標世三應六，後修正：

```text
火水未濟：坎宮游魂卦
世爻在四爻
應爻在初爻
```

經驗：

```text
If 本卦為火水未濟
Then 先按坎宮游魂處理，世四應初
```

## 二、日期與旬空修正

### 1. 日期不可自行更動

使用者多次提醒：

```text
日期請記好，不要擅自更動日期
```

經驗：

```text
If 使用者提供月建日辰
Then 嚴格以使用者提供為準，不可自行推改
```

### 2. 4/14為戊午日，卯日清單修正

原先卯日推算錯誤，後依農民曆修正為：

```text
4/23 丁卯（壬辰月）
5/5 己卯（壬辰月）
5/17 辛卯（癸巳月）
```

經驗：

```text
若使用者已查農民曆，優先採用使用者提供的日期資訊。
```

### 3. 戊午日旬空修正

曾將戊午日旬空誤判為辰巳，後修正為：

```text
戊午日在甲子旬
旬空為戌、亥
```

對小P初次感情卦的影響：

```text
四爻變爻丁亥入空，回頭克力道減弱
六爻兄弟亥水入空，競爭耗氣力量虛化
```

## 三、小P系列經驗主線

### 1. 慢熱、進神、未落實

小P系列多次出現：

```text
應爻進神
官鬼旬空或退神
世爻玄武或情感信號未實
```

整體經驗：

```text
她不是完全無感，而是慢熱、猶豫、信號尚未落實。
面對面互動比LINE有效。
過度盯訊息會造成自耗。
```

### 2. 不急是核心策略

多卦與月老籤共同指向：

```text
自然互動
不急著逼回應
在對的日子出現
留下記憶點
```

### 3. 應驗：戊午日互動自然

小P初卦中用神妻財午火當日得令，實際當天互動自然、有能量互補感。

經驗：

```text
If 用神得日辰同氣
Then 當日互動感受容易明顯，容易有共鳴或緣分感
```

### 4. 應驗：5/13現場取消，轉為私下陪同

原本問當天去找小P的互動，卦象為雷水解轉火澤睽，解為解除僵局，睽為不同場景或各有方向。

實際：

```text
轉為私下陪小P去弄指甲
晚點可能一起吃飯
```

經驗：

```text
雷水解可象徵原本卡住的關係鬆綁。
火澤睽可象徵不在原本預期場域，而是各自方向重新接上。
```

## 四、健康與牙齒案例經驗

### 1. 不植牙與植牙兩卦都出現辰未相刑

經驗：

```text
辰未相刑反覆出現，代表牙床骨、咬合、下顎骨、骨骼結構的慢性影響。
```

判斷：

```text
急性風險不一定高，但慢性累積不可忽視。
```

### 2. 植牙方向比不植牙好，但時機需看世爻狀態

自占006指出：

```text
植牙方向本身有利
但世爻旬空，當下身體能量尚未完全接收好處
```

經驗：

```text
If 方案本身應爻有力，但世爻空
Then 方向可行，但執行時機需等待世爻出空或自身狀態穩定
```

### 3. 拔牙日評估

4/29癸酉日：

```text
酉日使酉金世爻出空轉實
癸水壓制白虎官鬼午火
```

經驗：

```text
手術或牙齒處置日，可觀察白虎官鬼是否受制，以及世爻是否出空轉實。
```

## 七、失物案例經驗

### 小P手機遺失

關鍵象：

```text
妻財為手機用神
初爻代表地面低位
玄武代表陰暗、隱蔽、低處
丑土指東北、角落、地面
恆卦代表固定不動
夬卦代表需要主動突破搜尋
```

判斷經驗：

```text
手機傾向還在掉落地點附近的低處、角落、樹堆、座椅下或暗處。
若要找回，不能只問神或等待，必須回原地仔細找。
```

補充經驗：

```text
神明不給籤，有時可理解為「答案在自己行動裡」。
焦急不等於積極，朱雀退神可象徵只焦急但行動退縮。
```
