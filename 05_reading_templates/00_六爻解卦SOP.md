# 六爻解卦SOP（標準作業程序）

## 主題摘要

本 SOP 是日後進行六爻解卦時的標準流程。使用者提供卦象資訊後，解卦時以此流程為主，逐步完成排盤確認、用神設定、旺衰判斷、動爻檢查、靜卦判斷、世應分析、應期推算與結論輸出。

## 第零步：開始前準備

每次正式解卦前，需優先查閱下列資料：

```text
1. 05_reading_templates/00_六爻解卦SOP.md
2. 05_reading_templates/01_因果式解卦輸出模板.md
3. 02_rules/六神/00_六神總論.md
4. 02_rules/進階補充/00_六象意實戰資料庫.md
5. 02_rules/月日旺衰/01_旺衰分層判斷.md
6. 02_rules/月日旺衰/00_月建日建總論.md
```

## 第一步：接收卦象資訊

問卦者需要提供下列資訊：

| 資訊 | 必要性 | 說明 |
|---|---|---|
| 占問問題 | 必須 | 股票、感情、健康、求職等，問題越具體越好 |
| 占問對象 | 必須 | 自占還是代占？對象背景是誰？ |
| 卦卡狀態 | 必須 | 六爻爻象：少陰、老陰、少陽、老陽 |
| 本卦變卦 | 必須 | 卦名，例如履卦化乾卦 |
| 月建日辰 | 必須 | 例如癸巳月、癸巳日，不可省略或假設 |
| 職業背景 | 建議 | 有助於判斷用神與現實情境 |
| 性別年齡 | 建議 | 幫助判斷感情、健康、工作等問卦類型 |

### 2-7 六神安裝

六神順序固定為：

```text
青龍 → 朱雀 → 勾陳 → 騰蛇 → 白虎 → 玄武
```

依日干決定初爻起始六神，再依固定順序由初爻往上排。

| 日辰天干 | 初爻六神 |
|---|---|
| 甲、乙 | 青龍 |
| 丙、丁 | 朱雀 |
| 戊 | 勾陳 |
| 己 | 騰蛇 |
| 庚、辛 | 白虎 |
| 壬、癸 | 玄武 |

常用速查：

| 日干 | 初爻 | 二爻 | 三爻 | 四爻 | 五爻 | 六爻 |
|---|---|---|---|---|---|---|
| 甲、乙 | 青龍 | 朱雀 | 勾陳 | 騰蛇 | 白虎 | 玄武 |
| 丙、丁 | 朱雀 | 勾陳 | 騰蛇 | 白虎 | 玄武 | 青龍 |
| 戊 | 勾陳 | 騰蛇 | 白虎 | 玄武 | 青龍 | 朱雀 |
| 己 | 騰蛇 | 白虎 | 玄武 | 青龍 | 朱雀 | 勾陳 |
| 庚、辛 | 白虎 | 玄武 | 青龍 | 朱雀 | 勾陳 | 騰蛇 |
| 壬、癸 | 玄武 | 青龍 | 朱雀 | 勾陳 | 騰蛇 | 白虎 |

重點：癸日不是勾陳起初爻，癸日為玄武在初爻。

## 第三步：用神設定

依占問問題類型設定用神：

| 占問類型 | 用神 | 必要時同看 |
|---|---|---|
| 測人個性 | 應爻 | 六神補細節 |
| 感情發展 | 官鬼或妻財 | 女問男看官鬼，男問女看妻財 |
| 求職轉職 | 官鬼 | 父母可看文件履歷，妻財可看薪資 |
| 求財、股票 | 妻財 | 子孫為原神，兄弟為忌神 |
| 問健康 | 官鬼 | 官鬼為病因，子孫為康復，父母為勞損 |
| 考試功名 | 官鬼 | 子孫為忌神 |
| 問見面出行 | 父母 | 可輔看世應與應爻 |
| 測事業財運 | 官鬼、妻財 | 官鬼看事業，妻財看財 |
| 問聯繫訊息 | 父母 | 同看應爻、世應、朱雀與動爻 |
| 自占 | 世爻不主動參與生克 | 重要規則 |
| 代占 | 應爻正常參與生克 | 重要規則 |

## 第四步：旺衰判斷（分層次法）

按以下優先順序判斷，高層次大於低層次。

### 第一層：動變組合，最高

```text
動爻化進神 → 趨勢增強
動爻化退神 → 衰敗加重
動爻化進神但變爻逢破 → 檢查假破
變爻臨月建或合月建 → 動化旺，可視為假破或有力
```

### 第二層：月建高級別

| 關係 | 判斷 |
|---|---|
| 臨月建，同支 | 高層次旺 |
| 合月建，六合 | 高層次旺，但可能被合住，行動受阻 |
| 月破，相沖 | 高層次衰敗 |

### 第三層：月建一般層次

| 關係 | 判斷 |
|---|---|
| 月生、月扶 | 一般旺 |
| 月剋、月洩、月耗 | 一般衰或休囚 |

### 第四層：日辰高級別

| 關係 | 判斷 |
|---|---|
| 臨日辰 | 高層次旺 |
| 合日辰 | 高層次旺，但可能被合住 |
| 日辰沖靜爻 | 旺者為暗動，衰者為日破 |

### 第五層：日辰一般層次

```text
日生、日克、爻生日、爻克日 → 多作平相或低層參考，不直接主導旺衰
```

## 第五步：動爻檢查清單

每個動爻依序檢查：

```text
☐ 1. 變爻地支是什麼？進神還是退神？
☐ 2. 變爻在月建上是臨、合還是破？是否需要判假破？
☐ 3. 動爻對用神是生還是克？
☐ 4. 忌神動時，有沒有貪生忘克？
☐ 5. 有沒有通關？忌神是否引向中間爻，中間爻再生用神？
☐ 6. 靜爻被日沖時，旺相為暗動，衰敗為日破。
☐ 7. 動爻總體趨勢是化進、化退、化旺、化衰？
```

## 第六步：靜卦判斷三步走

適用於靜卦或動爻很少時。

### 第一步：察用忌

```text
用神持世 → 大吉
忌神持世 → 大凶
未持世 → 進入第二步
```

### 第二步：觀生克

```text
用神生世，且用神旺 → 吉
用神克世，特定問題如求財、測行人可用 → 用神旺且世得力 → 吉
無生克 → 進入第三步
```

### 第三步：用旺世興

```text
用神旺 + 世爻有氣 → 吉
用神旺 + 世爻衰 → 吉中帶憂
用神衰 → 凶，即使世旺也不救
```

> 逐項檢查後再下吉凶結論。

## 第七步：世應關係分析

| 世應關係 | 象意 |
|---|---|
| 世生應 | 問卦者主動付出，對對方有滋養 |
| 應生世 | 對方對問卦者有利 |
| 世克應 | 問卦者有主導權 |
| 應克世 | 對方對問卦者有壓制 |
| 世應相沖 | 雙方有衝突，需注意矛盾點 |
| 世應相合 | 雙方有牽絆，感情或合作較有利 |
| 世應比和 | 雙方狀態接近，較容易互動 |
| 世應相刑 | 雙方行事不按常理，容易有紛爭 |

## 第八步：應期判斷

### 一、先看是否有動爻

```text
動爻應期：逢值逢合
包含動爻地支、變爻地支、其六合之日/月/年
```

### 二、靜卦應期

```text
用神旺相 → 逢沖為應，沖起
用神衰敗 → 逢值為應，等用神得值
暗動 → 應期單位縮小，多以日論
```

### 三、旬空影響

| 情況 | 應期 |
|---|---|
| 用神旬空 | 逢值出空或逢沖沖空 |
| 世爻旬空 | 多在沖空或出空之時 |
| 原神旬空 | 逢值出空或逢沖沖空 |

### 四、應期單位

| 事件類型 | 應期單位 |
|---|---|
| 急性事件 | 時辰 |
| 日常事件、暗動 | 日 |
| 一般事件 | 月 |
| 長期事件 | 年 |

原則：

```text
應近不應遠
多個應期選項中，優先選最近者。
```

## 第九步：卦象補充

```text
1. 查本卦象意：03_hexagrams/對應卦名
2. 查變卦象意：03_hexagrams/對應卦名
3. 整合動爻、靜卦、卦義三層訊息
4. 卦義只作背景趨勢，不取代用神旺衰
```

## 第十步：結論輸出

解卦結論必須包含：

```text
1. 整體判斷：吉凶、有無機會、趨勢方向
2. 最大正面訊號是什麼
3. 最大警示訊號是什麼
4. 緩解或補充建議
5. 應期推算
6. 一句話總結
```

## 阿咪重點速記

- 解卦一定先照 SOP，不直接跳結論。
- 六神起法：甲乙青龍、丙丁朱雀、戊勾陳、己騰蛇、庚辛白虎、壬癸玄武。
- 癸日玄武在初爻，之後依序青龍、朱雀、勾陳、騰蛇、白虎。
- 月建、日建分開看；月主旺衰，日主沖動。
- 動爻先看變爻，再看對用神生克。
- 忌神動要檢查貪生忘克與通關。
- 靜卦用三步走：察用忌、觀生克、看旺衰。
- 最後一定輸出正面訊號、警示訊號、建議、應期與一句話總結。
