# 因果式解卦輸出模板

## 主題摘要

此模板用於輸出較完整、具有前因後果的六爻解卦內容。目標不是只給結論，而是把「為什麼這樣判斷」說清楚，讓每一個正面訊號、警示訊號、轉化關係與應期都有依據。

此模板適合正式解卦、複雜卦、多動爻、事業財運、感情、健康、失物、重大決策等需要完整推理脈絡的情況。

## 使用原則

正式解卦時，先照 `00_六爻解卦SOP.md` 完成排盤與檢查，再用本模板輸出。

```text
SOP = 判斷流程
因果式模板 = 輸出格式
```

## 重要規則

如果使用者提供的範例格式中有排盤、旬空、六親、卦宮或旺衰錯誤，必須在解卦時主動修正，不能為了照格式而延續錯誤。

```text
格式可以學
錯誤不能跟
```

```text
4. 旬空是否正確
5. 是否有需要修正的地方
```

## 三、占問類型與用神設定

### 建議格式

```text
🔮 解析——占問問題

占問類型：
用神設定：

- 用神：代表什麼
- 原神：代表什麼
- 忌神：代表什麼
- 仇神：代表什麼
- 世爻：問卦者自身狀態
- 應爻：對方 / 外部環境 / 事件對應方
```

### 目的

必須先讓使用者知道「為什麼取這個用神」。

例如：

```text
問業績 → 妻財為用神
問健康 → 官鬼為病因，子孫為康復
問腦波調頻 → 父母看調頻內容、引導、儀器與資訊；世爻看自己承受狀態；子孫看放鬆恢復；官鬼看壓力干擾
問感情 → 依男女與問題角度取官鬼、妻財、應爻
```

### 逐爻分析的順序

```text
1. 先分析用神
2. 再分析原神
3. 再分析忌神

5. 再分析應爻
6. 最後補充其他重要動爻、暗動爻、旬空爻
```

## 五、重點訊號總結區

此區是使用者特別偏好的格式，必須用「一、二、三、四……」整理，且每點都要有前因後果。

### 建議格式

```text
📝 綜合論斷

一、最大正面訊號：○○○

因為……，所以……。
但同時……，因此需修正為……。
這代表……。

二、原神 / 支撐訊號：○○○

因為原神……，又……，所以用神不是孤立無援。
這表示事情有後續支撐。

三、忌神是否被化解：○○○

忌神原本……，按理會……。
但因為它……，形成貪生忘克 / 通關 / 旬空 / 受制，
所以威脅被削弱或轉化。

四、最大警示訊號：○○○

雖然前面有吉象，但……。
這代表……可能會出現，例如……。
所以建議……。

五、世爻狀態：○○○

世爻代表問卦者。
世爻……，表示自身狀態……。
若世爻化退 / 旬空 / 受克，表示需要……。

六、六神補充：○○○

某六神臨用神 / 忌神 / 世爻，代表事情的氣氛是……。
例如騰蛇多波動、玄武多隱藏、白虎急烈、青龍正向、勾陳拖延、朱雀口舌表達。

七、本卦變卦：○○○

本卦代表目前背景，變卦代表後續走向。
所以這件事從……走向……。
```

## 六、因果式寫法規則

### 每一點都要遵守

```text
現象 → 規則 → 修正 → 象意 → 現實翻譯 → 建議
```

### 範例句型

```text
用神雖然月破，照一般規則屬於高層次衰敗。
但它同時發動，且變爻與月建相合，形成假破動化旺。
依動變組合優先於日月短期衝擊的原則，最後不作真破論。
所以這不是完全沒機會，而是「表面受阻、內部有轉機」。
```

```text
忌神原本克用神，理論上會壓制事情發展。
但忌神發動後轉而生原神，原神再生用神，形成通關。
因此忌神不是單純破壞，反而把壓力轉成推動力。
```

```text
世爻化退，代表問卦者本身有退縮、疲乏或後力不足。
所以即使用神有力，也需要主動維持節奏，不能中途鬆掉。
```

```text

```

## 阿咪重點速記

- 以後正式解卦可以使用「SOP + 因果式模板」。
- 排盤要完整，但遇到明顯錯誤要主動修正。
- 每個判斷都要有前因後果，不只講結論。
- 綜合論斷最好用一、二、三、四列出最大正面、原神、忌神、警示、世爻、六神、卦義。
- 核心句型：因為……所以……；但……因此修正為……。
