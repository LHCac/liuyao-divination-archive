# 六爻用神對招拆招要訣

## 主題摘要

本篇整理《六爻知先機》第七章「六爻用神對招拆招要訣」的問事分類取用與斷卦重點，包含感情婚姻、投資買賣事業財運、求職職場、學業考試、居家風水、醫藥疾病、求子孕產、是非官訟、夢境鬼神、失物、出行、行人尋人、流年運程、天時晴雨等十四大類。

資料來源：《六爻知先機》第七章「六爻用神對招拆招要訣」第094～114頁。

使用原則：

```text
1. 先確認問事類型。
2. 依類型取用神。
3. 再看世應、月日旺衰、動變、空破、合沖、原神忌神。
4. 表格只負責定位用神，不可單獨決定吉凶。
```

## 一、感情婚姻用神設定

### 1. 男占姻緣，未婚

| 爻位 / 六親 | 象意 |
|---|---|
| 世爻 | 自己 |
| 妻財 | 女友、喜歡的女性、姻緣用神 |
| 官鬼 | 情敵、對方已有男友時的男方象 |

#### 斷卦要點

- 最宜財動生世、合世，代表姻緣順成。
- 財動生合應爻或旁爻者，情屬他人。
- 兄爻持世，嚴詞相待，冷酷無情。
- 子孫持世，期待姻緣。
- 卦中有旁爻或變爻來生世爻，必定另有追求之人。
- 若是生財爻，則是對方有人追求。
- 妻財發動生合官爻，或伏官爻，表示對方已有男友。
- 玄武臨財，其女新潮外向。
- 財臨白虎，其女可能有身疾或強勢血氣象。
- 朱雀兄動，主口角爭吵。
- 勾陳財爻持世，有同入愛河或同居之象。
- 騰蛇財爻生合世，可能有前世注定、宿緣之象。
- 青龍官持世，財來生合且無沖剋，乃新郎之象。
- 青龍臨財，其女正派，生合世爻，為新娘正配之象。
- 財動剋世，女友強勢求去。

- 財化回頭剋，因對方反悔而不成。
- 世爻逢沖，或財爻逢沖，皆不成姻緣。

- 世爻、財爻得合，世動之化爻合財，或財動之化爻合世，且不逢空者，乃白頭偕老之慶。

## 六、醫藥疾病

| 爻位 / 六親 | 自占醫藥疾病 | 父母醫藥疾病 |
|---|---|---|
| 世爻 | 自己、宿疾 | 自己 |
| 官鬼 | 憂疑疾病 | 元神醫藥 / 病因需依題判斷 |
| 父母 | 積勞成傷 | 父母用神 |
| 兄弟 | 激氣詛咒 | 激氣詛咒 |
| 子孫 | 醫術醫藥 | 損傷元神 |
| 妻財 | 食療生活 | 剋傷父母 |
| 應爻 | 醫院、醫生 | 醫院 |

### 斷卦要點

- 自占疾病，以世爻為本身，官鬼為疾病。
- 喜日辰或動爻剋官，又喜日辰或動爻生世，主不藥而癒。
- 若日為官爻，或官鬼動來剋世，病必危亡；旺官者速，弱官者遲。
- 子孫爻為藥物，旺相而動，藥物對症，大吉。
- 子孫靜空休囚，或日沖月破，藥必無效。
- 應爻為醫生，應爻旺相，醫者精其術。
- 應爻空破墓絕，醫者可能庸醫。
- 病症看官鬼爻所屬五行，推其病位系統。
- 官鬼衰弱，主病輕；旺相，主病重。
- 官鬼發動，主病已發且煩躁不安；安靜，主病情穩定。
- 財爻為飲食，卦內無財，飲食不納。
- 兄化官，乃激氣抑鬱而致病。
- 父化官，乃積勞成傷。

### 官鬼伏藏病因

|---|---|

| 伏於兄弟 | 激氣之爭或受詛咒 |
| 伏於父母 | 憂心勞力或因長輩之事引起 |

### 官鬼爻位病位補充

- 凡官鬼所臨之處必有病。
- 官鬼在三爻，屬腰痛，逢空必腰軟。

## 八、是非官訟牢災

| 爻位 / 六親 | 自占是非官訟 | 占丈夫是非官訟 |
|---|---|---|
| 世爻 | 自己、我方 | 自己、我方 |
| 官鬼 | 官訟、憂疑、官方法律 | 丈夫 |
| 父母 | 文書、證物、契約 | 文書、證物 |
| 兄弟 | 激氣、奪財 | 激氣、奪財 |
| 子孫 | 解憂、證據、制官 | 官訟憂疑，需依題轉換 |
| 妻財 | 財脈、賄絡 | 夫之財脈 |
| 應爻 | 夥伴、對方 | 夥伴、對方 |

### 斷卦要點

- 以世為自己或我方，以應為對方，以官爻為官方、法律或法庭。
- 官鬼持世、剋世，或應動剋世，定然成訟。
- 子孫動剋官鬼，或子孫持世，必不成非。
- 官生世，得到官貴之人相助。
- 婦人占丈夫官司，官爻為丈夫；剋官爻則丈夫敗訴，生官爻則丈夫勝訴。
- 世生應，我求和。
- 應生世，他欲求和。
- 世應動空化空，雙方假意言和。
- 間爻為證人。生世合世，利於我；生應合應，利於他。
- 間爻沖世合應，證人與對方同謀陷害。
- 世爻入墓、化墓，防牢獄之禍。
- 世臨白虎入墓，防在獄中有病。
- 隨鬼入墓，恐因官訟而喪身。
- 卦中多現官爻、父爻，主事多變化，官司多纏訟難了結。
- 官司卦中有財爻發動，可能有用錢賄賂之象。
- 福動鬼散，以子孫生旺之月日，官司必散。
- 鬼動福靜，以官鬼入墓之月日，官司必散。

## 九、夢境、憂疑、鬼神

### 用神設定

| 取象 | 用神 |
|---|---|
| 自己 | 世爻 |
| 福神、解憂 | 子孫 |
| 憂疑阻滯、禍祟 | 官鬼 |

### 斷卦要點

- 庶人以鬼為禍祟，以子孫爻為福神。
- 福神能制官鬼，善解憂愁。
- 占官鬼，鬼爻旺相是神，休囚是鬼。
- 陽為神為男，陰為鬼為女。

## 十、占失物

### 用神設定

| 失物類型 | 用神 |
|---|---|
| 物之主人 | 世爻 |
| 文件、證書 | 父母爻 |
| 金銀首飾、車子、有價物 | 妻財爻 |
| 字畫、古董、觀賞之物 | 子孫爻 |
| 盜賊 | 官鬼爻 |
| 捕賊、找回制盜 | 子孫爻 |

### 斷卦要點

- 忌神持世，物難歸原主。
- 用神有化壞現象，尋覓無期。
- 化壞包括：用神受沖、受剋、入墓、伏藏而不得出。
- 六沖卦而無合，尋回渺茫。
- 重重合而帶剋，暗藏深處。
- 應爻持用神與世爻相合，是自借於他人。
- 騰蛇、玄武持世逢空，是自己忘記。
- 卦中官鬼動，乃盜賊偷走。
- 玄武忌神動剋用神，必是偷盜。
- 若帶鬼動，或官鬼與忌神互化，而官鬼助忌神剋用神，則是搶賊。
- 捕賊當看子孫爻，子孫能剋制官鬼，盜賊必擒獲。

## 阿咪重點速記

- 分類取用是第一步，但吉凶仍看旺衰、生剋、沖合、空破、動變。
- 感情女問男看官鬼，男問女看妻財；其他異性、情敵可看財官互動。
- 求財重財爻，子孫是財源，兄弟是劫財，父母是壓力文書，官鬼是阻礙風險。
- 求職看官鬼職位、應爻公司、妻財薪資、父母制度、兄弟同事。
- 疾病看世爻與官鬼，子孫是藥，應爻是醫院或醫生。
- 官司看世應雙方、官鬼官方、父母文書、子孫解憂證據。
- 失物依種類取財、父、子；官鬼動或玄武忌神動，要防偷盜。
- 出行與尋人，最怕用神空破、入墓、沖脫、受剋。
- 流年運程以世爻為核心，各爻來生即貴人，來剋即壓力或災象。
