# 書籍案例實戰技法提煉

## 主題摘要

本篇從《六爻知先機》第三篇「多層次斷卦訣」五個實際案例中，提煉可直接用於後續解卦的技法。重點不是複製案例內容，而是把案例中可重複運用的判斷方法拆出來，放入日後解析流程中。

收錄案例：

| 案例 | 卦象 | 主題 |
|---|---|---|
| 書籍案例 3-01 藕斷絲連 | 澤雷隨 → 天雷無妄 | 女占已離婚感情、情敵進神、伏神沖空 |
| 書籍案例 3-02 濃情蜜意 | 山天大畜 → 地天泰 | 女占感情、官鬼持世、伏神可出多層次 |
| 書籍案例 3-03 錯付的真心 | 乾為天 → 天澤履 | 男占感情、暗動、受剋受沖解救 |
| 書籍案例 3-04 戲鳳 | 風天小畜 → 巽為風 | 男占感情、神隱女友、多妻財與伏神情敵 |
| 書籍案例 3-05 天不長地不久 | 山雷頤 → 山地剝 | 女占婚姻、伏神與伏神無生剋、子女受傷 |

---

## 一、解卦流程補強

五個案例共同呈現的實戰流程：

```text
1. 先精準定用神。
2. 檢查旬空是否真空：旺者不為空，衰者才是真空。
3. 檢查伏神是否可出：飛生伏、飛合伏、飛神空破墓弱，皆可考慮出伏。
4. 檢查日合、日沖、暗動、沖脫與沖空。
5. 繪製意念流轉：動爻、變爻、可出伏神、日辰臨爻位的力量流向。
6. 以六親定人事主體，以六獸補性格與場景細節。
7. 最後將卦象逆轉為決策建議，而不是只下吉凶。
```

## 二、感情婚姻取用補強

### 女占感情 / 婚姻

| 取象 | 用神 |
|---|---|
| 自己 | 世爻 |
| 男友、丈夫、前夫、夫星 | 官鬼 |
| 情敵、元配、其他女性 | 妻財 |
| 子女、馭夫之星、忌神 | 子孫 |

常用判斷：

```text
If 女占感情官鬼持世
Then 多主濃情蜜意、心在對方身上；若臨騰蛇，可加宿世姻緣、自我合理化、糾纏難斷之象。

If 五爻尊位見青龍妻財且為應爻
Then 可取元配、正妻、名分較正的一方。

If 世爻與官鬼靜合
Then 多為內心仍有牽掛、執念或相互慰藉，但未必代表實際行動。

If 官鬼靜而不動、不來生合世爻
Then 男方未必有主動挽留或回心轉意的實際行動。

If 妻財動生合官鬼
Then 情敵與男方有實際牽連，女占感情多需防外情。
```

### 男占感情 / 婚姻

| 取象 | 用神 |
|---|---|
| 自己 | 世爻 |
| 確認關係中的女友、妻子、喜歡的女性 | 妻財 |
| 情敵、對方已有男友 | 官鬼 |
| 男命表達情愛、真心與溫柔輸出 | 子孫 |
| 未深交或不確定身份的對象 | 可先看應爻，再配合妻財 |

常用判斷：

```text
If 男占已有確認對象
Then 優先取妻財為女友或妻子，不只看應爻。

If 男占不熟悉、未深交、僅網友或泛泛對象
Then 可先看應爻，再看妻財是否成為具體女性象。

If 世爻剋子孫
Then 男命情愛表達受損，可能不是純粹真愛，而有佔有、壓迫、強迫或控制意味。

If 世爻生兄弟
Then 意念流向傷妻之神，需防不是愛，而是消耗、傷害或自我中心。
```

## 三、妻財多現的辨別法

妻財多現不可只取其中一個。需依爻位、世應、動靜、六獸、伏神與變爻分辨。

| 層次 | 可代表 |
|---|---|
| 世爻所化妻財 | 目前貼身、心念中直接牽引、眼前身邊之女性 |
| 應爻妻財 | 占問中最想知道答案的對象、外部對象、神隱對象 |
| 五爻妻財 | 尊位、正妻、元配、名分較正者 |
| 伏於官鬼旁的妻財 | 暗中牽連、隱藏情人、未浮上檯面的女性 |
| 動爻妻財化進神 | 情敵力量步步進逼、關係越來越強 |
| 朱雀妻財 | 善辯、炫媚、以言語或魅力介入 |
| 勾陳妻財 | 黏膩、長久依附、親密情婦、糾纏感 |
| 青龍妻財 | 正派、貌美、名分正、喜慶好感 |
| 玄武妻財 | 曖昧、暗情、桃色、享樂或隱密財色 |

## 四、伏神實戰規則補強

### 伏神可出條件

```text
飛神生伏 → 可出
飛神合伏 → 可出
飛神空亡 → 伏神易引拔
飛神受日沖成沖空 → 飛神不能用，伏神更易出
飛神入墓、化墓、化空、化回頭剋、化回頭沖 → 伏神可出
日辰沖開飛神 → 伏神可出
月建臨伏神 → 伏神得力，可出性增強
旺者不為空 → 伏神雖空但得月建或旺相，不作真空論
```

### 可出伏神的作用

```text
If 伏神可出
Then 可視為類動爻，可對本位卦爻放射生、剋、沖、合之力。
```

### 伏神與伏神之間

重要規則：

```text
伏神與伏神之間沒有生、剋、沖、合。
```

應用：

```text
If 伏神A與伏神B五行上看似相剋
Then 不可直接論伏神彼此相剋。
需看它們各自出伏後是否作用於本位卦爻。
```

此規則可用於判斷「關係斷線、彼此無感應、親情或情感無法直接連結」。

## 五、日辰臨爻位與暗動補強

### 日辰臨爻位

```text
If 日辰地支剛好臨卦中某爻
Then 日辰之生剋沖合，可視為該爻意念被當下環境強化。
```

例如：日辰申金臨五爻兄弟，可把「當下環境 / 當務之急」與五爻兄弟象合看；日辰巳火臨五爻子孫，可把日辰力量視為五爻子孫意念的放大。

### 暗動

```text
If 靜爻旺相受日沖
Then 成暗動。

If 暗動爻成立
Then 只有生剋之力，沒有沖合之力。
```

應用：

```text
暗動可以生用神、剋用神。
暗動不可拿來沖用神或合用神。
```

## 六、受剋與受沖的解救規則

五個案例多次確認：

```text
受剋之爻，必須用「生」解救。
受沖之爻，必須用「日合」解救。
他爻來合，解沖不真。
```

判斷式：

```text
If 用神被剋
Then 找原神、月日、動爻、可出伏神是否能生用神。

If 用神被沖
Then 找日辰是否合用神；沒有日合，沖的傷害仍在。
```

## 七、沖空與旺不為空

### 旺不為空

```text
If 爻逢旬空，但臨月建、得月建旺扶，或旺相有根
Then 旺者不為空，不作真空論。
```

### 沖空

```text
If 旬空之爻受日辰沖
Then 稱沖空，力量消失，吉凶不成象。
```

應用：

```text
沖空可代表證據被抹去、事情不成形、力量被沖散、承諾不落地。
```

## 八、進神、退神與變爻擴散

### 進神

```text
If 妻財動化進神
Then 情敵、財、關係、事件力量步步加強。
```

### 退神

```text
If 動爻化退神
Then 力量退縮、後力不足，但仍需看其是否帶來連續性影響。
```

案例補充：

```text
進退神往往有加重、加深、延續的意味，不宜只當普通變化看。
```

### 變爻擴散條件

```text
If 變爻與本位動爻有明確生剋沖合
Then 變爻先作用本位動爻。

If 變爻與本位動爻同五行，無生剋沖合
Then 可考慮變爻力量擴散到旁爻與可出伏神。
```

### 日辰桃花速查

| 日支 | 桃花地支 |
|---|---|
| 子 | 酉 |
| 卯 | 子 |

| 申 | 巳 |
| 亥 | 申 |
| 丑 | 戌 |
| 辰 | 丑 |
| 未 | 辰 |
| 戌 | 未 |

用法：

```text
If 日辰桃花地支剛好落在夫妻星、用神或情敵爻

```

注意：桃花只補象，不可單獨定外遇或成敗，仍要看用神、動靜、生剋與世應。

## 十、決策建言轉換原則

這五個案例的建言都不是單純判吉凶，而是將卦象逆推成人生選擇：

```text
If 用神無連結、世爻受沖、對方不動來合
Then 建議放下、停止耗損。

If 情敵進神、生合夫星，而夫星不主動回世
Then 建議承認局勢已變，不宜執著。

If 世爻剋子孫或生兄弟，顯示非真愛或有佔有心
Then 建議反觀自己的動機，不可把執念當愛。

If 子女爻受傷嚴重
Then 建議優先保護子女，而非只看夫妻情感。
```

## 阿咪重點速記

- 案例不是拿來背答案，是拿來提煉判斷流程。
- 感情卦最怕用神取錯；女問男看官鬼，男問女看妻財。
- 多妻財、多官鬼不可粗判，要分爻位、六獸、動靜、伏神、世應。
- 可出伏神等同動爻，是多層次斷卦核心。
- 伏神與伏神之間無生剋沖合。
- 暗動只有生剋，沒有沖合。
- 受剋用生救，受沖用日合救。
- 沖空會讓旬空爻力量消散。
- 桃花只作輔助取象，不單獨判斷外遇。
