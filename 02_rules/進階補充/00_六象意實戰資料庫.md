# 六象意實戰資料庫

## 主題摘要

本篇整理實戰卦例中常見的「六神 × 六親」、月日張力、動爻取捨、世應關係、持世爻、旬空與特殊卦象。此資料屬於實戰校正資料，用於輔助解卦，不取代用神、旺衰、原神忌神、動變與月日判斷。

## 三、動爻取捨原則

| 情況 | 取捨原則 | 說明 |
|---|---|---|
| 唯一動爻 | 直接分析 | 全卦核心能量集中於此 |
| 多動爻 | 以與用神關係最密切者為主 | 其他動爻作輔助 |
| 動爻生用神 | 大利 | 能量流向用神，事情有利 |
| 動爻克用神 | 大忌 | 能量壓制用神，事情受阻 |
| 動爻化退 | 動而路自封 | 主動後又退縮，進兩步退一步 |
| 動爻化進 | 能量持續增強 | 趨勢明確，力量向前 |
| 旬空爻發動 | 動而不實 | 表面有動，但落地較空 |
| 重重動爻 | 取最靠近用神者為主 | 其餘看整體趨勢 |

## 四、多動爻取捨細則

當卦中出現兩個以上動爻時，依下列優先順序判斷主從：

| 優先順序 | 判斷原則 | 說明 |
|---|---|---|
| 第一優先 | 動爻直接作用於用神 | 生用神或克用神的動爻最重要 |
| 第二優先 | 動爻作用於世爻、應爻 | 影響問卦者或對方的動爻次之 |
| 第三優先 | 動爻相互作用 | 動爻之間的生克形成連鎖 |
| 第四優先 | 爻位高低 | 外卦動爻看外在環境，內卦動爻看內在狀態 |
| 輔助判斷 | 六神加持 | 白虎急速、騰蛇虛幻、青龍正向、勾陳拖延等 |

### 常見組合

| 組合 | 象意 |
|---|---|
| 世爻動 + 應爻動 | 雙方都有動態，互相觸動 |
| 用神進神 + 忌神退神 | 事情越來越有利，阻力在消退，最吉組合 |
| 用神退神 + 忌神進神 | 事情越來越不利，阻力在增強，最凶組合 |
| 多爻同動 + 用神旬空 | 表面動盪多，但核心落空 |
| 世爻動化回頭克 | 自身行動反傷自己，期待落空 |
| 世爻動 + 妻財動 | 主動出擊，財運同步啟動 |
| 三爻以上同動 | 取最貼近用神者為主，其餘看整體趨勢 |

## 五、世應關係對照

| 世應關係 | 象意 | 常用問類 |
|---|---|---|
| 世生應 | 問卦者主動付出，滋養對方 | 測人、感情 |
| 世克應 | 問卦者有主導權，對對方有壓制力 | 測人 |
| 應生世 | 對方對問卦者有利，在乎情分 | 感情 |
| 應克世 | 對方對問卦者有壓力 | 測人、健康 |
| 世應比和 | 雙方能量相近，相處容易 | 合作、關係 |
| 世應相沖 | 雙方有衝突或距離感 | 各類關係 |
| 世應相合 | 雙方有牽絆，婚緣感情較有利 | 婚姻感情 |
| 世應相刑 | 雙方行事不按常理，有紛爭 | 測人需注意 |

## 七、旬空對照

| 情況 | 象意 |
|---|---|
| 用神旬空 | 事情虛化難落實，等出空後才有變化 |
| 忌神旬空 | 阻礙、競爭、隱患目前不存在 |
| 世爻旬空 | 問卦者能量虛化，被動或沒有把握 |

## If / Then 判斷

```text
If 多動爻出現
Then 先找直接生克用神者，再看世應、動爻連鎖與六神

If 用神進神且忌神退神
Then 事情越來越有利，阻力漸退

If 用神退神且忌神進神
Then 事情越來越不利，阻力漸增

If 用神旬空但卦中動爻很多
Then 表面事件多，核心仍可能落空

If 六神象意與旺衰生克衝突
Then 以旺衰生克定吉凶，六神只補細節
```

## 阿咪重點速記

- 六象意是「實戰補充」，不是取代基本規則。
- 多動爻先看用神，再看世應，再看動爻連鎖。
- 六神組合用來看細節氣氛與人事風格。
- 旬空、特殊卦象、持世爻可作校正訊號。
