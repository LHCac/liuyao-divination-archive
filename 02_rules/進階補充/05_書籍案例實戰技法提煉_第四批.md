# 書籍案例實戰技法提煉：第四批

## 主題摘要

本篇整理四個書籍實際案例中，可直接放入後續解卦流程的技法。此批案例補強重點包括：官非官司的太歲特殊用法、沖脫、官司與風水互參、考試功名用神、代占六親考試、售屋父母旺反賣不掉、伏神與伏神之間無生剋沖合、尊位判名次、旬空不等於真空與沖空。

收錄案例：

| 案例 | 卦象 | 主題 |
|---|---|---|
| 3-16 金融升等特考 | 風火家人 → 山火賁 | 考試功名、伏神官鬼、受剋無生得出、權威教材 |
| 3-17 愛的翅膀帶你飛翔 | 地山謙 → 雷山小過 | 代占女兒考試、子孫為用神、尊位判名次、沖空 |
| 3-18 售屋換宅占卜 | 雷山小過 → 澤山咸 | 售屋、父母旺反賣不掉、伏神間無生剋、宅氣受損 |
| 3-19 八字顯災劫、風水斷煞因 | 澤火革 → 澤雷隨 | 官非官司、太歲入日辰、沖脫、風水剪刀煞 |

## 一、考試功名用神設定

案例 3-16 與 3-17 補強考試類用神。

| 六親 | 考試功名象意 |
|---|---|
| 父母 | 學業成績、考卷、教材、證照、文書分數 |
| 官鬼 | 功名、錄取、升等、名次、資格 |
| 兄弟 | 競爭者、同學、同考者 |
| 子孫 | 才能發揮、放鬆、表現；代占子女時為子女本人 |
| 妻財 | 生活雜務、時間消耗、名次排列或分數排序中的財爻作用 |

### 實戰規則

```text
If 問自己考試
Then 父母為成績，官鬼為功名，兄弟為競爭者。

If 父母官鬼旺相相生
Then 成績與錄取機會佳。

If 官鬼伏神得出，且生父母、沖兄弟
Then 功名力量可顯，能在競爭中脫穎而出。
```

## 二、受剋無生，伏神反可保留力量

案例 3-16 示範：表面動爻剋伏神，但動爻本身被回頭剋且無救，伏神反而不受剋。

```text
If 動爻剋伏神
And 動爻自身受回頭剋
And 動爻無其他生扶解救
Then 動爻剋伏神之力不能成立，伏神可保留力量並出伏作用。
```

### 考試占應用

```text
If 子孫動剋伏神官鬼
But 子孫動化父母回頭剋且無救
Then 子孫剋官鬼不成立，伏神官鬼仍可代表功名可得。
```

## 三、青龍父母與權威教材

案例 3-16 補強考試占特殊象意。

```text
青龍子孫：新潮、速成、技巧型準備方式。
青龍父母：正規學術、系統教材、權威資料。
白虎官鬼：權威性、嚴格標準、考試功名壓力。
```

### 建議轉換

```text
If 考試卦中青龍子孫動化青龍父母回頭剋
Then 不宜只靠速成技巧，應回到正規教材與系統化準備。
```

## 四、代占六親考試：子孫為女兒 / 子女用神

案例 3-17 示範父親代占女兒考試。

```text
If 父母代占子女考試
Then 先取子孫爻為子女本人，再以父母爻看成績，官鬼爻看功名與名次。
```

### 子孫用神喜忌

```text
子孫宜得生得合。
子孫不宜受剋受沖。
父母動沖剋子孫，代表學業壓力重或考題生疏。
父母所化變爻剋子孫，同理。
```

## 五、旬空不一定是真空；沖空可成暗動

案例 3-17 補強旬空判斷。

```text
If 旬空之爻得月建生扶或自身旺相
Then 不作真空，仍有力量。

If 旬空但不是真空，又逢日辰沖
Then 可作沖空，仍具暗動之力。
```

### 實戰用法

```text
If 兄弟旬空但得月生，不是真空
And 日辰沖之
Then 兄弟仍可暗動生用神。
```

## 六、五爻尊位判名次

案例 3-17 補強尊位判名次。

```text
If 代占子女考試，子孫居五爻尊位
And 官鬼或父母能輾轉相生子孫
Then 可判名列前茅或得到獎狀肯定。
```

### 注意

```text
五爻尊位不等於必然第一名，但可作前段、前茅、高位、被看見。
```

## 七、售屋占：父母旺反賣不掉

案例 3-18 是售屋類重要規則。

```text
If 問房屋能否賣掉
Then 以父母爻為房屋用神。
父母旺相，反主房屋穩固不動、賣不掉。
父母受剋，才代表房屋可脫手、售屋願望可成。
```

### 區分

```text
問房屋本身好壞：父母旺可代表房屋穩。
問能否賣掉：父母旺反而表示房子留住，難售。
問何時拿到錢：再看妻財爻。
```

## 八、伏神與伏神之間無生剋沖合

案例 3-18 補強重要限制。

```text
伏神與伏神之間，不能互相發出生剋沖合之力。
```

### 實戰用法

```text
If 妻財伏神受剋、受沖
And 另一個子孫伏神理論上能生妻財
Then 不可用子孫伏神生妻財伏神來解救。
伏神彼此之間無力互助。
```

### 解救規則

```text
受剋之爻，必須得明顯生扶解救。
受沖之爻，必須得日合解救。
伏神間不能互相解救。
```

## 九、家宅風水三象

案例 3-18 與 3-19 都補強風水象。

| 六親 | 家宅風水象意 |
|---|---|
| 子孫 | 聚氣、人氣、和氣、居住舒適度 |
| 妻財 | 宅氣、財氣、房屋財利與可出售價值 |
| 兄弟 | 破氣、耗氣、破財、形煞、外部干擾 |
| 父母 | 房屋本體、產權、房契、文書 |
| 官鬼 | 煞氣、官非、壓力、病災、麻煩 |

### 應用

```text
If 兄弟落間爻
Then 可作明堂破氣，外部形煞或氣口受損。

If 妻財伏神受剋逢沖
Then 宅氣有損，售價與流通性受影響。
```

## 十、進神代表市場破氣加重

案例 3-18 中兄弟化進神。

```text
If 兄弟爻動化進神
Then 破氣、競爭、耗財、市場壓力逐步加重。
```

### 售屋應用

```text
If 兄弟進神剋妻財、沖妻財
Then 市場投機、價格下修、買方壓價力量逐漸加強。
```

## 十一、官非官司用神設定

案例 3-19 補強官非官司。

| 六親 | 官非官司象意 |
|---|---|
| 世爻 | 自己 / 代占者 |
| 用神六親 | 被代占者，如兄弟代占兄長則取兄弟爻 |
| 官鬼 | 官方、法律、法庭、官司、檢警、壓力 |
| 子孫 | 解憂之神，能剋官鬼則官司不成或可解 |
| 父母 | 文書、證據、契約、判決、法律文件 |
| 妻財 | 花費、賠償、資金、私下處理費用 |

## 十二、沖脫規則

案例 3-19 是沖脫的重要示範。

```text
If 動爻逢日辰沖
Then 可作沖脫。
沖脫之爻，其生剋之力皆不能起作用。
```

### 官司應用

```text
If 兄弟用神動化官鬼回頭剋
But 動爻逢日辰沖脫
Then 回頭剋力量轉移或失效，有化險可能。
```

---

## 十三、太歲在官司占的特殊意義

案例 3-19 補強太歲。

```text
一般占事：太歲尊而不親，不宜過度取用。
官非官司占：太歲可代表國法、公訴、法律權威與公權力。
```

### 太歲入日辰

```text
If 太歲地支同日辰
And 日辰沖脫用神凶動
Then 可作引用國法、法條、程序正義來化險。
```

### 建議轉換

```text
官司卦見太歲入日辰，不宜私下用錢亂處理。
應回歸證據、法條、程序與正式法律正軌。
```

---

## 十四、官司卦中的妻財警示

案例 3-19 提醒：妻財不只是錢，也可能是不當處理方式。

```text
If 妻財伏神生官鬼、合官鬼、剋父母文書
Then 用錢處理問題可能使官司更複雜，甚至引發新官非。
```

### 建議

```text
停止以金錢嘗試不正當干預。
回到證據、法條與正式文書。
```

## 阿咪重點速記

- 考試功名：父母看成績，官鬼看功名，兄弟看競爭。
- 代占子女考試：子孫先代表子女本人。
- 旬空得月生不是真空；不真空逢日沖可作沖空暗動。
- 五爻尊位可判名列前茅、排名前段、被看見。
- 售屋問能否賣掉，父母旺反而賣不掉；父母受剋才易脫手。
- 伏神與伏神之間無生剋沖合，不能互相救。
- 家宅風水：子孫聚氣、妻財宅氣、兄弟破氣。
- 官司占：官鬼為官方法庭，子孫為解憂神，父母為證據文書。
- 動爻逢日沖為沖脫，生剋力失效。
- 官司占中太歲可代表國法、公訴、公權力；太歲入日辰可用法條程序化險。
