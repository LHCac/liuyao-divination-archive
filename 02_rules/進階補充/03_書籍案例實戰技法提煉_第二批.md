# 書籍案例實戰技法提煉：第二批

## 主題摘要

本篇整理四個書籍實際案例中，可直接放入後續解卦流程的技法。此批案例補強重點包括：年命爻、第三人稱代占用神、不可出伏神、沖中逢合與合處逢沖、日辰合應沖世、真破、職場轉職風險與求財方案決策。

收錄案例：

| 案例 | 卦象 | 主題 |
|---|---|---|
| 3-07 在風中打轉的愛情 | 坤為地 → 雷地豫 | 女占感情、年命爻、六沖化六合 |
| 3-09 夕陽紅燦 | 澤風大過 → 天風姤 | 代占父親姻緣、不可出伏神、第三人稱用神 |

## 一、年命爻技法

年命爻是占卜者或用神出生年的地支。年命爻不一定出現在卦中，不能單獨決定吉凶，但能補充當事人深層意念、內心牽掛與過往關係線索。

### 使用規則

```text
If 年命爻出現在卦中
Then 可作當事人內心深層思維、隱性動機、心理牽掛。

If 年命爻生世爻或合用神
Then 可作內心仰慕、牽掛、想靠近或舊緣未斷。

If 年命爻沖情敵爻
Then 可作當事人介意第三方、排斥對方身邊其他人。

If 年命爻剋世爻
Then 可作內心掙扎、自我壓迫、理智與情感拉扯。
```

### 注意

```text
年命爻只補象，不定吉凶。
真正成敗仍以用神、世應、月日、動變、空破、合沖為主。
```

## 二、六沖卦與六合卦的正確認知

案例 3-07 示範：單看六合卦或六沖卦，不可直接定吉凶。

```text
六沖卦 → 不一定凶。
六合卦 → 不一定吉。
六沖化六合 → 常主始難後順。
六合化六沖 → 常主始順後難。
真正決定吉凶者，仍是卦爻五行生剋與用神狀態。
```

### 實戰用法

```text
If 六沖卦化六合卦
Then 可作事情一開始不流暢，後面逐漸順水推舟。

If 六合卦化六沖卦
Then 可作初期和合，後期分散或難續。

If 用神受傷無救
Then 即使六合卦，也不可直接判吉。

If 用神得生合有情
Then 即使六沖卦，也可能先難後成。
```

## 三、代占第三人稱用神轉換

代占不能機械以世爻代表被問者。需要先確認「誰來問、問誰、問什麼」。

### 常見轉換

| 占問對象 | 用神設定 |
|---|---|
| 女兒代占父親姻緣 | 世爻為女兒，父母爻為父親，妻財爻為父親女友，官鬼爻為父親情敵 |
| 父母代占女兒姻緣 | 子孫爻為女兒，官鬼爻為女兒男友 |
| 父母代占兒子姻緣 | 子孫爻為兒子，妻財爻為兒子女友 |
| 朋友感情代占 | 應爻可代表朋友，再依男女與問題調整財官 |

### 判斷提醒

```text
If 代占第三人稱
Then 先定位問卦者與被問者，不可一律用世爻代表主角。

If 被問者是父親、母親、長輩
Then 多先取父母爻為被問者，再依其性別與問題取財官。

If 被問者是子女
Then 先取子孫爻為被問者，再依其性別與問題取財官。
```

## 四、不可出伏神與可出伏神

案例 3-09 補強不可出伏神規則。

### 對照表

| 條件 | 判斷 |
|---|---|
| 飛神生伏神 | 可出 |
| 飛神合伏神 | 可出 |
| 月建臨伏神 | 可出性增強 |
| 日辰臨伏神 | 伏神力量加強 |
| 飛神空亡、沖空、入墓、化墓、化空、化回頭剋 | 伏神易出 |
| 飛神剋伏神，且無日月或動爻救援 | 不可出 |
| 飛神剋伏神，但日月強力生救伏神 | 視旺衰與救應再判斷 |

### 實戰用法

```text
If 伏神代表願望、孝心、希望
And 飛神剋伏神無救
Then 願望難以落地，希望落空。

If 伏神可出
Then 可視為類動爻，能對本位卦爻放射生剋沖合。
```

## 五、財爻多現 / 身份辨別補強

案例 3-09 的財爻三現補強多妻財辨別。

### 辨別原則

```text
同為妻財，不代表同一人。
要依爻位、六神、動靜、變爻、世應、與用神關係辨別身份。
```

| 財爻狀態 | 可能身份 |
|---|---|
| 青龍妻財居高位 | 元配、正妻、名分較正者 |
| 朱雀妻財在低位 | 年輕、善言、人緣佳、眼前新對象 |
| 動爻妻財化進神 | 其意念或影響力逐漸加強 |
| 妻財沖另一妻財 | 不同女性彼此排斥、阻礙或立場衝突 |
| 妻財受日合 | 緣分轉向他方、情屬他人或被外力牽走 |

## 六、沖中逢合與合處逢沖

案例 3-12 是重要方案決策法。

| 模式 | 次序 | 象意 | 結果傾向 |
|---|---|---|---|
| 沖中逢合 | 先沖後合 | 散而後聚、先失後得 | 較可持續 |
| 合處逢沖 | 先合後沖 | 聚而後散、先得後失 | 較難持續 |

### 實戰用法

```text
If 方案A先被世爻沖開，後被變爻合回
Then 可作沖中逢合：初期分歧，後期重新整合。

If 方案B先與世爻相合，後被變爻沖開
Then 可作合處逢沖：初期看似契合，後期失去延續性。
```

### 注意

```text
沖中逢合與合處逢沖要看力量流轉先後，不可只看地支是否同時有沖有合。
```

## 七、變爻擴散與方案選擇

案例 3-12 補強變爻擴散規則。

```text
If 變爻與本位動爻無生剋沖合
Then 變爻力量可擴散至旁爻或可出伏神。

If 變爻月破
Then 當月作用力弱，但出月可恢復。
```

方案決策時，可看世爻動後化出的變爻，是否把心意引向另一方案：

```text
世爻動 → 變爻
變爻沖某方案 → 放棄或淘汰該方案
變爻合某方案 → 轉向或保留該方案
```

## 八、日辰多重象意

案例 3-07 與 3-15 都補強日辰的多重象意。

| 問題類型 | 日辰可能代表 |
|---|---|
| 一般占事 | 當下環境、正在發生的事 |
| 商業求財 | 市場狀況、風潮、趨勢 |
| 感情婚姻 | 現任、當下出現的人、男方或新郎象需依六親判斷 |
| 職場轉職 | 產業環境、外部壓力、朋友慫恿、趨勢推動 |

### 實戰用法

```text
If 日辰剛好是用神五行或與用神同類
Then 可作當下環境正在推動該用神。

If 日辰合應又沖世
Then 外部目標被合住，而自身受沖，屬高層級警示。
```

## 九、日辰合應沖世：高層級警示

案例 3-15 示範：日辰同時合應爻、沖世爻。

```text
日辰合應爻 → 外部目標、公司、合作方被封住、被日辰牽制。
日辰沖世爻 → 自身被外部環境衝擊。
若世爻無日合解沖 → 傷害持續，警示加重。
```

### 實戰用法

```text
If 問轉職 / 合作 / 投資
And 日辰合應沖世
Then 外部目標看似被環境牽住，自身反受衝擊，需高度謹慎。

If 又見玄武兄弟伏藏、朱雀子孫發動
Then 防隱藏利益、話術企劃、口舌官非。
```

## 十、真破與出月應事

案例 3-15 補強真破。

```text
月破 + 旬空 = 真破。
真破之爻當下力量破敗。
但若得日辰生扶，出月後仍可能應事。
```

### 實戰用法

```text
If 子孫爻真破但得日生
Then 當下企劃、技術、行銷未成熟；但出月後可能發動應事。

If 真破爻臨朱雀子孫
Then 可作吹噓企劃、話術方案、口舌是非的後續引爆點。
```

## 十一、職場轉職風險判斷

案例 3-15 可放入職場轉職類 SOP。

### 取用神

| 爻位 / 六親 | 職場轉職代表 |
|---|---|
| 世爻 | 自己 |
| 應爻 | 新公司、任職之地、合作單位 |
| 官鬼 | 職位、主管、職權、法律壓力 |
| 父母 | 上司領導、制度、合約、文書 |
| 妻財 | 薪酬、獎金、公司財務 |
| 子孫 | 技術、企劃、產品、行銷 |
| 兄弟 | 競爭、耗財、團隊內耗、隱藏利益 |

### 高風險組合

```text
應爻臨騰蛇妻財 → 新公司或外部目標財務不透明、不正規。
父母臨玄武 → 上司或制度暗藏虛偽、隱情、合約風險。
玄武兄弟伏藏且可出 → 隱藏利益、詐財、內部耗損。
朱雀子孫發動 → 企劃話術、口舌是非、宣傳風險。
日辰合應沖世 → 外部目標被牽制，自身受傷無解。
```

## 十二、決策建言轉換原則

```text
If 沖中逢合者優於合處逢沖
Then 建議選擇前者，即使初期困難，後期較可整合。

If 日辰合應沖世，且世爻無日合解沖
Then 建議避免投入，防自身受損。

If 代占中代表願望的伏神不可出
Then 建議承認願望不易落地，不宜勉強撮合。

If 年命爻顯示疑慮，但用神生合有情
Then 建議不要被內心疑慮完全主導，仍需看現實互動與卦爻主體。
```

## 阿咪重點速記

- 年命爻看深層意念，不定吉凶。
- 六沖六合只看趨勢，不單獨定成敗。
- 代占第三人稱要先轉換用神，不可一律以世爻為主角。
- 飛剋伏無救，多為不可出伏神，願望難落地。
- 沖中逢合優於合處逢沖，前者失而復得，後者得而復失。
- 變爻與本位動爻無生剋沖合時，可擴散到旁爻或伏神。
- 日辰合應沖世是重大警示，特別適合用於職場、合作、投資風險判斷。
- 真破當下不成，但得日生者出月仍可應事。
