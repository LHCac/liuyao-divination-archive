# 書籍案例實戰技法提煉：第五批

## 主題摘要

本篇整理四個書籍實際案例中，可直接放入後續解卦流程的技法。此批案例以疾病、代占六親疾病、孕產求子、風水辨爻與逆向尋因為核心，補強疾病占中的用神、元神、仇神、藥物、飲食生活、雙官夾命、植物人垂象、風水病因排除與確認。

收錄案例：

| 案例 | 卦象 | 主題 |
|---|---|---|
| 3-20 陪伴是最長情的守護 | 天地否 → 天山遁 | 疾病占、雙官夾命、年命爻入官鬼、病入膏肓 |
| 3-21 醫玄共濟渡危厄 | 風天小畜 → 風火家人 | 代占六親疾病、用神肉體、元神魂魄、植物人垂象 |
| 3-23 古井填塞與因果業力 | 澤山咸 → 雷山小過 | 疾病占、逆向尋病因、藥物副作用、排除風水病因 |
| 3-24 求子路迢迢 | 山雷頤 → 風雷益 | 孕產求子、元神概念、貪生忘剋、風水辨爻 |

> 注意：疾病與孕產類占卜只能作為象意參考，不可替代醫師診斷、治療與專業醫療建議。

## 一、疾病占基本用神

| 六親 / 爻象 | 疾病占象意 |
|---|---|
| 世爻 | 本人、自身狀態 |
| 官鬼 | 疾病、病災、病灶、病勢 |
| 子孫 | 藥物、治療方式、醫療方案、解憂之神 |
| 妻財 | 飲食、生活方式、食療、生活習慣 |
| 父母 | 醫院、病歷、檢查、文書、醫療機構 |
| 兄弟 | 消耗、破氣、元神相關時需依問法轉換 |

### 疾病占基本吉凶

```text
喜日辰或動爻剋官鬼。
喜日辰或動爻生世。
官鬼受剋受沖而無救，病勢可制。
官鬼動來剋世，或日辰為官鬼剋世，病勢較危。
子孫旺相而動，藥物對症。
子孫休囚、空破、日沖月破或受剋，藥效不足。
```

## 二、自占疾病：必須打敗官鬼

案例 3-20 是重要示範。

```text
If 自占疾病
Then 官鬼是敵人，必須被子孫、日辰或動爻有效剋制。

If 官鬼受剋但得生
Then 官鬼得到解救，病勢不退。

If 官鬼受沖但得日合
Then 官鬼得到解救，病勢不退。
```

### 實戰判斷

```text
子孫剋官鬼 = 藥物能打病。
但官鬼得世爻妻財生 = 生活方式反而養病。
官鬼受子孫沖，但得日辰合 = 病勢被解救，仍難控制。
```

## 三、雙官夾命

案例 3-20 補強疾病凶象。

```text
If 二爻官鬼與四爻官鬼夾住三爻世爻
Then 雙官夾命，為病入膏肓、雙重病灶或病勢夾攻之象。
```

### 補強條件

```text
If 年命爻又落在官鬼爻
Then 疾病與本命深度相連，凶象加重。
```

## 四、年命爻在疾病占的用法

案例 3-20：年命爻在四爻午火朱雀官鬼。

```text
If 年命爻落官鬼
Then 此病與本命、體質、長期命理傾向有深度關聯。

If 年命爻合父母或伏神子孫
Then 可補充其心念、牽掛、求醫方向或對家人的牽連。
```

## 六、代占六親疾病：用神是肉體，元神是魂魄

案例 3-21 是核心案例。

```text
If 代占第三人稱疾病
Then 以被占者六親為用神。
用神代表肉體。
能生用神者為元神，代表魂魄、意識、生命精神。
沖剋用神者為病。
傷害元神者可作仇神。
```

### 植物人垂象

```text
If 用神旺，代表肉體尚存。
But 元神受剋受沖無解，尤其在上爻頭部位置
Then 可作意識受損、昏迷、植物人垂象。
```

### 上爻身體定位

```text
上爻可取頭部、腦部、意識高位。
上爻元神受沖無日合，為嚴重意識受損象。
```

## 七、仇神概念

案例 3-21 中伏神酉金官鬼為仇神。

```text
If 某爻專門傷害元神
Then 可作仇神。
仇神不是單純疾病，而是專傷魂魄、意識、元神之敵對力量。
```

### 解救規則

```text
元神受剋，得生可解。
元神受沖，需日合解。
若元神受沖而無日合，則無解。
```

## 八、丑土為刀兵、開刀手術

案例 3-21 補充醫療取象。

```text
丑土可取刀兵、開刀、手術、切割、醫療器械。

If 動爻化丑土，且丑土生仇神或官鬼
Then 再次開刀或手術可能加重負擔，需謹慎。
```

## 九、逆向尋病因

案例 3-23 是核心技法。

```text
疾病占不只看誰剋官鬼，也要看誰生官鬼。
由官鬼反推：官鬼 ← 生官鬼者 ← 生生官鬼者
即可找到病因來源。
```

### 範例結構

```text
官鬼（疾病）
← 妻財（飲食生活、副作用、生活習慣）
← 子孫（藥物、治療方式）

若子孫生妻財，妻財再生官鬼
Then 表示藥物或治療方式透過副作用、生活代謝問題，使疾病延續。
```

## 十、子孫不一定永遠吉：藥物也可能成病因之一

案例 3-23 補強。

```text
疾病占中，子孫為藥物，通常可剋官鬼為吉。
But If 子孫同時生妻財，妻財再生官鬼
Then 藥物可能透過副作用、併發症或不當使用，間接餵養疾病。
```

### 判斷重點

```text
看子孫是否直接剋官鬼有效。
看官鬼是否又得到其他爻生扶。
追查生官鬼者是否來自子孫轉化。
```

## 十一、進神退神也有重複加倍作用

案例 3-23 補強。

```text
卦爻化進神或退神，都隱含重複一次、雙倍影響力。
進神：力量推進、加重、擴大。
退神：力量後退，但仍有重複影響，不是完全消失。
```

## 十二、月破伏神出月不破，可出伏

案例 3-23 補強。

```text
If 伏神月破
Then 當月力弱，但出月不破。
月破伏神仍可作可出伏神，需看後續生剋沖合。
```

## 十三、風水辨爻法：疾病與家宅互參

案例 3-21、3-23、3-24 都用到風水辨爻法。

| 爻位 | 常見風水 / 身體象意 |
|---|---|
| 初爻 | 地基、土地、井、根基、低位神靈 |
| 二爻 | 爐灶、居家核心、廚房、生活火源 |
| 五爻 | 尊位、神位、菩薩、天頂、主導力量 |
| 六爻 | 水口、頭部、高位、遠方 |

### 排除錯誤病因

```text
If 問古井、地基、風水是否為病因
Then 需看對應爻是否進入官鬼生扶鏈。
If 對應爻不在病因能量鏈上
Then 可排除該風水假設。
```

## 十四、孕產求子用神

案例 3-24 補強孕產規則。

| 象 | 孕產占意義 |
|---|---|
| 子孫 | 胎孕、胚胎、子息用神 |
| 元神 | 能生子孫者，胎孕之元神 |
| 父母 | 傳統習俗、宗教信仰、醫療文書，也可能剋子孫 |
| 妻財 | 飲食、食補、食療、生活調理 |

### 基本規則

```text
子孫得生：吉。
子孫受剋：凶。
子孫逢沖：早產、流產、著床不穩。
子孫逢合：相對平安。
合之太過，如二重合、入墓逢合：難產。
六沖卦而無合：早產。
白虎動剋子孫：流血、早產、墮胎風險。
勾陳父母動剋子孫：跌傷、損胎、流產風險。
```

## 十五、孕產占中的元神

案例 3-24：日辰卯木兄弟是子孫之元神。

```text
能生用神者，稱為元神。
孕產占中，能生子孫胎孕者，就是胎孕元神。

If 元神為日辰
Then 日辰乃六爻主宰，四時俱旺，可有效解救受剋用神。
```

## 十六、貪生忘剋在孕產中的救援

案例 3-24 補強。

```text
If 某爻本可剋元神
But 其更優先去生父母或其他爻，形成貪生忘剋
Then 元神可得保留，胎孕仍有培育成功可能。
```

### 連環救援

```text
用神子孫得元神生，受剋可解。
元神又因忌神貪生忘剋而得存。
此為用神與元神雙重得救。
```

## 阿咪重點速記

- 疾病占：世是本人，官鬼是病，子孫是藥，妻財是飲食生活。
- 自占疾病要打敗官鬼；官鬼受剋得生、受沖得日合，病反而不退。
- 雙官夾命是嚴重凶象，若年命又在官鬼更重。
- 代占六親疾病：用神是肉體，元神是魂魄。
- 用神旺但元神受沖無解，尤其在上爻頭部，可作植物人垂象。
- 仇神是專門傷害元神的敵對力量。
- 丑土可取刀兵、開刀、手術。
- 逆向尋病因：從官鬼往回追誰生它。
- 子孫為藥物，但也可能因副作用間接養病。
- 月破伏神出月不破，仍可出伏。
- 孕產占以子孫為胎孕，能生子孫者為元神。
- 子孫得生吉，受剋凶，逢沖防早產流產，白虎動剋子孫防流血。
- 風水辨爻可用來確認或排除病因假設。
