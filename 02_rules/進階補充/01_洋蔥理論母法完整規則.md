# 洋蔥理論母法完整規則

## 主題摘要

本篇整理《六爻知先機》第六章「洋蔥理論母法」的核心規則，內容包含月建、日辰、六合、六沖、三合、反吟、伏吟、旬空、三刑、飛神伏神、進神退神、入墓、獨靜獨發與應期法則。

此檔作為進階總覽，實際斷卦時仍需回到各分類細則：月日旺衰、刑沖合、動爻變化、伏藏飛伏、卦象技法與應期規則。

資料來源：《六爻知先機》第六章「洋蔥理論母法」第076～093頁。

## 一、論月建

### 核心原則

```text
月建掌一月之權，司三旬之令。
月建主旺衰，是六爻判斷力量的提綱。
月建不入爻，也仍然有作用；若入卦，力量更明顯。
```

### 判斷重點

| 情況 | 判斷 |
|---|---|
| 爻臨月建 | 旺相當權，力量強 |
| 用神臨月建，且無他爻傷剋 | 凡占多吉，得力不輕 |
| 忌神臨月建，而用神休囚無救 | 凶力重，諸占不利 |
| 月建沖爻 | 月破 |

### 月破規則

| 規則 | 說明 |
|---|---|
| 月破 | 凡受月建沖破之爻，稱為月破 |
| 月破象意 | 破缺不全、力量不厚 |
| 月破作用 | 所生所剋皆無實力 |
| 出月不破 | 出本月後，月破力量可恢復，生剋重新有作用 |
| 月破又日沖 | 稱為日破，該爻不能用 |

### If / Then

```text
If 用神臨月建且不受傷
Then 用神有力，事情有根基

If 忌神臨月建且用神無救
Then 忌神勢強，凶象加重

If 用神月破
Then 當月力量破缺，需看出月、填實、合破或其他救應

If 月破又逢日沖
Then 日破，該爻難以作用
```

## 二、論日辰

### 核心原則

```text
日辰為六爻之主宰，司四時之旺相。
月建司一月之令，日辰則四時俱旺，操生殺之權。
日辰與月建同功，但作用偏向當下、觸發、沖合、暗動、沖脫。
```

### 日合

| 規則 | 說明 |
|---|---|
| 日合 | 與日辰相合之爻稱日合 |
| 吉象 | 日合多主貴人扶助、事情成就 |
| 忌神日合 | 忌神得合不吉，忌神力量被扶住 |
| 空合 | 旬空爻逢日合，主虛約難憑，反不成事 |

### 日沖

| 情況 | 名稱 | 判斷 |
|---|---|---|
| 靜爻逢日沖 | 暗動 | 有生剋之力，無沖合之力 |
| 動爻逢日沖 | 沖脫 / 動散 | 生剋之力不能作用 |
| 變爻逢日沖 | 沖脫 / 連根拔起 | 生剋之力不能作用 |
| 月破爻逢日沖 | 日沖月破 | 生剋之力不能作用 |

### If / Then

```text
If 靜爻旺相逢日沖
Then 可成暗動

If 動爻逢日沖
Then 動爻沖脫，生剋之力難以作用

If 變爻逢日沖
Then 變爻沖脫，本爻與變爻連根受損

If 旬空爻逢日合
Then 空合，承諾或助力偏虛
```

## 三、論六合

### 六合對照

| 六合 | 組合 |
|---|---|
| 子丑合 | 子、丑 |
| 寅亥合 | 寅、亥 |

| 辰酉合 | 辰、酉 |
| 巳申合 | 巳、申 |
| 午未合 | 午、未 |

### 六合象意

```text
合者，成也、聚也。
占謀事多有利，但仍需看用神旺衰與合住是否成阻。
```

### 六合類型

| 類型 | 說明 |
|---|---|
| 日月合爻 | 月建或日辰與爻相合 |
| 爻與爻合 | 卦中兩爻相合 |
| 爻動化合 | 動爻化出六合地支 |
| 六合卦 | 初四、二五、三上皆合 |

### 六合卦例

```text
天地否、澤水困、火山旅、雷地豫、水澤節、山火賁、地雷復、地天泰。
```

### 空合

```text
合爻逢旬空，稱為空合。
主虛約難憑、虛情假意、事難成。
```

## 四、論六沖

### 六沖對照

| 六沖 | 組合 |
|---|---|
| 子午沖 | 子、午 |
| 丑未沖 | 丑、未 |
| 寅申沖 | 寅、申 |
| 卯酉沖 | 卯、酉 |
| 辰戌沖 | 辰、戌 |
| 巳亥沖 | 巳、亥 |

### 六沖象意

```text
沖者，散也、敗也。
占一般事情多不利；但占災禍、官非、疾病，沖散反而可主災散無災。
```

### 相沖類型

| 類型 | 說明 |
|---|---|
| 日辰沖爻 | 靜爻旺則暗動，衰則日破或動散 |
| 月建沖爻 | 月破 |
| 爻與爻沖 | 相擊 |
| 動爻變沖 | 爻反吟或動變相沖 |
| 六沖卦 | 初四、二五、三上皆沖 |

### 合處逢沖與沖中逢合

| 情況 | 定義 | 結果 |
|---|---|---|
| 合處逢沖 | 先合後沖 | 聚而後散，先成後敗，終有不成 |
| 沖中逢合 | 先沖後合 | 散而後聚，先敗後成，終有變化 |

### 六合卦與六沖卦用法

| 卦變 | 象意 |
|---|---|
| 六合卦化六沖卦 | 先合後離 |
| 六沖卦化六合卦 | 先散後聚 |
| 六沖卦化六沖卦 | 上下不合，至親反目，始終不就 |

重要提醒：

```text
單看六合卦或六沖卦，不可直接定成敗。
卦之吉凶成敗，仍以用神之爻的生剋沖合為主。
卦化六合或六沖有背景參考價值，但不可取代用神判斷。
```

三合局有成局與不成局。

## 六、論反吟

### 反吟分類

| 類型 | 說明 |
|---|---|
| 卦反吟 | 卦化對沖 |
| 爻反吟 | 爻化相沖，如巳化亥、酉化卯 |

重要原則：

```text
卦的反吟伏吟影響力較小。
爻的反吟伏吟影響力較大。
```

### 反吟卦組合

| 反吟組合 |
|---|
| 乾變巽，巽變乾 |
| 姤變小畜，小畜變姤 |
| 坎變離，離變坎 |
| 未濟變既濟，既濟變未濟 |
| 艮變坤，坤變艮 |
| 剝變謙，謙變剝 |
| 震變兌，兌變震 |
| 歸妹變隨，隨變歸妹 |

### 反吟象意

```text
內卦反吟：內不安。
外卦反吟：外不寧。
內外反吟：內外不安。

反吟多主反覆不定、痛苦折磨、憂心呻吟。
但吉凶仍以用神為主。
```

### 判斷提醒

```text
If 用神旺相且不變沖剋
Then 雖有反覆，事情仍可能成

If 用神化回頭沖剋
Then 大凶，反吟凶象加重
```

| 甲寅旬：甲寅～癸亥 | 子、丑 |

### 旬空用法

| 規則 | 說明 |
|---|---|
| 旬空之爻 | 虛而不實，舉棋不定 |
| 旺者不為空 | 爻旺時空象減弱，不作真空論 |
| 衰者為真空 | 爻衰又空，力量虛弱不實 |
| 填實 | 空亡逢值填實，應吉凶 |
| 合空 | 虛約、虛情假意、難實踐 |
| 沖空 | 主移越、轉移、變遷，吉凶不成象 |
| 沖脫 | 動爻或變爻逢日沖，不能用，吉凶皆不成 |

### 特別提醒

```text
戊己日占卦，較容易出現沖空爻象。
動爻受日沖稱沖脫，本爻變爻連根生長，拔去本爻，變爻亦不能生存。
變爻受日沖也稱沖脫，拔去變爻，本爻亦不能生存。
```

## 十一、論飛神、伏神

### 伏神成立

```text
凡本卦中六親未全部出現，皆需定出伏神。
```

### 尋伏神方法

```text
1. 先看本卦屬於何宮。
2. 回到該宮首卦。
3. 由首卦中找出本卦缺少的六親。
4. 首卦所出現缺六親之爻位，即為伏神所在。
5. 本位卦爻稱為飛神或飛爻。
```

### 伏藏爻用法

| 情況 | 判斷 |
|---|---|
| 伏爻可受飛爻作用 | 飛爻可生伏、剋伏、合伏、沖伏 |
| 飛爻生伏 | 可出伏 |
| 飛爻合伏 | 可出伏 |
| 飛爻剋伏 | 一般不可出伏 |
| 日辰沖伏 | 不能出伏，如入囚牢 |
| 飛爻逢空、入墓、化墓、化空 | 伏爻可出 |
| 飛爻自化回剋、回沖 | 伏爻可出 |
| 日辰沖開飛爻 | 伏爻可出 |

### 伏爻出伏後

```text
伏爻若可出，對六爻皆有生剋沖合之力，亦可受日辰或其他動爻生剋沖合。

吉神不出伏則凶。
凶神不出伏則吉。
```

## 十二、入墓之爻

### 四墓對照

| 五行 / 地支 | 墓庫 |
|---|---|
| 寅卯木 | 未墓 |
| 巳午火 | 戌墓 |
| 亥子水 | 辰墓 |
| 申酉金 | 丑墓 |

### 三種入墓

| 類型 | 說明 |
|---|---|
| 入日墓 | 爻支入日辰之墓 |
| 入動墓 | 動爻為他爻之墓 |
| 入化墓 | 動爻自化入墓 |

### 入墓用法

```text
入墓之爻，如困於牢籠之中，不能正常作用。
入墓有可出墓與不能出墓之分。
```

### 重要修正

```text
「隨鬼入墓」未必必然大凶。
入墓之爻，必須休囚無力時才見凶危。
若旺而有扶，仍有解救。
```

## 十三、獨靜、獨發

| 名稱 | 定義 |
|---|---|
| 獨靜 | 五爻皆動，一爻不動 |
| 獨發 | 五爻皆靜，一爻發動 |

### 補充規則

```text
卦中一爻明動，一爻暗動，不算獨發。
六爻安靜，只有一爻暗動，也可稱獨發。
```

### 使用提醒

```text
獨靜、獨發主要用於觀事之成敗遲速。
古法有「獨發看發的爻，獨靜看靜的爻」。
但實務斷卦仍須以用神為重。
```

## 十四、論應期

### 應期速查

| 卦象狀態 | 應期規則 |
|---|---|
| 靜 | 逢沖應 |
| 動 | 逢合、逢值應 |
| 合住 | 待沖開應 |
| 月破 | 出月、填實、合破應 |
| 空亡 | 逢填實應 |
| 非特殊規範 | 依四時五行旺相休囚推斷 |

### 補充要點

```text
爻若逢合住，待沖開應之。
遠事定之以年月。
近事應之於日時。
不可因短期不驗，就說卦不驗；需依遠近事件調整應期單位。
```

## 十五、實戰總檢查表

正式斷卦時，可依序檢查：

```text
☐ 月建：用神、忌神是否臨月、月合、月破？
☐ 日辰：是否日合、日沖、暗動、沖脫？
☐ 六合：是否合住、空合、六合卦？
☐ 六沖：是否月破、日沖、爻沖、六沖卦？
☐ 三合：是否成局？是否有回頭剋破局？
☐ 反吟：卦反吟或爻反吟？重點看用神是否化沖剋。
☐ 伏吟：是否鬱滯不行、反覆不安？
☐ 旬空：是真空、旺不為空、合空、沖空、沖脫？
☐ 三刑：是否符合三刑成立條件？
☐ 進退：動爻是否化進神或退神？
☐ 飛伏：伏神是否可出？飛神是否生、合、剋伏？
☐ 入墓：入日墓、入動墓或入化墓？是否有出墓可能？
☐ 獨靜獨發：是否有成敗遲速提示？
☐ 應期：靜逢沖，動逢合值，空亡填實，合住待沖。
```

## 阿咪重點速記

- 月建主一月旺衰，日辰主當下生殺與觸發。
- 月破之爻當月力弱，日沖月破則不能用。
- 日合可成事，但旬空日合是空合，虛約難憑。
- 靜爻日沖為暗動；動爻、變爻日沖為沖脫，不能用。
- 六合六沖不能單獨定成敗，用神生剋才是核心。
- 三合局成則勢旺，但遇回頭剋可不成局。
- 反吟伏吟看用神，用神旺則可成，用神化沖剋則凶。
- 旬空旺者不為空，衰者才是真空。
- 三刑需有發動條件，不是看到地支就算刑。
- 飛生伏、飛合伏可出伏；飛剋伏多不出伏，除非飛爻被沖開或自受制。
- 入墓不必然大凶，休囚無力才危，旺而有扶仍可救。
- 應期：靜逢沖、動逢合值、空亡填實、合住待沖。
