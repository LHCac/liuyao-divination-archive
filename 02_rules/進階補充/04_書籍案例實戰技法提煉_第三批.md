# 書籍案例實戰技法提煉：第三批

## 主題摘要

本篇整理四個書籍實際案例中，可直接放入後續解卦流程的技法。此批案例補強重點包括：占此應彼、動爻方向意念、日辰臨世爻或應爻、飛神逢日沖出伏、空下伏神易引拔、真空不受生、收益與聲譽衝突、妻財三現與年命爻揭露真實動機。

收錄案例：

| 案例 | 卦象 | 主題 |
|---|---|---|
| 3-08 綿裡藏針 | 雷風恆 → 雷水解 | 女占感情、年命爻、妻財三現、受沖無解、空下伏神 |
| 3-10 愛的迷途與歸航 | 火雷噬嗑 → 山雷頤 | 女占婚姻、兩段卦、動爻方向意念、子女被忽略 |
| 3-11 愛的投資 | 澤雷隨 → 澤火革 | 投資占、占此應彼、伏神出伏、資金流向追蹤 |
| 3-14 演藝之路 | 火風鼎 → 火山旅 | 職場工作、日辰臨應爻、飛神逢日沖出伏、收益與聲譽衝突 |

## 一、占此應彼：問 A 事，卦顯 B 事

案例 3-11 是重要示範：占問表面是投資，卦象卻揭示感情糾葛與資金流向他人。

### 判斷條件

```text
If 占問主題用神不動、應爻不動或與主線無強關聯
And 世爻動向另一個關鍵爻位明顯輸送資源、情感或意念
Then 可懷疑占此應彼。
```

### 實戰用法

```text
投資卦：應爻代表合作方或求財之地。
If 應爻靜而不動，世爻卻動生合官鬼或其他高位爻
Then 需追蹤資金是否真正流向應爻，或被其他人吸收。
```

### 注意

```text
占此應彼不可輕用。
必須有明確意念流向支持，例如世爻動、生合他爻、剋父母文書、不走應爻路線。
```

## 二、動爻方向決定意念主語

案例 3-10 強調：同樣是六合，哪一方發動，主語完全不同。

### 核心規則

```text
動爻是表徵，靜爻是裡象。
動爻發出行為，靜爻承受意念。
同樣的合，方向不同，意義完全不同。
```

### 感情案例應用

```text
If 官鬼動合妻財
Then 是男方主動靠近、牽連其他女性。

If 妻財動生合官鬼
Then 是其他女性主動勾搭、吸引男方。
```

### 決策提醒

```text
不可只看「官財相合」就判出軌。
必須看誰動、誰靜、誰發出意念。
```

## 三、女占婚姻：妻財持世的辨別

案例 3-10 提醒：女占婚姻時，妻財若持世，不一定代表情敵。

```text
If 女占婚姻，妻財持世
Then 可代表本人愛慕男方、情意深、重視關係與財務安全。

If 妻財不持世、又與官鬼發生動合、生合、伏藏關係
Then 才較可能作情敵或其他女性。
```

### 補充

```text
妻財持世要先看是否為占問者本身。
不可機械判為情敵。
```

## 四、日辰臨世爻、日辰臨應爻

第三批案例多次補強：日辰臨某爻時，日辰之生剋沖合可視為該爻意念的放大。

### 日辰臨世爻

```text
If 日辰臨世爻
Then 日辰等同世爻意念，日辰對其他爻的生剋沖合，可視為占問者當下深層意念與行為方向。
```

應用：

```text
女占婚姻，日辰臨世並生官鬼 → 本人仍愛丈夫。
日辰臨世並剋父母家庭 → 本人想逃離家庭結構。
```

### 日辰臨應爻

```text
If 日辰臨應爻
Then 日辰等同對方意念，日辰之生剋沖合可判斷對方的行為、策略、動機與外部主導力量。
```

應用：

```text
職場占，日辰臨應爻經紀人 → 日辰代表經紀人與市場潮流。
若日辰沖起子孫暗動並剋世 → 對方的企劃能帶收益，但會衝擊本人形象或利益。
```

## 五、飛神逢日沖，伏神可出

案例 3-14 補強出伏條件。

```text
If 飛神被日辰沖動、沖開
Then 伏神有機會出伏。
```

### 實戰用法

```text
父母伏於子孫下：父母可作合約、文書。
If 飛神子孫逢日沖
Then 合約問題浮出檯面，可對旁爻放射生剋沖合。
```

### 與飛剋伏區分

```text
飛神剋伏神 → 通常伏神不易出，除非得日月動爻生救。
飛神逢日沖 → 飛神被沖開，伏神反而有出伏條件。
```

## 六、空下伏神易於引拔

案例 3-08 補強黃金策規則。

```text
If 飛神旬空
Then 伏神容易被引拔出伏。
```

### 實戰用法

```text
If 父母飛神旬空，伏神為玄武兄弟
Then 隱藏的陰謀、利益動機、消耗之象容易浮出。

If 伏神出伏後生子孫、剋妻財、沖官鬼
Then 可追蹤其背後真實動機與破壞範圍。
```

## 七、真空不受生

案例 3-14 補強：旬空又月破者，為真空，受生不實。

```text
If 爻旬空 + 月破
Then 可視為真空。
真空之爻不受生。
```

### 實戰用法

```text
If 兄弟爻真空
Then 競爭者力量虛，雖被他爻所生，也不成實力。

If 真空爻代表競爭者
Then 反而凸顯世爻優勢，表示本人有獨占鰲頭機會。
```

## 八、受沖無日合、受剋無生救

案例 3-08 明確示範傷害無解。

```text
受沖之爻，必須得日合才能解救。
受剋之爻，必須得生扶才能解救。
```

### 實戰用法

```text
If 白虎妻財受日沖，且無日合解沖
Then 喪事、血光、重大傷害象持續。

If 同時再受可出伏神剋，且無他爻生救
Then 傷害更重，不可輕判可解。
```

## 九、妻財三現的高階辨別

案例 3-08 提醒：同一卦中妻財多現，不可視為同一人。

| 妻財位置 | 可能象意 |
|---|---|
| 日辰妻財 | 當下出現的新女性、新娘、即時事件中的女性 |
| 初爻白虎妻財 | 喪事、血光、亡妻、低位但根源性女性象 |
| 六爻騰蛇妻財 | 宿世姻緣、遠方女性、名分外或幻想性女性 |
| 世爻妻財 | 女占時可為本人情意與財務欲望，不一定是情敵 |
| 應爻妻財 | 對方、外部女性、占問所指對象 |

### 補充

```text
妻財多現時，需結合六神、爻位、世應、年命爻、動靜、日辰與伏神來定身份。
```

## 十、年命爻揭露真實角色

案例 3-08 補強年命爻用法：年命爻可揭露本人或對方在事件中的真實角色。

```text
If 占問者年命爻落兄弟，且臨玄武
Then 可能有謀財、算計、隱藏利益動機。

If 對方年命爻落白虎妻財
Then 可指其心仍牽掛亡妻、喪事或過往財務關係。
```

### 注意

```text
年命爻不單獨定吉凶，但可補強身份辨別與動機判斷。
```

## 十一、資金流向追蹤法

案例 3-11 可作投資占重要 SOP。

### 流程

```text
1. 定世爻與應爻。
2. 找妻財爻代表資金。
3. 看世爻是否動。
4. 看世爻把財生給誰、剋誰、合誰。
5. 看伏神是否出伏，是否形成中間策略或話術。
6. 追蹤「伏神 → 世爻 → 其他爻」的完整能量鏈。
7. 若資金沒有流向應爻，需防假投資、傳聲筒、第三方受益。
```

### 警示組合

```text
應爻靜不動 + 世爻動生合五爻官鬼 + 父母文書受剋
= 投資恐無正式契約保障，資金可能流向非表面合作方。
```

## 十二、收益 vs 聲譽衝突

案例 3-14 可作職場與合作判斷。

```text
If 對方之意念或企劃能生妻財
But 同時剋世爻官鬼、沖擊世爻形象
Then 可判收益與聲譽衝突。
```

### 職場占應用

```text
子孫 → 才藝、企劃、行銷、產品。
妻財 → 薪酬、收益。
官鬼持世 → 職位、形象、專業身份。
父母伏神出伏 → 合約、文書、限制條款。
```

### 建議轉換

```text
If 合約可帶曝光但剋世爻形象、沖財、耗財
Then 不宜只看機會，要看專業聲譽與長期職涯。
```

## 十三、靜爻居外卦，不代表不受影響

案例 3-10 的子女爻示範：子孫靜而居外卦，未必代表不受影響。

```text
If 子孫爻靜而居外卦、無他爻生合照應
Then 可作被置於場外、被忽略、無人照顧其情緒需求。
```

### 應用

```text
婚姻卦中，夫妻互動與情敵象強烈，但子孫靜居外卦
Then 要提醒孩子不是沒受影響，而是被忽略。
```

## 十四、兩段卦對照法

案例 3-10 同一人前後兩段卦：離婚前與復合問。

```text
If 同一人對同一事件不同階段起卦
Then 可將兩卦對照，追蹤事件發展、動機轉變與結果驗證。
```

### 實戰用法

```text
第一卦看事件爆發前的根源。
第二卦看事件後續走向與真實目的。
若兩卦用神、動爻方向、子女象互相呼應，可信度提高。
```

## 阿咪重點速記

- 占此應彼要看意念流向，不是憑感覺轉題。
- 同樣六合，誰動誰主動，方向完全不同。
- 女占婚姻妻財持世，先看本人情意，不要直接判情敵。
- 日辰臨世爻，是世爻意念被放大；日辰臨應爻，是對方意念被放大。
- 飛神逢日沖，伏神可出。
- 飛神空亡，空下伏神易於引拔。
- 真空不受生。
- 受沖要日合解，受剋要生扶解。
- 投資占要追資金流向，錢流向誰，真相就在誰那裡。
- 職場占不能只看收入，也要看形象、聲譽、合約限制。
- 子女爻靜居外卦，不是不受影響，而是被忽略。
