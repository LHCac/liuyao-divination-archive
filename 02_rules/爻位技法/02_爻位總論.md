# 爻位總論

## 主題摘要

爻位是六爻卦的立體結構框架。六個爻位各自代表時間、空間、人物層次與事物階段。爻位結合五行生剋、六親、六神、旺衰與動變，可以讓卦象資訊更立體、更精細。

## 爻位基本對應

| 爻位 | 自然層次 | 人物層次 | 時間階段 | 空間位置 |
|---|---|---|---|---|
| 初爻 | 地下 | 員工、小孩、初始 | 開始、童年 | 地下、室內深處 |
| 二爻 | 地面 | 主管、成家 | 成長期 | 宅、房子、室內 |
| 三爻 | 人 | 中層副職 | 奔波期 | 室外近處 |
| 四爻 | 人 | 中層幹部 | 壯年 | 室外、門外 |
| 五爻 | 天 | 大領導、鼎盛 | 巔峰 | 道路、外地、重要位置 |
| 六爻 | 天外 | 退休、祖先、先天 | 結束 | 高處、天空、遠方 |

## 先天與後天雙軌

### 後天軸

```text
初爻 → 二爻 → 三爻 → 四爻 → 五爻 → 六爻
```

代表由下往上的成長、發展、推進，是主要判斷軸。

### 先天軸

```text
六爻 → 五爻 → 四爻 → 三爻 → 二爻 → 初爻
```

代表祖先、基因、先天影響、命運底色等由上而下的因素。

實務上，以後天為主，先天為輔。

## 爻位與陰陽

| 陰陽 | 象意 |
|---|---|
| 陽爻 | 正職、主位、外顯、剛強 |
| 陰爻 | 副職、輔助、內藏、柔順 |

例如問職場：

- 官鬼在五爻陽爻：核心領導、大領導、正式職位。
- 官鬼在初爻：基層職位、普通員工、剛開始。

## 常見應用

| 情境 | 爻位判斷 |
|---|---|
| 問事情開始 | 用神在初爻，事情剛萌芽 |
| 問事情結果 | 用神在六爻，接近尾聲或有退出象 |
| 問工作層級 | 官鬼在五爻，多主核心職位或高層 |
| 問房屋 | 二爻多主宅、室內、自住房 |
| 問出行 | 五爻多主道路、目的地、外部環境 |
| 問祖先或先天 | 六爻可輔看祖先、命運底色、高處之事 |

## If / Then 判斷

```text
If 用神在初爻
Then 事情才剛開始，尚未成形

If 用神在五爻
Then 事情在重要位置，力量或影響較大

If 用神在六爻
Then 事情接近尾聲，或有退出、結束、遠方、高處之象

If 問職場且官鬼在五爻陽爻
Then 可取核心領導、正職、重要職位

If 問職場且官鬼在初爻
Then 可取基層員工、低層職位、初入職

If 問房子且用神在二爻
Then 與自住房、室內、宅內問題相關

If 問出行且用神在五爻
Then 道路、目的地、外部環境很重要
```

## 阿咪重點速記

- 爻位是六爻的立體骨架。
- 初爻為始，六爻為終。
- 五爻多為重要位置、道路、外地、大領導。
- 二爻多為宅、室內、自住房。
- 六爻有結束、遠方、祖先、先天之象。
