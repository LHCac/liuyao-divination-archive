# 爻位靜態與動態

## 主題摘要

六爻爻位有靜態與動態兩種用法。靜態爻位是各爻固定代表不同層次、位置與身體部位；動態爻位是某些爻發動後，其象意可能轉移到另一個爻位，尤其同五行爻之間更容易出現同氣相求。掌握兩者可提升卦象細節判斷能力。

## 靜態爻位

### 房屋取象

| 爻位 | 房屋類象 |
|---|---|
| 初爻 | 地基、地下 |
| 二爻 | 宅、房間內部 |
| 三爻 | 門、室外近處 |
| 四爻 | 門外、院子 |
| 五爻 | 道路、遠處 |
| 六爻 | 遠方、天上 |

### 人體取象

| 爻位 | 人體類象 |
|---|---|
| 初爻 | 腳 |
| 二爻 | 膝蓋 |
| 三爻 | 肚子 |
| 四爻 | 肺 |
| 五爻 | 心臟 |
| 六爻 | 頭 |

## 動態爻位

當某爻發動時，尤其同五行爻之間，爻位代表的含義有時會轉移。

### 動態模式特點

- 同五行之間容易出現同氣相求。
- 原爻位上的六神、空破、旺衰等信息，可能一同轉移到新位置。
- 新位置的信息與原有信息結合，形成更細緻的取象。
- 此法較進階，需先熟練靜態爻位。

## 每個爻位都是小太極

一個爻位不是單一資訊，而是多層資訊共同存在：

```text
爻位本義
+
六親
+
六神
+
五行旺衰
+
月破、空亡、合沖
+
動變關係
```

因此每一個爻位本身就是一個小太極，需綜合判斷。

## 判斷順序

```text
爻位基本含義
↓
所臨六親
↓
所臨六神
↓
旺衰
↓
空亡、月破、合沖
↓
動變與是否轉移
```

## If / Then 判斷

```text
If 問房屋風水，問題落在二爻
Then 優先看宅內、房間內部問題

If 問健康，用神在四爻
Then 可輔助取肺部、胸腔相關象

If 問健康，用神在六爻
Then 可輔助取頭部相關象

If 某爻發動且另一個同五行爻存在
Then 可考慮爻位信息轉移

If 爻位轉移成立
Then 新位置的六親、六神、旺衰信息需加入判斷
```

## 阿咪重點速記

- 靜態爻位是基礎，先背房屋與人體取象。
- 動態爻位是進階，不可生搬硬套。
- 每個爻位都是小太極，需看爻位、六親、六神、旺衰、空破合沖。
