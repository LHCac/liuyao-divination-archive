# 左右互辯

## 主題摘要

左右互辯是一種進階取象技法，將卦中的伏藏之爻與變爻結合起來分析。左為伏藏，右為變爻。兩者雖然位置不同，但可以互補說明同一事物的不同面向，增加卦象資訊量。

## 左右定義

| 名稱 | 含義 |
|---|---|
| 左 | 伏藏之爻，伏在飛爻下方，游離於卦象主體之外 |
| 右 | 變爻，動爻發動後化出的爻，是動爻力量的終點 |

## 出現情境

### 情況一：飛爻直接發動

```text
飛爻發動
↓
化出變爻
↓
其下伏爻與變爻建立關聯
```

此時變爻與伏爻可互補取象。

### 情況二：同一六親另一爻發動

若飛爻未動，但卦中同一六親的另一個爻發動化出變爻，該變爻也可能與前面的伏藏爻形成關聯。

## 應用原則

- 伏藏與變爻之間有相通性。
- 可將兩者五行、六親、六神、爻位組合分析。
- 兩者共同描述一個事物的不同側面。
- 思維要靈活，但不可漫無邊際。
- 一切仍要圍繞所問之事。

## 應用流程

```text
Step 1：找出伏藏爻（左）與對應變爻（右）
Step 2：分別取兩者的五行、六親、六神、爻位象
Step 3：組合兩者描述同一事物的不同面向
Step 4：回到所問之事，不要離題延伸
```

## 生活取象練習

平常可練習把身邊事物轉成六爻概念：

| 看到的事物 | 可轉成六爻概念 |
|---|---|
| 車輛 | 父母爻，承載保護 |
| 花草 | 妻財爻或木象，視問題而定 |
| 道路 | 子孫爻，路徑流動 |
| 生病 | 官鬼爻，病源壓力 |
| 朋友來訪 | 兄弟爻，合或來生 |

## If / Then 判斷

```text
If 飛爻發動且下有伏爻
Then 變爻與伏爻可互補取象

If 卦中某六親兩現且一個動化出變爻
Then 該變爻可與另一個同六親伏藏爻相通

If 左右互辯成立
Then 將伏爻與變爻的五行、六親、六神、爻位共同分析

If 取象開始偏離原問題
Then 立即回到所問之事，不可漫無邊際
```

## 阿咪重點速記

- 左 = 伏藏；右 = 變爻。
- 左右互辯是進階取象，用來補充細節。
- 必須圍繞問題，不可隨意亂取象。
- 要把六親、六神、五行、爻位一起組合。
