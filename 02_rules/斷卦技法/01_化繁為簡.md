# 化繁為簡

## 主題摘要

化繁為簡是多動爻卦的核心斷法。當卦中出現多個動爻時，資訊量大、組合複雜，容易跑題。化繁為簡的核心是以用神為中心，先抓直接影響用神的動爻，再處理間接關係與旁證細節。

## 多動爻的挑戰

當卦中有 2 到 4 個動爻時，常見問題包括：

- 判斷分散，失去主線。
- 問財卻斷到祖上，問婚卻斷到學業。
- 被反吟、伏吟、三合局等結構吸引，忽略用神。
- 動爻很多，卻不知道哪一個最重要。

## 核心原則

```text
所有卦產生的目的只有一個：顯示所問之事。
用神是卦的中心。
任何動爻都應該圍繞用神來理解。
```

## 操作步驟

```text
Step 1：先確定用神
Step 2：找直接生克用神的動爻
Step 3：分辨原神、忌神、仇神
Step 4：看與用神關係最緊密的動爻
Step 5：其他動爻只作補充細節
Step 6：整合斷語，始終圍繞原問題
```

## 分則分，連則連

多個動爻有兩種可能：

| 情況 | 解讀方式 |
|---|---|
| 分則分 | 不同動爻分別說明不同面向 |
| 連則連 | 多個動爻串成一條事件發展線 |

判斷時要看動爻之間是否有明顯生克、合沖、化進退、三合局等連動關係。

## 動爻多寡的象意

| 動爻數量 | 象意 |
|---|---|
| 動爻少 | 事情較簡單、直接 |
| 動爻多 | 事情多變、複雜、牽涉面廣 |
| 全靜 | 事情表面平靜，需看月日、空破、暗動、伏藏 |

## If / Then 判斷

```text
If 多個動爻出現
Then 不要慌，先回到用神

If 動爻直接生用神
Then 這個動爻多為原神或助力，優先分析

If 動爻直接克用神
Then 這個動爻多為忌神或主要阻力，優先分析

If 動爻生忌神或克原神
Then 這個動爻可能是仇神，屬於間接阻力

If 動爻與用神無直接生克
Then 先從輕，取六親、六神、爻位作旁證細節

If 斷語開始偏離所問之事
Then 立刻回到用神，不可跑題
```

## 阿咪重點速記

- 多動爻不要被嚇到，先找用神。
- 直接生用神、克用神的動爻最重要。
- 其他動爻不是沒用，而是補充細節。
- 一卦只為一事而來，斷卦不可跑題。
