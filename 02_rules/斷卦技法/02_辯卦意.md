# 辯卦意

## 主題摘要

辯卦意是六爻斷卦的高階功夫。卦意就是整個卦象想表達的核心意圖。真正的重點不是堆砌神煞或口訣，而是在用神、旺衰、生剋、世應、動靜、合沖等資訊中，快速抓住卦的中心結構與重心。

## 卦意的核心

卦意可以理解為：

```text
這個卦到底在說什麼故事？
這個卦最想提醒哪個重點？
哪一個爻、哪一組關係是全卦的關鍵？
```

辯卦意要求：

- 基本功熟練。
- 能抓住用神與核心動爻。
- 能分清主次。
- 能結合所問之事。
- 能用生活常識翻譯卦象。
- 不死套公式。

## 三個層次

### 第一層：看旺衰生克

這是基本層。

重點：

- 用神是否旺相。
- 原神是否有力。
- 忌神是否發動。
- 用神是否空破、入墓、受克。

### 第二層：看整體結構

不只看單一爻，而是看全卦如何組合。

重點：

- 誰是關鍵爻。
- 世應如何互動。
- 動爻是否形成一條事件線。
- 忌神是否被制。
- 原神是否被傷。
- 是否有貪生忘克、合住、沖開等結構。

### 第三層：感知卦的意圖

問自己：

```text
卦在說的是吉凶？
還是在說過程？
是在說阻礙？
還是在說憂慮被解除？
```

同樣是克世，不同問題中意思可能不同。

例如：

- 問求官，官鬼克世可能是壓力與阻礙。
- 問危險或疾病，官鬼被克或克去身邊之鬼，可能反而是解除憂慮。

## 吉凶表達方式

### 表達吉時

- 用神旺相。
- 原神發動生用神。
- 忌神受制。
- 忌神化退、化空、化回頭克。
- 世爻旺相，能承受壓力。
- 整體結構形成助用神之勢。

### 表達凶時

- 用神空破。
- 用神休囚受克。
- 原神受制。
- 忌神旺相發動克用。
- 仇神生忌神又克原神。
- 用神太過無制，物極必反。
- 動變回頭克用神。

## 辯卦意的提醒

```text
卦是活的，只能解讀，不能預設。
```

不要看到某個符號就機械套公式。例如：

- 用神旺不一定永遠吉，太過可能反凶。
- 忌神克世不一定永遠凶，要看克的是什麼、問的是什麼。
- 動爻多不一定難斷，要抓住用神主線。
- 六神不能改變大吉凶，只能細化象意。

## If / Then 判斷

```text
If 一般邏輯判凶，但忌神被強力制住
Then 需重新看卦意，可能是凶中轉吉

If 世爻旺相，忌神雖發動來克
Then 不一定凶，需看是否為克去身邊之憂慮或病鬼

If 一般分析吉凶難辨
Then 回到所問之事本身，問這個卦到底在回答什麼

If 用神持世旺相但臨六神
Then 六神只補細節，不應推翻旺衰生克大方向

If 動爻克世但世爻旺相
Then 需依問題性質判斷：求官可能是壓力，問危險可能是解憂

If 卦中出現特殊技法很多
Then 不要迷失，先回到用神、旺衰、原神忌神
```

## 阿咪重點速記

- 辯卦意 = 抓住卦想說的核心故事。
- 高階技法不是堆神煞，而是基本功極熟後的靈活判斷。
- 先看旺衰生克，再看整體結構，最後感知卦意。
- 永遠緊扣問題，不可離題。
