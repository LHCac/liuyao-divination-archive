# 測來意

## 主題摘要

測來意是求測者不說明所問之事，由預測者從卦象中判斷對方最關心的問題。此技法考驗對卦意的綜合把握，核心是相信第一直覺、圍繞世爻與動爻分析，並結合求測者的實際情況。

## 測來意三步法

### 1. 相信第一直覺

看到卦後，第一條斷語常是心神感應的表達。不要立刻被過度分析覆蓋。

但直覺不是亂猜，仍要以卦象資訊為根基。

### 2. 結合求測者實際情況

卦外資訊也重要：

- 年齡
- 性別
- 衣著
- 談吐
- 面色
- 氣質
- 當下情緒
- 現實常識

```text
卦外信息為陽
卦內信息為陰
陰陽合看，判斷才完整
```

### 3. 圍繞世爻與動爻分析

- 世爻代表問事主體的狀態。
- 動爻代表最近最活躍、最牽動心神的事情。
- 動爻多時，先抓最有力、最有特點的動爻。
- 取象以六親基本含義為主，不要過度取偏象。

## 判斷流程

```text
Step 1：觀察求測者基本信息
Step 2：看世爻旺衰與所臨六親六神
Step 3：看最活躍的動爻
Step 4：用六親基本含義定位問題類別
Step 5：用第一直覺修正斷語
Step 6：回到卦象驗證，不可離卦亂猜
```

## 常見六親來意提示

| 六親 | 可能來意 |
|---|---|
| 官鬼 | 工作、官司、疾病、丈夫、壓力、小人 |
| 妻財 | 財運、錢財、感情、妻子、飲食、欲望 |
| 子孫 | 子女、健康解救、快樂、出行、財源、寵物 |
| 父母 | 學業、文書、房屋、車輛、長輩、證件 |
| 兄弟 | 朋友、同事、競爭、借錢、分財、兄弟姐妹 |

## 注意事項

- 預測者只是翻譯卦象，不是超能力者。
- 卦的產生圍繞所問之事，無關之事不一定體現。
- 若求測者故意試探、心不誠、無明確意念，卦象可能不清。
- 測來意不可為了神奇而亂斷，要保持客觀。

## If / Then 判斷

```text
If 官鬼多動且求測者為已婚女性
Then 可優先考慮丈夫、工作、疾病或官非壓力

If 世爻化父母且克他爻
Then 主體可能與文書、長輩、房屋、車輛或學業相關事項有糾紛

If 伏藏爻為子孫
Then 心中可能掛念子女、晚輩、健康解救或出行之事

If 動爻為妻財且旺相
Then 來意可能與錢財、感情、飲食、收益或員工工具有關

If 世爻休囚受克
Then 求測者當下狀態不佳，問題對其壓力較大
```

## 阿咪重點速記

- 測來意先看世爻，再看動爻。
- 第一直覺可以參考，但必須回到卦象驗證。
- 卦外資訊與卦內資訊要一起看。
- 不要為了神奇亂斷，重點是翻譯卦象。
