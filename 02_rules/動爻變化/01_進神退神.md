# 進神與退神

## 主題摘要

進神指動爻所化之爻，在同五行地支中向前推進，代表力量增強；退神指動爻所化之爻，在同五行地支中向後退，代表力量消減。進退神只在同五行地支之間成立，異五行則不論進退，改看生克關係。

## 判斷前提

```text
必須同五行，才論進神或退神。
五行不同，不論進退，只看生克、合沖、墓絕等關係。
```

例如：

```text
丑土化寅木
雖然十二地支順序往前
但土化木，五行不同
所以不論進神，而是看生克關係
```

## 基本象意

| 類型 | 含義 |
|---|---|
| 進神 | 力量增強、越來越旺、逐漸推進、事情往前走 |
| 退神 | 力量消減、越來越弱、退縮、事情往後退 |

## 吉凶判斷

| 情況 | 判斷 |
|---|---|
| 用神化進神 | 吉，事情越來越好 |
| 原神化進神 | 吉，生助力量越來越強 |
| 忌神化退神 | 吉，阻礙逐漸消退 |
| 仇神化退神 | 吉，間接傷害減弱 |
| 用神化退神 | 凶，所求越來越難 |
| 忌神化進神 | 凶，阻力越來越強 |

## 長期事件特別重要

進神、退神尤其適合判斷長期發展：

- 事業是否越做越好。
- 財運是否越來越旺。
- 感情是否持續升溫或冷卻。
- 病情是否加重或退去。
- 競爭對手是否越來越強或退場。

## If / Then 判斷

```text
If 動爻化同五行地支且地支往前
Then 為進神，力量增強

If 動爻化同五行地支且地支往後
Then 為退神，力量消減

If 五行不同
Then 不論進退，改看生克關係

If 用神化進神
Then 事情有好的發展，越來越有利

If 忌神化退神
Then 阻礙自動消退，利於用神

If 忌神化進神
Then 阻力加重，不利用神

If 用神化退神
Then 所求之事往不利方向發展，需謹慎
```

## 阿咪重點速記

- 同五行才論進退。
- 用神、原神喜化進神。
- 忌神、仇神喜化退神。
- 異五行不論進退，直接看生克。
