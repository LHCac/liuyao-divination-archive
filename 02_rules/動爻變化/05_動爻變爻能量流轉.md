# 動爻與變爻能量流轉

## 主題摘要

本篇整理動爻、變爻、暗動爻三者的能量流動規則，是多動爻卦象判斷的重要基礎。核心重點是：動爻可以向旁爻發力，變爻優先回應本位動爻，暗動爻只有生剋之力而無沖合之力。

資料來源：《六爻知先機》第五章「六爻意念流轉」第070～075頁。

---

## 一、動爻能量變化法則

| 編號 | 法則 | 說明 |
|---|---|---|
| 1 | 動爻可向旁爻放射力量 | 動爻可對其他五個本爻，以及可出伏之伏神，施放生、剋、沖、合之力 |
| 2 | 靜爻不能主動放射力量 | 靜爻只承受作用，不主動影響其他爻 |
| 3 | 動爻只能接受年月日與變爻力量 | 年月日、變爻可對動爻產生生剋沖合，但動爻不能反向作用於年月日或變爻 |
| 4 | 暗動爻只有生剋之力 | 暗動可以生剋，但沒有沖合之力 |
| 5 | 受剋之爻需用生解 | 被剋時，只有生扶能真正解剋 |
| 6 | 受沖之爻需用日合解 | 被沖時，只有日辰之合能真正解沖，他爻來合解之不真 |

### 動爻方向圖

```text
動爻
↓ 可放射生、剋、沖、合
其他五個本爻
可出伏之伏神

年月日、變爻
↓ 可對動爻放射生、剋、沖、合
動爻

注意：動爻不能反向對年月日、變爻施放力量。
```

---

## 二、暗動特別規則

暗動之爻雖有動意，但不是完整明動。

```text
暗動爻：有生剋之力
暗動爻：無沖合之力
```

| 判斷 | 說明 |
|---|---|
| 暗動生用神 | 可視為隱性力量生助用神 |
| 暗動剋用神 | 可視為隱性壓力傷害用神 |
| 暗動沖某爻 | 不成立，暗動無沖力 |
| 暗動合某爻 | 不成立，暗動無合力 |

---

## 三、解救規則

| 受損情況 | 正確解法 | 說明 |
|---|---|---|
| 爻受剋 | 用生解救 | 只有生扶才能解剋 |
| 爻受沖 | 用日合解救 | 只有日合能解沖，他爻合解不真 |

### 判斷提醒

```text
If 用神被剋
Then 找原神、月日生扶或動爻生扶，不用合來解剋

If 用神被沖
Then 優先看是否有日辰來合用神，不以旁爻合用神當真解
```

## 四、變爻能量變化法則

| 編號 | 法則 | 說明 |
|---|---|---|
| 1 | 變爻可對本位動爻施放力量 | 變爻可對原本那一爻的動爻產生生、剋、沖、合 |
| 2 | 若變爻與本位動爻有生剋沖合，力量專注於本位動爻 | 此時變爻不再影響其他旁爻或伏神 |
| 3 | 若變爻與本位動爻無生剋沖合，力量可擴散 | 可再考慮它對其他旁爻與可出伏神的影響 |
| 4 | 變爻旺衰只看年月日 | 動爻與伏爻不能反向影響變爻旺衰 |
| 5 | 變爻與變爻之間不互相作用 | 多動爻時，不可讓變爻A與變爻B互相生剋沖合 |

### 變爻流向圖

```text
情況 A：變爻與本位動爻有生剋沖合
變爻 → 專注作用於本位動爻
變爻 ✗ 不再影響其他旁爻

情況 B：變爻與本位動爻無生剋沖合
變爻 → 力量可擴散至其他旁爻
變爻 → 可影響可出伏神

情況 C：判斷變爻旺衰
年月日 → 決定變爻旺衰
動爻、伏爻 ✗ 不決定變爻旺衰

情況 D：多動爻
變爻 A ✗ 變爻 B
變爻之間彼此不生、不剋、不沖、不合
```

## 五、動爻與變爻關係定義

| 名詞 | 定義 | 象意 |
|---|---|---|
| 回頭生 | 變爻生動爻 | 動爻得後援，力量增強 |
| 回頭剋 | 變爻剋動爻 | 動爻被壓制，力量受損 |
| 進神 | 動爻化出同五行且向前推進的地支 | 趨勢增強、後續發展有力 |
| 退神 | 動爻化出同五行且向後退縮的地支 | 趨勢減弱、後力不足 |
| 伏吟 | 動爻與變爻地支相同 | 鬱滯不行、原地反覆 |
| 沖 | 動爻與變爻六沖 | 力量激盪、衝突、變化大 |
| 合 | 動爻與變爻六合 | 力量被封住、牽制、黏合 |

## 六、多動爻實戰檢查清單

每個動爻都要逐一檢查：

```text
☐ 1. 動爻與變爻是什麼關係？
   回頭生、回頭剋、進神、退神、伏吟、沖、合，或無明顯關係？

☐ 2. 若變爻對本位動爻有生剋沖合
   變爻力量只作用於本位動爻，不影響其他旁爻。

☐ 3. 若變爻對本位動爻無生剋沖合
   變爻力量才可能擴散到旁爻或可出伏神。

☐ 4. 變爻旺衰只看年月日
   不用動爻去反推變爻旺衰。

☐ 5. 多動爻時
   各變爻之間不互相生剋沖合。

☐ 6. 暗動爻
   只有生剋之力，沒有沖合之力。

☐ 7. 受剋與受沖的解救方式不同
   受剋找生扶，受沖找日合。
```

## 七、對舊解析的修正影響

| 舊式可能錯誤 | 修正後正確做法 |
|---|---|
| 說「動爻沖月建」 | 應改為「月建沖動爻」，動爻不能反向沖月建 |
| 說「動爻影響變爻旺衰」 | 變爻旺衰只看年月日 |
| 多動爻時讓變爻彼此生剋 | 變爻與變爻之間不互相作用 |
| 說「暗動爻沖某爻」 | 暗動只有生剋，沒有沖合 |
| 用他爻合來解沖 | 受沖只有日合能真解，他爻解之不真 |

## 八、If / Then 判斷

```text
If 動爻對旁爻發生生剋沖合
Then 可以判斷動爻主動影響其他本爻或可出伏神

If 靜爻未受日沖成暗動
Then 靜爻不主動影響其他爻

If 暗動爻生用神
Then 可視為隱性力量扶助用神

If 暗動爻沖用神
Then 不作暗動沖論，因暗動無沖合力

If 變爻回頭生動爻
Then 動爻後力增強

If 變爻回頭剋動爻
Then 動爻力量受壓制

If 變爻與本位動爻有生剋沖合
Then 變爻力量專注於本位動爻，不再分析其影響旁爻

If 變爻與本位動爻無生剋沖合
Then 可檢查其是否擴散影響旁爻或可出伏神

If 多動爻有多個變爻
Then 不可讓變爻之間互相生剋沖合

If 爻受剋
Then 以生扶解救

If 爻受沖
Then 以日合解救
```

## 阿咪重點速記

- 動爻可以對旁爻與可出伏神放射生剋沖合。
- 動爻不能反向影響年月日與變爻。
- 變爻若已對本位動爻發力，就不再影響旁爻。
- 變爻旺衰只看年月日，不看動爻。
- 變爻之間不互相作用。
- 暗動只有生剋，沒有沖合。
- 受剋用生解，受沖用日合解。
