# 反吟與伏吟

## 主題摘要

反吟與伏吟是動爻變化中的特殊警示。反吟代表動爻化出的變爻與本爻地支相沖，主事情反覆多變、去了又回；伏吟代表動爻化出與本爻相同的地支，主事情原地踏步、難以推進。兩者皆提示事情不穩或停滯，但最終吉凶仍要以用神旺衰為核心。

## 反吟

### 定義

```text
動爻化出與本爻相沖的地支
例如：巳化亥、卯化酉
```

### 象意

- 反覆多變
- 去了又回
- 循環往復
- 事情不穩
- 好事多磨
- 來回折騰

### 各事類應象

| 問題類型 | 反吟可能象意 |
|---|---|
| 財運 | 財來財去，收支反覆 |
| 婚姻 | 將成又破，分合反覆 |
| 工作 | 調來調去，安排變動 |
| 疾病 | 反覆難癒，病情起伏 |
| 合作 | 談成又變，往返協商 |

## 伏吟

### 定義

```text
動爻化出與本爻相同的地支
例如：子化子、午化午
```

### 象意

- 原地踏步
- 動了沒變
- 難以前進
- 心有餘力不足
- 計畫卡住
- 事情停滯

## 重要原則

反吟、伏吟都帶有「吟」的痛苦、難受、糾結之象，但不等於必凶。

```text
反吟/伏吟只是提示過程不順
最終吉凶仍看用神旺衰與生克救應
```

## If / Then 判斷

```text
If 動爻化出與本爻相沖地支
Then 反吟，事情反覆，去而復回

If 動爻化出與本爻相同地支
Then 伏吟，事情停滯，原地打轉

If 反吟或伏吟出現
Then 提高警覺，但仍以用神旺衰定吉凶

If 用神旺相遇反吟
Then 雖反覆但仍可能最終可成

If 用神休囚遇反吟
Then 反覆後難成，偏凶

If 用神遇伏吟且無沖開救應
Then 事情卡住，難以推進
```

## 阿咪重點速記

- 反吟 = 反覆，去了又回。
- 伏吟 = 停滯，原地打轉。
- 兩者都不是直接判凶，要回到用神旺衰。
- 反吟看反覆，伏吟看不動。
