# 假破：動變組合大於日月組合

## 主題摘要

假破是指某爻或變爻看似被月建或日辰沖破，但實際上其破的力量被更高層次的動變趨勢化解或覆蓋。核心邏輯是：動變組合的長期趨勢，有時大於日月沖破的短期衝擊。

## 一、核心邏輯

```text
動變組合 > 日月組合
```

尤其是動爻化進神時，若變爻雖被月建或日辰沖破，但整體動變趨勢是前進增強，則可能是假破，不作真破論。

## 二、假破第三種類型：動化進神

| 項目 | 內容 |
|---|---|
| 現象 | 動爻化出的變爻地支，在月建或日辰上屬於破 |
| 但是 | 動爻化出的是進神，代表力量向前推進 |
| 判斷 | 動變進神趨勢蓋過日月沖破 |
| 結論 | 假破，不為真破，用神仍有力 |

## 三、進神地支對照

| 動爻 | 進神變爻 | 說明 |
|---|---|---|
| 寅 | 卯 | 木前進 |
| 卯 | 巳 | 火前進 |
| 午 | 未 | 土前進 |
| 丑 | 寅 | 土前進資料用法，需配合系統檢核 |
| 辰 | 午 | 土前進資料用法，需配合系統檢核 |
| 未 | 辰 | 土前進資料用法，需配合系統檢核 |
| 申 | 酉 | 金前進 |
| 亥 | 子 | 水前進 |

> 注意：進神判斷以資料所採用系統為準。若與「同五行才論進退」系統衝突，實戰時需以案例驗證與所用派別一致性為準。

## 四、層次優先順序

| 層次 | 力量來源 | 核心 |
|---|---|---|
| 最高 | 動變組合：進神、退神、化旺、化衰 | 長期趨勢 |
| 高 | 月建臨、合、破 | 月令大勢 |
| 中 | 日辰臨、合、沖 | 短期觸發 |
| 一般 | 月建生、剋、休囚 | 輔助判斷 |
| 最低 | 日建生、剋、洩耗 | 多作低層或平相參考 |

## 五、假破判斷流程

```text
變爻被月建或日辰相沖
↓
動爻是否化進神？
↓
是 → 假破，不為真破
↓
否 → 看變爻是否臨月建或合月建
↓
是 → 假破，化旺
↓
否 → 真破，用神或動爻後力受損
```

## 六、假破三種類型整理

| 類型 | 現象 | 結論 |
|---|---|---|
| 第一種 | 變爻臨月建或合月建 | 假破，動化旺 |
| 第二種 | 待補充 | 待實戰驗證 |
| 第三種 | 動爻化進神，變爻雖被月日沖 | 假破，進神趨勢蓋過沖破 |

## 七、實戰檢查清單

```text
1. 動爻化出什麼地支？
2. 是進神還是退神？
3. 變爻地支在月建、日辰上是否被沖破？
4. 若變爻被破，但動爻化進神 → 假破
5. 若變爻被破，且動爻化退神 → 真破加退神，凶上加凶
```

## If / Then 判斷

```text
If 變爻被月日沖破，但動爻化進神
Then 優先考慮假破，不作真破論

If 變爻被月日沖破，但變爻臨月或合月
Then 也可考慮假破，屬化旺

If 變爻被月日沖破，又化退神
Then 真破加退神，凶象加重

If 用神動化進神但變爻逢破
Then 不急判用神破，先檢查是否假破
```

## 阿咪重點速記

- 假破核心：動變趨勢有時大於日月短期沖破。
- 化進神遇破，常先看假破。
- 化退神遇破，凶象更重。
- 變爻臨月、合月，也可能是假破化旺。
