# 暗動詳解：世爻特別規則與用神生世

## 主題摘要

暗動是靜爻被日辰沖起後形成的隱性發動。最常見形式是靜爻在月建條件下旺相，又被日建相沖。暗動力量強，常是斷卦的關鍵。自占卦中，世爻即使暗動或明動，也主要承受卦象，不主動參與生克其他爻。

## 一、暗動最常見形式

| 條件 | 內容 |
|---|---|
| 條件一 | 靜爻在月建條件下旺相 |
| 條件二 | 該靜爻被日建相沖 |
| 結果 | 暗動，力量強大，是有用之動 |

```text
靜爻旺相 + 日建相沖 = 暗動
靜爻衰敗 + 日建相沖 = 日破 / 動散
```

## 二、自占世爻特別規則

在自占卦中，世爻代表問卦者自身，其能量主要用來承受卦象，而不是主動出力影響其他爻。

| 情況 | 判斷 |
|---|---|
| 自占卦世爻暗動 | 不主動參與生克其他爻 |
| 自占卦世爻明動 | 不主動參與生克其他爻 |
| 代占卦中應爻動 | 正常參與生克，無此限制 |

## 三、修正觀念

| 舊判斷 | 修正後 |
|---|---|
| 世爻暗動可生用神 | 自占中世爻暗動不生用神，用神旺衰仍由月日動變判定 |
| 世爻動可克用神 | 自占中世爻只承受卦象，不主動克用神 |
| 代占應爻也不參與生克 | 代占應爻代表被占者，可正常參與生克 |

## 四、用神暗動生世

用神動而生世，是非常吉利的結構。用神暗動生世，可視為用神被隱性激活後主動來生世。

### 判斷條件

| 條件 | 內容 |
|---|---|
| 用神旺相 | 得月建生扶、臨月、合月等 |
| 用神暗動 | 旺相用神被日建沖起 |
| 用神生世 | 用神五行生世爻 |

### 象意

```text
用神主動靠近問卦者
財來就我
工作機會靠近我
對方或所求之事來生我
```

## 五、暗動形式整理

| 暗動形式 | 判斷條件 | 力量 |
|---|---|---|
| 最常見形式 | 靜爻月旺 + 日沖 | 最強 |
| 第二種形式 | 動爻生靜爻 + 日沖靜爻 | 強 |
| 其他形式 | 待補充 | 待驗證 |

## 六、自占與代占對照

| 類型 | 世爻 / 應爻能否參與生克 |
|---|---|
| 自占 | 世爻不能主動參與生克，只承受卦象 |
| 代占 | 應爻代表被占者，可正常參與生克 |

## If / Then 判斷

```text
If 靜爻旺相且被日建沖
Then 暗動，有用之動

If 靜爻衰敗且被日建沖
Then 日破，動散無力

If 自占卦世爻暗動
Then 世爻只承受卦象，不主動生克其他爻

If 自占卦世爻明動
Then 世爻仍不主動生克其他爻，只與自身變爻作用

If 用神旺相暗動且生世
Then 用神主動來生世，為非常吉利的結構

If 代占卦中應爻動
Then 應爻可正常參與生克，不套用自占世爻限制
```

## 阿咪重點速記

- 暗動 = 旺相靜爻被日沖起。
- 日沖衰爻是日破，不是暗動。
- 自占世爻不主動生克其他爻。
- 用神暗動生世，是很吉的結構。
