# 變動總論

## 主題摘要

「神兆機於動」，動爻是卦象中最重要的資訊載體。一個爻發動後，必然產生變爻。動爻與變爻之間的生克、合沖、空破、墓絕、進退、反吟伏吟，決定此爻最終能發揮的力量大小與方向。

## 動變關係速查

| 動變關係 | 說明 |
|---|---|
| 變爻生動爻 | 回頭生，動爻後力足，力量更強 |
| 變爻克動爻 | 回頭克，動爻力量被截斷，快速而直接 |
| 動爻克變爻 | 動爻能量不受阻，可正常發揮 |
| 動爻生變爻 | 動爻生出後續力量，後繼有力 |
| 動變同五行 | 可能為伏吟、進神、退神，需看地支關係 |
| 動變相沖 | 反吟，事情反覆多變 |
| 動爻化空化破 | 動爻力量打折，事情可能落空 |
| 動爻化絕化墓 | 動爻之氣被截斷或被收藏，需謹慎 |

## 回頭克的重要性

變爻回頭克動爻，其力量特別強，因為變爻只針對本位動爻作用，力量專一。

### 判斷

```text
If 動爻為用神，變爻回頭克
Then 用神力量被直接截斷，偏凶

If 動爻為忌神，變爻回頭克
Then 忌神被克去，反而偏吉
```

## 沖中帶克

例如：

- 卯木化酉金：回頭沖 + 回頭克。
- 巳火化亥水：回頭沖 + 回頭克。

這類動變關係力量更快、更直接，是需要高度注意的組合。

## 動變象法

當動變之間沒有明顯生克時，仍要看象法：

1. 動爻所在爻位代表什麼位置。
2. 動爻六親代表什麼人事物。
3. 動爻六神帶出什麼性質。
4. 變爻的六親、五行、爻位代表後續結果。

例如：

```text
四爻為門
化出父母
父母可代表車輛
→ 可取「坐車出門」之象
```

## If / Then 判斷

```text
If 動爻為用神且化回頭生
Then 用神後力足，事情有持續推進力

If 動爻為用神且化回頭克
Then 用神受損，力量被截斷

If 動爻為忌神且化回頭克
Then 忌神自受克制，對用神反而有利

If 動爻化空亡
Then 動爻力量落空，事情可能不成或暫時無效

If 動爻化墓、化絕
Then 動爻氣被收藏或截斷，需看是否有沖開或救應

If 變爻旺相不空破
Then 動爻後力足，可持續發揮作用

If 變爻無力或空破
Then 動爻後繼無力，力量減弱
```

## 阿咪重點速記

- 動爻是卦中最重要的動態資訊。
- 動爻必看變爻，因為變爻代表後續方向。
- 回頭克很強，用神怕回頭克，忌神喜回頭克。
- 動變沒有生克時，要用爻位、六親、六神組合取象。
