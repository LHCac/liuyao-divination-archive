# 沖：月沖、日沖、爻沖

## 主題摘要

沖是六爻中地支相對、力量直達的作用。沖本身沒有固定吉凶，要結合生克、用忌、旺衰與所問之事判斷。沖有兩種主要效果：一是沖破，使力量受損；二是沖動或沖實，使靜爻被啟動、空亡被沖實。

| 沖對 | 關係 |

| 辰 ↔ 戌 | 土土相沖，同五行沖 |
| 巳 ↔ 亥 | 火水相沖，相克沖 |

## 沖的兩種效果

| 效果 | 說明 |
|---|---|
| 沖破 | 被沖者受損，力量削弱，如月破 |
| 沖動、沖實 | 靜爻被沖而動，空亡爻被沖而不空 |

## 沖的嚴重程度

```text
沖中帶克 > 同五行沖 > 純沖
月沖（月破）> 日沖 > 爻沖
```

## 月沖：月破

月建沖某爻，稱月破，力量損傷大。

### 象意

- 衝突
- 斷裂
- 傾斜
- 破財
- 傷病
- 改變
- 破損

月沖中若又帶克，凶性更重；同五行沖較輕。

## 日沖：暗動與沖實

日辰沖靜爻，常形成暗動。

### 作用

- 靜爻被日辰沖動，成為暗動。
- 空亡爻被日辰沖，稱為沖實，該爻不再完全空亡。
- 日沖力量不如月沖，但比普通爻沖重要。

## 爻沖

動爻沖他爻，稱爻沖。

### 作用

- 動爻把力量沖向目標爻。
- 可沖動靜爻。
- 可解合，沖開原本合住的爻。
- 可影響被沖爻的力量與狀態。

## 合中逢沖

合遇沖可解合。

```text
原本被合住的爻
若遇沖
則束縛解除
```

## If / Then 判斷

```text
If 月建沖某爻
Then 該爻月破，力量大損，需看是否沖中帶克

If 日辰沖靜爻
Then 該爻暗動，可視為有動象，但力量通常弱於正式動爻

If 日辰沖空亡爻
Then 空亡被沖實，該爻不再完全空

If 動爻沖他爻
Then 被沖爻受動爻影響，可能被沖動、沖破或解合

If 合中逢沖
Then 原本合住的爻被沖開，合的束縛解除

If 空亡爻被沖
Then 沖實，不空，可按正常狀態論，但仍需看旺衰
```

## 阿咪重點速記

- 沖不是一定凶，重點看沖破還是沖動。
- 月沖最重，稱月破。
- 日沖靜爻可成暗動。
- 空亡逢沖可沖實。
- 合逢沖可解合。
