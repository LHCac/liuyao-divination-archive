# 三刑

## 主題摘要

三刑是地支之間一種特殊的相刑關係，代表不順、制約、麻煩、傷損、管理、刑罰與修理。三刑在卦中出現時應留意其細節象意，但不可單憑三刑直接斷吉凶，仍須以五行生剋、用神旺衰、喜忌為主。

## 三刑分類

| 刑法 | 地支組合 | 主要特點 |
|---|---|---|
| 無禮之刑 | 子刑卯、卯刑子 | 不知禮義、衝突不敬、失禮冒犯 |
| 恃勢之刑 | 丑刑戌、戌刑未、未刑丑 | 仗勢欺人、三角糾紛、互相壓制 |
| 無恩之刑 | 寅刑巳、巳刑申、申刑寅 | 忘恩負義、背信棄義、恩將仇報 |
| 自相刑 | 辰刑辰、午刑午、酉刑酉、亥刑亥 | 自我傷害、內耗、內部矛盾 |

## 核心象意

三刑常見象意：

- 口舌
- 不順
- 麻煩
- 難受
- 凡事多費周折
- 限制、管理
- 傷疤、手術
- 官司、刑罰
- 修理、檢查
- 規章制度
- 拖延、卡關

## 三刑的雙重性

三刑不一定全凶。

在某些需要治理、管控、修理、治療、執法的事情中，刑反而有正面意義。

例如：

| 所問事項 | 三刑可能含義 |
|---|---|
| 醫療 | 手術、治療、檢查、修復 |
| 法律 | 執法、規範、審判、處罰 |
| 管理 | 制度、規章、約束、整頓 |
| 修理 | 拆修、調整、矯正 |

## 生中有刑與克中有刑

| 情況 | 判斷 |
|---|---|
| 生中有刑 | 吉中帶凶，好事不圓滿，過程有麻煩 |
| 克中有刑 | 凶上加凶，阻礙、傷損、壓力更重 |

## If / Then 判斷

```text
If 卦中出現三刑組合
Then 必須留意不順、麻煩、制約、傷損等細節象

If 用神與忌神相刑
Then 所問之事有阻礙、麻煩纏身

If 官鬼爻涉及刑
Then 可取官司、法律、管理、疾病、壓力之象

If 父母爻涉及刑
Then 可取文件、合約、證照、房屋、車輛方面有問題

If 自相刑出現
Then 可取自我消耗、內部矛盾、自己困住自己

If 所問之事為醫療、手術、法律、管理
Then 刑有時可作正面信息，代表治療、執法或整頓

If 生中有刑
Then 有希望但過程不順

If 克中有刑
Then 凶上加凶，需要特別注意
```

## 阿咪重點速記

- 三刑重點是「不順、麻煩、制約、傷損」。
- 三刑不是直接判凶，要看用神喜忌。
- 問醫療、法律、管理時，刑可能代表正面治理。
- 自相刑多看內耗、自傷、內部矛盾。
