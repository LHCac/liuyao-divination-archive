# 合

## 主題摘要

合是六爻中地支之間相互吸引、融合、牽絆的關係，包含六合與三合。合的核心象意是聚合、穩定、牽絆、得到，但被合住的爻也可能力量受阻，因此合既可能是成就，也可能是束縛，必須依用神喜忌與問題類型判斷。

| 辰 + 酉 | 金 |

| 水局 | 申、子、辰 | 水 |

## 合的核心象意

| 情境 | 合的含義 |
|---|---|
| 用神被合 | 受牽絆，暫時無法自由行動 |
| 忌神被合 | 忌神受制，對用神威脅降低 |
| 用神旺相逢合 | 聚合、得到、穩定成就 |
| 用神休囚逢合 | 被困、被拖住、更加無力 |
| 動爻逢合 | 動者被止，力量被封住或打折 |
| 六合卦 | 事情聚合，緣分到位，穩定性增強 |

## 合的雙重性

合有兩面：

```text
好的一面：聚合、穩定、得到、成就
壞的一面：牽絆、束縛、封閉、不能自由發揮
```

因此不能看到合就直接判吉，要看：

1. 合住的是用神還是忌神。
2. 用神本身旺還是弱。
3. 問題需要穩定還是需要流動。
4. 是否有沖來解合。

## 合中有沖

合而遇沖，則解合。

```text
合住的爻被沖開
原本受牽絆的力量可以重新活動
```

## 病症應用

| 病情 | 合沖喜忌 |
|---|---|
| 近病 | 喜沖，不喜合。沖可使病氣散開。 |
| 久病 | 喜合，不喜沖。合可聚氣，避免元氣散失。 |

## 卦象變化

| 卦象變化 | 含義 |
|---|---|
| 六合卦化六沖 | 先聚後散 |
| 六沖卦化六合 | 先散後聚 |

## If / Then 判斷

```text
If 忌神被合
Then 忌神受制，對用神威脅降低，偏吉

If 用神旺相逢合
Then 可主穩定到手、聚合成就

If 用神休囚被合
Then 用神被困，更加無力

If 動爻被日辰合住
Then 動而受制，力量打折

If 六合卦問婚姻
Then 緣分聚合，有成之象，但仍需看用神旺衰

If 六合卦化六沖
Then 先有好兆，後面可能散局

If 六沖卦化六合
Then 先散亂，後有聚合機會

If 問久病且用神逢合
Then 氣得以聚住，偏吉

If 問近病且用神逢合
Then 病恐纏綿難去，需謹慎
```

## 阿咪重點速記

- 合 = 聚合，也 = 牽絆。
- 用神旺而合，可能得到；用神弱而合，可能被困。
- 忌神被合，多為忌神受制。
- 合遇沖可解合。
- 近病喜沖，久病喜合。
