# 勾陳

## 主題摘要

勾陳屬中央土之象，偏陰土，主遲緩、糾纏、穩定、不動、陳舊、土地與牢固。六爻中，勾陳常用來判斷事情是否拖延、卡住、反覆糾纏，也常與房產、土地、工程、牢獄、腸胃、腫瘤、結石等象意相關。

## 基本屬性

| 項目 | 內容 |
|---|---|
| 五行 | 土，偏陰土 |
| 方位 | 中央 |
| 核心特性 | 遲緩、糾纏、穩定、不動、陳舊 |

## 核心象意

### 事情狀態

- 進展緩慢
- 久拖不決
- 被勾住、被絆住
- 固定不動
- 舊事、舊物、陳年問題
- 穩定但缺乏變化

### 人物

- 慢性子的人
- 敦厚誠實的人
- 保守、穩重、反應慢的人
- 不容易改變立場的人

### 事物與場所

- 土地
- 房地產
- 工程
- 房子
- 牢獄
- 固定建築
- 被束縛之處

### 身體取象

- 腸胃不好
- 腸胃蠕動慢
- 腹脹
- 腫瘤
- 結石
- 慢性病、久病

## 特殊爻位提示

- 三、四爻臨勾陳：可加強腸胃、腹部、內部卡滯之象。
- 五爻為路爻，若五爻臨勾陳且受傷，可能取跌倒、道路阻滯之象。

## If / Then 判斷

```text
If 用神臨勾陳
Then 事情進展緩慢，可能被糾纏或拖延

If 忌神臨勾陳且不動
Then 阻礙固定，短期難以改變

If 官鬼臨勾陳
Then 工作穩定但晉升慢，或有土地、房產、工程、牢獄、官司之象

If 問健康且三四爻臨勾陳
Then 注意腸胃、腹脹、腫瘤、結石、慢性卡滯問題

If 五爻臨勾陳且受剋
Then 可留意道路、跌倒、行走不順之象
```

## 阿咪重點速記

- 勾陳核心是「被勾住，所以慢」。
- 問事情進度：臨勾陳多主拖延、糾纏。
- 問財產：勾陳常與土地、房產、不動產有關。
- 問健康：腸胃、結石、腫瘤、慢性病。
