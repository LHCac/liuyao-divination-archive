# 青龍

## 主題摘要

青龍為六神之首，位居東方，五行屬木，對應春天、早晨、生髮與開始。六爻中，青龍主喜慶、貴人、親情、遺傳、生機、開始與有益之物。青龍臨用神多取吉象；臨忌神則可能因「生」的力量讓麻煩增長。

## 基本屬性

| 項目 | 內容 |
|---|---|
| 方位 | 東方，左青龍 |
| 季節 | 春天 |
| 時辰 | 早晨 |
| 五行 | 木 |
| 核心特性 | 生髮、開始、喜慶、貴人、親情、遺傳 |

## 核心象意

### 吉性取象

- 喜慶：結婚、生子、慶祝。
- 貴人：願意幫助自己的人。
- 親情：有血緣、遺傳、親生關係。
- 生機：事情開始發展，有希望。
- 仁義：肯付出、有善意。
- 才華與高貴：用神旺相時可主有才、有名望。

### 凶性取象

當青龍臨忌神或凶神時，吉神轉凶：

- 因酒色成災。
- 因飲食、享樂而生病。
- 做善事反被誤會。
- 表面好看，實際帶來麻煩。
- 忌神得青龍生髮，麻煩增長。

## 分類取象

### 人事

- 貴人相助
- 親生父母、血緣關係
- 仁義之人
- 有情有義者
- 政府、官貴、發號施令之處

### 身體

- 生殖器官
- 神經系統
- 遺傳病症
- 痛癢，因木主神經感知

### 事物

- 飲食，尤其對人有益的食物
- 先進技術，特別是子孫臨青龍
- 先進文化、學術，特別是父母臨青龍
- 佛學、玄學緣分，尤其世爻臨青龍時

### 風水

- 面南而立，左手邊為青龍方。
- 青龍方主希望、變化、生機。
- 建房或布局時，青龍方宜有生動之氣，不宜被壓制。

## If / Then 判斷

```text
If 青龍臨用神且旺相
Then 主喜慶、生機、貴人、親情相助

If 青龍臨忌神且發動
Then 表面吉利，實際可能因好事、飲食、酒色、善意而惹麻煩

If 子孫臨青龍且問子女
Then 可取男孩、生育、生髮之象，仍需配合卦象判斷

If 父母臨青龍
Then 可取親生父母、有血緣遺傳關係的父母

If 用神旺相又臨青龍
Then 多遇貴人，事情有生機與助力

If 世爻臨青龍且問玄學佛學
Then 可取與玄學、佛學、善法有緣之象
```

## 阿咪重點速記

- 青龍的核心是「生髮開始」。
- 青龍不是永遠吉，要看臨用神還是忌神。
- 臨用神：喜慶、貴人、生機。
- 臨忌神：好事變麻煩，或麻煩被滋生擴大。
- 問親情、血緣、遺傳、子嗣時，青龍很重要。
