# 玄武

## 主題摘要

玄武位居北方，五行屬水，主藏匿、寒冷、黑暗、隱私、儲存與暗中操作。六爻中，玄武代表隱蔽、偷盜、曖昧、欺騙、投機，也有聰明機警、內秀、能量儲備的正面含義。取象時需依用神喜忌判斷。

## 基本屬性

| 項目 | 內容 |
|---|---|
| 方位 | 北方，後玄武 |
| 季節 | 冬季 |
| 五行 | 水 |
| 顏色 | 黑色 |
| 時辰 | 夜晚，子時前後 |
| 溫度 | 寒冷 |
| 核心特性 | 藏匿、儲存、聚氣、隱私、黑暗 |

## 核心象意

### 吉性取象

- 聰明機警
- 辦事隱密
- 不露聲色
- 內秀有才
- 肚有知識
- 聚藏能量，為未來蓄勢

### 凶性取象

- 偷盜
- 曖昧私情
- 欺騙
- 虛假
- 隱瞞
- 投機
- 暗中操作

## 具體取象

### 風水與空間

- 房子後方
- 靠山
- 地下室
- 暗室
- 儲物間
- 臥室，尤其三爻
- 廁所、浴室
- 光線不足之處

### 身體

- 生殖器官病症
- 黑痣
- 膚色黝黑
- 尿液
- 痰
- 潰瘍糜爛
- 流膿
- 眼睛視力不佳，尤其五爻、六爻臨玄武時

### 人事與事件

- 私情
- 不公開的關係
- 盜賊
- 欺騙者
- 暗中交易
- 投機財
- 秘密計畫

## If / Then 判斷

```text
If 玄武臨用神且旺相
Then 可取聰明、儲備實力、城府深、暗中布局

If 玄武臨忌神
Then 可取欺騙、偷盜、曖昧、藏匿問題

If 玄武臨父母爻且問風水
Then 房子光線可能不佳，或只採北面之光

If 玄武臨三爻
Then 可取臥室、私密空間、男女之事

If 玄武臨五爻或六爻且受剋
Then 眼睛、視力、看不清楚之象

If 問財運且財爻臨玄武
Then 可取投機之財、暗中操作之財，但需防不正或欺瞞

If 官鬼臨玄武
Then 可取暗病、隱患、盜賊、欺騙、曖昧或不可公開之壓力
```

## 阿咪重點速記

- 玄武核心是「藏、暗、隱」。
- 臨用神：聰明、內秀、暗中布局。
- 臨忌神：欺騙、偷盜、曖昧、隱瞞。
- 問財：財臨玄武可能是投機財或暗財。
- 問感情：玄武常提示私情、曖昧、隱藏關係。
