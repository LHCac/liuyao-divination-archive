# 朱雀

## 主題摘要

朱雀位於南方，五行屬火，代表光明、熱情、外向表達與消息傳遞。六爻中，朱雀主語言、文字、消息、文書、口才、口舌、爭訟、飛翔，也可代表事情被公開、被看見，以及未來發展方向。

## 基本屬性

| 項目 | 內容 |
|---|---|
| 方位 | 南方，前朱雀 |
| 季節 | 夏天 |
| 五行 | 火 |
| 核心特性 | 光明、熱情、外向、發揚、表達、飛揚 |

## 核心象意

### 表達與消息

- 語言表達
- 口才
- 文字
- 消息
- 新聞
- 文書
- 合約
- 公告、公開資訊

### 口舌與爭訟

- 口角
- 爭論
- 官司
- 文書糾紛
- 言語傷害
- 謠言傳播，需與騰蛇、玄武同看

### 發展與方向

- 事情被公開
- 前景與未來方向
- 光明與希望
- 活潑外向、神采飛揚

### 飛行取象

- 飛鳥
- 飛機
- 高飛之物
- 航空、飛行、快速傳播

## 身體取象

- 眼睛，尤其五爻臨朱雀時。
- 火氣、發炎、熱象需與火爻、官鬼同看。
- 言語器官、口腔、喉嚨，依問題與爻位輔助判斷。

## 風水取象

- 朱雀方為房子正前方。
- 朱雀方宜開闊、光線充足。
- 若朱雀方受阻或採光不足，可能影響前途、子嗣、發展方向。

## If / Then 判斷

```text
If 用神臨朱雀且旺相
Then 有消息、口才好、事情容易公開表達

If 用神臨朱雀但休囚受剋
Then 消息不順，或因言語、文書造成問題

If 官鬼臨朱雀
Then 可能有口舌訴訟、文書糾紛、官方消息壓力

If 官鬼臨朱雀且問職業
Then 可取語言、文字、教師、媒體、文書相關工作

If 五爻臨朱雀且受剋或臨病象
Then 眼睛可能有問題

If 朱雀動而克用神
Then 口舌、謠言、文書、消息可能傷害事情結果
```

## 阿咪重點速記

- 朱雀核心是「向外表達」。
- 問消息、文書、合約、口舌時，朱雀很重要。
- 朱雀臨官鬼，多注意口舌訴訟或文書壓力。
- 朱雀方代表前方、未來與光線。
