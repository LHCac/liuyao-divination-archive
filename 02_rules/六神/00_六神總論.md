# 六神總論

## 主題摘要

六神是六爻預測中依照日辰天干安裝在六爻旁的輔助神位，包含青龍、朱雀、勾陳、騰蛇、白虎、玄武。六神可以提供人物性格、事件細節、環境狀態、吉凶性質與取象方向，但不能取代用神旺衰、五行生剋與世應關係。

## 六神順序

六神順序固定不變：

```text
青龍 → 朱雀 → 勾陳 → 騰蛇 → 白虎 → 玄武
```

不管搖出什麼卦，六神順序都固定；會變的是從哪一個六神開始安裝。

## 依日干起六神

| 日干 | 初爻起始六神 |
|---|---|
| 甲、乙 | 青龍 |
| 丙、丁 | 朱雀 |
| 戊 | 勾陳 |
| 己 | 騰蛇 |
| 庚、辛 | 白虎 |
| 壬、癸 | 玄武 |

## 六神核心象意

| 六神 | 五行 | 核心象意 |
|---|---|---|
| 青龍 | 木 | 吉祥、喜慶、生髮、貴人、親情、才華 |
| 朱雀 | 火 | 口舌、文書、消息、表達、爭訟、飛翔 |
| 勾陳 | 土 | 穩定、遲緩、糾纏、土地、房產、牢固 |
| 騰蛇 | 火 | 虛驚、怪異、謠言、夢幻、纏繞、精神困擾 |
| 白虎 | 金 | 血光、傷病、喪事、剛硬、行動力 |
| 玄武 | 水 | 陰暗、隱秘、盜賊、曖昧、欺騙、暗中操作 |

## 六神定位

六神可視為天干在六爻中的變體應用：

- 天干偏先天，地支偏後天。
- 六神提供先天輔助信息。
- 地支、月建、日辰、用神旺衰仍是主要判斷根基。

## 斷卦使用順序

```text
用神旺衰
↓
原神、忌神、仇神
↓
世應關係
↓
動爻與變爻
↓
六神補充細節
```

## If / Then 判斷

```text
If 用神臨青龍且旺相
Then 吉象加強，主喜慶、生機、貴人

If 用神臨白虎
Then 需注意健康、傷病、血光、手術風險

If 用神臨玄武
Then 事情可能暗中進行，或有隱情、欺騙、曖昧

If 官鬼臨朱雀
Then 可能有口舌訴訟、文書糾紛、消息壓力

If 官鬼臨騰蛇
Then 可能有虛驚、謠言、怪病、精神壓力

If 忌神臨青龍且發動
Then 表面看似好事，但可能因好事、生機、酒色、飲食而生麻煩
```

## 阿咪重點速記

- 六神是輔助取象，不是斷卦核心。
- 六神順序固定，但起始六神依日干而變。
- 青龍、朱雀、勾陳、騰蛇、白虎、玄武各自提供事件細節。
- 斷卦不能只看六神吉凶，必須先看用神旺衰與生剋。
