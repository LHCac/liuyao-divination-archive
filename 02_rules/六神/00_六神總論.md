# 六神總論

## 主題摘要

六神，又稱六獸，是六爻預測中依照日辰天干安裝在六爻旁的輔助神位，包含青龍、朱雀、勾陳、騰蛇、白虎、玄武。六神可以提供人物性格、事件細節、環境狀態、吉凶性質與取象方向，但不能取代用神旺衰、五行生剋與世應關係。

## 六神使用核心原則

六神乃附和之神：

```text
卦之吉者，逢青龍而更吉。
卦之凶者，值騰蛇、白虎而更凶。
```

重要提醒：

```text
六親是決斷吉凶之用神。
六神 / 六獸是辨象輔助，用來呈現狀態與細節。
判斷吉凶仍必須以六親五行、用神旺衰、生剋沖合為主。
```

## 六神順序

六神順序固定不變：

```text
青龍 → 朱雀 → 勾陳 → 騰蛇 → 白虎 → 玄武
```

不管搖出什麼卦，六神順序都固定；會變的是從哪一個六神開始安裝。

## 六神定位

六神可視為天干在六爻中的變體應用：

- 天干偏先天，地支偏後天。
- 六神提供先天輔助信息。
- 地支、月建、日辰、用神旺衰仍是主要判斷根基。

## 斷卦使用順序

```text
用神旺衰
↓
原神、忌神、仇神
↓
世應關係
↓
動爻與變爻
↓
六神補充細節
```

## If / Then 判斷

```text
If 用神臨青龍且旺相
Then 吉象加強，主喜慶、生機、貴人

If 用神臨白虎
Then 需注意健康、傷病、血光、手術風險，也可能代表強力執行或操刀專業

If 用神臨玄武
Then 事情可能暗中進行，或有隱情、欺騙、曖昧、酒色、偏門因素

If 官鬼臨朱雀
Then 可能有口舌訴訟、文書糾紛、消息壓力、火災燙傷等象

If 官鬼臨騰蛇
Then 可能有虛驚、謠言、怪病、精神壓力、陰私疑慮

If 忌神臨青龍且發動
Then 表面看似好事，但仍需以六親生剋定吉凶，可能因好事、生機、酒色、飲食而生麻煩

If 六神象意與用神旺衰衝突
Then 以用神旺衰、生剋沖合定吉凶，六神只補細節
```

## 阿咪重點速記

- 六神是輔助取象，不是斷卦核心。
- 六親五行、用神旺衰、生剋沖合才是吉凶主體。
- 六神順序固定，但起始六神依日干而變。
- 青龍、朱雀、勾陳、騰蛇、白虎、玄武各自提供事件細節。
- 吉卦遇青龍更吉；凶卦遇蛇虎更凶。
- 斷卦不能只看六神吉凶，必須先看用神旺衰與生剋。
