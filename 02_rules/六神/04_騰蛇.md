# 騰蛇

## 主題摘要

騰蛇屬中央土分化出的陰火之象，主虛驚、怪異、纏繞、謠言、夢幻、虛假信息與精神神經方面的困擾。六爻中，騰蛇常用來判斷事情是否有虛假成分、意外驚嚇、怪異難明、傳言干擾或心理壓力。

## 基本屬性

| 項目 | 內容 |
|---|---|
| 五行 | 火，偏陰火 |
| 方位 | 中央分化之象 |
| 核心特性 | 虛驚、怪異、纏繞、謠言、夢幻 |

## 核心象意

### 事情狀態

- 虛驚一場
- 怪異、不尋常
- 纏繞難解
- 事情真假難辨
- 夢境、預兆、幻象
- 意外干擾

### 人物

- 說謊者
- 行事怪異者
- 情緒不穩者
- 讓人摸不清的人
- 製造謠言或混亂的人

### 事物

- 虛假消息
- 謠言
- 夢境
- 怪事
- 不明原因的驚嚇
- 虛幻、不實、包裝過度的東西

### 身體取象

- 神經系統
- 幻覺
- 精神困擾
- 驚嚇
- 睡眠與夢境問題
- 怪病，需搭配官鬼、用神、月日判斷

## 勾陳與騰蛇的差別

| 六神 | 重點 |
|---|---|
| 勾陳 | 穩定、遲緩、糾纏、土地、舊事、固定阻礙 |
| 騰蛇 | 虛驚、謠言、怪異、夢幻、虛假、精神干擾 |

簡單分辨：

```text
事情卡住、拖延、土地房產 → 優先看勾陳
事情虛假、怪異、驚嚇、謠言夢境 → 優先看騰蛇
```

## If / Then 判斷

```text
If 用神臨騰蛇
Then 事情可能有虛假、怪異、纏繞或意外驚嚇成分

If 官鬼臨騰蛇
Then 可能有謠言、虛驚、怪病、精神壓力

If 騰蛇動而克用神
Then 虛假信息、謠言、驚嚇或意外干擾會影響結果

If 問感情且應爻臨騰蛇
Then 對方狀態可能真假難辨，或關係中有猜疑與不安

If 問健康且官鬼臨騰蛇
Then 需注意神經、精神、幻覺、驚嚇、怪病等象
```

## 阿咪重點速記

- 騰蛇核心是「虛驚與怪異」。
- 問事情真假：騰蛇很重要。
- 問謠言、夢境、精神壓力：騰蛇很重要。
- 騰蛇動克用神，要防虛假消息或意外干擾。
