# 六獸結合六親辨象

## 主題摘要

本篇整理六獸與六親組合後的實戰取象。六獸結合六親可用來細分人、事、物的狀態與象意，但不可單獨決定吉凶。吉凶仍須以用神、六親五行、旺衰、生剋沖合、動變與世應為主。

## 使用原則

```text
六親 = 吉凶主體與用神定位
六獸 = 狀態、氣氛、細節、人物風格
```

判斷時先看：

```text
用神旺衰 → 原神忌神 → 世應 → 動變 → 六獸補象
```

```text
If 用神旺相又臨青龍
Then 吉象加強，可取喜慶、貴人、好感、正向發展

If 用神衰敗又臨白虎或騰蛇
Then 凶象加重，可取急迫、虛驚、壓力、傷病、詭譎等象

If 忌神臨青龍

If 六獸組合象意過於極端
Then 需回到問題類型、用神旺衰與現實背景取合理象，不可無限放大
```

## 阿咪重點速記

- 六獸 × 六親 = 細節象意資料庫。
- 六獸不能單獨定吉凶。
- 同一組合要依問題類型取象，例如感情、工作、健康、財運會取不同面向。
- 象意太重時要回到用神旺衰與現實背景，不要過度推論。
