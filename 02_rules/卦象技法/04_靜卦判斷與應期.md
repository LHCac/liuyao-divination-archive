# 靜卦判斷三步走與應期判斷

## 主題摘要

靜卦不是沒有資訊，而是要用「察用忌、觀生克、看旺衰」三步法判斷吉凶，再用動靜、旬空、暗動等規則推應期。靜卦尤其重視用神、忌神是否持世，以及用神與世爻的關係。

## 一、靜卦判斷三步走

```text
第一步：察用忌
↓
第二步：觀生克
↓
第三步：看旺衰
```

## 二、第一步：察用忌

先看用神或忌神是否持世。

| 情況 | 判斷 |
|---|---|
| 用神持世 | 大吉，用神即問卦者本人決心與能量 |
| 忌神持世 | 大凶，問卦者自身就是阻礙 |
| 其他爻持世 | 進入第二步 |

## 三、第二步：觀生克

觀察用神與世爻是否存在生克關係。

| 生克關係 | 判斷 | 條件 |
|---|---|---|
| 用神生世 | 吉 | 用神必須旺相，否則只是空架子 |
| 用神克世 | 特定情況可吉 | 求財、測行人等；用神旺相或平相，世爻得力 |
| 無生克 | 進入第三步 | 看旺衰 |

### 特別警示

自占卦中，世爻不主動參與對其他爻的生克。

```text
世爻代表問卦者，只承受卦象影響。
六爻中不以「世爻生用神」或「世爻克用神」作主要斷法。
即使世爻發動，也主要與自己的變爻發生作用。
```

## 四、第三步：看旺衰

| 條件 | 判斷 |
|---|---|
| 用神旺相 + 世爻興 | 吉 |
| 用神旺相 + 世爻衰 | 吉中帶憂 |
| 用神衰敗 | 凶，即使世爻旺相也不能救 |

## 五、靜卦應期判斷法則

經典原則：

```text
動而逢值逢合為應
靜而逢值逢沖為應
```

## 六、動爻應期

| 應期方式 | 說明 |
|---|---|
| 逢值 | 動爻地支對應的日、月、年 |
| 逢值 | 變爻地支對應的日、月、年 |
| 逢合 | 與動爻地支六合的日 |
| 逢合 | 與變爻地支六合的日 |

例：寅木動化子水

```text
逢值：寅日、子日
逢合：亥日（寅亥合）、丑日（子丑合）
```

## 七、靜爻應期：沖起與沖破

| 靜爻狀態 | 沖的性質 | 應期 |
|---|---|---|
| 旺相靜爻 | 沖起，越沖越旺 | 逢沖之日為應，反應最快 |
| 衰敗靜爻 | 沖破，衰敗被沖散 | 逢值之日為應 |

## 八、應期單位判斷

| 應期單位 | 適用情況 |
|---|---|
| 時辰 | 急性事件，例如小孩生病、即時治病 |
| 日 | 靜卦暗動、日常事務 |
| 月 | 一般事件 |
| 年 | 久出不歸、長期事件 |

原則：

```text
應近不應遠
多個應期選項中，優先選最近者。
```

## 九、旬空對應期的影響

| 旬空情況 | 應期方式 |
|---|---|
| 用神旬空 | 逢值出空或逢沖沖空 |
| 原神旬空 | 同上 |
| 世爻旬空 | 多在沖空或出空之時 |

## 十、完整流程

```text
靜卦判斷三步走：
1. 察用忌：用神持世吉，忌神持世凶
2. 觀生克：用神生世吉；特定問題用神克世可吉
3. 看旺衰：用旺世興吉；用神衰敗則凶

應期判斷：
1. 動爻看逢值逢合
2. 靜爻旺看逢沖，靜爻衰看逢值
3. 旬空看出空或沖空
4. 暗動主快，應期單位偏日
5. 應近不應遠
```

## If / Then 判斷

```text
If 靜卦用神持世
Then 大吉，用神即問卦者自身能量

If 靜卦忌神持世
Then 大凶，問卦者自身成為阻礙

If 用神旺相且生世
Then 吉，所求之事來助我

If 用神衰敗
Then 即使世爻旺相，也難以成事

If 靜爻旺相逢沖
Then 沖起，逢沖為應

If 靜爻衰敗逢沖
Then 沖破，逢值為應

If 用神旬空
Then 看出空或沖空之日
```

## 阿咪重點速記

- 靜卦先看用神、忌神是否持世。
- 用神生世吉，但用神要旺才有用。
- 用旺世興為吉；用神衰敗則難救。
- 應期：動看值合，靜看值沖，旬空看出空沖空。
