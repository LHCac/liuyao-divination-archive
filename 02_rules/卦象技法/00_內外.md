# 內外

## 主題摘要

內外是六爻預測中用來區分主客體、本地與外地、自身與他人、已知與未知的結構概念。內外不是固定死板的分類，而會隨著預測角度、問題範圍與太極場不同而改變。

## 六大內外結構

| 層次 | 內 | 外 |
|---|---|---|
| 卦位 | 內卦，初爻至三爻 | 外卦，四爻至六爻 |
| 日月與卦 | 卦中諸爻，本體 | 日月，外在影響力 |
| 八卦特點 | 中間爻 | 上下兩爻 |
| 世應 | 世爻，自己、本地、本方 | 應爻，對方、外地、他方 |
| 本伏 | 本卦出現之爻 | 伏藏之爻，游離主體之外 |
| 本變 | 本卦，現況 | 變卦，變化結果 |

## 常見應用

### 本地與外地

問出行、遷居、外派時：

```text
內卦 = 本地、目前所在地
外卦 = 外地、目的地、外部環境
```

### 外部力量

日月沖合某爻，常代表外部人事或時間氣場對此爻造成影響。

### 世應內外

```text
世爻 = 自己、本方、本地
應爻 = 對方、他方、外地
```

### 伏藏內外

伏藏之爻游離於本卦主體之外，常有遠離、分居、隔絕、不顯之意。

## 卦包概念

| 卦象結構 | 象意 |
|---|---|
| 陽爻包陰爻 | 外硬內軟之物；問懷孕時可輔取女孩象 |
| 陰爻包陽爻 | 外軟內硬之物；問懷孕時可輔取男孩象 |

此法屬輔助取象，不能單獨作為結論。

## 使用提醒

內外要依太極場調整。

例如：

```text
以房間為太極：房間外就是外
以整個院子為太極：院子外才是外
以城市為太極：外縣市才是外
```

## If / Then 判斷

```text
If 問本地與外地發展
Then 內卦可看本地，外卦可看外地

If 日月沖合某爻
Then 可取外部力量引發變化

If 用神在內卦
Then 事情偏在自身、本地、內部範圍

If 用神在外卦
Then 事情偏在外部、遠方、他方範圍

If 用神伏藏
Then 事情游離主體之外，有分離、隔絕、不顯之象

If 陽爻包陰爻
Then 可輔取外硬內軟之物，問胎孕可輔看女孩象

If 陰爻包陽爻
Then 可輔取外軟內硬之物，問胎孕可輔看男孩象
```

## 阿咪重點速記

- 內外不是固定的，要看問題太極場。
- 內卦多看本地、內部、自己範圍。
- 外卦多看外地、外部、對方範圍。
- 日月是卦外力量，伏藏是游離主體之外。
