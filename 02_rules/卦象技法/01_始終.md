# 始終

## 主題摘要

始終是六爻中描述事情發展過程與結果的結構概念，用來判斷事情從哪裡開始、在哪裡結束，以及目前處於哪個階段。掌握始終，有助於判斷來龍去脈、應期早晚與最終結局。

## 三種始終形式

### 一、本卦為始，變卦為終

```text
本卦 = 事情目前狀態
變卦 = 事情最終結果
```

常見組合：

| 卦變 | 含義 |
|---|---|
| 卦變回頭生 | 終助始，最終有利 |
| 卦變回頭克 | 終傷始，最終不利 |
| 六合變六沖 | 先聚後散 |
| 六沖變六合 | 先散後聚 |

### 二、動爻為始，變爻為終

```text
動爻 = 發動力量、起因
變爻 = 結果落點、終點
```

核心規則：

- 變爻可以生克本位動爻。
- 動爻不去生克本位變爻。
- 變爻回頭生動爻，多主後力足。
- 變爻回頭克動爻，多主後路受阻。

### 三、初爻為始，六爻為終

爻位時間軸：

| 爻位 | 時間與人物象意 |
|---|---|
| 初爻 | 開始、根基、童年、員工、地下 |
| 二爻 | 主管、成家、成長期、宅內 |
| 三爻 | 中層、副職、奔波期 |
| 四爻 | 中層幹部、壯年、門外 |
| 五爻 | 鼎盛、大領導、道路、外地 |
| 六爻 | 退出、退休、祖先、天上、結束 |

## 判斷快慢與階段

| 情況 | 象意 |
|---|---|
| 動爻在初爻 | 事情剛開始 |
| 動爻在三、四爻 | 事情正在過程中、奔波期 |
| 動爻在五爻 | 事情到重要階段、鼎盛位置 |
| 動爻在六爻 | 事情接近尾聲、將結束或退出 |

## If / Then 判斷

```text
If 問事情結果
Then 看本卦為現況，變卦為結局

If 本卦六合化六沖
Then 先聚後散，開始有成象，後面易散

If 本卦六沖化六合
Then 先散後聚，開始不穩，後面可聚

If 動爻化回頭生
Then 終助始，後力足

If 動爻化回頭克
Then 終傷始，後路受阻

If 動爻在初爻
Then 事情才剛開始

If 動爻在六爻
Then 事情接近尾聲

If 問官職工作且用神在初爻
Then 可能剛入職、低層、初始階段

If 問官職工作且用神在五爻
Then 可能在核心、領導、重要位置
```

## 阿咪重點速記

- 本卦看現況，變卦看結果。
- 動爻看起因，變爻看落點。
- 初爻為始，六爻為終。
- 六合化六沖：先聚後散；六沖化六合：先散後聚。
