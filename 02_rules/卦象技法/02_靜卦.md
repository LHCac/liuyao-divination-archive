# 靜卦

## 主題摘要

靜卦是六爻全部安靜、沒有動爻發動的卦象。靜卦不是沒有資訊，而是資訊較隱微，需從月建、日辰、旬空、月破、暗動、伏藏、相合相沖、旺衰生克等細節中找出卦的玄機。

## 靜卦特點

- 六爻全部安靜，無動爻。
- 事情不形於外表。
- 變化緩慢。
- 暫時停滯。
- 外靜內動，陰中有陽。

世界沒有完全靜止的事物，因此靜卦仍然可以斷，只是要觀微知著。

## 靜卦觀察重點

| 觀察點 | 說明 |
|---|---|
| 月建生克 | 月令對各爻旺衰的影響 |
| 日辰沖合 | 日辰沖某爻可能形成暗動 |
| 旬空 | 空亡爻力量削弱，或事不顯 |
| 月破 | 被月建沖者為月破，力量大損 |
| 暗動 | 靜爻被日辰沖，雖靜似動 |
| 伏藏 | 不在卦中的爻，暗藏信息 |
| 相合相沖 | 卦中爻之間的吸引、牽絆、衝突 |
| 旺衰生克 | 靜卦仍以用神旺衰為核心 |

## 暗動

靜爻被日辰沖，稱為暗動。

暗動的特點：

- 有動象，但不是正式動爻。
- 力量通常弱於明動。
- 在靜卦中是重要能量來源。
- 可提示事情表面未動，內部已有變化。

## 靜卦判斷流程

```text
Step 1：定用神
Step 2：看用神旺衰（月建、日辰）
Step 3：找暗動（日沖靜爻）
Step 4：看旬空、月破
Step 5：找伏藏
Step 6：看合沖關係
Step 7：綜合六親、六神、爻位取象
```

## If / Then 判斷

```text
If 六爻全靜
Then 為靜卦，重點看月建、日辰、旬空、月破、暗動、伏藏

If 靜爻被日辰沖
Then 為暗動，可視為有動象，但力量稍弱於明動

If 用神月建旺相且日辰生扶
Then 即使全靜，也可斷吉或有希望

If 用神空亡或月破
Then 即使全靜，也主事情有問題

If 靜卦中用神伏藏
Then 事情不顯，需要看伏爻旺衰與飛伏關係
```

## 阿咪重點速記

- 靜卦不是沒資訊，而是資訊藏得深。
- 靜卦最重要看月日、空破、暗動、伏藏。
- 日沖靜爻為暗動，是靜卦中的動力來源。
- 靜卦仍然以用神旺衰為核心。
