# 八卦取象應用

## 主題摘要

八卦取象應用是以八卦本身的人物、方位、人體部位等象意，補充六爻斷語細節。這是五行生克之外的另一個取象維度，尤其常用於判斷人物特徵、方位、病症部位與事件細節。

## 八卦人物象意

| 卦 | 人物象意 |
|---|---|
| 乾 | 老父、男性長輩、公家領導、重要人物 |
| 坤 | 老母、女性長輩、大眾、群體 |
| 坎 | 中男、盜賊、水性之人、隱藏之人 |
| 離 | 中女、文人、有才者、光明外顯者 |
| 震 | 長男、行動力強者、主動者 |
| 巽 | 長女、商人、隨和者、流通者 |
| 艮 | 少男、穩重者、停止者 |
| 兌 | 少女、口才好者、喜悅之人 |

## 應用原則

八卦取象是補充細節，不取代用神旺衰與五行生克。

使用順序：

```text
先看用神旺衰與生克
↓
再看用神、原神、忌神所在卦象
↓
補充人物、方位、身體部位或事件細節
```

## If / Then 判斷

```text
If 問病症且用神在坤卦
Then 可輔助取脾胃、腹部問題

If 問病症且用神在坎卦
Then 可輔助取腰、腎、水液系統問題

If 問病症且用神在巽卦
Then 可輔助取大腿、股部問題

If 問身體且忌神在離卦
Then 可輔助取心臟或眼部受損

If 問方位且原神在離卦
Then 可取南方有助力或貴人

If 問方位且忌神在坤卦
Then 可取西南方向不利

If 問財且用神在乾卦
Then 可取與長輩、公家、領導或重要事物有關

If 問婚且用神在乾卦
Then 可取對方年齡較大、男性氣強或較有主導性
```

## 阿咪重點速記

- 八卦取象是細節補充，不是吉凶核心。
- 問病可看用神、忌神落在哪個卦，輔助定位身體部位。
- 問方位可用後天八卦方位。
- 問人物特徵可用八卦人物象意。
