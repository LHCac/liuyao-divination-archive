# 兄弟

## 主題摘要

兄弟爻代表與「我」同類者，包括兄弟姐妹、朋友、同事、同行、競爭者、阻力與分財者。兄弟爻最大的特點是延伸性與競爭性，能擴大影響範圍，也可能分奪財源。

## 根本含義

兄弟 = 同我者。

在六爻中，「我」指卦宮，也就是所問之事的太極場。凡是與此事同類、同輩、同行、相似、競爭或延伸者，都可歸入兄弟象。

## 常見取象

| 角度 | 兄弟代表 |
|---|---|
| 人際 | 兄弟姐妹、朋友、同事、同學、同輩 |
| 競爭 | 同行、競爭者、競爭公司、搶資源者 |
| 財運 | 分財、借錢、破財、財被分奪 |
| 身體 | 手足、手臂、肢體延伸 |
| 建築 | 院牆、圍牆、支脈、延伸出去的結構 |
| 事業 | 同類企業、同行市場、競爭環境 |

## 六親生剋邏輯

```text
父母生兄弟
兄弟生子孫
官鬼克兄弟
兄弟克妻財
```

### 解讀

- 父母生兄弟：有源頭才有同類延伸。
- 兄弟生子孫：朋友多、路好走；困難中生出快樂與動力。
- 官鬼克兄弟：公事克私情，規則壓制朋黨與競爭。
- 兄弟克妻財：競爭分財、朋友借財、同行搶財。

## 兄弟爻的喜忌

### 問財

兄弟爻多為忌。

- 兄弟動：競爭者強、財源被分奪。
- 兄弟旺：同行多、分財嚴重。
- 兄弟克妻財：求財不利。

### 問朋友、人緣

兄弟爻可為喜。

- 兄弟旺而得生：朋友多、支援多。
- 父母生兄弟：願意為朋友操心，人脈廣。

### 問競爭

兄弟爻代表競爭者。

- 兄弟旺：對手強。
- 兄弟被官鬼克：對手可能受規則、法律、主管壓制。

## 特殊取象

- 兄弟臨木：延伸性更強，人脈、手足、發展分散更明顯。
- 兄弟在六爻：遠方、臂膀、延伸到外部，影響範圍較廣。
- 兄弟持世：自己與對方相似，或自己處於競爭、同類、分財位置。

## If / Then 判斷

```text
If 問財，兄弟爻旺相發動
Then 競爭激烈、財源被分奪，不利求財

If 問人際，兄弟爻旺且得生
Then 朋友多、人脈廣、有人協助

If 問生意競爭，兄弟動但被官鬼克制
Then 對手可能受規則、法律、主管壓制，反而對自己有利

If 兄弟持世
Then 自己可能處於競爭者、同類者、朋友或分財角色；問財多不利，問友可利

If 六爻兄弟臨木
Then 延伸力強，可主遠方手足、人脈廣，也可能財源被廣泛分散
```

## 阿咪重點速記

- 兄弟 = 同類、朋友、競爭者、阻力。
- 問財忌兄弟，因兄弟克財。
- 問朋友、人脈，兄弟可為助力。
- 問競爭，兄弟就是同行或對手。
