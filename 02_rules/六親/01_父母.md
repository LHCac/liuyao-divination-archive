# 父母

## 主題摘要

父母爻代表天地、源頭、保護、庇護、文書、學習、車輛、房屋與長輩。父母爻是六親中最具包容與保護性的一爻，常作為學業、考試、文件、證照、車輛出行、父母健康等問題的用神。

## 根本含義

父母 = 生我者。

在六爻中，父母有以下核心特性：

- 保護性
- 包容性
- 庇護性
- 源頭與根基
- 文書與知識
- 勞心與操心

## 常見取象

| 角度 | 父母代表 |
|---|---|
| 人物 | 父親、母親、長輩、老師、保護者 |
| 物件 | 房屋、衣服、雨傘、車輛、包裹、快遞 |
| 機構 | 單位、學校、庇護員工或學生之處 |
| 學習 | 學業、考試、知識、學歷、書籍、試卷 |
| 文書 | 合約、證照、文件、公告、申請資料 |
| 精神 | 辛苦、勞心、操心、責任感 |
| 建築 | 主房、房屋本體、承載保護之物 |

## 六親生剋邏輯

```text
官鬼生父母
父母生兄弟
父母克子孫
妻財克父母
```

### 解讀

- 官鬼生父母：職位、規章、官方系統帶來文書與名分。
- 父母生兄弟：有源頭與保護，才有同類延伸。
- 父母克子孫：規矩、學習、保護壓制玩樂與自由。
- 妻財克父母：欲望、財物、飲食可能損害庇護與長輩。

## 重要應用

### 問學業考試

父母爻為用神。

- 父母旺相：學業有成、成績較好、文書順利。
- 父母空亡：考試落空、成績不理想、文件暫時不得。
- 父母受克：讀書受阻、文書有問題。

### 問車輛出行

父母可代表車輛、承載與保護。

- 父母旺相：車輛穩、路程較平安。
- 父母受克：車輛問題、行程阻礙。
- 父母空破：文件、票券、車況或行程安排落空。

### 問父母健康

父母爻為父母本人。

- 父母旺相受生：健康較穩。
- 妻財動克父母：不利父母健康。
- 父母入墓、受克、空破：需注意身體與病情延遲。

### 父母持世

- 愛操心。
- 成熟老成。
- 責任感強。
- 辛苦奔波、勞心勞力。
- 保護欲強。

## If / Then 判斷

```text
If 問學業考試，父母爻旺相
Then 學業有成、成績較好、考試或文書順利

If 問文書證照，父母爻空亡
Then 文書暫時無法取得，需等填實或出空

If 問出行車輛，父母爻受剋
Then 車輛、票券、行程或路途有問題

If 問父母健康，妻財爻發動克父母
Then 父母健康受損，不宜輕忽

If 世爻持父母
Then 此人愛操心、責任感強、勞心勞力

If 問工作，父母持世且旺相
Then 可能學歷、專業、文書能力有利工作
```

## 阿咪重點速記

- 父母 = 保護、文書、學業、車輛、房屋、長輩。
- 問學業、證照、合約、車輛出行，常取父母為用神。
- 父母持世：操心、責任、辛苦。
- 妻財克父母：問父母健康時很不利。
