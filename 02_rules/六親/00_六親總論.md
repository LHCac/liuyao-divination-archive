# 六親總論

## 主題摘要

六親是把五行生剋關係人倫化、生活化後形成的判斷工具，包含父母、兄弟、子孫、妻財、官鬼五種實際關係。六親配置以「卦宮五行」為中心，不是以世爻五行為中心，是六爻取象與斷卦的核心系統。

## 六親的本質

六親本質上是五行生剋的擬人化表達：

| 與卦宮五行的關係 | 六親 |
|---|---|
| 同我者 | 兄弟 |
| 我生者 | 子孫 |
| 生我者 | 父母 |
| 我克者 | 妻財 |
| 克我者 | 官鬼 |

> 注意：這裡的「我」是卦宮，也就是所問之事的太極場，不是問卦者本人。問卦者通常由世爻代表。

## 六親相生循環

```text
父母 → 兄弟 → 子孫 → 妻財 → 官鬼 → 父母
```

### 意義

- 父母生兄弟：有源頭與庇護，才有同類延伸。
- 兄弟生子孫：困難、競爭、同類互動中生出動力與快樂。
- 子孫生妻財：有路、有動力，才有財源。
- 妻財生官鬼：欲望、財物、飲食會引來約束、工作、疾病或禍端。
- 官鬼生父母：官位、制度、死亡與結束後，又生出文書、名分、新秩序。

## 六親相剋速查

| 相剋 | 常見理解 |
|---|---|
| 父母克子孫 | 規矩、學業、保護壓制玩樂與自由 |
| 子孫克官鬼 | 藥克病、自由克約束、神佛克鬼祟 |
| 官鬼克兄弟 | 公事克私情、制度壓制競爭與朋黨 |
| 兄弟克妻財 | 競爭分財、朋友借財、同行搶財 |
| 妻財克父母 | 欲望損庇護、財損文書、妻財損父母健康 |

## 取象靈活性

同一個人事物，在不同問題中可能屬於不同六親。

例子：

| 事物 | 問題角度 | 可能六親 |
|---|---|---|
| 車輛 | 代步、保護、承載 | 父母 |
| 車輛 | 財產、汽車買賣 | 妻財 |
| 同事 | 同級競爭 | 兄弟 |
| 同事 | 結婚對象 | 妻財或官鬼，依性別與問題而定 |
| 晚輩 | 家庭後代 | 子孫 |
| 晚輩 | 公司員工 | 妻財，因為被我使用、為我服務 |

## 太極場觀念

```text
卦宮 = 這件事情的太極場
世爻 = 問卦者在這件事中的角色
應爻 = 對方、外部對象或相對面
六親 = 這件事情裡的角色分類
```

同一個人在不同太極場中，六親角色會改變。家中是兄弟，工作中可能是領導而成為官鬼；公司中是員工，對老闆而言可能是妻財。

## If / Then 判斷

```text
If 問工作升遷
Then 官鬼多代表職位、工作、主管、約束，妻財可代表薪資與業績

If 問孩子
Then 子孫為孩子，父母可代表孩子的學業、學校、文書

If 問生意合夥
Then 應爻可看合夥人，兄弟可看同行競爭者

If 同一個人在不同問題中出現
Then 依太極場重新定位六親，不可固定死板

If 某事物兼具多種六親屬性
Then 依問題真正關心的角度取用神
```

## 阿咪重點速記

- 六親配置看卦宮五行，不看世爻五行。
- 六親其實是五行生剋的人倫化版本。
- 斷卦前先確認「此事的太極場」與「問題角度」。
- 同一物可有不同六親身份，取象要靈活。
