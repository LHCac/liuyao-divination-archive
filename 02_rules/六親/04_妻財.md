# 妻財

## 主題摘要

妻財爻代表一切可以為我所驅使、所使用、所掌握之物，包括財物、收入、妻子、女性、員工、工具、飲食與欲望。妻財與財運、婚姻、欲望、飲食、業績最直接相關。

## 根本含義

妻財 = 我克者，也就是可以被我掌握、使用、支配的人事物。

妻財的核心特點：

- 為我服務
- 供我使用
- 被我驅使
- 財物與資源
- 欲望與私心
- 飲食與享受

## 常見取象

| 角度 | 妻財代表 |
|---|---|
| 財運 | 財物、收入、存款、金錢、資產 |
| 婚姻 | 妻子、女性、男問感情中的對象 |
| 工作 | 員工、工具、被使用的人力資源 |
| 飲食 | 食物、入口之物、享受 |
| 欲望 | 貪念、私心、物欲、享樂 |
| 物件 | 各種工具、財產、可支配之物 |

## 六親生剋邏輯

```text
子孫生妻財
妻財生官鬼
兄弟克妻財
妻財克父母
```

### 解讀

- 子孫生妻財：財源來自道路、動力、客源、流通。
- 妻財生官鬼：財物與欲望引來工作、約束、疾病或禍端。
- 兄弟克妻財：競爭者搶財、朋友借財、同行分財。
- 妻財克父母：欲望損害保護、文書、學業與父母健康。

## 重要應用

### 問財

妻財爻多為用神。

- 妻財旺相持世：自己掌握財權，財運較旺。
- 妻財空亡或月破：財源暫時落空，需等填實、沖開或旺起。
- 兄弟發動克妻財：競爭分財，不利求財。
- 子孫旺相生妻財：財源長遠，財路穩定。

### 問婚姻感情

男問感情、婚姻，妻財常代表女方或妻子。

- 妻財旺相臨青龍：感情較吉，對方條件或情感較佳。
- 妻財受克：感情受損，女方狀態不穩。
- 妻財空亡：對方不實、不現，或感情暫時落空。

### 問健康

妻財可代表飲食。

- 妻財動生官鬼：病從口入，飲食不當引病。
- 妻財克父母：問父母健康時不利。

### 問升遷工作

妻財可代表業績、效益、薪資。

- 妻財旺相生官鬼：業績推動升職或工作機會。
- 世爻持妻財：重視成果、收益、實際利益。

## If / Then 判斷

```text
If 問財，妻財爻旺相且持世
Then 財運較旺，自己能掌握財權

If 問財，妻財爻空亡或月破
Then 財源不穩或暫時落空，需等填實或沖開

If 問財，兄弟爻發動克妻財
Then 競爭者搶財、朋友借財或財被分走

If 子孫爻旺相生妻財
Then 財源長遠，財路穩定

If 問健康，妻財爻動生官鬼
Then 可能病從口入，飲食不當引發疾病

If 問升遷，妻財動生官鬼
Then 可由業績、效益、財務成果推動職位提升
```

## 阿咪重點速記

- 妻財 = 財、妻、欲望、飲食、員工、工具。
- 問財最怕兄弟動克妻財。
- 問長遠財運，要看子孫是否生妻財。
- 妻財生官鬼：可代表業績生職位，也可代表飲食生病、財多招禍。
