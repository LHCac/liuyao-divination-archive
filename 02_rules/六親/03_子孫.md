# 子孫

## 主題摘要

子孫爻代表後輩、孩子、快樂、自由、動力、道路、財源頭、藥物與解救力量。子孫克官鬼，因此在問病、官司、災禍時常是重要的解救爻；子孫又生妻財，因此問財時也是財源與財路。

## 根本含義

子孫 = 我生者。

子孫像孩子，最大特點是：

- 快樂
- 自由
- 活力
- 動態
- 不喜歡被管束
- 具有先天真陽之氣

## 常見取象

| 角度 | 子孫代表 |
|---|---|
| 人物 | 子女、孫子女、晚輩、後輩 |
| 精神 | 快樂、自由、玩樂、活潑、開朗 |
| 動態 | 道路、車流、人流、血管、腸道、流動路徑 |
| 財富 | 財源頭、財路、賺錢管道 |
| 醫療 | 藥物、治療、病之解救 |
| 訴訟 | 官司解救、化解官禍 |
| 靈學 | 僧道、術士、神佛、靈感、第六感 |

## 六親生剋邏輯

```text
兄弟生子孫
子孫生妻財
父母克子孫
子孫克官鬼
```

### 解讀

- 兄弟生子孫：困難、競爭、朋友與同類之中生出動力與快樂。
- 子孫生妻財：路通才有財，財源來自動力與出口。
- 父母克子孫：學習、規矩、保護壓制玩樂與自由。
- 子孫克官鬼：藥克病、自由克約束、神佛克鬼祟。

## 重要應用

### 問病

官鬼為病，子孫為藥。

- 子孫旺相：有救，病可解。
- 子孫發動克官鬼：治療有效。
- 子孫空亡休囚：藥力不足，解救力弱。
- 子孫被父母克：治療受阻，或規矩、文書、壓力壓制恢復力。

### 問官司訴訟

子孫克官鬼，可化解官禍。

- 子孫旺相動：官司可解，有機會脫身。
- 子孫弱或空亡：解救力不足。

### 問財

子孫是財源頭，妻財是財本身。

- 子孫旺相生妻財：財源長遠，財路通。
- 子孫受克：財路不順。
- 只有妻財旺但子孫弱：眼前有財，但來源不穩。

### 問孩子

- 子孫旺相：孩子狀況較好，求子較吉。
- 子孫持世：活潑開朗、愛自由。
- 子孫空亡：孩子問題暫時落空，或求子暫未成。

### 問修行靈學

子孫代表先天之氣、神佛、僧道術卜。

- 子孫旺相：與靈學、術法、修行有緣，靈感較強。

## If / Then 判斷

```text
If 問病，子孫爻旺相且能克官鬼
Then 有藥可救，病情有解

If 問病，子孫爻空亡休囚
Then 解救力弱，藥效不足或難以治癒

If 問訴訟，子孫爻動且旺相
Then 官司可解，官禍有化解機會

If 問財，子孫旺相生妻財
Then 財源長遠，財路穩定

If 問出行，子孫旺相
Then 路途順暢；子孫受剋則路上有阻

If 問修行靈學，子孫旺相
Then 與神佛、術法、靈感、第六感有緣
```

## 阿咪重點速記

- 子孫 = 快樂、自由、孩子、道路、財源、藥。
- 問病與官司，子孫是解救爻。
- 問財，子孫是財源頭，妻財是財本身。
- 問學業時子孫太旺可能不利，因玩樂自由壓過規矩學習。
