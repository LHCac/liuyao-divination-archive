# 官鬼

## 主題摘要

官鬼爻代表約束、工作、職位、領導、官府、疾病、死亡、鬼神、小人、盜賊與不可掌控之事。官鬼是六親中最複雜的一爻：問工作時常為用神，問病時多為病源，問官司時代表官非壓力，問女方感情時可代表男性對象。

## 根本含義

官鬼 = 克我者。

「官」偏陽間的管束與制度，如官府、領導、工作、職位、規章。

「鬼」偏陰暗、不可測、不可掌控之事，如疾病、死亡、鬼神、盜賊、小人、陰邪。

## 常見取象

| 角度 | 官鬼代表 |
|---|---|
| 社會 | 官府、政府、領導、主管、權力、制度 |
| 工作 | 職位、工作、職責、壓力、升遷 |
| 健康 | 疾病、病源、身體壞掉之處、腐爛 |
| 官司 | 官非、訴訟、法律壓力、官方約束 |
| 靈學 | 鬼神、陰邪、不可測之事 |
| 人物 | 小人、盜賊、男性、管束者 |

## 六親生剋邏輯

```text
妻財生官鬼
官鬼生父母
子孫克官鬼
官鬼克兄弟
```

### 解讀

- 妻財生官鬼：財物、業績、飲食、欲望引出工作、疾病、禍端。
- 官鬼生父母：職位與制度帶來文書、名分、規章、榮耀。
- 子孫克官鬼：藥克病、自由克約束、神佛克鬼祟、官司可解。
- 官鬼克兄弟：制度克制競爭者，公事克私情。

## 重要應用

### 問工作升遷

官鬼多為用神。

- 官鬼旺相：職位力量明顯，工作機會或壓力強。
- 官鬼動生父母：升職後有文書、名分、制度、證照或公告。
- 妻財動生官鬼：因業績、收益、實際成果推動升遷。
- 官鬼自化進神：職位自身往前推進，可能涉及關係、政治手段或職務進階。

### 問疾病

官鬼多為疾病爻。

- 官鬼旺相克世：病情較重。
- 子孫旺相克官鬼：有藥可解，治療有效。
- 官鬼臨玄武：暗病、隱疾、泌尿生殖、陰暗病象。
- 官鬼臨騰蛇：怪病、驚嚇、精神神經、陰邪疑慮。

### 問官司

官鬼代表官非壓力。

- 官鬼旺而世弱：官司不利，壓力大。
- 子孫旺相動克官鬼：官禍可解。
- 官鬼臨朱雀：口舌、訴訟、文書糾紛。

### 問鬼神或小人

- 官鬼臨玄武：盜賊、小人、陰謀、隱匿問題。
- 官鬼臨騰蛇：陰邪、驚擾、怪異、虛驚。
- 官鬼臨青龍：正直官員、正式單位、貴氣之官。

## If / Then 判斷

```text
If 問工作升遷，官鬼旺相且動生父母
Then 升職有望，並帶來文書、制度、名分或公告

If 問升遷，妻財動生官鬼
Then 可透過業績、收益、實際成果推動職位提升

If 問病，官鬼旺相克世
Then 病情較重，需看子孫是否能克官鬼

If 問病，子孫旺相克官鬼
Then 有藥可救，治療有解

If 官鬼臨玄武且發動
Then 有盜賊、小人、暗病、隱情或陰暗操作之象

If 官鬼臨騰蛇且發動
Then 有虛驚、怪病、陰邪、謠言、精神壓力之象

If 問生意競爭，兄弟動但被官鬼克制
Then 對手可能被法律、規則、主管壓制，自己轉危為安
```
