# 爻之旺衰：月建與日建的綜合判斷

## 主題摘要

判斷爻的旺衰不是單純加減分，而要分層次。臨、合、沖破屬於高層次關係，往往決定卦象大勢；一般生剋、洩耗只是輔助。斷卦時需先看高層次，再看一般層次。

## 一、爻得月建旺相的四種情況

| 層次 | 類型 | 定義 | 示例 |
|---|---|---|---|
| 高層次旺相 | 旺相一 | 爻臨月令，同支 | 寅木爻逢寅月 |
| 高層次旺相 | 旺相二 | 爻得月令六合 | 寅木爻逢亥月，亥寅合 |
| 一般旺相 | 旺相三 | 爻受月令五行相生 | 寅木爻逢子月，水生木 |
| 一般旺相 | 旺相四 | 爻與月令同五行 | 寅木爻逢卯月，同屬木 |

```text
高層次旺相 > 一般層次衰敗
```

例：寅木爻逢寅月酉日，雖被日辰酉金剋，但寅木臨月建，屬高層次旺相，總體仍以旺相論。

## 二、爻得月建衰敗的四種情況

| 層次 | 類型 | 定義 | 示例 |
|---|---|---|---|
| 高層次衰敗 | 衰相一 | 月破，爻被月令相沖 | 亥水爻逢巳月，巳亥沖 |
| 一般衰敗 | 衰相二 | 爻受月令五行剋制 | 寅木爻逢酉月，金剋木 |
| 一般衰敗 | 衰相三 | 爻剋月令，耗氣 | 寅木爻逢丑月，木剋土 |
| 一般衰敗 | 衰相四 | 爻生月令，洩氣 | 寅木爻逢午月，木生火 |

```text
高層次衰敗（月破） > 一般層次旺相
```

例：寅木爻逢申月子日，雖得日辰子水相生，但申月沖寅成月破，高層次衰敗大於日辰一般生扶，總體仍以衰敗論。

## 三、旺衰判斷公式

```text
先看：臨月、合月、沖月（月破）
↓
若有高層次關係，先定大勢
↓
再看：日建的臨、合、沖
↓
最後看：月生日生、月剋日剋、洩耗等一般層次
↓
綜合判定總體旺衰
```

## 四、泛用優先順序

| 優先 | 關係 | 層次 |
|---|---|---|
| 1 | 月臨、月合、月破 | 最高層，決定大勢 |
| 2 | 日臨、日合、日沖 | 高層，次於月 |
| 3 | 月生、月剋、月耗、月洩 | 一般層次 |
| 4 | 日生、日剋 | 一般層次，較低 |
| 5 | 日洩、日耗 | 多作平相 |

## 五、動爻化旺、化衰

| 變爻情況 | 象意 |
|---|---|
| 變爻臨月建 | 動爻化旺，用事爻獲得旺相徵兆 |
| 變爻合月建 | 動爻化旺，有用動爻的一種形式 |
| 變爻被月建沖破 | 動爻化衰，動而不實 |
| 變爻被月建剋制 | 動爻力量打折，後續不穩 |

## 六、實戰檢查清單

```text
1. 用神跟月建是臨、合、沖破嗎？
2. 用神跟日辰是臨、合、沖嗎？
3. 兩者綜合後，用神總體旺還是衰？
4. 月臨高層旺 + 日沖一般衰 → 總體仍旺
5. 月破高層衰 + 日生一般旺 → 總體仍衰
```

## If / Then 判斷

```text
If 爻臨月或合月
Then 高層次旺相，優先判旺

If 爻月破
Then 高層次衰敗，優先判衰

If 高層次旺相與一般衰敗同時出現
Then 以高層次旺相為主

If 高層次衰敗與一般旺相同時出現
Then 以高層次衰敗為主

If 動爻變爻臨月或合月
Then 動爻化旺，後續力量強

If 動爻變爻被月破
Then 動爻化衰，需再看是否假破或有救應
```

## 阿咪重點速記

- 旺衰要分層，不是單純加減分。
- 臨月、合月、月破最重要。
- 高層次能壓過一般層次。
- 動爻也要看變爻旺衰，因為變爻代表後續力量。
