# 月建與日建總論：平相、日破與暗動

## 主題摘要

月建與日建同功同權，共同構成日月組合，影響卦中爻的旺衰與發動。但兩者作用範圍不同：月建主旺衰與月破，日建主短期觸發，尤其能沖動靜爻，形成暗動或日破。

## 一、日建與月建的核心差異

| 功能 | 月建 | 日建 |
|---|---|---|
| 不同五行生爻 | 旺相 | 平相，不加分不減分 |
| 不同五行剋爻 | 衰敗 | 平相，不加分不減分 |
| 爻生建 | 休囚，爻氣外洩 | 平相 |
| 爻剋建 | 休囚，爻氣受耗 | 平相 |
| 相合 | 高層次旺相 | 高層次旺相 |
| 相沖 | 月破，高層次衰敗 | 沖動靜爻，分暗動或日破 |
| 臨同支 | 高層次旺相 | 高層次旺相 |
| 沖動靜爻 | 無此功能 | 日建獨有功能 |

## 二、平相的觀念

日建對爻位的「洩」與「耗」，不計入旺衰判斷，歸為平相。

```text
爻生日建 → 平相
爻剋日建 → 平相
日建生爻、日建剋爻 → 仍可參考，但層次低於臨、合、沖
```

例：

```text
寅木爻逢午月：木生火，爻洩氣，月令下為休囚
寅木爻逢午日：日令洩爻，不作休囚，視為平相
```

## 三、日建獨有功能：沖動靜爻

日辰沖靜爻，不能一律說暗動，需先判斷該靜爻本身旺衰。

```text
靜爻被日沖
↓
爻本身旺相 → 暗動，大利，有用之動
爻本身衰敗 → 日破，動散，無用之動
```

## 四、暗動

| 項目 | 內容 |
|---|---|
| 定義 | 靜爻旺相，又被日辰沖動 |
| 象意 | 隱藏力量被激活，常是卦中關鍵轉折 |
| 實戰 | 用神靜爻旺相 + 日沖 = 用神被激活 |

## 五、日破

| 項目 | 內容 |
|---|---|
| 定義 | 靜爻衰敗，又被日辰沖散 |
| 象意 | 動了等於沒動，表示事情難成立 |
| 實戰 | 用神靜爻衰敗 + 日沖 = 用神動散，偏凶 |

## 六、判斷流程

```text
Step 1：看爻與月建關係
Step 2：判斷爻本身旺衰基礎
Step 3：若被日辰沖，判斷暗動或日破
Step 4：旺相被日沖 = 暗動
Step 5：衰敗被日沖 = 日破
```

## 七、實戰修正

過去若把「日洩用神」直接判成用神變弱，需修正為：

```text
日洩用神 = 平相，不計入旺衰扣分
日洩世爻 = 平相，不計入旺衰扣分
日沖靜爻 = 要分暗動或日破
```

## If / Then 判斷

```text
If 爻生日建或爻剋日建
Then 視為平相，不直接扣旺衰分

If 爻生月建或爻剋月建
Then 視為休囚，屬於月令下的衰象

If 靜爻旺相且被日辰沖
Then 暗動，有用之動

If 靜爻衰敗且被日辰沖
Then 日破，動散無力

If 用神旺相暗動
Then 用神被激活，事情有快速推動可能

If 用神衰敗日破
Then 用神被沖散，事情難以落實
```

## 阿咪重點速記

- 月建主旺衰，日建主沖動。
- 月建的洩耗要算休囚；日建的洩耗多作平相。
- 日沖靜爻不是一定暗動，要先看旺衰。
- 旺者逢日沖為暗動，衰者逢日沖為日破。
