## 使用原則

斷卦時，六神屬於輔助取象工具，不取代用神旺衰與五行生剋判斷。

建議順序：

```text
問題類型
↓
取用神
↓
看用神旺衰
↓
看原神、忌神、仇神
↓
看動爻、變爻、世應
↓
最後用六神補充細節取象
```
