# 伏藏與飛伏

## 主題摘要

當六爻卦中缺少某一六親時，需要從本宮首卦中找出該爻，寫在對應爻位飛爻的左側，稱為伏藏。飛爻是覆蓋在上的爻，伏爻是被藏在下的爻。伏藏代表所問之事不顯、不順、有阻礙，但仍需看伏爻旺衰與飛伏關係判斷成敗。

## 基本概念

| 名稱 | 說明 |
|---|---|
| 伏爻 | 被藏在飛爻下方的爻，代表不顯、不出、潛藏的人事物 |
| 飛爻 | 覆蓋伏爻的本卦之爻，像蓋在種子上面的土壤 |
| 伏藏 | 用神或某六親不出現在本卦中，需從本宮首卦借出 |

## 種子與土壤比喻

伏爻像種子，飛爻像土壤。

| 情況 | 含義 |
|---|---|
| 伏爻旺相 + 飛爻生伏 | 種子好、土壤也好，吉，等出現即應 |
| 伏爻旺相 + 飛爻克伏 | 種子好但土壤不利，有阻力，需等待 |
| 伏爻休囚空破 | 種子本身壞掉，凶，即使飛爻生也難成 |
| 伏爻弱又被飛爻克 | 雙重不利，凶象重 |

## 伏爻的作用原則

- 伏爻可以受到日月與卦中其他爻的生克。
- 伏爻在未出現之前，不能主動生克卦中其他爻。
- 用神伏藏，代表所問之事有阻礙、不順、不顯。
- 伏藏不一定凶，若伏爻旺相且飛爻生扶，仍可成。

## 伏藏應期

伏爻出現常見應期：

1. 伏爻逢沖，被沖出。
2. 飛爻逢沖，飛移開，伏爻得出。
3. 伏爻逢合，因合而被引出。
4. 伏爻出空、填實、得日月扶起。

## If / Then 判斷

```text
If 用神伏藏
Then 所問之事有阻礙、不順、不顯，但不必直接判凶

If 伏爻旺相且飛爻生伏
Then 吉，等待伏爻出現或應期到來即可成

If 伏爻休囚空破
Then 伏爻本身無力，事情難以翻轉

If 飛爻克伏爻但伏爻旺相
Then 有阻力但仍可撐住，需等飛爻被沖開或伏爻被引出

If 伏爻逢沖或飛爻逢沖
Then 伏爻有顯現、出現、應事的機會
```

## 阿咪重點速記

- 伏藏 = 事情藏住、不顯、有阻礙。
- 飛爻像土壤，伏爻像種子。
- 伏爻旺才有希望；伏爻弱空破，土再好也難成。
- 伏爻未出時不能主動作用，但可受日月與卦爻影響。
