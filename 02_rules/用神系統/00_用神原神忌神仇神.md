# 用神、原神、忌神、仇神

## 主題摘要

用神是所問之事的核心代表；原神是生用神之爻，像用神的元氣與來源；忌神是克用神之爻；仇神則是生忌神、克原神之爻。四者構成用神周圍的能量網絡，是判斷吉凶、轉機與阻礙的關鍵。

## 四者關係

```text
原神 → 生 → 用神
忌神 → 克 → 用神
仇神 → 生 → 忌神
仇神 → 克 → 原神
```

## 原神

原神是生用神的爻，如人之元氣、水之源頭。

### 原神的作用

- 用神休囚時，第一件事要找原神。
- 原神旺相發動生用神，稱為「旱苗得雨、雪中送炭」。
- 原神可讓用神轉弱為強，逢凶化吉。
- 原神若空破、休囚、被合、被制，則無法生用神。

## 忌神

忌神是克用神的爻。

### 忌神的作用

- 忌神發動克用神，代表所問之事受損。
- 忌神旺而用神弱，事情容易往凶的方向發展。
- 凶象重時，忌神可能先制原神，再克用神，形成斬草除根之象。

## 仇神

仇神是生忌神、克原神之爻。

### 仇神的作用

- 仇神生忌神，間接加強對用神的傷害。
- 仇神克原神，削弱用神的生助來源。
- 仇神與忌神同時有力時，凶象加重。

## 判斷順序

```text
先定用神
↓
看用神旺衰
↓
用神弱，先找原神
↓
看忌神是否發動克用神
↓
看仇神是否生忌神或克原神
↓
綜合月建、日辰、動爻、變爻判斷
```

## If / Then 判斷

```text
If 用神休囚
Then 先找原神，看是否能生起用神

If 原神旺相發動生用神
Then 用神可得救，事情有轉機

If 原神空破或被制
Then 生助無力，事情難以扭轉

If 忌神發動克用神
Then 用神受損，事情不順

If 忌神發動克用神，且仇神同時克原神
Then 忌神斷源，大凶，類似斬草除根

If 原神發動，但用神本身空破休囚太重
Then 來生也未必有效，需防爛根難救
```

## 阿咪重點速記

- 原神 = 生用神者，是用神的來源。
- 忌神 = 克用神者，是直接傷害。
- 仇神 = 生忌神、克原神，是間接加害。
- 用神弱時，先看原神有沒有救。
- 忌神旺、原神被制，通常凶象加重。
