# 用神、原神、忌神、仇神

## 主題摘要

用神是所問之事的核心代表；原神是生用神之爻，像用神的元氣與來源；忌神是克用神之爻；仇神則是生忌神、克原神之爻。四者構成用神周圍的能量網絡，是判斷吉凶、轉機與阻礙的關鍵。

## 四者關係

```text
原神 → 生 → 用神
忌神 → 克 → 用神
仇神 → 生 → 忌神
仇神 → 克 → 原神
```

補充關係圖：

```text
仇神 → 剋 → 原神 → 生 → 用神
仇神 → 生 → 忌神 → 剋 → 用神
```

## 用神

用神是占問人事物的主要核心，也是斷卦的鎖鑰。

斷卦時，先依占問事由類別取用神，再以用神為中心，衡量月建、日辰、動爻、變爻、全卦五行生剋沖合，最後決斷吉凶。

重要原則：

```text
用神是斷卦核心。
六親決定吉凶主體。
六神只輔助辨象，不取代六親五行。
```

## 用神多現

若用神在卦中多現，不應草率捨棄其中一個。任一用神都可能是主題意念的一部分，需要依旺衰、動靜、世應、月日、爻位與六神分辨其層次。

```text
If 用神多現
Then 不直接捨棄，逐一分辨其代表的人、事、層次與真假
```

注意：資料補充中特別提醒，古書中「捨其閒爻而用持世」等五條固定取捨法，不宜機械採用。實務上應以卦中實際功能與主題意念判斷。

## 原神 / 元神

原神，又稱元神，是生用神的爻，如人之元氣、水之源頭。

| 名稱 | 定義 | 作用 |
|---|---|---|
| 原神 / 元神 | 生用神之爻 | 扶助用神，使事情有來源與支撐 |

### 原神的作用

- 用神休囚時，第一件事要找原神。
- 原神旺相發動生用神，稱為「旱苗得雨、雪中送炭」。
- 原神可讓用神轉弱為強，逢凶化吉。
- 原神若空破、休囚、被合、被制，則無法生用神。

### 原神歌

```text
元神發動志揚揚，
用伏藏兮也不妨，
須要生扶兼旺相，
最嫌化剋及逢傷。
```

白話：

```text
原神若發動且旺相，能積極生助用神。
即使用神伏藏，只要原神有力，仍有轉機。
但原神最怕被剋、化剋、空破或受傷。
```

## 忌神

忌神是克用神的爻。

| 名稱 | 定義 | 作用 |
|---|---|---|
| 忌神 | 剋用神之爻 | 傷害用神，使事情受阻或受損 |

### 忌神的作用

- 忌神發動克用神，代表所問之事受損。
- 忌神旺而用神弱，事情容易往凶的方向發展。
- 凶象重時，忌神可能先制原神，再克用神，形成斬草除根之象。
- 忌神宜靜不宜興，若忌神發動，需看是否被剋、被合、旬空、化退、貪生忘剋或通關。

### 忌神歌

```text
看卦先須看忌神，
忌神宜靜不宜興，
忌神急要逢傷剋，
若遇生扶用受刑。
```

白話：

```text
看卦不可只看用神，也要看忌神。
忌神安靜較好，忌神發動則容易傷用。
忌神若被制，凶力可減。
忌神若反得生扶，用神容易受害。
```

## 仇神

仇神是生忌神、克原神之爻。

| 名稱 | 定義 | 作用 |
|---|---|---|
| 仇神 | 剋原神、反生忌神 | 間接傷害用神 |

### 仇神的作用

- 仇神生忌神，間接加強對用神的傷害。
- 仇神克原神，削弱用神的生助來源。
- 仇神與忌神同時有力時，凶象加重。

## 忌神與原神同動：貪生忘剋

若忌神與原神同時發動，或忌神動後轉向生原神，可能形成「貪生忘剋」。此時忌神原本要克用神，但因貪生原神，反而使原神力量增加，間接讓用神更穩。

```text
忌神 → 生 → 原神 → 生 → 用神
```

判斷時不可見忌神動就立即判凶，需檢查：

```text
1. 忌神是否同時能生原神？
2. 原神是否能再生用神？
3. 原神本身是否旺相、有力、未空破？
4. 是否形成通關？
```

## 判斷順序

```text
先定用神
↓
看用神旺衰
↓
用神弱，先找原神
↓
看忌神是否發動克用神
↓
看忌神是否被制、旬空、化退、貪生忘剋或通關
↓
看仇神是否生忌神或克原神
↓
綜合月建、日辰、動爻、變爻判斷
```

## If / Then 判斷

```text
If 用神休囚
Then 先找原神，看是否能生起用神

If 原神旺相發動生用神
Then 用神可得救，事情有轉機

If 原神空破或被制
Then 生助無力，事情難以扭轉

If 忌神發動克用神
Then 用神受損，事情不順

If 忌神發動但同時生原神，且原神再生用神
Then 可能形成貪生忘剋或通關，忌神不一定成凶

If 忌神發動克用神，且仇神同時克原神
Then 忌神斷源，大凶，類似斬草除根

If 原神發動，但用神本身空破休囚太重
Then 來生也未必有效，需防爛根難救

If 用神多現
Then 逐一看旺衰、動靜、世應、爻位與六神，不機械捨棄
```

## 來源補充

本檔補入《六爻知先機》第065～066頁關於斷卦基本法則與用神、元神、忌神、仇神的整理：斷卦以用神為中心，六親五行為吉凶主體；六神只作辨象；用神多現時任一用神皆是主題意念；忌神與元神同時發動時可形成貪生忘剋。

## 阿咪重點速記

- 用神 = 占問核心，是斷卦鎖鑰。
- 原神 / 元神 = 生用神者，是用神的來源。
- 忌神 = 克用神者，是直接傷害。
- 仇神 = 生忌神、克原神，是間接加害。
- 用神弱時，先看原神有沒有救。
- 忌神旺、原神被制，通常凶象加重。
- 忌神動不一定凶，需查貪生忘剋與通關。
- 用神多現，不可機械捨棄，要看各爻在卦中的功能。
