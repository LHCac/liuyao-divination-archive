# 用神判斷

## 主題摘要

用神是六爻預測的核心，是所問事情或人物在卦中的代表。用神不是單純的六親名稱，而是從六親、世應、卦中最具代表性的爻中，提取出能代表所問之事的那一個核心爻。定準用神，是斷卦吉凶的第一步。

## 用神的本質

- 用神代表所問之事、所問之人、所問之物。
- 用神先產生於問卦者心中的意念，再由卦象呈現。
- 卦象是心神與宇宙全息的呈現，所以「無事不占，心不誠不占」。
- 若只是被旁人勸說而問，自己沒有明確意念，容易出現神散，用神不明。

## 六類用神速查

| 類別 | 常見代表 |
|---|---|
| 父母類 | 父母、長輩、學業、考試、文章、車輛、房子、書籍、證件、合同 |
| 兄弟類 | 兄弟姐妹、朋友、同事、同行、競爭者、同學 |
| 子孫類 | 子女、晚輩、動物、僧道、公安、徒弟、藥、財源 |
| 妻財類 | 配偶戀人、錢財、貨物、員工、下屬、工具、飲食 |
| 官鬼類 | 丈夫、工作、事業、領導、考學名次、盜賊、疾病、官司 |
| 世應類 | 問自己看世爻，問對方看應爻，競爭比賽看世應雙方 |

## 對事不對人原則

問別人的事情時，要取「那件事」的用神，不是只取那個人的代表爻。

| 問題 | 用神 |
|---|---|
| 問兒子工作 | 官鬼，因為問的是工作 |
| 問朋友財運 | 妻財，因為問的是財 |
| 問老公考試 | 父母或官鬼，依問題重點為成績、名次或職位而定 |
| 問女兒婚姻 | 若問婚姻對象與婚事，依性別與事類取財官、世應共同看 |

## 用神兩現

同一六親出現兩次時，稱用神兩現。

### 判斷吉凶

以最有特點的爻為主用神：

- 旬空
- 月破
- 發動
- 伏藏
- 逢沖
- 逢合
- 持世
- 臨應
- 臨日月

### 判斷細節

兩個同類用神可以互相補充，形成事情的動態軌跡。

```text
一個爻看主體吉凶
另一個爻看細節、過程、應期或另一面向
```

## If / Then 判斷

```text
If 問財運
Then 妻財爻為用神，並輔看子孫財源與兄弟分財

If 問工作或求官
Then 官鬼爻為用神，並輔看父母文書與妻財業績

If 問學業考試
Then 父母爻為用神，問名次時也可輔看官鬼

If 問自身狀態
Then 世爻為核心，依問題再取對應六親

If 問他人事情
Then 取代表那件事的六親，不可只看那個人的人倫身份

If 用神兩現
Then 先取最有特點者為主用神，另一爻補充細節與應期

If 問事不誠或意念不明
Then 用神散亂，卦象容易失準
```

## 阿咪重點速記

- 用神是卦的主軸，先定用神再斷吉凶。
- 問事要「對事不對人」。
- 用神兩現，先取最有特點者；另一個補充細節。
- 世爻不一定永遠代表自己，世爻代表事件中心或問卦者在此事中的角色。
