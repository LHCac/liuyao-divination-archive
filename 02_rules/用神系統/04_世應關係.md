# 世應關係

## 主題摘要

世爻代表問事主體、自己或自己一方；應爻代表對方、他方或相對力量。世應是全卦的重要太極點，判斷自己與對方、本地與外地、主體與客體、競爭雙方時非常重要。

## 世爻的意義

世爻是六爻預測的首要觀察點。

世爻可代表：

- 問卦者本人
- 自己一方
- 事件中心人物
- 本地、本方、本體
- 問事主體的狀態

古法有「卦揖其主」之說，一卦首先顯示問事主體的狀況。

若世爻休囚、空破、被克，甚至動化回頭克，代表主體本身有危機，其他斷語需往後放。

## 應爻的意義

應爻通常代表：

- 對方
- 他方
- 相對力量
- 外部環境
- 合作方
- 競爭方
- 目的地
- 市場

在特殊情境中，應爻也可代表局外第三方或意料之外的介入者，需要依問題判斷。

## 世應關係基本法

| 世應關係 | 含義 |
|---|---|
| 世應相生 | 雙方關係較和諧，彼此有助力 |
| 世應相合 | 賓主相投，有牽連、合作、感情、牽絆之象 |
| 世應相剋 | 兩情不睦，互相壓制或不順 |
| 世應相沖 | 雙方對立，事情難成或有爭執 |
| 世克應 | 自己能掌握對方，自己較主動 |
| 應克世 | 對方壓制自己，自己壓力較大 |
| 世旺應衰 | 自己一方較強 |
| 應旺世衰 | 對方較強，自己需謹慎 |

## 以世爻為用神的情況

下列問題以世爻為主，因為事情直接牽涉問卦者自身損益：

| 問題 | 取用方向 |
|---|---|
| 占自己疾病 | 世爻為自身，兼看官鬼病因、子孫康復 |
| 問壽數 | 世爻為生命根本 |
| 問出行吉凶 | 世爻為自己是否平安、能否承受出行 |
| 諸凡損益自身者 | 先看世爻旺衰、空破、受生受克 |

## 以應爻為用神的情況

下列問題以應爻為主，因為事情重點在他方或不直接牽涉自身損益：

| 問題 | 取用方向 |
|---|---|
| 占無尊卑稱呼、未曾深交之朋友 | 應爻為對方 |
| 九流術士、一般外部人物 | 應爻為他方 |
| 仇人、競爭者 | 應爻為對立方 |
| 諸凡不直接牽涉自身損益者 | 可先取應爻，再配合六親定位 |

## 世應常見應用

| 問題類型 | 世爻 | 應爻 |
|---|---|---|
| 婚姻感情 | 自己 | 對方、愛人、對象 |
| 財運生意 | 自己、公司、本方 | 市場、客戶、競爭者、合夥人 |
| 出行遷居 | 本地、自己 | 外地、目的地 |
| 官司競爭 | 己方 | 對方 |
| 合作 | 自己一方 | 合作方 |
| 求職 | 求職者 | 公司、面試方、對方單位 |
| 測陌生人 / 未深交朋友 | 問卦者觀察角度 | 被測者本人 |

## 用神合應爻

若用神合應爻，可能代表用神被對方或第三方吸引、取得、牽制。

常見象：

- 財被他人取。
- 情被他人奪。
- 機會被對方掌握。
- 用神不向世爻，而向應爻。

## If / Then 判斷

```text
If 世爻旺相安靜
Then 問事主體穩固，可繼續分析其他用神

If 世爻休囚空破
Then 主體有危，需先處理自身問題

If 問自己疾病、壽數、出行、自身損益
Then 以世爻為用神或首要觀察點

If 問未深交朋友、外部人物、仇人、競爭者
Then 可取應爻為用神或首要觀察點

If 世應相生相合
Then 賓主相投，雙方較容易配合

If 世應相剋相沖
Then 兩情不睦，雙方有阻隔或衝突

If 用神合應爻
Then 可能有第三方介入，或用神被對方牽走

If 問官司，世旺應衰
Then 己方較有力

If 問官司，應旺世衰
Then 對方較有力，己方壓力大
```

## 來源補充

本檔補入《六爻知先機》第064頁關於世應用神的整理：世爻代表自己，應爻代表他人；世應相生相合為賓主相投，世應相剋相沖為兩情不睦。

## 阿咪重點速記

- 世爻先看，因為它代表問事主體狀態。
- 應爻是對方、他方、目的地、市場或競爭者。
- 問自身損益，多看世爻；問他人或未深交對象，多看應爻。
- 世應相生相合較和，相剋相沖較爭。
- 用神合應，常要防機會、財、情被對方牽走。
