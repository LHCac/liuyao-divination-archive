# 世應關係

## 主題摘要

世爻代表問事主體、自己或自己一方；應爻代表對方、他方或相對力量。世應是全卦的重要太極點，判斷自己與對方、本地與外地、主體與客體、競爭雙方時非常重要。

## 世爻的意義

世爻是六爻預測的首要觀察點。

世爻可代表：

- 問卦者本人
- 自己一方
- 事件中心人物
- 本地、本方、本體
- 問事主體的狀態

古法有「卦揖其主」之說，一卦首先顯示問事主體的狀況。

若世爻休囚、空破、被克，甚至動化回頭克，代表主體本身有危機，其他斷語需往後放。

## 應爻的意義

應爻通常代表：

- 對方
- 他方
- 相對力量
- 外部環境
- 合作方
- 競爭方
- 目的地
- 市場

在特殊情境中，應爻也可代表局外第三方或意料之外的介入者，需要依問題判斷。

## 世應常見應用

| 問題類型 | 世爻 | 應爻 |
|---|---|---|
| 婚姻感情 | 自己 | 對方、愛人、對象 |
| 財運生意 | 自己、公司、本方 | 市場、客戶、競爭者、合夥人 |
| 出行遷居 | 本地、自己 | 外地、目的地 |
| 官司競爭 | 己方 | 對方 |
| 合作 | 自己一方 | 合作方 |
| 求職 | 求職者 | 公司、面試方、對方單位 |

## 世應關係判斷

| 世應關係 | 含義 |
|---|---|
| 世應相生 | 雙方關係較和諧，彼此有助力 |
| 世應相沖 | 雙方對立，事情難成或有爭執 |
| 世應相合 | 有牽連、合作、感情、牽絆之象 |
| 世克應 | 自己能掌握對方，自己較主動 |
| 應克世 | 對方壓制自己，自己壓力較大 |
| 世旺應衰 | 自己一方較強 |
| 應旺世衰 | 對方較強，自己需謹慎 |

## 用神合應爻

若用神合應爻，可能代表用神被對方或第三方吸引、取得、牽制。

常見象：

- 財被他人取。
- 情被他人奪。
- 機會被對方掌握。
- 用神不向世爻，而向應爻。

## If / Then 判斷

```text
If 世爻旺相安靜
Then 問事主體穩固，可繼續分析其他用神

If 世爻休囚空破
Then 主體有危，需先處理自身問題

If 世應相生
Then 雙方關係較和諧，事情有配合空間

If 世應相沖
Then 雙方對立，事情難成或有爭執

If 用神合應爻
Then 可能有第三方介入，或用神被對方牽走

If 問官司，世旺應衰
Then 己方較有力

If 問官司，應旺世衰
Then 對方較有力，己方壓力大
```

## 阿咪重點速記

- 世爻先看，因為它代表問事主體狀態。
- 應爻是對方、他方、目的地、市場或競爭者。
- 世應相生較和，相沖較爭。
- 用神合應，常要防機會、財、情被對方牽走。
