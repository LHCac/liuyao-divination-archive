# 旺衰判斷

## 主題摘要

旺衰是六爻判卦的根本基礎。爻得月建、日辰、動爻生扶，或生多克少，稱為旺相；爻被月建、日辰、動爻克制，或生少克多，稱為休囚。用神旺衰決定事情成敗的大方向，是一切斷語的出發點。

## 旺衰定義

| 狀態 | 條件 | 含義 |
|---|---|---|
| 旺相 | 得日月生扶，或動爻相生，生多克少 | 有力，事情有希望 |
| 休囚 | 被日月克制，或動爻相克，生少克多 | 無力，事情有阻礙 |

## 判斷層次

```text
月建
↓
日辰
↓
動爻
↓
十二長生
↓
空亡、月破、墓絕、合沖
```

### 1. 月建

月令是最大的能量場，同五行者旺，被月令克制者衰。

### 2. 日辰

日辰為次要但非常直接的影響，可生、克、沖、合、扶起某爻。

### 3. 動爻

卦中動爻代表事情正在發動的力量，動爻生用神為助，動爻克用神為阻。

### 4. 十二長生

十二長生用來輔助判斷爻的本質能量，如長生、帝旺為有氣，死、絕、墓為弱或藏。

## 應用原則

- 用神旺相，即使遇到動爻克泄，仍可能成事，只是過程有阻。
- 用神休囚，即使遇到動爻生助，也未必能成大事。
- 用神休囚空破，事情基本困難，需先看原神是否能救。
- 原神、用神喜旺相；忌神、仇神不宜旺相發動。

## 斷卦心態

斷卦者必須保持平靜客觀，不可因希望事情成功就處處看成有希望，也不可因擔心失敗就處處看成問題。

## If / Then 判斷

```text
If 用神旺相且遇克
Then 仍有成事可能，阻礙只是過程

If 用神休囚且遇生
Then 事倍功半，難以大成

If 用神休囚空破
Then 事情基本難成，首要看原神能否補救

If 原神旺相發動生用神
Then 用神雖弱仍有轉機

If 忌神旺相發動克用神
Then 阻力大，事情不順

If 用神得月建、日辰、動爻多重生扶
Then 力量強，但也需防過旺反凶
```

## 阿咪重點速記

- 旺衰是六爻判斷的根本。
- 月建最重要，其次日辰，再看動爻。
- 用神旺，受阻仍可成；用神弱，受生也未必成。
- 旺衰判斷要客觀，不要被主觀期待影響。
