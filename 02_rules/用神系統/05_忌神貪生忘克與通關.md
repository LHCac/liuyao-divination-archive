# 忌神貪生忘克與通關

## 主題摘要

忌神發動不一定直接判凶。若忌神同時可以克用神，又可以生其他能生用神的爻，可能出現「貪生忘克」；若忌神與用神之間有中間爻承接力量，則形成「通關」。兩者都可能把忌神的傷害轉化為生助用神的力量。

## 一、貪生忘克

### 定義

當一個動爻同時具備：

```text
可以克制某爻，例如用神
又可以生扶另一爻，例如原神
```

此時動爻可能優先選擇「生」，而暫時放棄「克」，稱為貪生忘克。

## 二、貪生忘克公式

```text
動爻對用神：忌神，可克用神
動爻對原神：可生原神
↓
忌神貪生忘克
↓
忌神轉向生原神，不再克用神
↓
忌神力量流失，用神得保
```

## 三、通關

### 定義

忌神與用神之間，如果有一個中間爻具備：

```text
能被忌神生
又能生用神
```

此中間爻就是通關爻，能把忌神力量轉化成生助用神的力量。

## 四、通關圖解

```text
忌神 → 生 → 通關爻 / 原神 → 生 → 用神
```

忌神原本要克用神，但因中間爻承接，力量被轉向，形成凶轉吉。

## 五、實戰模型

```text
忌神巳火父母動 → 原本克用神酉金
但兄弟丑土成為中間爻 / 原神
巳火 → 生丑土 → 生酉金
↓
忌神力量被轉化，用神得保
```

## 六、貪生忘克與通關對照

| 項目 | 貪生忘克 | 通關 |
|---|---|---|
| 主角 | 忌神動爻自身 | 忌神、通關爻、用神三者 |
| 機制 | 忌神選擇生原神，放棄克用神 | 中間爻吸收忌神力量，再生用神 |
| 條件 | 忌神同時可克用神、又可生原神 | 中間爻能被忌神生，又能生用神 |
| 結果 | 忌神力量流失，用神得保 | 忌神力量轉化為生用神 |
| 性質 | 凶轉吉可能 | 凶轉吉可能 |

## 七、檢查清單

看到忌神發動時，先不要急著判凶：

```text
1. 忌神除了克用神，還有沒有可生的爻？
2. 該可生之爻是否為原神或能轉生用神？
3. 忌神與用神之間是否有通關爻？
4. 通關爻是否有力、暗動、旺相或得生？
5. 若都沒有，再按忌神克用神判凶
```

## If / Then 判斷

```text
If 忌神動爻只克用神，無其他可生通道
Then 忌神克用，偏凶

If 忌神動爻可克用神，也可生原神
Then 考慮貪生忘克，忌神未必克用

If 忌神生通關爻，通關爻再生用神
Then 忌神力量被轉化，有凶轉吉機會

If 通關爻旺相或暗動
Then 通關力量較強，轉化更有效

If 通關爻空破休囚
Then 通關無力，忌神仍可能傷用神
```

## 阿咪重點速記

- 忌神動，不要馬上判凶。
- 先看忌神是否貪生忘克。
- 再看中間是否有通關爻。
- 忌神 → 通關 → 用神，是凶轉吉的重要結構。
