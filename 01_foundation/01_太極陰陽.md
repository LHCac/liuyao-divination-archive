# 太極陰陽

## 主題摘要

太極是易經與六爻的根本概念，代表萬事萬物尚未成形之前的氣場蓄積。陰陽由太極分化而出，再進一步生四象、八卦與萬象。六爻中的動爻、變爻、旺衰反斷，都離不開陰陽轉化的邏輯。

## 核心概念

### 太極

- 太極不是「什麼都沒有」，而是有形之前的準備狀態。
- 天地之間物物一太極，每一件事、每一個問題都可以形成自己的太極場。
- 一卦搖出後，就是把所問之事放進一個卦象太極場中觀察。

### 陰陽

- 陰陽是太極之氣分化後形成的兩種狀態。
- 陽中有陰，陰中有陽。
- 陽極生陰，陰極生陽，這就是物極必反。
- 六爻中的動變關係，本質上就是陰陽互變。

## 六爻中的應用

| 數字 | 名稱 | 陰陽 | 是否動爻 | 是否產生變爻 |
|---|---|---|---|---|
| 9 | 老陽 | 陽 | 是 | 變陰爻 |
| 6 | 老陰 | 陰 | 是 | 變陽爻 |
| 7 | 少陽 | 陽 | 否 | 無變爻 |
| 8 | 少陰 | 陰 | 否 | 無變爻 |

### 暗動

暗動通常指某爻被日辰沖動，因此出現動象。但暗動不是老陰、老陽，所以沒有變爻，力量也通常比正式動爻弱。

## 反斷觀念

陰陽有物極必反的特性，所以在斷卦時不能只看「旺就是好、衰就是壞」。

- 用神太旺，到處逢生而無制，可能反斷為凶。
- 用神衰弱至極，卦中多克而無生，有時可反斷為吉。

## If / Then 判斷

```text
If 某爻為老陽 9
Then 該爻為動爻，變為陰爻

If 某爻為老陰 6
Then 該爻為動爻，變為陽爻

If 某爻為少陽 7 或少陰 8
Then 該爻為靜爻，不產生變爻

If 某爻被日辰沖動
Then 可視為暗動，但無變爻

If 用神過旺且沒有制衡
Then 可考慮物極必反，反斷為凶

If 用神過衰且衰到極點
Then 可考慮陰極陽生，反斷為吉
```

## 阿咪重點速記

- 太極 = 事情形成前的氣場，不是空無。
- 一卦就是一個太極場，主斷所問之事。
- 動爻必有變爻，暗動沒有變爻。
- 老陽變陰，老陰變陽。
- 過旺、過衰都可能進入反斷邏輯。
