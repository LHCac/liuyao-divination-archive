## 斷卦時的使用方式

占卦後，先看：

- 問題屬性：感情、財運、事業、健康、人際等。
- 卦宮五行：決定六親配置的中心。
- 月建、日辰：判斷各爻旺衰。
- 動爻與變爻：判斷事情變化方向。
- 五行生剋：判斷用神是否受生、受剋、入墓、逢沖。
