# 木

## 主題摘要

木主舒展、延伸、生長、遺傳與交際，是五行中最能表現「向外發展」的一行。六爻取象時，木不只是樹木，也代表擴張、傳播、人際影響、肝膽筋骨、神經感知與生長力量。

## 木的核心氣性

| 特性 | 說明 |
|---|---|
| 舒展 | 像春天萬物伸展，代表放開、發展、活動 |
| 延伸 | 向外擴張，影響範圍變大 |
| 生長 | 事情開始成長，有發展空間 |
| 遺傳 | 子代繼承親代特徵，與傳承、基因、子嗣相關 |
| 交際 | 萬物互相影響，代表人際、傳播、關係延伸 |

## 木的六爻取象

### 人事

- 人際關係
- 朋友往來
- 傳播、擴散
- 成長、學習
- 影響力擴大
- 家族、遺傳、傳承

### 健康

- 肝膽
- 筋
- 神經系統
- 痛、癢、抽筋、伸展不順

### 事業與環境

- 教育
- 文化
- 設計
- 農業
- 成長型產業
- 需要擴張與人脈的事業

## 木與五行關係

```text
水生木
木生火
金克木
木克土
```

### 判斷重點

- 水生木：木得滋養，事情能繼續生長。
- 木生火：發展後走向公開、曝光、釋放。
- 金克木：發展受限制、被規範、被切斷。
- 木克土：木可突破穩定結構，也可能消耗基礎。

## 木旺與木弱

| 狀態 | 可能取象 |
|---|---|
| 木旺 | 發展快、人際廣、擴張強、影響力變大 |
| 木過旺 | 過度蔓延、分散、難以收束、競爭者變多 |
| 木弱 | 發展不足、缺乏人脈、肝膽筋神經較弱 |
| 木受金克 | 被制度壓制、被阻斷、成長受傷 |

## If / Then 判斷

```text
If 某爻臨木且旺相
Then 此爻有延伸、擴張、發展、傳播之力

If 問人際，世爻臨木且不受剋
Then 人緣、人脈、交際能力較佳

If 兄弟爻臨木旺
Then 朋友多、競爭者多，也可能分財

If 木爻受金克
Then 發展受限制，健康上注意肝膽筋神經

If 木爻發動生火
Then 事情可能從發展走向公開、曝光或快速釋放
```

## 阿咪重點速記

- 木的關鍵字：舒展、延伸、生長、遺傳、交際。
- 問人際：木旺常代表關係延展、人脈增加。
- 問財：兄弟木旺可能代表競爭與分財。
- 問健康：肝膽、筋、神經、痛癢。
