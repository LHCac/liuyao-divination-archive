# 水

## 主題摘要

水為五行之首，主潤下、收藏、智慧、暗藏與寒性。六爻取象時，水不只是具體的水，也代表往下、流動、藏匿、內心、暗處、財源流動、智慧與泌尿腎臟系統。

## 水的核心氣性

| 特性 | 說明 |
|---|---|
| 潤下 | 水往低處流，代表下行、滋潤、基礎源頭 |
| 收藏 | 北方主藏，水有藏匿、收納、隱蔽之性 |
| 暗 | 看不見的地方、內心想法、曖昧、偷盜、暗中操作 |
| 智 | 智慧藏於內，不一定外露 |
| 寒 | 水氣為寒，常與冬天、寒冷、受寒有關 |

## 水的六爻取象

### 人事

- 內心想法
- 隱情、秘密
- 曖昧、不明朗關係
- 偷盜、暗中操作
- 智慧、策略、思考

### 財務

- 流動財
- 現金流
- 財源活絡
- 但水太旺時，也可能代表財來財去、難以留住

### 健康

- 腎臟
- 泌尿系統
- 生殖系統
- 血液循環
- 受寒、寒症

## 水與五行關係

```text
金生水
水生木
土克水
水克火
```

### 判斷重點

- 水得金生，代表來源、資源、補給。
- 水生木，代表滋養生長、使事情開始發展。
- 土克水，代表阻擋、壓制、蓄水、入墓。
- 水克火，代表冷卻、壓制熱度、打斷快速外放。

## 水過旺與水過弱

| 狀態 | 可能取象 |
|---|---|
| 水旺 | 流動強、智慧強、隱藏性強、事情暗中發展 |
| 水過旺 | 氾濫、失控、拖延、情緒深陷、財難聚 |
| 水弱 | 缺乏滋潤、思路不足、資源不足、腎水不足 |
| 水入墓 | 藏而不出、被壓制、財入庫、暫時不顯 |

## If / Then 判斷

```text
If 妻財爻臨水且旺相
Then 財源活絡、流動性強，但也要看是否能留住

If 官鬼臨水又臨玄武
Then 有陰暗、偷摸、不正當或隱情之象

If 水爻受剋或休囚
Then 健康上需注意腎臟、泌尿、血液、受寒

If 水爻過旺
Then 事情可能氾濫失控，或暗中問題擴散

If 水爻入辰墓
Then 藏而不出，需等沖開或日月引動
```

## 阿咪重點速記

- 水的關鍵字：藏、暗、智、寒、潤下、流動。
- 問財：水旺可代表現金流，但過旺則財難聚。
- 問感情：水常代表曖昧、隱情、內心不明。
- 問健康：腎臟、泌尿、血液、寒症。
