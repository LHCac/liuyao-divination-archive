# 階段性與地支五行

## 主題摘要

五行運化具有階段性，萬物不是永遠直線前進，而是在生、長、收、藏、再生之間循環。十二地支中的四正代表純氣，四隅代表季節過渡與氣的轉換，尤其辰戌丑未四個土地支具有墓庫與緩衝功能。

## 一、萬物發展的階段性

萬物發展如同四季交替：

```text
生 → 長 → 收 → 藏 → 再生
```

例子：植物結果

- 果肉為土，主承載、收藏、遲緩。
- 果實內有芽籽，芽籽為木，主延伸、傳承、新生命。
- 因此果實是「土 + 木」共存，代表舊階段收藏新階段的開始。

## 二、四正之位

四正之位氣最純：

| 地支 | 方位 | 季節 | 五行 |
|---|---|---|---|
| 子 | 正北 | 冬 | 水 |
| 卯 | 正東 | 春 | 木 |
| 午 | 正南 | 夏 | 火 |
| 酉 | 正西 | 秋 | 金 |

四正只含一個主要地支，氣較純正，方位明確。

## 三、四隅之位

四隅是季節過渡地帶，氣較雜，具有銜接與轉換作用。

| 四隅 | 地支 | 過渡 | 功能 |
|---|---|---|---|
| 東北 | 丑、寅 | 冬 → 春 | 丑土藏水、金入墓，寅木開始生發 |
| 東南 | 辰、巳 | 春 → 夏 | 辰為水庫，使水入庫不阻火，巳火初起 |
| 西南 | 未、申 | 夏 → 秋 | 未為木庫，釜底抽薪，申金開始收斂 |
| 西北 | 戌、亥 | 秋 → 冬 | 戌為火庫，使火入庫不阻水，亥水開始流動 |

## 四、四庫速查

| 地支 | 墓庫 | 作用 |
|---|---|---|
| 丑 | 金庫 | 金入墓，不阻木，利冬春轉換 |
| 辰 | 水庫 | 水入庫，不阻火，利春夏轉換 |
| 未 | 木庫 | 木入墓，火失源頭，利夏秋轉換 |
| 戌 | 火庫 | 火入庫，不阻水，利秋冬轉換 |

## 五、艮卦與子嗣傳承

艮卦納丑土與寅木：

- 丑土為濕土，內藏水，像母體與收藏之地。
- 寅木為新生命的生發。
- 丑土生寅木，形成子嗣、傳承、新事物的象。

問子嗣、墓地、祖先、傳承時，艮卦與丑寅關係需要特別留意。

## 六、墓地風水取象

好墓地需要：

- 土壤：土主收藏。
- 草木：木主延伸、庇佑子孫。

若墓地寸草不生，代表木氣不足，子孫傳承與延伸之力弱。

## If / Then 判斷

```text
If 月建為丑月
Then 丑為濕土、金庫，金入墓，木開始蓄力

If 月建為辰月
Then 辰為水庫，水入庫不阻火，木旺生火之路較通

If 月建為未月
Then 未為木庫，木入墓，金氣開始醞釀

If 月建為戌月
Then 戌為火庫，火入墓，水氣開始蓄力

If 問子嗣且艮卦出現
Then 可取丑土母體、寅木新生命之象

If 問墓地風水
Then 看是否有土壤收藏與草木延伸，兩者關係子孫傳承
```

## 阿咪重點速記

- 四正：子午卯酉，氣純。
- 四隅：丑寅、辰巳、未申、戌亥，氣雜而過渡。
- 丑=金庫，辰=水庫，未=木庫，戌=火庫。
- 艮卦與丑寅關係，常和子嗣、墳墓、傳承有關。
