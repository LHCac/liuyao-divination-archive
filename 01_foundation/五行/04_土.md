# 土

## 主題摘要

土是五行的基礎與核心，主不動、緩慢、收藏、承載與生育。六爻取象時，土不只是泥土，也代表穩定、基礎、房地產、倉庫、財庫、墓庫、消化系統與事情的承載力。

## 土的核心氣性

| 特性 | 說明 |
|---|---|
| 不動 | 穩定、固定、停滯、守成 |
| 緩慢 | 使事情速度放慢，不至於過激 |
| 收藏 | 萬物歸土，藏納、庫藏、墓庫 |
| 承載 | 萬物依附於土，代表基礎與平台 |
| 生育 | 土能孕育萬物，代表資源承接與轉化 |

## 土的六爻取象

### 人事

- 穩定
- 保守
- 遲緩
- 停滯
- 基礎條件
- 承擔責任
- 倉庫、收藏、保管

### 財務

- 財庫
- 房地產
- 土地
- 不動產
- 積蓄、存款
- 財聚不散

### 健康

- 脾胃
- 消化系統
- 肌肉
- 代謝吸收
- 身體基礎狀態

## 四庫

辰、戌、丑、未都是土，但各自有墓庫含義：

| 地支 | 墓庫 |
|---|---|
| 辰 | 水庫 |
| 戌 | 火庫 |
| 丑 | 金庫 |
| 未 | 木庫 |

## 土與五行關係

```text
火生土
土生金
木克土
土克水
```

### 判斷重點

- 火生土：外放後沉澱，事情開始穩定。
- 土生金：基礎累積後出現收斂、制度、成果。
- 木克土：發展力量突破基礎，也可能使穩定被破壞。
- 土克水：阻擋水流，代表壓制、蓄水、入墓。

## 土旺與土弱

| 狀態 | 可能取象 |
|---|---|
| 土旺 | 穩定、能承載、財庫強、基礎穩 |
| 土過旺 | 遲滯、僵化、阻塞、難以流通 |
| 土弱 | 基礎不穩、難承擔、反覆、消化吸收差 |
| 土受木克 | 穩定被突破，基礎受損或壓力來自擴張 |

## If / Then 判斷

```text
If 用神臨辰戌丑未
Then 需先檢查是否入墓、入庫或財庫

If 用神入墓
Then 事情被收藏、壓住、暫時不顯，逢沖可開墓

If 問財，世爻臨土且土為財庫
Then 有財聚、存款、不動產、多路來財之象

If 土爻休囚無力
Then 事情基礎不穩，容易反覆

If 土爻過旺
Then 事情可能停滯、遲緩、卡住，需要流通之氣
```

## 阿咪重點速記

- 土的關鍵字：基礎、緩慢、收藏、穩定、財庫、墓庫。
- 問財：土常與財庫、不動產、存款有關。
- 問健康：脾胃、消化、肌肉、吸收。
- 辰戌丑未要特別看墓庫與沖開。
