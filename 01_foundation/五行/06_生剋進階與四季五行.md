# 生剋進階與四季五行

## 主題摘要

本篇整理五行進階概念：生剋的多樣性、五行俱全、氣的本質、四季五行旺衰。這些概念用來避免斷卦時只用單線條的「生就是吉、克就是凶」來判斷，而是要結合喜忌、月令、合化、沖刑、旺衰與整體平衡。

## 一、生剋的多樣性

五行生剋不是單純一對一的直線關係，實際斷卦時還會出現：

- 三合局
- 六合
- 三刑
- 六沖
- 合化
- 貪生忘克
- 合局生用
- 忌神受制反吉
- 用神過旺反凶

### 貪生忘克

某爻若被強力生扶，可能顧著受生，而暫時不去克其他爻。這種情況需看動爻、月日、合局與實際問題。

### 太過需制

```text
用神過旺，不一定吉。
忌神出現制約過旺用神，有時反而是平衡。
```

## 二、五行俱全

萬物皆含五行，五行俱全不是數量平均，而是「以某一行為主，其他四行輔助存在」。

例：一棵樹

| 五行 | 在樹中的表現 |
|---|---|
| 木 | 樹本身、生長延伸 |
| 金 | 堅硬、支撐、收斂 |
| 水 | 滋潤、汁液、生命流動 |
| 火 | 花朵、向上、釋放、開放 |
| 土 | 果實、承載、收藏 |

## 三、氣為本質

五行是氣的概念，不只是具體物質。

```text
先有氣，後有物。
陽為氣，陰為物。
氣先至，物後現。
```

斷卦時應該看「氣的特性」，不是只看字面物質。

例如火不只代表火焰，也代表：

- 光明
- 熱情
- 溫度
- 紅色
- 尖形
- 曝光
- 快速變化

## 四、四季五行

| 季節 | 旺氣 | 主要含義 |
|---|---|---|
| 春 | 木旺 | 舒展、生髮、筋、成長 |
| 夏 | 火旺 | 炎上、速度、熱度、顯現 |
| 秋 | 金旺 | 收斂、結果、規範、肅殺 |
| 冬 | 水旺 | 藏匿、積蓄、寒冷、智慧 |
| 四季末月 | 土旺 | 緩衝、承載、過渡、墓庫 |

## If / Then 判斷

```text
If 某爻五行沒有出現在卦中
Then 不代表該氣完全不存在，只是顯現較弱，仍需看月日辰補充

If 用神過旺且無制
Then 可考慮太過反凶，需要制約或洩氣

If 卦中出現三合局
Then 合局成後，局中五行力量大增，需重新評估旺衰

If 問某事與季節關係
Then 先看月令所旺之行，再看用神是否得令

If 五行極端失衡
Then 事情難以長久穩定
```

## 阿咪重點速記

- 五行是氣，不是物。
- 生剋要看喜忌與平衡，不可死判。
- 用神過旺可能反凶，忌神受制可能反吉。
- 月令是判斷旺衰的根本之一。
