# 十二長生

## 主題摘要

十二長生是描述五行之氣在十二地支中盛衰循環的系統，用來評估某一爻在月建、日辰或特定地支下的本質能量狀態。六爻使用十二長生時，主要看五行本身的長生，不必分陰陽干支順逆。

## 十二長生階段

| 階段 | 說明 | 能量狀態 |
|---|---|---|
| 長生 | 剛剛出生，有活力 | 旺 |
| 沐浴 | 洗禮階段，略不穩 | 中 |
| 冠帶 | 成長茁壯 | 旺 |
| 臨官 | 正值壯年，力量強 | 旺 |
| 帝旺 | 巔峰狀態 | 最旺 |
| 衰 | 開始走下坡 | 中 |
| 病 | 體弱 | 弱 |
| 死 | 生命終止 | 極弱 |
| 墓 | 入庫封存 | 很弱、被收藏 |
| 絕 | 完全消失 | 最弱 |
| 胎 | 開始孕育 | 微弱 |
| 養 | 孕育成形 | 漸強 |

## 五行長生、帝旺、墓地速查

| 五行 | 長生 | 帝旺 | 墓 |
|---|---|---|---|
| 木 | 亥 | 卯 | 未 |
| 火 | 寅 | 午 | 戌 |
| 土 | 申 | 午 | 辰、戌 |
| 金 | 巳 | 酉 | 丑 |
| 水 | 申 | 子 | 辰 |

> 註：部分傳統系統中土與火同論，土長生於寅、帝旺於午、墓於戌；六爻資料中亦有「水土長生於申」的用法。實戰時需依所採系統與卦例驗證一致性。

## 氣與象的關係

- 氣：內在生命力、本質旺衰。
- 象：外在表現形式。

兩者有時一致，有時不同步。

例如：

```text
外表旺，但氣入死絕 → 外強中乾
外表弱，但氣有長生 → 內在仍有生機
```

六爻判斷應以氣為根本，不可只被外象迷惑。

## 六爻使用原則

- 只看五行本身的長生，不分陰陽順逆。
- 十二長生需與月令旺衰、日辰生剋、動爻變爻共同判斷。
- 不能單憑長生、帝旺就判吉，也不能單憑死絕就判凶。
- 若總體旺衰出問題，即使臨帝旺，也可能外強中乾。

## 其他維度的類比

| 維度 | 十二長生類比 |
|---|---|
| 爻位 | 初爻像童年，五爻像鼎盛，六爻像退出 |
| 六神 | 青龍像生長，朱雀像發榮，勾陳像減緩，騰蛇像萎縮，白虎像病死，玄武像消失 |
| 六親 | 兄弟像我之出現，子孫像成長，妻財像成家，官鬼像衰弱，父母像成為祖先 |

## If / Then 判斷

```text
If 爻的五行處於長生、冠帶、臨官、帝旺
Then 本質有力，但仍需看月日與用忌

If 爻的五行處於衰、病、死、絕
Then 本質較弱，需外力生扶

If 爻的五行入墓
Then 被庫收藏，需看日辰是否沖墓、動爻是否引出

If 爻外表旺相但長生狀態為死絕
Then 可能外強中乾，不可輕信

If 十二長生與月令旺衰矛盾
Then 需綜合日辰、動變、用忌，不可單項斷卦
```

## 阿咪重點速記

- 十二長生是精細旺衰工具，不是唯一標準。
- 六爻看五行長生，不分陰陽順逆。
- 長生、帝旺代表有氣；死、絕、墓代表弱或藏。
- 氣比象重要，外表強不一定真的強。
