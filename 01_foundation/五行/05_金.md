# 金

## 主題摘要

金主收斂、肅殺、聚集與結果。六爻取象時，金不只是金屬，也代表規範、法律、制度、停止、收束、成果、傷害、肺部、骨骼與皮膚。

## 金的核心氣性

| 特性 | 說明 |
|---|---|
| 收斂 | 使外放之氣收束，事情進入結果或停止 |
| 肅殺 | 秋天萬物收斂，花葉脫落，生長停止 |
| 聚集 | 各自收斂、聚斂成形，不易重新融合 |
| 規範 | 法律、制度、標準、約束力 |
| 結果 | 生長後的收成、定案、成果 |

## 金的六爻取象

### 人事

- 規矩、制度
- 法律、契約
- 主管壓力
- 收斂、結束
- 傷害、刀兵、手術
- 競爭中的強制力

### 財務與事業

- 金融
- 法律
- 機械
- 金屬
- 制度管理
- 收穫結果

### 健康

- 肺
- 骨骼
- 皮膚
- 呼吸系統
- 外傷、刀傷、血光，尤其火金相剋時

## 金與五行關係

```text
土生金
金生水
火克金
金克木
```

### 判斷重點

- 土生金：基礎累積後形成成果、制度、收束。
- 金生水：收斂後進入收藏與深層積累。
- 火克金：制度被破壞，或有傷害、血光、急性衝突。
- 金克木：限制發展、修剪枝葉、切斷延伸。

## 金旺與金弱

| 狀態 | 可能取象 |
|---|---|
| 金旺 | 有規範、有結果、能收束、執行力強 |
| 金過旺 | 太冷硬、壓制成長、傷害性強、難融合 |
| 金弱 | 缺乏規範、難收尾、肺皮骨較弱 |
| 金受火克 | 傷害、血光、衝突、制度被破壞 |

## If / Then 判斷

```text
If 金爻為用神且得月令生扶
Then 所問之事有收穫、結果、定案之象

If 金爻受火剋
Then 健康上注意肺、骨、皮膚，並留意外傷血光

If 官鬼臨金且剋世
Then 可能有主管、法規、競爭者或制度壓力

If 金爻發動化水
Then 事情由收斂轉為收藏、積累或暗藏

If 秋月申酉月金旺
Then 金爻力量強，木爻受制較重
```

## 阿咪重點速記

- 金的關鍵字：收斂、肅殺、聚集、制度、結果。
- 問健康：肺、骨骼、皮膚、呼吸系統。
- 問官司或工作：金可代表法律、規範、主管壓力。
- 火金相剋時，要留意血光、傷害、衝突。
