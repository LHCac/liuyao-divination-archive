# 五行總論

## 主題摘要

五行是古人對天地萬物運化規律的歸納。金、木、水、火、土並不只是五種物質，而是五種「氣的特性」。六爻斷卦時，五行用來判斷爻與爻之間的生剋、旺衰、轉化與取象。

## 五行不是物質，而是氣

初學者容易把五行理解成：

- 金 = 金屬
- 木 = 木頭
- 水 = 水
- 火 = 火焰
- 土 = 泥土

但在六爻中，更重要的是理解它們背後的氣性：

| 五行 | 氣性核心 | 基本方向 |
|---|---|---|
| 水 | 潤下、收藏、暗、智 | 往下、藏匿、流動、智慧 |
| 木 | 舒展、延伸、遺傳、交際 | 生長、擴展、影響他人 |
| 火 | 炎上、速度、光明、釋放 | 快速、公開、發散、顯現 |
| 土 | 不動、緩慢、收藏、基礎 | 穩定、緩衝、承載、孕育 |
| 金 | 收斂、肅殺、聚集、結果 | 收束、規範、停止、結果 |

## 五行相生

```text
水生木
木生火
火生土
土生金
金生水
```

### 斷卦理解

- 生我者：幫助我、滋養我。
- 我生者：我付出、我洩氣、我產生的結果。
- 相生不一定全吉，要看誰是用神、忌神。

## 五行相剋

```text
水克火
火克金
金克木
木克土
土克水
```

### 斷卦理解

- 克我者：壓力、管束、阻礙、疾病、官非。
- 我克者：我能掌握、支配、取得的對象。
- 相剋不一定全凶，有時忌神被克反而有利。

## 五行與人體取象

| 五行 | 人體方向 |
|---|---|
| 水 | 腎臟、泌尿、生殖、血液、受寒、智力 |
| 木 | 肝膽、筋、神經、痛癢、舒展能力 |
| 火 | 心臟、血液、眼睛、發燒、炎症 |
| 土 | 脾胃、肌肉、消化、吸收、身體基礎 |
| 金 | 肺、骨骼、皮膚、呼吸、收斂能力 |

## 五行與職業、場地取象

| 五行 | 職業或場地方向 |
|---|---|
| 水 | 物流、水利、漁業、航海、流動性行業 |
| 木 | 農業、教育、文化、設計、出版、成長型產業 |
| 火 | 電力、光熱、媒體、表演、網路曝光、炎熱行業 |
| 土 | 房地產、土地、建築、倉儲、基礎產業 |
| 金 | 金融、法律、機械、金屬、規範制度 |

## If / Then 判斷

```text
If 某爻五行旺相，且得月建日辰生扶
Then 此爻力量強，生剋作用明顯

If 某爻五行休囚無力
Then 此爻難以發揮作用，需看是否有動爻或日月救助

If 只把五行看成具體物質
Then 斷卦容易狹隘，需回到氣性取象

If 五行能形成平衡
Then 事情較能長久穩定

If 某一行過旺或過衰
Then 需注意失衡、反斷或偏象
```

## 阿咪重點速記

- 五行是「五種氣」，不是五種物體。
- 相生相剋要看用神與忌神，不可直接判吉凶。
- 土是五行的基礎與容器，辰戌丑未常與墓庫有關。
- 斷卦取象要從五行氣性延伸，不要被字面限制。
