# 火

## 主題摘要

火主炎上、速度、光明、釋放與顯現。六爻取象時，火不只是火焰，也代表快速變化、公開曝光、情緒激烈、血液心臟、眼睛、發燒炎症，以及事情從隱藏走向明顯。

## 火的核心氣性

| 特性 | 說明 |
|---|---|
| 炎上 | 向上、尖形、紅色、熱度、外放 |
| 速度 | 變化快、反應快、傷害也可能來得快 |
| 光明 | 使事情被看見，代表公開、曝光、清楚 |
| 釋放 | 能量向外發散，難以被束縛 |
| 動力 | 使事物快速推進、變化、爆發 |

## 火的六爻取象

### 人事

- 公開、曝光
- 名聲、光彩
- 情緒激動
- 快速推進
- 衝動、暴躁
- 明顯化、被看見

### 健康

- 心臟
- 血液
- 眼睛
- 發燒
- 炎症
- 血光之災，尤其火金相剋時

### 事業與環境

- 電力、光熱、能源
- 媒體、表演、曝光產業
- 網路流量、影像、照明
- 高溫、速度、競爭激烈的環境

## 火與五行關係

```text
木生火
火生土
水克火
火克金
```

### 判斷重點

- 木生火：發展到一定程度後開始釋放、公開、爆發。
- 火生土：快速釋放後逐漸緩和，事情開始沉澱。
- 水克火：熱度被壓制、速度被冷卻。
- 火克金：容易有傷害、衝突、血光、制度被破壞。

## 火旺與火弱

| 狀態 | 可能取象 |
|---|---|
| 火旺 | 速度快、曝光強、熱度高、事情明顯 |
| 火過旺 | 衝動、炎症、爆衝、失控、血光風險 |
| 火弱 | 缺乏動力、曝光不足、心力不足、溫度不夠 |
| 火受水克 | 熱情下降、計畫被冷卻、進度受壓 |

## If / Then 判斷

```text
If 官鬼爻臨火且旺相
Then 工作壓力、職位、名聲或主管力量明顯

If 火爻受水克
Then 健康上注意心臟、血液、眼睛、發燒炎症

If 火爻與金爻同時發動相剋
Then 需留意血光、受傷、開刀、衝突之象

If 火爻化土
Then 事情由快速釋放轉向緩和沉澱

If 火為忌神且旺相
Then 傷害來得快，需快速處理
```

## 阿咪重點速記

- 火的關鍵字：速度、光明、釋放、炎上、公開、尖形。
- 問事業：火旺可能有曝光、名聲、升遷、壓力。
- 問健康：心臟、血液、眼睛、炎症。
- 火金相剋要特別注意血光之象。
