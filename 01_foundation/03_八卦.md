# 八卦

## 主題摘要

八卦是太極、兩儀、四象繼續演化後形成的具體象徵系統，也是六十四卦與六爻納甲的基礎。在六爻中，八卦不只是象徵自然物，也代表卦宮、五行、方位、家族角色與六親配置的根本依據。

## 演化脈絡

```text
太極
↓
兩儀：陰、陽
↓
四象：老陰、老陽、少陰、少陽
↓
八卦：乾、坤、震、巽、坎、離、艮、兌
```

## 先天八卦與後天八卦

### 先天八卦

- 又稱伏羲八卦。
- 偏向宇宙原始結構與先天秩序。
- 常見方位：乾南、坤北、離東、坎西。

### 後天八卦

- 又稱文王八卦。
- 六爻實務主要使用後天八卦。
- 常見方位：離南、坎北、震東、兌西。

## 卦宮決定六親

六爻配置六親時，核心不是世爻五行，而是「卦宮五行」。

例如：

```text
天地否屬乾宮
乾宮五行為金
所以：
金 = 兄弟
水 = 子孫
土 = 父母
火 = 官鬼
木 = 妻財
```

這是六爻初學時最容易混淆，但非常關鍵的規則。

## 卦宮五行對六親的基本邏輯

以卦宮五行為「我」：

| 關係 | 六親 |
|---|---|
| 同我者 | 兄弟 |
| 我生者 | 子孫 |
| 生我者 | 父母 |
| 克我者 | 官鬼 |
| 我克者 | 妻財 |

## 艮卦特殊含義

艮為山，也常有停止、阻隔、墳墓、子孫、傳承之象。

- 艮納丑土與寅木。
- 丑土為濕土，內藏水，可生寅木。
- 寅木有延伸、生長、遺傳、傳承之意。
- 問墓地、祖先、子嗣、遺傳時，艮卦或艮象需要特別留意。

## 六沖卦提示

若某卦為六沖卦，例如乾為天、坎為水等，爻與爻之間有相沖之象，常主事情散、衝突、難聚焦，需依問題類型細看。

## If / Then 判斷

```text
If 搖出的卦屬乾宮金
Then 金為兄弟，水為子孫，土為父母，火為官鬼，木為妻財

If 搖出的卦屬震宮木
Then 木為兄弟，火為子孫，水為父母，金為官鬼，土為妻財

If 艮卦出現且與子孫、世爻、墓地問題有關
Then 可加重傳承、子嗣、墳墓、停止、阻隔等取象

If 卦為六沖卦
Then 需留意事情不穩、難聚、易散、易衝突
```

## 阿咪重點速記

- 六爻主要使用後天八卦。
- 八卦五行：乾兌金、坎水、震巽木、離火、坤艮土。
- 六親配置以卦宮五行為中心，不是以世爻五行為中心。
- 六爻中的「我」多指所問之事的卦宮太極，不一定是問卦者本人。
