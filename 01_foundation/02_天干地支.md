# 天干地支

## 主題摘要

天干地支是易經與六爻系統中用來標示時間、空間、五行氣勢的重要工具。六爻預測中，主要使用地支來配合月建、日辰、爻位與納甲；天干則常用於排列旬空。

### 六爻用途

- 天干主要用於排旬空。
- 每旬十天，十二地支中會有兩支落在旬外，稱為空亡或旬空。

## 地支三重功能

1. **代表時間**：月份、時辰、日期氣勢。
2. **代表空間**：方位與氣場位置。
3. **代表五行**：判斷爻的生剋旺衰。

## 四正與四隅

- 四正：子、午、卯、酉，氣較純，方位明確。
- 四隅：丑寅、辰巳、未申、戌亥，氣較雜，屬於過渡地帶。

## If / Then 判斷

```text
If 月建為寅月
Then 木氣旺，木爻得令，金爻受制

If 某爻臨子、午、卯、酉
Then 此爻氣較純，方位與五行特性明顯

If 某爻旬空
Then 該爻力量暫弱，事情有落空、不現、延後之象

If 某爻臨辰、戌、丑、未
Then 需進一步判斷是否為墓庫或財庫
```

## 阿咪重點速記

- 六爻主要用地支，天干多用來排旬空。
- 地支五行必背：寅卯木、巳午火、申酉金、亥子水、辰戌丑未土。
- 月建與日辰是斷卦旺衰的核心。
- 子午卯酉是四正，氣較純；辰戌丑未常與墓庫、財庫有關。
