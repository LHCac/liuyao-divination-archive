# 子平真詮核心觀念

## 資料定位

本篇整理使用者提供的八字資料，並依《子平真詮》格局用神系統重新校正文字與分類。

本資料屬於八字基礎總論，重點不是直接斷命，而是建立後續八字分析的核心觀念。

## 一、命理應回到干支五行，不宜迷信神煞

《子平真詮》的核心精神，是以天干地支、五行生剋、月令格局、用神相神為主，不以神煞、空亡、符咒、方位等旁門作主要判斷。

### 重點

```text
八字分析的根本：天干、地支、五行、生剋、制化、格局、用神。
神煞、空亡、特殊口訣可作輔助取象，但不可凌駕格局本體。
```

### 實戰用法

```text
斷八字時，先看月令與日主旺衰，再看格局、用神、相神、喜忌。
不可一看到神煞、空亡、桃花、驛馬，就直接斷吉凶。
```

## 二、格局用神是子平法的骨架

子平法重視「月令取格」。月令代表出生時節的主氣，是命局中最重要的環境力量。

用神通常從月令所藏、所透之干來定格，再看其是否清純、有根、有相神輔助。

### 原則

```text
用神以月令為主。
月令透出天干者，較容易成格。
月令藏而不透者，需看行運是否透清。
```

### 注意

「用神」在不同八字流派中有不同用法。

```text
子平真詮的用神：偏向月令格局用神。
後世扶抑法的用神：偏向平衡日主強弱的喜用神。
```

本資料庫後續整理時，會盡量區分：

```text
格局用神
扶抑喜用神
調候用神
通關用神
病藥用神
```

避免混用。

## 三、命局不是死劇本，人的心念與選擇很重要

使用者提供的資料中提到「執心為道」。此觀點可作為本資料庫的倫理總則。

八字可描述一個人的先天環境、性格傾向、喜忌結構與大運流年，但不宜把命盤當成固定劇本。

### 實戰態度

```text
八字用來理解傾向，不用來否定選擇。
八字用來看優勢與弱點，不用來貼死命運標籤。
八字用來趨吉避凶，不用來製造恐懼。
```

## 四、命局分析不宜拘泥格局名稱

資料中提到「拘泥格局」與「以訛傳訛」，這點非常重要。

很多八字學習者容易犯兩種錯：

```text
1. 看到某個格局名稱，就直接套結論。
2. 聽到特殊口訣，就不再回到五行生剋檢查。
```

正確方式應該是：

```text
先看月令。
再看天干透出。
再看地支根氣。
再看生剋制化。
再看用神是否有相神輔助。
最後才判格局高低與行運喜忌。
```

## 五、用神、相神、忌神三者要一起看

八字不是只找一顆用神就結束。

### 用神

命局中作為格局核心的主星。

### 相神

輔助用神成事的關鍵星。

```text
用神像主角。
相神像幫主角完成任務的人。
沒有相神，用神容易孤立無援。
```

### 忌神

破壞格局、干擾用神、攻擊相神的星。

### 救應

當忌神出現時，若有另一星制住忌神、化掉忌神，格局仍可保留。

```text
財格怕比劫奪財。
若官星透出制比劫，則為救應。
```

## 六、判命要動態看，不可只看原局

原局是基本盤，大運流年是觸發器。

```text
原局有好格局，行忌運也會受阻。
原局格局有缺，行運補足也可發展。
地支藏用神，逢運透清，力量才容易完全發揮。
地支藏忌神，逢運透出，也可能破格。
```

## 七、八字與六爻互參定位

本資料庫後續會把八字與六爻分開使用：

```text
八字：看底層結構、長期傾向、性格模式、大運流年。
六爻：看單一事件、當下氣場、短期變化、對方心態、應期。
```

重大問題可互參：

```text
八字看長期方向是否適合。
六爻看當下這件事能不能做、何時做、阻礙在哪。
```
