# 三元、胎元與起息法

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 三元：天元、地元、人元的命盤架構
- 起胎元法：子平命盤先推胎息之位
- 論起息法：正、變、通三法尋得命局息神

本篇歸入八字基礎系統，用於補充命盤架構、胎元與息神等進階參考。

---

# 一、三元架構

## 核心概念

八字可分天元、地元、人元三層。

```text
天元：天干。
地元：地支。
人元：地支所藏天干。
```

## 三元象意

### 天元

```text
外顯表現。
主動張力。
表面行為。
明面上的十神。
```

### 地元

```text
內在環境。
實際情境。
根基與場域。
命局承載力量。
```

### 人元

```text
藏干。
內心深層心理。
真實情感。
暗藏動機。
未顯露的力量。
```

## 實戰應用

```text
天干看外顯。
地支看環境。
藏干看內心與根氣。
天地人三元合看，才能看出表裡差異。
```

# 二、胎元

## 核心概念

胎元是配合受胎月份的輔助參考柱，可用來分析先天素質與命局補充。

## 取法

資料提供的取法：

```text
胎元取生月月干向前一位。
地支向前三位。
```

常見說法可簡化為：

```text
月干進一位，月支進三位。
```

後續實務需依採用流派與排盤工具校對。

## 象意

```text
先天素質。
受胎之氣。
命局補充五行。
先天張力。
早期身體與氣質參考。
```

## 實戰提醒

胎元只作輔助，不可取代四柱原局。

```text
胎元補足命局缺五行，可作參考。
胎元傷用神，亦可作先天隱患象。
仍需合看月令、日主、用神與大運。
```

# 三、起息法

## 核心概念

息神屬進階命理研究，用來尋命主發展與歸宿之神。

資料提供三法：

```text
正法：日主天干合處、地支合處尋息。
變法：時上天干合處、時下地支合處尋息。
通法：地支相通尋息。
```

## 使用定位

```text
進階命理研究。
生死歸壽參考。
命局完整性補充。
不可作一般解盤主線。
```

## 實戰提醒

起息法深度較高，且不同流派差異較大。

本資料庫使用規則：

```text
先用四柱、月令、格局、用神、大運流年判斷。
起息法只作高階補充，不作主判。
涉及健康壽元，不作絕對斷語。
```
