# 貴人吉神總整理

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 十干學堂：聰明秀氣生發的入門吉神
- 論天赦：上天開恩赦罪的吉日
- 論福星貴人：登科進祿、一生福祿無缺
- 論天廚貴人：管食祿的吉神，一生食祿不缺
- 論月德貴人：月德、月德合的安適小吉
- 論天德貴人：化凶為吉、代表天穰加持的吉神
- 論太極貴人：聯繫天地第一的方道貴人
- 起天官貴人：福祿豐厚、官運亨通之吉神
- 天乙貴人：命中最重要的福山吉神
- 論三奇貴人：天上、地下、人中三奇

本篇歸入格局與特殊神煞系統。貴人吉神只作輔助取象，不可取代月令、日主強弱、格局用神與大運流年。

---

# 一、貴人吉神使用總原則

```text
吉神不等於一定吉。
吉神被沖破，力量減弱。
吉神若落忌神，也需謹慎。
吉神配合用神，力量較明。
吉神只作輔助，不作主判。
```

實戰順序仍是：

```text
先看月令。
再看日主強弱。
再看格局用神。
再看大運流年。
最後才用貴人吉神補充取象。
```

# 二、天乙貴人

## 核心概念

天乙貴人是重要福神之一，以日干查年月日時地支。

常見口訣：

```text
甲戊庚牛羊。
乙己鼠猴鄉。
丙丁豬雞位。
壬癸兔蛇藏。
六辛逢馬虎。
此是貴人方。
```

## 象意

```text
貴人相助。
福分深厚。
逢凶化解。
人際資源。
社會支持。
```

## 實戰提醒

```text
天乙為喜用，貴人力較明。
天乙被沖破，力量減弱。
天乙落空亡，助力不實。
不可單憑天乙斷富貴。
```

---

# 三、天德貴人

## 核心概念

天德貴人以出生月份查取，是化凶為吉、減災保護的吉神。

## 象意

```text
化凶為吉。
少病災。
心態安穩。
安全保護。
凶中藏吉。
```

## 使用提醒

天德屬保護性吉神，若與用神、喜神配合較佳；若被刑沖破害，力量減弱。

---

# 四、月德貴人與月德合

## 核心概念

月德貴人以出生月份查取，主安適、福分、平穩與善良。

月德合力量略弱於月德，但仍有輔助化解之力。

## 象意

```text
一生安適。
心性善良。
少災少害。
遇難有解。
平穩小吉。
```

---

# 五、天赦

## 取法

```text
春戊寅。
夏甲午。
秋戊申。
冬甲子。
```

## 象意

```text
上天開恩赦罪。
免牢獄或法律糾紛之象。
遇事可化解。
保守性吉神。
```

## 實戰提醒

天赦不能保證無災，只代表命局中有化解、寬赦、轉圜之象。

---

# 六、福星貴人

## 核心概念

福星貴人以日主查時支找食神，主食祿、財祿、登科進祿。

## 象意

```text
食祿旺。
財運較豐。
生活資源不缺。
子孫宮有福祿象。
登科進祿。
```

## 與天廚貴人差異

```text
福星貴人：偏登科進祿、福祿與功名。
天廚貴人：偏食祿、衣食、生活供養。
```

---

# 七、天廚貴人

## 核心概念

天廚貴人以日干的食神所在地支為天廚，主食祿旺，不愁吃穿。

## 象意

```text
食祿。
生活品質。
口福。
衣食不缺。
壽命與福氣象。
```

## 實戰提醒

天廚本質仍與食神相關，需看食神是否為喜用、是否被偏印奪食。

---

# 八、天官貴人

## 核心概念

天官貴人以官星本旬遁見為取，以日干查地支。

## 象意

```text
官運亨通。
做事易成。
福祿豐厚。
遇凶可化。
官場或工作貴人。
```

## 實戰提醒

若命局官星為喜，天官貴人更有用；若官殺為忌，仍需看制化，不可只以天官為吉。

---

# 九、太極貴人

## 核心概念

太極貴人主聰明、學術、玄學、研究、天地道理之悟性。

常見口訣：

```text
甲乙生人子午中。
丙丁雞兔定亨通。
戊己兩干臨四季。
庚辛寅亥祿豐隆。
壬癸巳申偏喜美。
值此應當福氣鍾。
```

## 象意

```text
聰明。
學術研究。
玄學宗教。
哲理思考。
高層次理解力。
廣博多識。
```

## 實戰提醒

太極貴人為喜時，適合研究型、學術型、玄學型、哲思型方向；若落忌神，可能變成想太多、孤高、不接地氣。

---

# 十、學堂

## 核心概念

學堂位即五行長生位。

資料重點：

```text
甲乙木長生在亥。
丙丁火長生在寅。
庚辛金長生在巳。
壬癸水長生在申。
戊己土長生依派別不同，需看流派。
```

## 象意

```text
聰明。
學業。
秀氣生發。
入門吉神。
功名考試。
學術天賦。
```

## 實戰提醒

學堂有助學習，但是否考試有利，仍要看印星、食傷、官星與流年考試運。

# 十一、三奇貴人

## 三奇分類

```text
天上三奇：甲、戊、庚。
地下三奇：乙、丙、丁。
人中三奇：壬、癸、辛。
```

## 象意

```text
特殊才華。
格局清奇。
富貴氣象。
高層次機會。
```

## 使用條件

```text
命局需清純。
三奇需成局有力。
不宜混雜煞忌。
若命局濁亂，空奇無益。
```

# 十二、本篇速記

```text
天乙：重要貴人，遇難有助。
天德、月德：保護化解，少災安適。
天赦：赦罪解厄，保守性吉神。
福星：食祿功名，登科進祿。
天廚：衣食口福，不愁吃穿。
天官：官運、職位、工作貴人。
太極：學術玄思，研究悟性。
學堂：學業聰明，秀氣生發。
三奇：清奇格局，需命局清純。
吉神都要看喜忌與是否沖破，不能單獨斷吉。
```
