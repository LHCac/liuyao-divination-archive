# 祿神、羊刃、拱祿與暗祿補充

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 十天干之祿：命帶祿神則日元健旺、功名易成
- 論暗祿：地支六合暗帶入祿神之術
- 論拱祿：日時首尾相鄰、虛夾祿神之祿格
- 論羊刃：多有剛烈、功成則退的刃庫之星

本篇歸入格局與特殊格局系統。祿神、羊刃與拱祿可用於補充日主強弱、格局氣勢與特殊祿格，但仍需遵守：先月令、先正格、後特殊格。

---

# 一、十天干之祿

## 核心概念

祿神即天干之臨官位，代表日主得氣、得根、得俸祿、得安身之處。

## 十干祿位

```text
甲祿在寅。
乙祿在卯。
丙祿在巳。
丁祿在午。
戊祿在巳。
己祿在午。
庚祿在申。
辛祿在酉。
壬祿在亥。
癸祿在子。
```

## 象意

```text
日元健旺。
根氣明顯。
自我能力較強。
功名事業較易成。
身強可追財官。
弱身逢祿可補身，但仍需看是否受沖破空亡。
```

## 實戰提醒

```text
祿神為喜：日主有根，能承財官。
祿神被沖：根基受動，職位、資源、身體元氣易波動。
祿神入空亡：有祿而不實，需看救應。
祿神太多：身旺過度，反需財官食傷來疏導。
```

# 二、暗祿

## 核心概念

暗祿是命局地支不直接見祿神，但有與祿神六合之字，可暗中帶入祿氣。

例如：

```text
甲日祿在寅。
寅與亥合。
若命局不見寅而見亥，亥可暗合寅，稱暗祿。
```

## 特點

```text
暗祿力量小於明祿。
暗中有助。
大運流年引動時，力量可小幅提升。
適合判斷命局無明祿但仍有暗助之象。
```

## 實戰提醒

暗祿不能當成明祿使用，只能作輔助。

```text
明祿：直接有根。
暗祿：間接有助。
暗祿被沖合破，力量更弱。
```

# 三、拱祿

## 核心概念

拱祿，是日時兩柱相鄰並列，中間虛夾出祿神。

資料重點：

```text
日時相互緊鄰。
中間夾出祿神。
需日時天干同字方成。
主富貴、財官雙美。
```

## 實戰定位

拱祿屬進階祿格，必須研討命局整體。

```text
要看日主強弱。
要看虛夾之祿是否被沖破。
要看月令是否支持。
要看財官印食是否配合。
```

## 使用提醒

若正格已明，不必硬套拱祿。拱祿可作：

```text
祿氣補充。
富貴取象。
日時晚年、成果、子息宮補充。
```

# 四、羊刃

## 核心概念

羊刃又稱陽刃、刃星，是日主帝旺之氣，剛烈、過旺、容易傷人的力量。

資料中說：陽干祿前一位為羊刃，飛刃為羊刃對沖位。

常見羊刃：

```text
甲羊刃在卯。
丙、戊羊刃在午。
庚羊刃在酉。
壬羊刃在子。
```

> 陰干是否取羊刃，各派不同。本資料庫以常用子平法：陽干羊刃為主，陰干日刃另行輔助參考。

## 象意

```text
剛烈。
衝動。
競爭。
攻堅。
刀兵血光。
功成則退。
權力、軍警、工程、技術、危機處理。
```

## 喜忌

```text
喜官殺制刃。
喜有制度、紀律、專業約束。
身旺刃強，需財官食傷疏導。
忌羊刃無制。
忌羊刃逢沖合失控。
忌飛刃引動傷害。
```

## 實戰提醒

```text
羊刃為喜：可加強日元、提升攻堅能力。
羊刃為忌：容易剛烈傷人、衝動破財、血光意外。
羊刃遇七殺有制：可成殺刃相停，化為權柄。
羊刃遇歲運沖合：需特別謹慎交通、手術、衝突、官非。
```

# 五、本篇速記

```text
祿神：日主臨官位，代表根氣與俸祿。
暗祿：六合暗帶祿神，力量弱於明祿。
拱祿：日時虛夾祿神，需命局整體配合。
羊刃：日主帝旺過剛之氣，有制成權，無制成禍。
祿刃都不可單獨斷吉凶，需看月令、格局、喜忌、行運。
```
