# 從格、建祿、稼馬與特殊格局補充

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 從格與陰陽相配：從兒格、從財格
- 建祿格與月建逢合化
- 稼馬論：飛天稼馬與財星喜忌
- 一世安然、財命有氣；身強殺淺假殺為權，假殺格

本篇屬特殊格局與格局補充，依資料庫規則：先正格、後特殊格；特殊格局必要時才主用。

---

# 一、從格總論

## 核心概念

從格是日主極弱，無力抵抗全局強旺之勢，只能順勢而從。

常見從格：

```text
從官殺格。
從財格。
從兒格，又稱從食傷格。
```

## 從格成立條件

```text
日主極弱。
全局某一方五行或十神極旺。
日主無根或根極弱。
不見強印比扶身。
氣勢清純不混雜。
```

## 從格最忌

```text
行印運或比劫運，使日主還魂。
日主一旦得根變強，從勢破裂。
從格破，反而大凶。
```

## 實戰提醒

從格不可輕判。若日主仍有根、有印比、有救，就不宜作從格。

# 二、從財格

```text
全局財星極旺。
日主極弱而無根。
命局順財勢而從。
```

喜：

```text
財運。
食傷生財。
官殺順財氣。
```

忌：

```text
印比扶身。
日主得根。
破壞財勢清純。
```

# 三、從兒格

從兒格即從食傷格。

```text
食神傷官極旺。
日主弱而順其所生之勢。
全局以食傷流通為主。
```

喜：

```text
食傷。
財星承接食傷之氣。
順勢流行之運。
```

忌：

```text
印星制食傷。
比劫使日主轉強。
官殺逆食傷之勢。
```

# 四、建祿格與月建逢合化

## 核心概念

建祿格，是日主得月令祿地，日主身強氣旺。

資料中「建稼格」多可視為「建祿格」或「建祿得月建」之義。

## 特點

```text
日主得月建之力。
身強氣旺。
需要財官食傷來導出力量。
不宜印比再助。
```

## 月建逢合化

若月建被合化、合走或轉移，可能影響格局根基。

```text
月令是命局基地。
月建被合化，等於根基被牽動。
若合化為喜，可能轉成新格局。
若合化為忌，可能失去原本建祿根基。
```

## 喜忌

```text
喜財官透出，使強旺日主有用武之地。
喜食傷洩秀生財。
忌比劫印綬再助旺。
忌月建被不當合化。
```

# 五、稼馬論與飛天稼馬

## 核心概念

稼馬論涉及財星、祿馬、特殊暗拱暗合等古法取象。

資料重點可轉化為：

```text
稼：日主居於有根有力之地。
馬：財星或動能之地。
稼馬同現，且身旺能承財，主財富流動。
```

## 財星喜忌

```text
財有根旺，且身強能承財，為吉。
財旺身弱，成財多身弱，反為負擔。
財格忌比劫奪財。
財星死絕或被沖破，財力不穩。
```

## 飛天稼馬

飛天稼馬屬特殊格局，多依暗拱暗合、虛邀財官而成。

使用規則：

```text
必要時才看。
先檢查正格是否已可解釋。
若條件不清純，不宜硬套。
```

# 六、假殺格：身強殺淺，假殺為權

## 核心概念

假殺格是身強而七殺力量較淺，七殺無法真正壓制日主，反可成為管理、紀律、權威之象。

```text
身強殺淺，假殺為權。
```

## 成立條件

```text
日主身強。
七殺透出但力量不深。
七殺無多重根氣或壓身太過。
命局有食傷制殺、印化殺，或其他制化。
```

## 與真殺區別

```text
身弱殺強：真殺壓身，為禍。
身強殺淺：假殺為權，可化壓力為管理能力。
```

## 行運

```text
喜制殺、化殺、使殺有用之運。
忌財生殺太過，使假殺變真殺。
忌日主突然失根變弱。
```
