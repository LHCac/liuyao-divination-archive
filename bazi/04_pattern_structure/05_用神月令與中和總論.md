# 用神、月令與中和總論

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 取用憑於生月：月令是取用神的根本
- 用神論：命局核心，不可損傷
- 五行絕處逢生：卷一結語
- 繼善篇序：命由天定，推算有法

本篇歸入格局用神系統，作為後續八字解盤的總原則。

---

# 一、取用憑於生月

## 核心概念

月令是命局的提綱，是取用神最重要的根本。

```text
取用憑於生月。
當推究於深淺。
發覺在於日時。
要消詳於強弱。
```

意思是：

```text
先看出生月份的月令。
再看節氣深淺。
再看日時如何補充或修正。
最後合看全局強弱與平衡。
```

## 月令深淺

```text
初氣：剛入節，前月餘氣尚存，月令力量較淺。
中氣：過渡階段，需看具體干支。
本氣：月令正氣最旺，格局力量最明。
```

## 實戰提醒

```text
月令給大方向。
日時給細節修正。
天干透出決定格局清濁。
地支根氣決定力量深淺。
不可只看月令，也不可忽略月令。
```

# 二、用神是命局核心，不可損傷

## 核心概念

每個命局中必有一個主象，稱為用神。用神是命局中最需要保護、最不可損傷的核心力量。

```text
用神旺，命局容易順。
用神受傷，命局容易阻滯。
```

## 取用神三大依據

```text
1. 月令：當令之氣最強，是第一依據。
2. 日主強弱：身強取洩、耗、剋；身弱取生、扶。
3. 整體格局平衡：命局偏枯處，以用神調和。
```

## 用神與喜神不同

```text
用神：命局核心主象，負責成格與主線。
喜神：輔助用神、幫助命局順行的力量。
相神：使格局成立、幫用神成事的關鍵神。
忌神：破壞用神或相神的力量。
```

## 用神受傷

```text
被刑。
被沖。
被剋。
被合去。
被忌神混雜。
被大運流年破壞。
```

用神受傷時，需看有無救應、通關、制化。

# 三、五行絕處逢生

## 核心概念

五行的「絕」不等於完全消失，而是氣勢轉換、潛伏待生的階段。

```text
絕處未必真絕。
絕地也可能藏有轉機。
命局需看全局，不可只見一字便斷死絕。
```

## 哲學意義

```text
五行循環不斷。
旺極會衰，絕後可生。
墓絕之地未必皆凶。
命理判斷重在中和與轉化。
```

## 實戰提醒

```text
用神入墓絕，不一定無用。
若行運透出、沖開、合動、得生，仍可發用。
忌神入墓絕，反可能減凶。
```

# 四、命理學習心態：知命而趨吉避凶

《繼善篇》序強調，人稟天地之氣而生，命屬陰陽，推算干支五行，是為了理解禍福吉凶的走向。

本資料庫採用的態度：

```text
命理用來理解趨勢，不用來製造恐懼。
命理用來調整選擇，不用來否定人生。
命理用來知命、順命、改習慣、趨吉避凶。
```

# 五、後續解盤使用規則

```text
1. 先看月令。
2. 再看節氣深淺。
3. 看天干透出與地支根氣。
4. 定格局用神。
5. 分用神、喜神、相神、忌神。
6. 看用神是否受傷。
7. 看是否有救應、制化、通關。
8. 看大運流年是否助用或傷用。
9. 最後回到中和，而非死套旺衰。
```
