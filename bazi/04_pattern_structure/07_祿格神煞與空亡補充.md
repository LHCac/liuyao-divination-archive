# 祿格神煞與空亡補充

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 論十惡大敗：祿入空亡、財散家敗之日
- 論天羅地網：辰為天羅、戌為地網之納音神煞
- 論天地轉煞：物極而反的納音神煞（子平不用）
- 論帝座：時下納音旺處的子孫宮神煞
- 論四大空亡：五行不全的納音神煞
- 論四廢日：四季中五行能量最弱的廢日
- 論六甲空亡：心神落空、不善世俗經營的象徵
- 論截路空亡：人行路上遇水流斷，難以前進
- 華蓋：孤寂、聰明、才藝之星

本篇歸入格局與特殊神煞資料。依資料庫規則：神煞、納音、空亡類只作輔助取象，不可凌駕月令、日主強弱、格局用神與五行生剋。

---

# 一、神煞使用總原則

## 核心原則

```text
先正格，後神煞。
先月令，後納音。
先五行生剋，後吉凶名稱。
神煞只作輔助取象，不作主判。
```

## 特別提醒

許多神煞屬納音論命系統，並非子平正法。

例如：

```text
天羅地網
天地轉煞
帝座
四大空亡
```

這些資料可作背景知識，但實戰解盤時不可單憑神煞斷富貴貧賤、生死吉凶。

# 二、十惡大敗

## 核心概念

十惡大敗日，是六甲旬中十個日值祿入空亡的日子。

古法認為：

```text
日主祿神入空亡。
凡事不成。
財散家敗。
```

## 使用方式

```text
主要看日柱。
需結合日主強弱、用神喜忌、財官印食配置。
不可只因日柱犯十惡大敗就斷一生敗壞。
```

## 實戰轉化

若命局中祿神為喜用，又落空亡，可能象徵：

```text
根氣不穩。
資源落空。
努力難以聚財。
家業或財務需更謹慎經營。
```

若祿神不是用神，或原局有其他根氣救應，則不宜重斷。

# 三、六甲空亡

## 核心概念

六十甲子分六旬，每旬末兩支為空亡。

```text
甲子旬空戌亥。
甲戌旬空申酉。
甲申旬空午未。
甲午旬空辰巳。
甲辰旬空寅卯。
甲寅旬空子丑。
```

## 象意

```text
心神落空。
志向漂浮。
世俗經營力較弱。
不拘小節。
有修道、學術、玄學、藝術傾向。
```

## 實戰提醒

空亡需分陰陽、宮位、十神與喜忌。

```text
用神空亡：事情容易落空或延遲。
忌神空亡：忌神力量減弱，反可減凶。
財星空亡：財務、資源、感情現實感需注意。
官星空亡：名分、職位、責任感可能不穩。
印星空亡：庇護、學習、母系資源可能有落空象。
```

# 四、截路空亡

## 核心概念

截路空亡以日干點算五鼠遁時，見壬癸水即是截路空亡。

古象：

```text
人行路上遇水流斷，難以前進。
```

## 象意

```text
事不順。
有志難伸。
前進路上遇阻。
出外、求財、求官需謹慎。
```

## 實戰提醒

```text
需看出現在何柱、何宮位。
若截路空亡影響用神，阻力較明顯。
若只是神煞背景，且格局用神有力，則不宜重斷。
```

# 五、四廢日

## 核心概念

四廢日是四季中五行能量最弱的日子。

常見分類：

```text
春：庚申、辛酉。
夏：壬子、癸亥。
秋：甲寅、乙卯。
冬：丙午、丁巳。
```

## 象意

```text
五行失令。
做事無力。
體質活力不足。
生命力或推動力較弱。
```

## 實戰提醒

四廢日要看此日主五行是否為命局用神。

```text
用神逢廢：不利較明。
忌神逢廢：反可減凶。
若有印比扶助、地支根氣、大運補救，不可直接斷廢而無用。
```

# 六、四大空亡

## 核心概念

四大空亡源自納音論命，認為六十甲子中只有甲辰、甲戌兩旬五行俱全，其餘四旬五行不全。

```text
甲子、甲午、甲寅、甲申四旬，稱四大空亡。
```

## 使用定位

此屬納音神煞，子平正法不常用。

可作：

```text
納音論命背景知識。
對納音派論法的理解。
輔助取象。
```

不可作主判。

# 七、天羅地網

## 核心概念

天羅地網屬納音神煞。

資料重點：

```text
辰為天羅，戌為地網。
亦有戌亥為天羅、辰巳為地網之說。
火命人遇辰戌為天羅地網。
```

## 使用定位

```text
納音神煞背景知識。
子平正法不使用天羅地網作主判。
不可單憑天羅地網斷困厄。
```

---

# 八、天地轉煞

## 核心概念

天地轉煞屬納音論神煞，代表物極而反的概念。

資料指出：

```text
旺連天干曰天轉。
旺連納音曰地轉。
```

## 使用定位

```text
非子平正法。
只作納音系統背景知識。
不可納入主線判命。
```

---

# 九、帝座

## 核心概念

帝座取時下納音旺處，主子孫宮相關神煞。

例如：

```text
甲子時納音屬金，金旺於酉，則酉為帝座。
```

## 使用定位

```text
納音神煞。
可作子孫宮背景參考。
子平正法不以此為主。
```

# 十、華蓋

## 核心概念

華蓋取三合局第三字，也就是墓庫字。

```text
申子辰局，華蓋在辰。
寅午戌局，華蓋在戌。
巳酉丑局，華蓋在丑。
亥卯未局，華蓋在未。
```

## 象意

```text
孤寂。
聰明。
才藝。
靈性。
宗教、玄學、藝術、學術。
高標準與自視甚高。
```

## 實戰提醒

```text
華蓋為喜：利藝術、學術、研究、修行、專業孤高。
華蓋為忌：孤僻、封閉、難融入、人際距離感。
需看華蓋所在宮位與十神。
```
