# 日祿歸時與六辛朝陽補充

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》整理頁：

- 時歸日祿生平不喜官星；陰若朝陽忌丙丁，六辛朝陽格

本篇歸入格局與特殊格局系統。依先前規則：特殊格局必要時才看，先正格、後特殊格。

---

# 一、時歸日祿

## 核心概念

時歸日祿，指日主之祿神落在時柱。

```text
祿為日主旺地。
時柱主晚年、成果、子息、後期收成。
日祿歸時，表示後期有祿、有根、有收成之象。
```

## 成格條件

```text
日主之祿在時支。
祿神需清純。
祿神不宜被沖破。
不喜官星破祿。
命局不可混雜破格。
```

## 為何不喜官星

官星剋日主，而祿是日主旺氣。

```text
官星出現，剋日主之祿。
祿被官剋，清純被破。
```

所以古句說：

```text
時歸日祿，生平不喜官星。
```

## 實戰提醒

若命局已可用正官、財官、印綬等正格論，不必硬套日祿格。

日祿歸時可作輔助象：

```text
晚年有根。
後期有技能、資源、職位或成果。
子息、晚景、專業積累可看時柱喜忌。
```

# 二、六辛朝陽格

## 核心概念

六辛朝陽格，是辛日主逢特定時支，古法多論辛日遇戌時或相關朝陽結構，要求格局專純不雜。

資料重點：

```text
辛日主逢戌時。
天干透印綬。
格局專純不雜。
忌丙丁。
```

## 忌丙丁的原因

丙丁為辛金之官殺。

```text
丙火為辛之正官。
丁火為辛之七殺。
丙丁出現，官殺剋辛，破壞朝陽格清純。
```

古句：

```text
陰若朝陽，忌丙丁。
```

## 成格條件

```text
辛日主。
時柱符合朝陽格條件。
天干不見丙丁官殺破格。
印綬配合。
格局清純。
行運不破清氣。
```

## 實戰提醒

六辛朝陽屬特殊格局，必須遵守：

```text
先正格，後特殊格。
普通格局能解釋，就不硬套。
若丙丁明透，朝陽格多破。
若官殺為命局正用，則需回到正格判斷，不可只以朝陽忌丙丁定凶。
```

# 三、本篇速記

```text
日祿歸時：日主祿神在時柱，晚年有根，有成果象。
日祿格忌官星：官剋日主，也剋祿神。
六辛朝陽：辛日特殊格，重清純。
六辛朝陽忌丙丁：丙丁為辛之官殺，出現則破格。
特殊格局只作必要時輔助，不可凌駕正格與五行生剋。
```
