# 用神與格局總論

## 資料定位

本篇整理使用者提供的「論用神、用神成敗救應、用神純雜、用神變化、因成得敗因敗得成」等資料，並歸入八字格局用神系統。

## 一、用神是命局格局的核心

在《子平真詮》系統中，用神主要從月令取出。月令是出生月份的主氣，代表命局最核心的環境力量。

### 基本原則

```text
用神以月令為主。
月令透出天干者，格局較明顯。
月令藏而不透者，力量較隱，需看行運是否透清。
用神有根有氣，才能真正發力。
```

### 補充說明

後世常說：

```text
日主強，宜財官食傷洩制。
日主弱，宜印比生扶。
```

這是扶抑法的喜用神思路。子平真詮的格局用神，則更重視月令所成之格。

因此後續整理需區分：

```text
格局用神：看月令成格。
扶抑用神：看日主強弱平衡。
調候用神：看寒暖燥濕。
通關用神：看五行交戰如何流通。
病藥用神：看命局病處與藥處。
```

## 二、常見格局分類

使用者提供資料中提到的格局類型，可先分為：

```text
正官格
財格
印綬格
食神格
偏官格，又稱七煞格
傷官格
陽刃格
建祿月劫格
雜氣格
外格，如從格、化格、一行格
```

## 三、用神要有根氣

用神若只在天干虛透，地支沒有根，力量較浮。

```text
天干透出：代表外顯、可見、容易發揮。
地支有根：代表內在有支撐，能量穩。
藏而不透：代表有潛力，但未完全顯露。
```

## 四、用神成敗

### 成格

「成」代表格局完整，用神能發揮作用。

常見條件：

```text
用神清楚。
用神有根。
用神不被沖剋合去。
有相神輔助。
忌神不破格。
行運能助格局。
```

### 敗格

「敗」代表忌神破壞用神或相神，使格局受損。

常見情況：

```text
用神被合去。
用神被沖破。
用神被忌神剋制。
忌神透干破格。
用神與忌神混雜不清。
```

## 五、救應

「救應」是命局中雖有忌神破格，但另有一星制化忌神，使格局保存。

### 例子

```text
財格怕比劫奪財。
若有官星制比劫，則官星為救應。

官格怕傷官破官。
若有印星制傷官，則印星為救應。

七煞攻身為忌。
若有食神制煞，或印星化煞，則為救應。
```

## 六、因成得敗、因敗得成

### 因成得敗

原本用神取用得當，格局看似成立，但因其他干支干擾而破局。

```text
用神被合去。
相神被破。
忌神透出。
生剋順序不利。
氣候不配，用神失力。
```

### 因敗得成

原局看似有破格之處，但因其他干支制化干擾之神，反而讓格局成立。

```text
忌神被制。
破格之神被合化。
相神出現救應。
行運透清用神。
格局反而清純。
```

## 七、相神緊要

相神是輔助用神發揮的關鍵。

```text
用神是主角。
相神是幫主角做成事情的人。
忌神是破壞主角任務的人。
救應是出來制住破壞者的人。
```

### 實戰提醒

```text
不要只找用神。
還要看用神是否有相神扶助。
也要看忌神是否被制化。
```

## 八、用神純雜與格局高低

### 用神純

```text
用神清楚。
喜神相神有力。
忌神不混。
生剋流通。
格局高，事情容易專一有成。
```

### 用神雜

```text
喜忌交錯。
用神被破。
忌神混入。
合化不清。
格局較低，人生起伏較多。
```

### 重點

格局高低不只看用神數量，而是看清濁純雜。

```text
清而有力為佳。
雜而交戰則低。
```

## 九、用神變化

用神不是永遠不變。天干合化、地支刑沖合會，都可能讓原本的用神發生變化。

### 常見變化

```text
天干五合可能使透出之神改變性質。
地支刑沖可能使藏干受損或被引出。
大運流年透出藏干，可能使格局變清或變濁。
```

### 實戰提醒

```text
用神變化後，要重新看新格局的喜忌。
不可死守原格局名稱。
```

## 十、行運成格與變格

大運流年可補足原局，也可破壞原局。

```text
原局不成格，行運補足用神或相神，可成格。
原局已成格，行運透出忌神或破相神，可變格。
```

### 干支有別

```text
天干：影響外顯之神、透出之神、事件表面。
地支：影響根氣、藏干、月令氣勢、事件深層。
```

同一大運，天干與地支要分開看，再合併判斷。

## 十一、氣候得失

氣候得失是指季節寒暖燥濕對用神力量的影響。

```text
夏火炎燥，需水調候。
冬水寒冷，需火暖局。
木火燥烈，需水潤土承。
金水寒濕，需火土調和。
```

氣候不是單獨斷吉凶，而是看用神是否得時得地、能否發揮。

## 十二、實戰分析順序

```text
1. 看日主與月令。
2. 看月令透出何干，是否成格。
3. 看用神是否有根。
4. 看相神是否出現。
5. 看忌神是否破格。
6. 看有無救應。
7. 看用神純雜。
8. 看氣候是否配合。
9. 看行運是否成格或變格。
10. 最後才看格局高低與人生應象。
```
