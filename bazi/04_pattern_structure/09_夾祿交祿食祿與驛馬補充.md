# 夾祿、交祿、食祿與驛馬補充

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 論夾祿：命局地支小橋夾出祿神之格
- 論十天干食祿：命帶食祿旺、財運豐厚
- 驛馬：命帶驛馬代表移動與奔波
- 論交祿：兩干勢均力敵的互換祿神

本篇歸入格局與特殊格局系統。夾祿、交祿、食祿與驛馬皆可作命局補充取象，但仍需以月令、日主強弱、用神喜忌與大運流年為主，不可單獨定吉凶。

---

# 一、夾祿

## 核心概念

夾祿是命局地支形成相鄰結構，中間虛夾出日主祿神。

資料重點：

```text
夾祿：天干共日主冠帶與帝旺相鄰。
中間夾出祿神。
夾祿主富貴、財官雙美。
需配合命局整體，不可單獨看。
夾祿與拱祿不同，不可混淆。
```

## 與拱祿差異

```text
拱祿：多重日時兩柱相鄰，虛拱祿神。
夾祿：重地支前後相鄰，中間夾出祿神。
```

## 實戰提醒

```text
夾祿屬進階祿格。
需看日主是否能受祿。
需看祿神是否為喜用。
需看祿神有無被沖破空亡。
若正格已明，夾祿只作輔助，不作主判。
```

# 二、交祿

## 核心概念

交祿是兩柱天干互換祿神，形成兩干勢均力敵、互相活動之象。

資料中提到的類型：

```text
本主交祿：日主天干與年柱天干互換祿神。
年月交祿。
月日交祿。
日時交祿。
全局祿元互換。
```

## 象意

```text
兩干互相牽動。
兩柱關係密切。
祿氣互換。
資源、身份、職位、能力彼此借力。
```

## 實戰提醒

```text
交祿不可單獨看富貴。
需察全局吉凶貴賤。
若互換之祿為喜，主互助。
若互換之祿為忌，主牽制或爭奪。
```

# 三、十天干食祿

## 核心概念

食祿是日主所生之食神，主飲食、財祿、子孫、才藝與生活享受。

十干食神對應：

```text
甲食丙。
乙食丁。
丙食戊。
丁食己。
戊食庚。
己食辛。
庚食壬。
辛食癸。
壬食甲。
癸食乙。
```

## 象意

```text
飲食。
財祿。
子孫。
才藝。
產出。
生活品質。
福氣與口福。
```

## 食祿旺

```text
食神旺而為喜，主食祿不休，財運豐厚。
食神能生財，代表才華、技術、產品、服務能轉成收入。
食神對應子孫宮，可輔助看子孫緣分。
```

## 食祿受剋

```text
食神受偏印剋，稱倒食或梟神奪食。
才華產出受阻。
財源受到影響。
子息、食祿、生活品質需注意。
```

## 實戰提醒

食祿不是越旺越好。

```text
日主身旺，食神可洩秀生財。
日主身弱，食神太旺反洩身。
食神為喜則吉，為忌則耗身。
```

# 四、驛馬

## 取法

驛馬取三合局第一字的沖位：

```text
申子辰局，驛馬在寅。
寅午戌局，驛馬在申。
巳酉丑局，驛馬在亥。
亥卯未局，驛馬在巳。
```

## 象意

```text
移動。
奔波。
出差。
外勤。
搬遷。
轉換。
旅行。
跨地區發展。
```

## 喜忌

```text
驛馬臨喜用：移動有利，出差、變動、外地發展可帶來機會。
驛馬臨忌神：奔波勞碌，變動帶來壓力。
驛馬遇空亡：漂泊不定，目標感不足。
驛馬被合住：移動受阻，想動卻卡住。
驛馬被沖動：變動加劇，需看喜忌。
```

## 實戰應用

驛馬可用於：

```text
判斷是否適合外地發展。
看職業是否多移動、外勤、出差。
看搬遷、旅行、轉職、環境變動。
看大運流年是否引動移動象。
```

# 五、本篇速記

```text
夾祿：地支相鄰，虛夾祿神，與拱祿不同。
交祿：兩干互換祿神，兩柱互相牽動。
食祿：食神為飲食財祿，旺則可生財，受剋則食祿受損。
驛馬：移動奔波，喜用則動中得利，空亡則漂泊，合住則受阻。
```
