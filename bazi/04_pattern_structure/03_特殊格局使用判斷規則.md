# 特殊格局使用判斷規則

## 資料定位

本篇作為八字分析時的操作規則，用來區分「正格優先」與「特殊格局必要時才看」的判斷順序。

## 一、總原則

```text
先正格，後特殊格。
先實神，後虛邀。
先月令，後古格名目。
先五行生剋制化，後暗沖暗合。
```

特殊格局不是不能看，而是不能一開始就硬套。

## 二、八字分析優先順序

每次看八字，先依此順序：

```text
1. 看日主與月令。
2. 看月令所藏與天干透出。
3. 判斷能否成立正格。
4. 看日主強弱、根氣、透干。
5. 看格局用神、相神、忌神。
6. 看成敗救應。
7. 看寒暖燥濕與調候。
8. 看大運流年是否助格或破格。
9. 正格不足以完整解釋時，才檢查特殊格局。
10. 特殊格成立後，仍需回到五行生剋與行運驗證。
```

## 三、什麼情況才需要看特殊格局

### 1. 正格不明顯

```text
月令用神不透。
正格難以成立。
財官印食不明顯。
命局呈現特殊集中氣勢。
```

### 2. 命局有明確特殊結構

例如：

```text
多子、多午、多辰、多丑等重複地支。
申子辰、寅午戌等三合局成勢。
日時形成古格指定組合。
祿神明確歸於時柱。
四庫辰戌丑未藏財官印並逢透出或引動。
```

### 3. 命局不見明官財，但有暗沖暗合線索

```text
局中無官星，但古格以暗沖暗合取官。
局中無財官透出，但藏干與支局有虛邀可能。
```

但這類必須非常謹慎。

## 四、什麼情況不要硬套特殊格局

### 1. 正格已經可以解釋

如果月令正官、財、印、食神、七殺、傷官等格局清楚，就先用正格。

```text
正官格清楚，不必硬說暗沖取官。
財格清楚，不必硬說飛天祿馬。
食傷生財明顯，不必硬說井欄叉。
```

### 2. 明見官殺財星填實

很多暗沖暗合格局要求命局不見明官殺或明財官。

```text
倒沖格見明官殺，多破。
飛天祿馬見午或官星填實，多破。
子遙巳、丑遙巳見官星明透，多不宜作虛邀。
井欄叉見丙丁火明透，多不取暗沖。
```

### 3. 命局混雜不清

```text
天干地支混雜。
忌神多透。
格局不清。
行運無法驗證。
```

此時不要為了套古格而忽略基本旺衰與生剋。

### 4. 特殊格條件只符合一半

例如只看到：

```text
有一個子。
有一個午。
日主剛好是某日。
時柱剛好有某支。
```

但沒有成局、無清純、無行運配合，就不可直接定特殊格。

## 五、特殊格局使用檢查表

判斷特殊格前，必須問：

```text
1. 正格是否已能解釋？
2. 月令用神是否清楚？
3. 特殊格要求的日時地支是否完整？
4. 命局是否清純？
5. 是否忌見的官殺、財星已明透？
6. 是否有沖破、合走、刑害破局？
7. 大運是否配合此特殊格？
8. 用普通五行生剋論，是否也能得出同樣結論？
```

若第 1 題答案是「正格已清楚」，特殊格只作輔助，不作主判。

## 六、特殊格局的正確定位

特殊格局應作：

```text
輔助取象。
補充命局特殊性。
理解古法命例。
與正格互參。
```

不應作：

```text
硬套富貴。
跳過日主旺衰。
跳過月令格局。
跳過用神相神。
跳過調候。
只憑名字斷命。
```

## 七、判斷語氣規則

分析時應使用分層語氣：

### 正格成立時

```text
此命局先以正格論，特殊格只作參考。
```

### 特殊格有跡象但不完整時

```text
此局有某特殊格的影子，但條件未完全清純，不宜直接定格。
```

### 特殊格條件完整時

```text
此局具備某特殊格成立條件，但仍需與月令、旺衰、調候與行運合看。
```

## 八、阿咪速記

```text
正格是主線，特殊格是補充。
明透優先於暗邀。
月令優先於格名。
五行生剋優先於古訣。
普通格局能解釋，就不要硬套特殊格。
特殊格只有在條件完整、命局清純、正格不足時才主用。
```
