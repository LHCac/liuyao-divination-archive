# 干支起源、五行方位與六十甲子

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 引（序言）：《淵海子平》的傳承與緣起
- 五行所生之始：天地萬物生成之理
- 天地干支所出：黃帝制干支的歷史起源
- 論干支字義：植物生命週期與十干的符號邏輯
- 十天干十二地支所屬方位：五行方位與四季對應
- 十二支生肖：地支與十二生肖的對應原理
- 六十花甲子：天干地支的六十組合與納音

本篇歸入天干地支系統，作為八字基礎理論、干支記憶與五行方位分類的資料。

# 一、《淵海子平》的傳承定位

《淵海子平》序文說明，子平命理以出生年月日時四柱推命，源流可追溯至唐代李虛中，後經宋儒發揚形成完整體系。

## 學習定位

```text
八字不是現代新創。
四柱以年月日時為命盤基礎。
學習應重視口訣、架構與實務應用。
```

## 本資料庫態度

```text
重視傳統，但不迷信口訣。
重視原理，但不硬套神煞。
以月令、日主、格局、用神、行運為主線。
```

# 二、五行所生之始

## 核心概念

五行是金、木、水、火、土，為天地萬物的基本分類。

傳統理論以：

```text
太極生陰陽。
陰陽分五行。
五行生萬物。
```

來描述天地萬物的生成。

## 命理應用

```text
五行不是單純元素，而是自然運行分類。
木主生發。
火主發散。
土主承載。
金主收斂。
水主收藏與流動。
```

# 四、十干字義與植物生命週期

十天干可用植物生命週期理解：

```text
甲：種子破殼，初生發芽。
乙：幼苗彎曲，尚未伸直。
丙：陽氣顯明，火力向上。
丁：成長穩定，光熱持續。
戊：茂盛厚實，土氣承載。
己：收束整理，成形可用。
庚：更新更替，成熟轉變。
辛：收斂精華，修飾成器。
壬：孕育潛藏，流動包容。
癸：歸藏至極，等待新生。
```

## 使用方式

此法適合記憶天干性質，不能取代十神與五行旺衰判斷。

# 七、六十花甲子與納音

## 核心概念

十天干配十二地支，形成六十組合，六十年一循環。

```text
十天干 × 十二地支 = 六十甲子。
```

六十甲子又稱六十花甲。

## 納音定位

納音五行是另一套系統，例如甲子、乙丑為海中金。

本資料庫使用原則：

```text
納音可作背景知識。
子平正法不以納音作主判。
解盤仍以干支本氣、月令、格局、用神為主。
```
