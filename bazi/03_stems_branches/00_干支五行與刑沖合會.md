# 干支五行與刑沖合會

## 資料定位

本篇整理使用者提供的「十干十二支、陰陽生剋、十干合而不合、天干五合、刑沖會合、十二地支分類、旺相休囚死、十二長生」等內容。

## 一、一氣分陰陽，陰陽化五行

八字的基礎是天干地支。

傳統命理認為天地之間一氣流行，一氣有動靜：

```text
動為陽。
靜為陰。
陰陽再分四象。
四象生五行。
五行再分陰陽，形成十天干與十二地支。
```

## 六、五行生剋

### 相生

```text
木生火
火生土
土生金
金生水
水生木
```

### 相剋

```text
木剋土
土剋水
水剋火
火剋金
金剋木
```

### 重要觀念

```text
生不是絕對吉。
剋不是絕對凶。
生我者補我。
我生者洩我。
剋我者制我。
我剋者受我控制，但也耗我力。
```

### 合不一定化

天干相合是否能化，要看月令、根氣、環境是否支持。

```text
有合，不一定能化。
能化，需得月令氣勢與命局配合。
```

## 八、合而不合

兩干相合，但可能因隔位、被別干搶合、受制等情況，形成「合而不合」。

### 常見情況

```text
中間有阻隔。
被別干先合。
一方力量受制，不敢合。
合而不化，只留牽絆之象。
```

實戰上，要看天干排列、遠近、力量與月令條件，不可只看到五合就直接判化。

## 十四、刑沖會合解法順序

使用者資料中提到一個重要提醒：刑沖合會不能死套，要看五行正論。

建議實戰順序：

```text
1. 先看月令與旺衰。
2. 再看合會是否成勢。
3. 再看生剋是否流通。
4. 再看刑沖破害是否傷到用神或忌神。
5. 最後看是否有救應與通關。
```

注意：不同古籍對優先順序略有差異，本資料庫以「格局用神是否受益或受損」為最終判準。

## 十五、旺相休囚死

五行在四季中有旺、相、休、囚、死之分。

常見概念：

```text
當令者旺。
我生者相。
生我者休。
我剋者囚。
剋我者死。
```

不同流派表述略有差異，使用時不可機械套用，需配合根氣、透干、通關、全局五行。

## 十六、十二長生

十二長生是五行在十二地支中的氣勢循環：

```text
長生、沐浴、冠帶、臨官、帝旺、衰、病、死、墓、絕、胎、養
```

### 使用原則

```text
十二長生是取象與看氣勢的工具。
不可只用長生帝旺就斷吉，也不可只用死絕墓就斷凶。
仍要回到命局全局、根氣、用神喜忌。
```

## 十七、十天干得時不旺、失時不弱

天干強弱不能只看是否得月令。

```text
得時但無根，仍可能虛浮。
失時但有根，仍可能有力。
```

### 例子

```text
甲木生寅月，若地支完全無根且受剋，未必真旺。
甲木生酉月，若地支寅卯多根，仍可有力。
```

因此判強弱要綜合：

```text
月令
地支根氣
天干透出
生扶剋洩
合沖刑害
大運流年
```
