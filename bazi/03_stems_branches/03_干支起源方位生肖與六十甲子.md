# 干支起源、方位生肖與六十甲子

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 引（序言）：《淵海子平》的傳承與緣起
- 五行所生之始：天地萬物生成之理
- 天地干支所出：黃帝制干支的歷史起源
- 論干支字義：植物生命週期與十干的符號邏輯
- 十天干十二地支所屬方位：五行方位與四季對應
- 十二支生肖：地支與十二生肖的對應原理
- 六十花甲子：天干地支的六十組合與納音

本篇歸入天干地支基礎系統，用於建立八字干支、五行、方位、生肖、六十甲子與納音的基礎架構。

# 一、《淵海子平》的傳承與緣起

## 核心概念

子平命理並非現代新創，而是由唐代李虛中以年月日時推命的系統發展而來，後經宋儒傳承與整理，形成四柱八字體系。

## 學習定位

```text
四柱八字以出生年、月、日、時為基礎。
核心是天干地支、陰陽五行與四柱結構。
學習口訣是入門工具，但實務仍需回到生剋制化與格局用神。
```

---

# 二、五行所生之始

## 核心概念

五行是金、木、水、火、土，是天地萬物的基本分類。

傳統思想中：

```text
太極分陰陽。
陰陽生五行。
五行生萬物。
天干地支源自對天地規律的觀察。
```

## 實戰提醒

五行不是單純元素，而是自然運行狀態與力量分類。

```text
木：生發、條達、成長。
火：光明、炎上、發散。
土：承載、轉化、中和。
金：收斂、規範、肅殺。
水：潤下、流動、收藏。
```

# 三、天地干支所出

## 十天干

```text
甲、乙、丙、丁、戊、己、庚、辛、壬、癸。
```

## 十二地支

```text
子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥。
```

## 陰陽分類

```text
奇數位為陽。
偶數位為陰。
天干象圓而動。
地支象方而靜。
```

## 實戰應用

```text
天干主外顯、流動、表面。
地支主內在、根氣、環境。
藏干主人元、心理、暗藏力量。
```

# 四、十天干字義與植物生命週期

十天干可用植物生命週期幫助記憶。

```text
甲：破殼而出，生命開始萌發。
乙：彎曲未伸，生命仍柔弱曲折。
丙：光明發熱，生氣向上而盛。
丁：壯實成形，火氣漸凝。
戊：茂盛厚重，土氣成形。
己：收束整理，萬物成型。
庚：更替變革，金氣肅殺開始。
辛：辛苦成熟，精華收斂。
壬：孕育流動，水氣包藏。
癸：歸藏待發，循環將再起。
```

## 實戰提醒

干支字義主要是記憶與取象工具，不可直接當成吉凶判斷。

# 五、五行方位與四季

## 方位五行

```text
東方甲乙木。
南方丙丁火。
中央戊己土。
西方庚辛金。
北方壬癸水。
```

## 四季地支

```text
春：寅、卯、辰，木旺。
夏：巳、午、未，火旺。
秋：申、酉、戌，金旺。
冬：亥、子、丑，水旺。
四季月：辰、戌、丑、未，土寄旺。
```

## 土的特殊性

```text
土居中央。
土寄旺四季。
萬物皆賴土承載與轉化。
```

## 實戰應用

```text
判斷月令旺衰。
判斷日主是否得令。
判斷五行偏枯與調候。
判斷命局是否缺少承載與轉化之土。
```

# 七、六十花甲子與納音

## 六十甲子

六十甲子由十天干與十二地支依序相配而成。

```text
十天干 × 十二地支，形成六十組干支組合。
六十年一循環。
四柱中的年、月、日、時皆由干支組成。
```

## 納音五行

納音有自己的五行系統，例如甲子海中金等。

## 使用定位

```text
六十甲子是排四柱的基礎。
納音屬另一套系統。
子平正法不以納音作主線論命。
納音可作文化、神煞或古法背景參考。
```

# 八、本篇速記

```text
子平命理以年月日時四柱為基礎。
五行源自陰陽變化，代表自然力量分類。
天干圓而動，地支方而靜。
天干看外顯，地支看根氣，藏干看人元。
東木、南火、中土、西金、北水。
春木、夏火、秋金、冬水，辰戌丑未為四季土。
生肖只輔助記憶，不可單獨論命。
六十甲子是時間循環基礎，納音不作子平主判。
```
