# 節氣、月令、藏干與五行基礎

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，主要來源：

- 月令
- 論節候歌
- 又論節氣歌
- 四孔大節訣
- 論未來月朔節氣奧訣
- 天地干支暗藏總訣
- 地支藏干歌
- 五行相生相剋
- 二至隨陽相生之理
- 五行生旺衰絕吉凶
- 五行生剋制化各有所喜所害例

本篇歸入天干地支系統，作為排盤、月令、節氣、藏干、旺衰與五行生剋的基礎資料。

# 一、命理月份以節氣為準

八字排月柱，不以農曆初一換月，而以二十四節氣中的「節」為換月點。

```text
立春入寅月。
驚蟄入卯月。
清明入辰月。
立夏入巳月。
芒種入午月。
小暑入未月。
立秋入申月。
白露入酉月。
寒露入戌月。
立冬入亥月。
大雪入子月。
小寒入丑月。
```

## 實戰提醒

```text
交節前後差一分鐘，月柱可能不同。
月柱不同，月令不同，格局與用神也可能不同。
```

---

# 二、月令是提綱

月令代表出生月份的主氣，是判斷日主旺衰與格局用神的第一依據。

```text
月令為命局提綱。
得月令者，日主有季節力量支持。
失月令者，需看根氣、印比與全局補救。
月令中藏用神，若得透出或行運引動，吉驗更明。
```

---

# 三、節氣深淺與司令之干

同一個月份中，節氣初、中、末力量不同。

```text
初氣：剛交節，前月餘氣仍在。
中氣：月令逐漸穩定，但需看藏干輪值。
本氣：月令主氣最明顯。
```

天地干支暗藏總訣指出，每個節氣期間有不同天干輪流用事，這是判斷「誰當令」的精密方法。

實戰上：

```text
先看月令大方向。
再看節氣深淺。
再看藏干司令。
最後合看天干透出與地支根氣。
```

# 四、地支藏干

地支藏干是判斷根氣、格局清濁、暗藏力量的重要依據。

| 地支 | 藏干 |
|---|---|
| 子 | 癸 |
| 丑 | 己、癸、辛 |
| 寅 | 甲、丙、戊 |
| 卯 | 乙 |
| 辰 | 戊、乙、癸 |
| 巳 | 丙、戊、庚 |
| 午 | 丁、己 |
| 未 | 己、丁、乙 |
| 申 | 庚、壬、戊 |
| 酉 | 辛 |
| 戌 | 戊、辛、丁 |
| 亥 | 壬、甲 |

> 有些古籍版本會列卯藏甲乙，本資料庫以常用子平藏干表為主，卯以乙木為主氣。

# 五、五行相生相剋

## 相生

```text
木生火。
火生土。
土生金。
金生水。
水生木。
```

## 相剋

```text
木剋土。
土剋水。
水剋火。
火剋金。
金剋木。
```

## 六親對應

```text
生我者：父母、印星。
我生者：子孫、食傷。
剋我者：官鬼、官殺。
我剋者：妻財、財星。
同我者：兄弟、比劫。
```

# 六、五行生剋制化的喜害

五行得生不一定永遠吉，被剋也不一定永遠凶，必須看該五行是喜用還是忌神。

```text
用神得生：吉。
用神被剋：凶。
忌神得生：忌神更旺，凶。
忌神被剋：反可能為吉。
```

實戰判斷：

```text
先定用神喜忌。
再看五行生剋制化。
最後看是否過強、過弱、偏枯或有通關。
```

# 七、二至陰陽消長

二至指夏至與冬至。

```text
夏至一陰生：夏至後白晝漸短，陰氣漸長。
冬至一陽生：冬至後白晝漸長，陽氣漸長。
```

這是五行十二長生、旺衰消長的天文背景。

理解方式：

```text
五行不是靜止狀態。
四季陰陽不斷消長。
旺極會衰，衰極可生。
```

# 八、十二長生與旺衰

十二長生：

```text
長生、沐浴、冠帶、臨官、帝旺、衰、病、死、墓、絕、胎、養。
```

使用原則：

```text
長生、臨官、帝旺：氣勢較強。
衰、病、死、絕：氣勢較弱。
墓、胎、養：藏蓄、轉換、待生。
```

不可只因一個字在死絕，就直接斷凶；仍需看根氣、月令、用神喜忌與行運引動。

# 九、曆法推算資料定位

四孔大節訣、論未來月朔節氣奧訣，屬高階曆法推算資料。

在現代實務中：

```text
排盤以可靠曆法或排盤工具為主。
古法推算訣可作校驗與理解節氣原理。
遇交節時刻，需特別精確。
```
