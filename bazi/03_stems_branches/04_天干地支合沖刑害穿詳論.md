# 天干地支合沖刑害穿詳論

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 天干相合：五合化氣的原理與應用
- 十二地支六合：陰陽相合的天文原理
- 十二地支三合：長生帝旺墓庫的力量
- 十二支相沖：對沖激動的力量
- 十二支相穿：破壞六合的嫌惡力量
- 十二支相刑：無恩無禮無義之刑

本篇歸入天干地支系統，用於後續分析命局干支互動、大運流年引動與格局成敗。

---

# 一、天干五合

## 五合口訣

```text
甲己合化土。
乙庚合化金。
丙辛合化水。
丁壬合化木。
戊癸合化火。
```

## 合化條件

```text
化氣需要月令配合。
需要全局氣勢支持。
合而不化，只論合住、牽制、情意，不論化出新五行。
```

## 實戰提醒

```text
合可成情，也可成絆。
合可化忌為喜，也可合走用神。
天干合化後，需重新看用神是否改變。
```

# 二、地支六合

## 六合口訣

```text
子丑合土。
寅亥合木。
卯戌合火。
辰酉合金。
巳申合水。
午未合土，亦有日月合之說。
```

## 六合作用

```text
引動。
牽制。
合住。
化合。
合作與關聯。
```

## 實戰提醒

```text
六合力量不如三合局強。
六合受沖，合力減弱。
是否化氣，仍看月令與全局。
合住用神可能不利，合住忌神可能有利。
```

# 三、地支三合

## 四組三合

```text
申子辰合水局。
亥卯未合木局。
寅午戌合火局。
巳酉丑合金局。
```

## 核心概念

三合取五行長生、帝旺、墓庫三者匯聚。

```text
長生：氣之開始。
帝旺：氣之最盛。
墓庫：氣之收藏。
```

## 三合力量

```text
三字齊全，力量最大。
缺一為半合。
帝旺字最重要。
三合局可改變五行氣勢，影響用神判斷。
```

## 實戰提醒

```text
三合成喜用局，力量加大為吉。
三合成忌神局，忌神力量加重。
行運流年補足三合，常為事件觸發點。
```

# 四、地支六沖

## 六沖口訣

```text
子午沖。
卯酉沖。
寅申沖。
巳亥沖。
辰戌沖。
丑未沖。
```

## 象意

```text
衝突。
變動。
不穩定。
搬遷。
關係變化。
職場變動。
身體或情緒被激動。
```

## 喜忌判斷

```text
喜神被沖為凶。
忌神被沖為吉。
沖開墓庫需看開出的是喜神還是忌神。
合可解沖，沖可解合。
```

## 實戰提醒

沖不是絕對壞。沖是動，動後結果看喜忌與救應。

# 五、地支六穿，亦稱六害

## 六穿口訣

```text
子未穿。
丑午穿。
寅巳穿。
卯辰穿。
申亥穿。
酉戌穿。
```

## 核心概念

穿是破壞六合的力量，又稱害。

```text
穿的力量弱於沖。
主嫌惡、不合、小障礙。
多為暗中不順、人際不協作。
```

## 實戰應用

```text
看人際關係中的嫌隙。
看合夥關係的小阻礙。
看命局中細微不順。
若穿害到用神，才明顯不利。
```

# 六、地支相刑

## 三刑

```text
寅巳申：無恩之刑。
丑戌未：無禮之刑。
子卯：無義之刑。
```

## 自刑

```text
辰、午、酉、亥自刑。
```

## 象意

```text
傷害。
法律糾紛。
意外事故。
健康問題。
內耗。
互相折磨。
```

## 力量層次

資料中指出：

```text
刑重於穿，但通常輕於沖，仍需視命局而定。
```

## 實戰提醒

```text
刑到用神，傷害較明。
刑到忌神，可能有制忌之效，但仍主過程不舒服。
自刑多見內耗、自責、反覆、精神壓力。
```

---

# 七、合沖刑害穿判斷順序

```text
1. 先看月令與用神喜忌。
2. 再看合局是否成勢。
3. 看沖是否引動喜忌、開庫或破庫。
4. 看刑是否傷用或制忌。
5. 看穿害是否造成暗中不合。
6. 看大運流年是否引動原局。
```

# 八、本篇速記

```text
天干五合：合不一定化，需月令配合。
六合：牽制、引動、合作，力量小於三合。
三合：長生帝旺墓庫成局，力量強。
六沖：動盪、衝突、變化，喜忌決定吉凶。
六穿：破壞六合，小障礙與嫌隙。
三刑：傷害、法律、意外、內耗。
所有合沖刑害，都必須回到用神喜忌判斷。
```
