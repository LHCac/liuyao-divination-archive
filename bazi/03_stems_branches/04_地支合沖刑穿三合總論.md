# 地支合沖刑穿與三合總論

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 十二地支六合：陰陽相合的天文原理
- 十二地支三合：長生、帝旺、墓庫的力量
- 十二支相沖：對沖激動的力量
- 十二支相刑：無恩、無禮、無義之刑
- 十二支相穿：破壞六合的嫌惡力量
- 天干相合：五合化氣的原理與應用

本篇歸入天干地支系統，用於後續判斷命局中的合、沖、刑、穿、三合、天干五合化氣。

---

# 一、天干五合

## 五合口訣

```text
甲己合化土。
乙庚合化金。
丙辛合化水。
丁壬合化木。
戊癸合化火。
```

## 合化條件

```text
有合不一定化。
化氣需要月令配合。
合而不化，只論合住、牽制、互動，不論化出新五行。
```

## 實戰提醒

```text
若合化成功，五行性質可能改變，需重新看用神喜忌。
若合而不化，則看被合之神是否被牽制、合去、合住。
```

# 二、地支六合

## 六合口訣

```text
子丑合土。
寅亥合木。
卯戌合火。
辰酉合金。
巳申合水。
午未合日月，常作合土論。
```

## 六合作用

```text
引動。
牽制。
合住。
化合。
關係黏著。
```

## 實戰提醒

```text
六合力量通常不如三合局強。
六合被沖，合力減弱。
合是否化，要看月令與全局氣勢。
```

---

# 三、地支三合

## 三合局

```text
申子辰合水局。
亥卯未合木局。
寅午戌合火局。
巳酉丑合金局。
```

## 三合原理

三合取五行的長生、帝旺、墓庫三者相聚，力量較強。

```text
長生：氣之開始。
帝旺：氣之最盛。
墓庫：氣之收藏。
```

## 半合

```text
缺一為半合。
帝旺字最重要。
若有帝旺字，半合力較明。
若無帝旺字，力量較弱。
```

## 實戰提醒

```text
三合局可改變五行氣勢。
若三合成用神，格局得助。
若三合成忌神，忌神成勢。
大運流年補齊三合，常是事件觸發點。
```

# 四、地支六沖

## 六沖口訣

```text
子午沖。
卯酉沖。
寅申沖。
巳亥沖。
辰戌沖。
丑未沖。
```

## 沖的象意

```text
動。
剋。
衝突。
變動。
不穩定。
分離。
環境改變。
```

## 喜忌判斷

```text
喜神被沖為凶。
忌神被沖為吉。
用神被沖，要看有無救應。
合可解沖，沖可解合。
```

## 實戰應用

```text
命局動盪。
搬遷轉職。
婚姻宮沖。
大運流年衝破原局。
沖開墓庫或沖破根氣。
```

# 五、地支相刑

## 三刑與自刑

```text
寅巳申：無恩之刑。
丑戌未：無禮之刑。
子卯：無義之刑。
辰、午、酉、亥：自刑。
```

## 刑的象意

```text
傷害。
糾結。
法律糾紛。
意外事故。
健康壓力。
內在自我折磨。
```

## 實戰提醒

```text
刑重於穿，但通常輕於沖，仍須視命局而定。
刑到用神不利。
刑到忌神可能激發問題，也可能制忌，需看全局。
自刑多帶內耗、自我困住、重複犯錯之象。
```

# 六、地支相穿，又稱六害

## 六穿口訣

```text
子未穿。
丑午穿。
寅巳穿。
卯辰穿。
申亥穿。
酉戌穿。
```

## 穿的本質

穿是破壞六合的力量，作用通常弱於沖。

## 象意

```text
嫌惡。
不合。
小障礙。
暗中不順。
合作不舒服。
關係中有刺。
```

## 實戰提醒

```text
穿不是大凶，通常是細微卡點。
若穿到用神或婚姻宮，關係不適感較明。
若穿到忌神，可能減少忌神穩定性。
```

---

# 七、合沖刑穿判斷優先順序

實戰時不可只看名詞，要看喜忌。

```text
1. 先看月令與日主強弱。
2. 看合沖刑穿作用到哪一個十神。
3. 看該十神是用神、喜神、相神還是忌神。
4. 看作用在哪一柱宮位。
5. 看大運流年是否引動。
6. 看是否有合解沖、沖解合、通關救應。
```

# 八、本篇速記

```text
天干五合：合不一定化，化需月令配合。
六合：牽制、引動、合住，力量不如三合。
三合：長生、帝旺、墓庫成局，力量最強。
沖：動盪與變動，喜被沖凶、忌被沖吉。
刑：傷害糾結、法律意外與內耗。
穿：嫌惡不合，小障礙，弱於沖。
一切合沖刑穿都要回到用神喜忌判斷。
```
