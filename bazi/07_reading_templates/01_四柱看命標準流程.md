# 四柱看命標準流程

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，主要來源：

- 喜忌篇總論：四柱看命次序
- 子平舉要歌
- 以日為主
- 詳解定真訣

本篇歸入八字解析模板，用於建立後續看八字時的固定 SOP。

---

# 一、四柱看命總次序

喜忌篇總論指出，看命不能跳步驟，應先定基本盤，再看格局與喜忌。

## 標準流程

```text
1. 排四柱：年、月、日、時。
2. 定日主：以日干為命盤中心。
3. 看月令：確認日主是否得令。
4. 看天干透出：財、官、印、食傷、比劫是否明透。
5. 看地支藏干：有無暗藏財官印食傷比劫。
6. 識別格局：正格優先，特殊格必要時才看。
7. 取用神：以月令、日主強弱、全局平衡為根本。
8. 分喜忌：用神、喜神、相神、忌神。
9. 看成敗救應：忌神是否破格，有無救應。
10. 看大運流年：是否助用或傷用。
```

# 二、以日為主

子平法以日干為日主，也就是命盤主人。

```text
年柱：祖上、早年、家世根基。
月柱：父母、成長環境、青年與職涯根基。
日柱：自己、配偶宮、壯年與核心狀態。
時柱：子女、晚年、成果與後期收成。
```

六親對應：

```text
同我者：比劫，兄弟、同輩、朋友、競爭者。
生我者：印星，父母、長輩、學習、庇護。
我生者：食傷，子女、才華、產出、表達。
我剋者：財星，妻財、資源、金錢、現實管理。
剋我者：官殺，職位、壓力、制度、伴侶名分。
```

---

# 三、月令與日主強弱是第一關

子平舉要歌與月令資料都強調：先看日主是否得月令，再看格局用神。

```text
日主得令：先天有氣，身強機率高。
日主失令：先天氣弱，需看印比、地支根氣補救。
```

但不能只看得令失令：

```text
得令但無根，也可能虛浮。
失令但有根、有印比，也未必弱。
```

---

# 四、有形與暗藏都要看

喜忌篇提到，有形透出與暗藏伏藏都要列入判斷。

```text
天干透出：外顯、明面、容易被看見。
地支藏干：內在、根氣、暗中力量。
暗拱、暗夾、暗沖：必要時作輔助參考。
```

注意：暗藏不是一定比明透強，而是代表「未顯但有根」。是否有力，要看月令、刑沖合會與大運流年是否引動。

---

# 五、小局與大局要互相印證

詳解定真訣資料提醒：不能只看局部十神，也不能只看全局口號。

```text
小局：某一組十神互動，如財官印、食神制殺、傷官配印。
大局：整體五行氣勢、格局成敗、寒暖燥濕。
```

實戰上：

```text
小局成立，要看大局是否支持。
大局有氣，要看小局是否有用神落點。
```

# 六、後續解盤輸出格式

每次看八字可照以下格式輸出：

```text
一、排盤確認
二、日主與月令
三、天干透出與地支藏干
四、日主強弱
五、格局判斷
六、用神、喜神、相神、忌神
七、成敗救應
八、財官印食傷比劫重點
九、大運流年影響
十、問題類型分析
十一、具體建議
```

```text

先明透，再藏干。
先大局，再小局。
先用神，再喜忌。
先原局，再大運流年。
```
