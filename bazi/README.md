# bazi：八字資料系統

本資料夾存放八字命理資料，與既有六爻資料分開管理。

## 資料夾結構

```text
bazi/
├── 00_index/              八字內部索引
├── 01_foundation/         八字基礎概念
├── 02_ten_gods/           十神系統
├── 03_stems_branches/     天干地支、藏干、刑沖合害
├── 04_pattern_structure/  格局、旺衰、用神喜忌
├── 05_luck_cycles/        大運、流年、流月
├── 06_question_types/     八字問事分類
├── 07_reading_templates/  八字解析模板
├── 08_case_archive/       八字案例庫
└── 99_sources/            八字來源資料
```

## 使用原則

八字主要用來看長期結構，包括性格、感情模式、事業方向、財運結構、健康體質、大運流年與人生策略。

若是單一事件成敗，優先使用六爻；若是長期命盤傾向，優先使用八字；若是重大選擇，可用八字看底層方向，再用六爻看當下事件成敗。
