# 六親妻子與人生問事

## 資料定位

本篇整理使用者提供的「論妻子、宮分用神配六親」等資料，用於八字問事分類。

## 一、八字六親基本定位

八字六親可由十神與宮位兩個角度合看。

### 十神角度

```text
財星：男命妻緣、財務、現實資源。
官星：女命夫緣、婚姻規範、名分與關係承諾。
食傷：子息、才華、表達、晚輩。
印星：母親、長輩、學習、庇護。
比劫：兄弟姊妹、朋友、同輩、競爭者。
```

### 宮位角度

```text
年柱：祖上、長輩、早年環境。
月柱：父母、家庭背景、青年環境、事業根基。
日柱：自己與配偶宮。
時柱：子女、晚年、成果、後代。
```

## 二、論妻子與婚姻

傳統子平法中：

```text
男命以財星為妻星。
女命以官殺為夫星。
日支為配偶宮。
```

### 男命看妻緣

```text
財星清而有根：妻緣較明。
財星被比劫奪：感情競爭、破財、妻緣受損。
財多身弱：欲望多、財妻壓力重，未必能承受。
日主強能任財：較能掌握財與婚姻資源。
```

### 女命看夫緣

```text
官星清純：婚姻名分較正。
官殺混雜：感情對象混雜、關係不穩。
傷官破官：對婚姻規範不滿，容易衝突。
印星護官或制傷：可緩和婚姻衝突。
```

## 三、子息判斷

食神、傷官常作子息星。

```text
食傷有力且不被嚴重破壞，子息緣較明。
食傷受印重重制伏，子息表達或子女緣可能受限。
時柱為子女宮，需合看時柱喜忌。
```

## 四、行運對婚姻子息的影響

行運會引動婚姻與子息。

```text
男命行財運，容易引動感情、婚姻、財務事件。
女命行官殺運，容易引動對象、感情、婚姻名分。
行食傷運，容易引動子息、創作、表達，也可能衝擊官星。
行比劫運，男命需防奪財、感情競爭或破財。
```

## 五、不可機械論斷

婚姻不能只看一顆星。

必須合看：

```text
配偶星
配偶宮
日主強弱
格局用神
大運流年
合沖刑害
原局清濁
```

## 六、與六爻互參

```text
八字看長期感情模式與婚姻傾向。
六爻看某一位對象當下心態、關係發展、主動與否、短期應期。
```

例如：

```text
八字顯示感情容易焦慮、官殺混雜或財星受劫。
六爻再看某個具體對象是否真心、是否主動、是否有其他人。
```
