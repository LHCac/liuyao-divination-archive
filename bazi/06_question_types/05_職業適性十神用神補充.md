# 職業適性與十神用神補充

## 資料定位

本篇整理使用者補充的八字四柱資料，來源：

- 從八字看適合的職業

本篇歸入八字問事分類，用於職業、事業方向、用神對應職能與特殊格局職業傾向分析。

> 使用原則：職業判斷不可只看單一用神。需合看日主強弱、格局、用神喜忌、十神位置、大運流年、現實技能與個人興趣。

---

# 一、各用神對應職業

## 正官為用神

```text
公職。
官職。
公司重要人員。
管理者。
按部就班、制度明確、規範性強的職業。
```

適合特質：

```text
守規矩。
有責任感。
能承擔制度與職務。
適合組織內發展。
```

## 七殺為用神

```text
武術教練。
法律界。
軍警。
危機處理。
具權威與高壓性質的行業。
```

適合特質：

```text
抗壓。
果斷。
能處理危險與競爭。
需要專業權威。
```

## 正印為用神

```text
文學。
教師。
宗教。
慈善。
奉獻愛心。
安定性職業。
有驛馬星時，也可看外交、外派、跨地區交流。
```

適合特質：

```text
學習。
教學。
照顧。
文書。
知識傳承。
```

## 偏印為用神

```text
醫學。
學術研究。
副業兼職。
特殊技術。
玄學研究。
跨領域工作。
```

適合特質：

```text
獨立思考。
研究。
非典型路線。
專門技術。
副業或多重技能。
```

> 原資料提到偏印心性較刻薄，現代解讀應改為：偏印較理性、距離感強、思維特殊，不宜直接貼負面標籤。

# 二、特殊格局職業方向

## 殺印格

```text
書記。
企業高管。
制度內高壓管理。
專業資格與權責結合的工作。
```

## 傷官生財格

```text
技術要求高的商業。
專業變現。
創作變現。
顧問、技術服務、商業策劃。
```

## 食神制殺格

```text
權威性高的職業。
專業技術制壓力。
危機處理。
法律、軍警、醫療急難、工程攻堅。
```

# 三、職業判斷流程

```text
1. 先看命局格局與用神。
2. 看用神是哪一個十神。
3. 看用神落在哪一柱、是否有根。
4. 看大運是否扶持該用神。
5. 看食傷是否代表技能輸出。
6. 看財星是否代表變現方式。
7. 看官殺是否代表制度、壓力與職位。
8. 再結合現實學歷、技能、興趣與工作環境。
```
