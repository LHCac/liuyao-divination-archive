# 健康疾病與五行病象補充

## 資料定位

本篇整理使用者補充的八字四柱資料，來源包含：

- 五行對應臟腑與疾病
- 八字看疾病的基本原則

本篇歸入八字問事分類，用於健康、疾病、五行病象與大運流年引動的輔助判斷。

> 重要提醒：八字健康判斷只能作象意參考，不能替代醫師診斷、檢查與治療。若有身體不適，應以醫療專業為準。

# 一、五行對應臟腑與疾病

## 木

```text
木：肝、膽、筋骨、四肢、關節、筋脈、眼、神經。
木過旺或過衰，容易在肝膽、頭頸、四肢、關節、筋脈、眼睛、神經方面有敏感象。
```

## 火

```text
火：心臟、小腸、血脈、循環系統、臉部、牙齒、腹部、舌頭。
火過旺或過衰，容易在心臟、小腸、肩、血液、臉部、牙齒、腹部、舌頭方面有敏感象。
```

## 土

```text
土：脾、胃、腸、消化系統、肌肉、胸腹。
土過旺或過衰，容易在脾胃、腸胃、胸、肺、肚腹方面有敏感象。
```

## 金

```text
金：肺、大腸、氣管、呼吸系統、皮膚、鼻、喉嚨。
金過旺或過衰，容易在大腸、肺、喉嚨、皮膚、痔瘡、鼻氣管方面有敏感象。
```

## 水

```text
水：腎、膀胱、泌尿系統、腰、耳、下焦、生殖泌尿。
水過旺或過衰，容易在腎、膀胱、腳、頭、肝、泌尿、陰部、腰部、耳朵、子宮、疝氣方面有敏感象。
```

# 二、八字看疾病的基本原則

## 1. 五行代表五臟

```text
火：心。
水：腎與泌尿。
土：脾胃。
木：肝膽。
金：肺與呼吸。
```

> 原資料中「胃對應水」可能為筆誤，本資料庫依常用五行臟腑：土主脾胃，水主腎膀胱。

## 2. 偏印與七殺

```text
偏印：可作經絡、管道、特殊病因、隱性問題之象。
七殺：可作實際病痛、壓力、外來攻擊之象。
```

判斷時可看：

```text
偏印、七殺在哪一柱。
落在哪個五行。
是否為忌神。
是否受到大運流年引動。
```

## 3. 五行太多

資料提出：某五行三個或三個以上，該五行有病，同時它所剋的五行也可能有病。

實戰轉化：

```text
某五行太旺，容易出現該五行過盛之病象。
該五行所剋的五行也會受壓。
例如木太旺，木本身肝膽筋脈象過強，也可能剋土，影響脾胃消化。
```

## 4. 生多為剋

```text
生扶太過，反成壓力。
某五行太旺，去生另一五行，若被生者太少或無力承受，也可能成病。
```

例如：

```text
水太多生木，木無根則水盛木漂。
土太多生金，金弱則土厚金埋。
```

## 5. 五行缺少

```text
命局完全缺某五行，該五行對應系統可能較敏感。
```

但不可直接斷病，需看：

```text
該五行是否為用神。
是否在藏干中存在。
大運流年是否補足。
是否有其他五行通關調候。
```

## 6. 通根受剋

```text
某五行雖有通根，但通根位受到周圍干支剋制太重，也容易出現病象。
```

這類常見於：

```text
用神有根但被沖。
根氣被合去。
根氣被刑害破。
大運流年再引動剋制。
```

## 7. 干沖與地支六沖

```text
天干相沖、地支六沖，會引動被沖五行。
若被沖的是用神或身體相關五行，病象更明。
```

## 8. 大運流年引動

```text
原局只是體質傾向。
大運流年才是觸發點。
同一病象會隨運勢變化而加重或緩和。
```

# 三、健康判斷流程

```text
1. 看日主強弱與五行偏枯。
2. 看哪個五行過旺、過衰、缺失。
3. 看偏印、七殺是否為忌，落在哪一柱。
4. 看用神是否受沖剋合化。
5. 看地支是否有六沖、三刑、自刑。
6. 看大運流年是否引動病象五行。
7. 結論只給保養方向與注意事項，不作醫療診斷。
```

# 四、本篇速記

```text
木看肝膽筋骨眼神經。
火看心血循環小腸臉舌。
土看脾胃消化肌肉腹部。
金看肺大腸呼吸皮膚鼻喉。
水看腎膀胱泌尿腰耳下焦。

七殺可看實際病痛。
偏印可看管道經絡與隱性問題。
五行太過、不及、缺失、受沖，都可能形成健康象。
原局看體質，大運流年看觸發。
```
