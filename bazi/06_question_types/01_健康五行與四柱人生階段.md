# 健康五行與四柱人生階段

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》整理頁：

- 五行與身體疾病：金主肺、土主脾胃、木主肝膽、火主血氣、水主腎臟
- 看命入勢：五行提綱，四柱分根苗花果

本篇歸入八字問事分類，用於健康象意與人生階段判斷。

> 重要提醒：命理健康分析只能作象意參考，不能替代醫師診斷、檢查與治療。

---

# 一、五行與身體疾病對應

## 核心概念

五行可對應身體部位與臟腑系統，用來觀察命局中可能較敏感、較需保養的方向。

## 五行臟腑對應

| 五行 | 身體系統 | 常見象意 |
|---|---|---|
| 金 | 肺、呼吸系統、大腸、皮膚 | 肺病、呼吸、皮膚、血疾相關象 |
| 土 | 脾胃、消化系統、皮膚、肌肉 | 消化問題、脾胃、皮膚、腿部病象 |
| 木 | 四肢、肝膽、筋、神經調節 | 四肢疼痛、抽筋、肝膽問題 |
| 火 | 血氣、心臟、眼睛、三焦 | 血氣、心火、眼睛、循環相關象 |
| 水 | 腎臟、泌尿系統、骨、下焦 | 腎、尿路、泌尿、腰腳相關象 |

## 常見判斷句

```text
金弱遇火炎：血疾、肺病、呼吸或皮膚象較明顯。
土虛逢木旺：脾胃消化、腸腹寒病象較需注意。
木弱遇金剋：四肢、筋骨、抽筋、肝膽象較需注意。
火逢強水剋：血氣不達、眼目、頭部循環象需注意。
水火不交：寒熱失調、上下不交、冷病象。
```

## 判斷原則

```text
某五行弱，又遇剋它的五行旺，對應身體系統較敏感。
某五行旺且為喜，對應系統較有力。
某五行旺而為忌，可能形成過盛、上火、壅塞或壓力。
```

## 實戰提醒

健康象意要合看：

```text
日主旺衰。
五行偏枯。
寒暖燥濕。
用神忌神。
大運流年是否引動。
是否有沖刑破害傷到相關五行。
```

不可只因某一五行弱，就直接斷病。

# 二、看命入勢：四柱分根苗花果

## 核心概念

四柱可分為根、苗、花、果，對應人生階段與人事宮位。

| 四柱 | 生命比喻 | 人事宮位 | 年齡範圍 |
|---|---|---|---|
| 年柱 | 根 | 祖上宮、祖先、家世、早年根基 | 0–16歲 |
| 月柱 | 苗 | 父母宮、幼年生長環境、青年基礎 | 17–32歲 |
| 日柱 | 花 | 自身、配偶宮、婚姻情感、壯年狀態 | 33–48歲 |
| 時柱 | 果 | 子息宮、晚年歸宿、成果、子女緣 | 49歲以後 |

## 基本用法

```text
年柱為根：看祖上、早年、家底、祖先資源。
月柱為苗：看父母、成長環境、青年發展、社會起步。
日柱為花：看自身狀態、婚姻情感、壯年生活。
時柱為果：看晚年、子息、成果、最終歸宿。
```

## 喜用落宮

```text
年柱為喜用：早年根基較好，家族或祖上助力較明。
月柱為喜用：父母環境、青年發展、職涯起步較有利。
日柱為喜用：自身與婚姻宮較穩，壯年較能展開。
時柱為喜用：晚年成果、子息、後期收穫較佳。
```

## 忌神落宮

```text
年柱為忌：早年或家族根基壓力較多。
月柱為忌：父母、青年、職涯起步或成長環境較有課題。
日柱為忌：自身狀態、婚姻與壯年生活需調整。
時柱為忌：晚年、子女、成果收束需多經營。
```

## 年運對應階段

資料中提到「年運走到哪一柱對應的階段，就是那階段發生之時」。

實務上可理解為：

```text
大運流年引動年柱，多應早年、家族、祖上、根基議題。
引動月柱，多應父母、職場、青年根基、成長環境。
引動日柱，多應自己、婚姻、伴侶、居住與壯年生活。
引動時柱，多應子女、晚年、成果、長期收成。
```

## 實戰流程

```text
1. 先定日主與格局用神。
2. 看喜忌落在哪一柱。
3. 用根苗花果判斷人生哪一階段較順或較有課題。
4. 看大運流年引動哪一柱。
5. 再合看十神與宮位，判斷事件類型。
```
