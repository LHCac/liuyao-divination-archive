# 太歲流年與歲君沖犯

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 太歲乃眾殺之主，入命未必為災，太歲論
- 歲傷日干禍身輕；日犯歲君災殃必重，年運剋沖論

本篇歸入大運流年系統，用於判斷流年太歲、流年天干與日主天干之間的互動。

---

# 一、太歲的本質

太歲是流年的年柱，是當年司令，也是一年中最強的時間力量。

```text
太歲為年之主令。
太歲能引動命局已有干支。
太歲本身不直接等於災。
吉凶取決於它引動的是喜用還是忌神。
```

## 太歲入命

若流年太歲與命局已有干支相同或強烈呼應，稱太歲入命或太歲引動命局。

### 太歲入命為吉

```text
太歲引動命局喜用神。
喜用得力。
五行得助。
相關宮位與十神被正面啟動。
```

### 太歲入命為凶

```text
太歲引動命局忌神。
忌神得勢。
刑沖戰鬥加重。
羊刃、七殺、傷官等凶性被引動而無制。
```

## 判斷原則

```text
太歲不是一律凶。
太歲只是放大命局已有傾向。
命局喜用被太歲引動，則吉。
命局忌神被太歲引動，則凶。
```

# 二、歲傷日干與日犯歲君

## 歲傷日干

歲傷日干，指流年天干剋日主天干。

古句：

```text
歲傷日干，有禍必身輕。
```

可理解為：

```text
年力在上，日力在下。
流年來剋日主，日主承受外來壓力。
若命局有救應，禍不重。
```

### 可能象意

```text
外在壓力。
制度壓力。
長輩、主管、環境壓力。
健康或情緒被年份氣場影響。
```

但仍需看日主強弱與用神喜忌。

## 日犯歲君

日犯歲君，指日干剋流年天干。

古句：

```text
日犯歲君，災殃必重。
```

其邏輯是：

```text
日主反剋流年之君。
臣犯君，容易形成衝撞、對抗、冒犯權威之象。
```

### 可能象意

```text
與制度衝突。
與主管、長輩、官方規則對抗。
容易冒進。
容易因逞強而招惹麻煩。
```

# 三、救應原則

資料提到：庚辛來傷甲乙，若丙丁先見，則甲乙安全。

可轉化為一般規則：

```text
流年剋日主，不必立即斷凶。
若命局中已有制化流年之神，則有救。
若流年之剋剛好剋忌神，反可能為吉。
```

例如：

```text
庚辛金剋甲乙木。
若命局有丙丁火制庚辛。
則金剋木之災可減。
```

# 四、流年判斷流程

```text
1. 先看流年天干地支。
2. 看是否與日主天干形成剋犯。
3. 看流年是否引動原局喜用或忌神。
4. 看是否有合、沖、刑、害、破。
5. 看命局是否有救應。
6. 看大運是否同時助吉或助凶。
7. 最後判斷事件落在哪個十神與宮位。
```

# 五、阿咪速記

```text
太歲入命未必為災，看喜忌。
歲傷日干：年份壓我，有壓力但不一定重。
日犯歲君：我去犯年份，較容易衝撞規則。
有救應就不必怕，無救應才看凶重。
流年吉凶必須合原局與大運，不可單句斷命。
```
