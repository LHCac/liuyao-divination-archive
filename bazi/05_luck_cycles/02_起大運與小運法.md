# 起大運與小運法

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，來源為《淵海子平透析》相關整理頁：

- 起大運法
- 行小運法

本篇歸入大運流年系統，用於排命盤時確定大運起運年齡、小運與幼年運勢參考。

---

# 一、起大運法

## 核心概念

大運是每十年一柱的運勢主軸，起運時間決定一個人何時進入第一步大運。

## 順逆規則

```text
陽男陰女：順數至下一個節氣。
陰男陽女：逆數至上一個節氣。
```

## 起運歲數計算

```text
天數 ÷ 3 = 起運歲數。
餘 1 天 = 4 個月。
餘 2 天 = 8 個月。
```

資料中另有古法說法：

```text
每一天折算一歲。
每一個時辰折算四個月。
```

實務上需依所採排盤系統統一換算。

## 大運排列

```text
每柱大運管十年。
依月柱干支順排或逆排。
通常排七至八柱大運供一生分析。
```

## 實戰提醒

現代排盤建議使用可靠排盤工具計算精確交節時刻，再以此校對起運時間。

---

# 二、行小運法

## 核心概念

小運是大運的補充，主要用於童年到大運起步前的每一年運勢。

## 起法

```text
小運從一歲起計。
男命從月柱地支順排。
女命從月柱地支逆排。
每個小運管一年。
```

## 使用定位

```text
推算幼年運勢。
解釋童年重大事件。
補足大運起步前的運勢空白。
與太歲流年同看。
```

## 現代使用提醒

現代命理多以大運與流年為主，小運可作參考，不宜單獨決定吉凶。

---

# 三、大運、小運、流年合看

```text
大運：十年大環境。
流年：當年觸發點。
小運：幼年或輔助細節。
原局：一切吉凶的根本。
```

判斷順序：

```text
1. 先看原局格局喜忌。
2. 看大運是否助用或傷用。
3. 看流年是否引動原局與大運。
4. 幼年階段可補看小運。
5. 不可只憑小運單獨斷大事。
```

# 四、阿咪速記

```text
陽男陰女順排，陰男陽女逆排。
起運看出生日至節氣的距離。
天數除以三，得起運歲數。
大運十年一柱。
小運一年一換，主要補幼年。
現代看命以原局、大運、流年為主，小運輔助。
```
