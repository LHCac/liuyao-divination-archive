# 行運取用總論

## 資料定位

本篇整理使用者提供的行運資料，包括財格取運、正官取運、印綬取運、食神取運、偏官取運、傷官取運、陽刃取運、建祿月劫取運、支中喜忌逢運透清、行運成格變格、喜忌干支有別等。

## 一、行運不是單獨看吉凶

大運流年必須回到原局判斷。

```text
原局喜什麼，行運來補，則吉。
原局忌什麼，行運引動，則凶。
原局格局未成，行運補足用神相神，可成格。
原局格局已成，行運破用神相神，可變格。
```

## 二、天干與地支要分開看

行運一柱有天干與地支，作用不同。

```text
天干：外顯、透出、事件表面、十神顯化。
地支：根氣、藏干、月令氣勢、深層結構。
```

同一大運不能只看干，也不能只看支，必須干支合參。

## 三、支中喜忌逢運透清

地支藏干如果是喜用，逢大運或流年透出天干，力量容易顯化。

地支藏干如果是忌神，逢運透出，也容易破格。

```text
藏用神逢運透出：格局發秀。
藏忌神逢運透出：格局受破。
```

## 四、行運成格與變格

### 成格

```text
原局缺相神，行運補相神。
原局用神藏支，行運透清。
原局有病，行運來藥。
原局日主弱，行運扶身承財官。
```

### 變格

```text
原局清純，行運混雜。
原局用神被行運合去。
原局相神被行運沖剋。
原局忌神被行運透出。
```

## 五、財格取運

財格取運要看日主能不能任財。

```text
日主強：喜財官旺地。
日主弱：喜比劫印綬扶身。
財多身弱：行比劫印綬運，反能承財。
財格忌比劫重重奪財。
```

### 提醒

```text
財旺不等於一定發財。
若日主太弱，財反成壓力、慾望、債務、資源不可承受。
```

## 六、正官格取運

正官格以官星清純為貴。

```text
喜財生官。
喜印護身。
喜身旺可任官。
忌傷官破官。
忌七煞混雜。
忌官星被合去。
```

## 七、印綬格取運

印綬格以印護身、官印相生為佳。

```text
喜官旺印旺。
忌財破印。
官煞透而混雜時，需看食傷制化。
身輕印重，財可制印但不可破格太過。
```

## 八、食神格取運

```text
食神生財：喜財旺、身旺。
食神格忌官煞破局。
食神太旺而煞輕，行印運可調和。
財要有根，才成食神生財之美。
```

## 九、偏官七煞格取運

```text
食神制煞：喜食神有力，忌印奪食。
印化七煞：喜印旺護身，忌財壞印。
財生煞：必須有食制或印化配合。
忌官煞混雜。
忌身弱不能任煞。
```

## 十、傷官格取運

```text
傷官生財：喜財運。
傷官配印：喜印運。
傷官旺而無財印：需看日主強弱，身弱宜扶身。
傷官格忌正官混入，尤其傷官見官無救。
```

## 十一、陽刃格取運

```text
陽刃喜官煞駕刃。
煞刃相停為貴。
行運宜官煞旺地。
財運可生煞，也可能成助力。
忌食傷剋盡官煞。
忌比劫再助刃旺。
```

## 十二、建祿月劫取運

```text
建祿月劫多為日主強旺。
喜財官食傷。
忌比劫印綬再助身。
比劫旺運防奪財、競爭、破財。
```

## 十三、外格取運

外格如從格、化格、一行格，成立條件嚴格。

```text
若外格成立，行運宜順其氣勢。
若行運逆其氣勢，易破格。
若原局不純，不宜勉強作外格。
```

## 十四、行運判斷流程

```text
1. 先定原局格局與用神。
2. 看用神有無相神輔助。
3. 看原局病處在哪。
4. 看大運天干透出何神。
5. 看大運地支是否補根或破根。
6. 看大運是否讓藏干喜忌透清。
7. 看流年是否引動大運與原局。
8. 最後判斷事件類型與吉凶程度。
```

## 十五、心性提醒

行運吉凶不是命運定死，而是環境五行與命局互動造成的機會與阻礙。

```text
吉運：謙遜守正，善用機會。
凶運：收斂慾望，修正習慣，避開忌神所象之事。
```
