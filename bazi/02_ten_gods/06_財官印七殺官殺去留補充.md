# 財官印、七殺與官殺去留補充

## 資料定位

本篇整理使用者上傳的八字四柱補充資料，主要來源：

- 財官印的喜忌：財生官、官印相生
- 印綬格的喜忌：印星被財壞印
- 官殺去留：去殺留官、官殺混雜
- 七殺格：食傷制殺與身旺任殺

本篇歸入十神系統，用於後續分析財官印結構、印綬格、官殺混雜與七殺格。

---

# 一、財官印良性循環

財、官、印若能形成順生，為命局貴氣的重要結構。

```text
財生官。
官生印。
印生日主。
```

這代表：

```text
財星提供資源。
官星形成名位與制度。
印星承接官星並護身。
日主能承接財官印的成果。
```

## 成格條件

```text
財星有根，不被比劫奪。
官星清純，不被傷官破。
印星有力，不被財星壞印。
日主能承受財官印循環。
```

---

# 二、財多身弱與財壞印

財旺不一定吉。

```text
日主身旺，財可為用。
日主身弱，財旺反成負擔。
印為用神時，財星太旺會剋印，形成財來壞印。
```

## 財來壞印的應象

```text
學業、資格、文書受阻。
長輩、母系、庇護資源受損。
精神支持下降。
健康元氣受損之象。
原本官印相生的格局被破。
```

## 印綬格喜忌

```text
喜官運：官能生印。
喜印運：印星有力護身。
忌財運：財剋印，若印為用則破格。
忌印星死絕又逢財剋。
```

# 三、官殺混雜與去留

正官與七殺同時透出，稱官殺混雜。格局不清，需去一留一。

```text
去殺留官：保留正官，去除七殺。
去官留殺：保留七殺，去除正官。
```

## 去殺留官

適用於正官清而可用、七殺為混雜破格時。

方法：

```text
合殺。
制殺。
沖殺。
化殺。
```

## 去官留殺

適用於七殺有力、有制化，正官反而混雜時。

方法：

```text
合官。
沖官。
制去官星干擾。
保留七殺成權。
```

## 關鍵

```text
留下來的官或殺必須有根有力。
去掉的那一方不能再被大運流年引動破局。
歲運也能完成去留，不限原局。
```

# 四、七殺格兩路：制殺與任殺

七殺格有兩條主要路線：

```text
食傷制殺。
身旺任殺。
```

## 食傷制殺

```text
七殺旺而有壓力。
食神傷官有力，能制殺。
日主不一定要極旺，但不能太弱。
```

喜：

```text
食傷有力。
殺有根但不過度。
行制殺運或扶身運。
```

忌：

```text
印星奪食，破壞制殺工具。
比劫奪食，削弱食傷。
財運生殺太過，殺反攻身。
```

## 身旺任殺

```text
日主極旺。
七殺不必完全制盡。
七殺可成為權柄、紀律、執行力。
```

喜：

```text
殺有力但不亂。
身旺能承。
財可生殺，但不可過度。
```

忌：

```text
身弱殺強。
官殺混雜。
財生殺太過。
殺無制化。
```

# 五、實戰判斷流程

```text
1. 看財官印是否形成順生循環。
2. 看日主能否承接財官印。
3. 看印是否被財壞。
4. 看官星是否被傷官破。
5. 看官殺是否混雜。
6. 若混雜，判斷去官留殺或去殺留官。
7. 看七殺是制殺路線還是任殺路線。
8. 看大運流年是否完成去留、制化或破格。
```

# 六、阿咪速記

```text
財官印順生是富貴主線。
財多身弱，財反成壓力。
印綬格最怕財來壞印。
官殺混雜，要去一留一。
七殺不是一定凶，制得住或任得起就是權。
制殺靠食傷，化殺靠印，任殺靠身旺。
```
