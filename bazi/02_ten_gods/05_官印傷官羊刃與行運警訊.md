# 一、官印辯證：名位需身旺能承

## 核心概念

官星代表名位、職責、制度與壓力。印星代表資格、庇護、文書、資源與承接能力。

```text
官旺身弱，名位反成壓力。
身旺逢官，才能承擔職責。
官印相生，才有名位與資格配合。
```

資料金句：

```text
名標金榜，須還身旺逢官星。
```

## 官貴太盛的危險

```text
官太旺而身弱，官全面剋身。
表面有名位，實際受制於責任與制度。
職位越高，壓力越重。
```

## 印被傷的危機

```text
財剋印：印失去生身與承官的能力。
傷官破官，又傷印：官印皆受損。
印被傷，仕途、學業、資格、庇護不穩。
```

# 二、官星正氣，忌見刑沖

## 核心概念

正官是命局的正氣之神，主社會地位、名聲、規矩、責任、紀律。

古句：

```text
官星正氣，忌見刑沖。
```

## 官星忌諱

```text
忌刑：三刑動官，主官非糾紛。
忌沖：六沖破官，主職位不穩。
忌合去：官被合化成他神，主地位或名分變化。
忌傷官破官：才氣衝撞制度。
忌官殺混雜：名分不清，壓力混雜。
```

## 保護官星

```text
印星護官。
財星生官但不可過旺混雜。
官星清純獨透。
日主身旺能任官。
大運流年不刑沖官星。
```

# 三、傷官行官運警訊

## 核心概念

傷官的本性是剋官。若命局以傷官為用，再行官運，容易形成用神與運勢衝突。

古句：

```text
傷官復行官運，不測災來。
```

## 可能應象

```text
口舌是非。
工作糾紛。
制度衝突。
官非訴訟。
職位不穩。
與主管、權威、規範對抗。
```

## 例外與救應

```text
傷官配印：印能制傷、護官，災減。
傷官生財：若財能通關生官，衝突可轉化。
官星為忌時，傷官制官反可能為吉。
```

所以不可看到傷官行官運就一律斷凶，要看官星是喜是忌、有無印財通關。

# 四、羊刃沖合歲君

## 核心概念

羊刃是日主帝旺之地，氣勢過剛，是雙面刃。

羊刃遇太歲沖合，容易引動劇烈變動。

```text
羊刃沖太歲：與歲君為敵，主意外、血光、官非、重大變動。
羊刃合太歲：合而不合，引動刑沖矛盾。
```

## 羊刃的危險與轉化

```text
羊刃無制：剛猛失控，易衝動、破財、傷身。
羊刃駕殺：身旺、刃強、殺有制，可成權柄。
羊刃合官：吉凶不定，需看合化是否有情。
```

## 羊刃行運提醒

```text
行運引動羊刃，需防衝動決策。
涉及手術、車禍、官非、衝突時要謹慎。
若有官殺制刃或印化刃，災性可減。
```

# 五、比劫與食傷喜忌

## 比劫

```text
身弱喜比劫：幫日主承擔財官。
身旺忌比劫：比劫奪財、分財、競爭加重。
財格逢比劫：需官星制比劫，或食傷轉化。
```

## 食神

```text
食神喜身旺，能洩秀生財。
食神格忌偏印奪食。
食神可制殺，化壓力為才華與策略。
```

## 傷官

```text
身旺傷官可洩秀。
傷官生財可把才華變現。
傷官見官若無救，易衝撞制度。
傷官配印，可化才華為學術、專業、名聲。
```
