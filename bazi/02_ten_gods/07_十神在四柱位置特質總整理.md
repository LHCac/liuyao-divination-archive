# 十神在四柱位置特質總整理

## 資料定位

本篇整理使用者補充的八字四柱資料，來源包含：

- 十神在四柱的特質與影響
- 十神在四柱的特質與影響（補充：偏財、偏印、比肩、劫財、偏官、正印）

本篇用途：用來判斷十神落在年柱、月柱、日柱、時柱時，對早年、青年、婚姻、中晚年、子女與事業的不同影響。

> 使用原則：四柱位置象意只能作輔助。實際解盤仍需先看月令、日主強弱、十神是否為喜用、是否受沖剋合化，以及大運流年是否引動。

---

# 一、四柱位置基本定位

```text
年柱：祖上、早年、家庭背景、父母長輩、童年環境。
月柱：父母宮、青年、兄弟姊妹、職涯起步、社會環境。
日柱：自己與配偶宮，中年、婚姻、核心關係。
時柱：子女、晚年、成果、後期生活與最終收成。
```

# 二、正官在四柱

## 年柱正官

```text
出身較受規範。
家庭管束較明。
正官為喜用，少年較易得名聲或長輩助力。
正官為忌，容易受家族限制，發展被規矩壓住。
```

## 月柱正官

```text
重面子、重規範、重責任。
女性可能較早遇到婚姻或穩定關係議題。
正官過旺剋身，壓力大。
失令但有印，可能受父母庇護。
```

## 日柱正官

```text
配偶條件或責任感較明。
正官為喜用，婚姻較穩。
正官為忌，婚姻或伴侶壓力較重。
```

## 時柱正官

```text
工作態度穩，容易被器重。
晚年重穩定與規範。
女性晚年可能得丈夫或穩定關係照顧。
正官有力為喜，晚年與事業成果較佳。
```

# 三、七殺／偏官在四柱

## 年柱七殺

```text
早年家庭壓力較明。
家庭條件未必穩定。
七殺過旺剋日主，需注意身心壓力。
```

## 月柱七殺

```text
冷靜理性、決斷力強。
能承擔重任。
若有食神制殺，表現突出。
若身弱殺重，青年壓力較大。
```

## 日柱七殺

```text
配偶個性強、行動力高、工作效率強。
七殺為忌，伴侶可能暴躁、冷漠或關係緊張。
```

## 時柱七殺

```text
個性耿直聰明。
若地支有印星或祿刃，又有食神制殺，可成高壓成就格。
晚年或子女宮帶七殺，需看制化是否得宜。
```

# 五、偏財在四柱

```text

偏財為忌，早年容易養成奢侈、貪玩、重享受的習慣。
```

## 月柱偏財

```text
有外財機會。
擅長掌握商機與人脈資源。
過旺容易因貪財、冒險或人情財陷入困境。
```

## 日柱偏財

```text
婚姻或感情容易有現實利益、外緣或忠誠度問題。
偏財為喜用，中年財運與事業較佳。
```

## 時柱偏財

```text
晚年可能因財與子女有矛盾。
偏財為喜用，晚年財富與享受增加。
```

# 十一、劫財在四柱

## 年柱劫財

```text
祖蔭較少，靠自身奮鬥。
劫財為忌，早年多挫折困難，與父親緣薄。
```

## 月柱劫財

```text
性格衝動好勝。
有攻擊性與競爭性。
過旺則工作社交衝突頻繁，感情也較不穩。
```

## 日柱劫財

```text
婚姻易出現感情問題或婚變象。
日支劫財為喜用，夫妻親密但仍有爭吵。
```

## 時柱劫財

```text
晚年與子女關係不融洽。
劫財為忌，子女不孝或溝通不良。
劫財為喜用，子女有才華，但與父母有代溝與矛盾。
```

# 十二、使用規則

```text
1. 先判斷該十神是喜用還是忌神。
2. 再看落在哪一柱。
3. 再看是否有根、有氣、有制化。
4. 再看大運流年是否引動該柱。
5. 不可只因某十神落某柱就直接斷定吉凶。
```
