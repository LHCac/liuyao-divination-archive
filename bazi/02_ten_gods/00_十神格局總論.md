# 十神格局總論

## 資料定位

本篇整理使用者提供的正官、偏官、印綬、食神、傷官、財、陽刃、建祿月劫等格局資料。

## 一、十神不是固定吉凶

八字十神常分為四吉神與四凶神，但這只是初步分類。

### 四吉神

```text
正官
正財、偏財
正印、偏印
食神
```

### 四凶神

```text
七煞
傷官
劫財
梟神、偏印在特定情境下可作凶神
```

### 核心觀念

```text
吉神非永遠吉。
凶神非永遠凶。
吉神配置失當，也會破格。
凶神制化有情，也能成貴格。
```

## 二、正官格

正官是剋制日主的十神，象徵法制、紀律、規範、秩序、職位與名譽。

### 正官格核心

```text
官星清純最貴。
喜財星生官。
喜印綬護身。
忌傷官破官。
忌七煞混雜。
忌刑沖破害傷官星。
```

### 財印並透

若財印並透且兩不相礙，可形成較高格局。

```text
財能生官。
印能護身。
官能成名位。
```

## 三、正官取運

```text
喜財印。
喜身旺之地。
忌食傷破官。
忌煞混雜。
官星透出不宜逢合去。
```

## 四、財格

財是我所剋之物，代表資源、金錢、妻緣、管理資源與現實利益。

### 財格核心

```text
財旺需身能任財。
日主強，喜財官旺地。
日主弱，喜印比扶身。
財格忌比劫重重奪財。
財太露，容易被劫。
```

### 財多身弱

```text
財多身弱，不一定真能得財。
行比劫印綬扶身之運，反而較能承財。
```

## 五、財格取運

```text
日強財格：喜財官。
日弱財格：喜比印扶身。
財多身弱：行比劫旺地反有用。
比劫太重：防奪財、破財、競爭。
```

## 六、印綬格

印綬是生助日主的十神，代表學識、文書、保護、長輩、資源、精神依靠。

正印與偏印皆可成格，不宜只用吉凶名義粗判。

### 印綬格核心

```text
官透為美。
財印不相礙為美。
身輕印重，可用財制印。
身旺印旺，不怕印多，只要官星清純仍可貴。
印制食傷，可保護官星。
```

## 七、印綬取運

```text
喜官旺印旺之鄉。
財運通常為忌，尤其財破印時凶。
若官煞透而競透，可喜食傷制煞。
```

## 八、食神格

食神是日主所生之神，代表才華、表達、享受、技能、產出與福氣。

### 食神格核心

```text
身強食旺財透，為美格。
食神生財，財要有根。
食神格忌官煞直接干擾。
食神可制煞，成食神制煞之局。
```

## 九、食神取運

```text
食神生財：喜財旺、身旺。
喜食傷之地，但不可過度混雜。
忌官煞破格。
食神太旺而煞輕，行印運反可調和。
```

## 十、偏官七煞格

偏官即七煞，是剋制日主的強力十神。

七煞不是一定凶，制化得宜反能成大格。

### 七煞格核心

```text
食神制煞為上。
印綬化煞次之。
財生煞必須有食制或印化配合。
忌官煞混雜。
忌身弱不能任煞。
忌財壞印。
```

### 七煞取運

```text
煞用食制：忌印奪食。
煞用印化：不喜財透破印。
財生煞：需食神或印綬配合。
```

## 十一、傷官格

傷官是日主所生之神，較食神猛烈，象徵才華、叛逆、突破、表達、技術、自由意識。

### 傷官格核心

```text
傷官生財者喜財運。
傷官配印者喜印運。
傷官旺而無財印者，要看日主強弱配置。
傷官最忌正官混入，形成傷官見官。
```

## 十二、陽刃格

陽刃屬日主極旺之氣，剛烈、強勢、競爭、衝動。

### 陽刃格核心

```text
陽刃喜官煞駕刃。
煞刃相停為貴。
行運宜官煞旺地。
財地可生煞，亦可有用。
忌食傷剋盡官煞。
忌比劫過旺使刃更重。
```

## 十三、建祿月劫格

建祿、月劫多為日主強旺之格。

### 核心

```text
月令為比肩或劫財。
日主本已強旺。
需另取財官食傷為用。
喜財官食傷。
忌比劫印綬再助旺。
```

## 十四、外格與雜格

外格如從格、化格、一行格，成立條件從嚴。

```text
必須氣勢純粹。
不可混雜。
稍有逆勢之神，可能不成外格。
成立後行運宜順其勢，忌逆其勢。
```

雜格如倒沖、井欄叉、刑合等，古法多有特殊說法，本資料庫暫列參考，實戰仍以正格、五行、生剋制化為主。

## 十五、十神使用總原則

```text
1. 不以十神名稱直接斷吉凶。
2. 先看此神是不是格局用神。
3. 再看它是否有根、有力、清純。
4. 再看是否有相神輔助。
5. 再看是否被忌神破壞。
6. 最後看行運是否助成或破壞。
```
