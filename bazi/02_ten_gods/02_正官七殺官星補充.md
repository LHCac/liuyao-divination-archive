# 一、正官定義與陰陽原理

## 核心概念

正官是與日主陰陽相異的剋我之神。

例如：

```text
甲木見辛金，辛為甲之正官。
乙木見庚金，庚為乙之正官。
```

正官象徵規範、制度、職位、名分、責任、管理、公務、社會秩序。

## 正官核心條件

```text
正官喜清純。
正官喜中和。
正官怕刑沖破害。
正官怕傷官破官。
正官怕七殺混雜。
正官太多則責任過重，反成壓力。
```

## 看命先看月令提綱

正官若在月令主氣，且為命局用神，稱為真神得用。

```text
月令正官為用，富貴較真。
行官旺之鄉，若不破格，事業較易成。
行傷官之地，或刑沖破官之運，需防破格。
```

## 正官需要中和

正官太弱，無力成制度與名位。

正官太多，責任過重，容易由官化殺。

所以正官最重中和：

```text
官星清而有力為佳。
官星太過或不及，都需通變。
```

# 二、月令正官性格申論

## 核心概念

月令正官，代表天命偏向公共事務、制度管理、服務大眾、承擔責任。

## 性格象意

月令正官者常見：

```text
守法依例。
服從制度。
自制力強。
重視規矩。
有責任感。
講求名分。
內斂保守。
不喜失序。
```

但若格局不佳或正官過重，也可能：

```text
死板。
無彈性。
少應變。
壓抑自己。
責任感過重。
```

## 職業象意

適合：

```text
公務體系。
大型機構。
行政管理。
法規制度。
合規工作。
教育管理。
社會服務。
```

## 正官見七殺

月令正官若見七殺混入，為官殺混雜。

```text
正官主正統制度。
七殺主壓力、權術、非常任務。
兩者混雜，天命容易不純。
```

需看是否有去官留殺、去殺留官，或有印食制化。

# 三、偏官即七殺：定義與制伏原理

## 核心概念

偏官即七殺，是與日主陰陽相同的剋我之神。

例如：

```text
甲木見庚金，庚為甲之七殺。
乙木見辛金，辛為乙之七殺。
```

七殺象徵壓力、挑戰、權力、危機、競爭、特殊任務、非常手段、魄力與決斷。

## 七殺不是絕對凶

七殺若無制，容易凶暴、壓身、帶來災禍。

七殺若制化得宜，反可成權貴、專業權威、臨危決斷之才。

```text
無制為殺。
有制有化，可作偏官。
```

## 制殺方式

常見制化方式：

```text
食神制殺。
傷官合殺或制殺。
印綬化殺。
羊刃駕殺。
合殺留清。
```

## 七殺格使命象

月令七殺，常代表天命給予壓力與磨練，使人增益韌性。

其人生常需面對例外事件、非常任務、壓力場域，並在高壓中建立專業權威。

# 四、七殺基本格局與制化要點

## 快速規則

```text
七殺只宜一位，多則須制。
喜身旺。
喜羊刃。
喜食神制殺。
忌身弱。
忌財生殺太過。
忌七殺太多而無制。
身弱殺強，宜印綬化殺生身。
月令七殺，最喜見食神。
```

## 七殺只宜一位

七殺為強烈壓力之星，一位有制，可成權。

七殺太多，壓力過度，必須有制伏或化解。

```text
一殺有制為權。
殺多無制為災。
```

## 身強身弱的差異

### 身旺殺旺

```text
能任殺。
壓力變動力。
危機變權柄。
磨練成專業。
```

### 身弱殺強

```text
壓力過重。
容易恐懼、焦慮、受制。
需要印綬化殺生身。
或先行扶身運。
```

## 財生殺的問題

財可生殺。

若身旺、殺有制，財生殺可形成權柄與資源。

若身弱，財生殺會使壓力更大。

# 五、月令七殺性格申論

## 核心概念

月令七殺者，性格多積極有膽識，機智、有權術、能決斷，遇壓力不亂。

## 正面特質

```text
膽識。
魄力。
決斷。
機智。
抗壓。
戰鬥力。
不怕困難。
能處理非常事件。
```

若身強殺旺且制化得宜，常可成：

```text
武職人才。
專業權威。
危機處理者。
管理強人。
法律、軍警、醫療急難、工程、專案攻堅類人才。
```

## 負面特質

若身弱殺強或殺無制：

```text
壓力難控。
脾氣急躁。
恐懼轉攻擊。
衝動。
容易走偏門。
容易被壓力吞沒。
```

## 羊刃與七殺

羊刃與七殺同時出現，威力倍增。

```text
身旺，羊刃七殺可成馭殺為權。
身弱，羊刃七殺刑沖太烈，傷害性大。
```

# 六、七殺命例解析實戰重點

## 七殺與偏官合看

資料中特別提醒：

```text
七殺即偏官，偏官即七殺。
不要機械分名稱。
有氣為用、制化得宜，可稱偏官。
為忌失控，則作七殺敗殺論。
```

## 日主自坐七殺

例如：

```text
甲申
乙酉
戊寅
己卯
癸未
```

日支是命主直接心性位置，若日主自坐七殺，常見反應靈敏、機智、決斷，但也需看是否身能任殺。

## 命例分析要點

```text
先看七殺在哪裡。
再看七殺是一位還是多位。
再看日主身強身弱。
再看制殺位置：食神、傷官、印、羊刃。
再看財是否生殺加壓。
再看行運是制殺、化殺、助殺，還是扶身。
```

# 七、官星太過：官多化殺

## 核心概念

官星太多則雜，正官過多會失去清純，轉為七殺壓力。

```text
官星太過則化殺。
官多責任重。
官多壓力大。
官多需制化或減輕。
```

## 官多的常見應象

```text
責任太重。
制度壓力大。
職務牽制多。
心理壓力重。
健康容易受壓力影響。
事業中規範與任務過多。
```

## 官多的制化

```text
食傷可制官殺壓力。
印可化殺生身。
比劫可幫身承壓，但不可混亂。
行運若再行官殺旺地，壓力加重。
行能制化官殺之運，反可舒解。
```

## 官多行官運

官已多，再行官運，不一定吉。

```text
若無制化，責任更重，易有災病禍事。
若有食傷或印綬制化，則可轉為職位與權責提升。
```

# 八、正官、七殺、官多速記

```text
正官：陰陽相異之剋，主制度、名分、職位、規範。
七殺：陰陽相同之剋，主壓力、危機、權柄、磨練。
正官喜清純，怕傷官與七殺混雜。
七殺喜制化，怕身弱殺重無制。
官多化殺，責任過重，需食傷或印綬制化。
身旺可任官殺，身弱先求扶身或化殺。
月令正官是真神得用，主公共事務與制度管理。
月令七殺是高壓磨練，主非常任務與專業權威。
```

## 九、後續解盤使用規則

```text
1. 先分正官與七殺，不混稱。
2. 再看是否月令主氣。
3. 看官殺是否清純或混雜。
4. 看日主能否任官殺。
5. 看有無食神制殺、印化殺、羊刃駕殺。
6. 官殺太過時，先判制化，再判吉凶。
7. 不可看到七殺就斷凶，也不可看到正官就斷吉。
```
