# 二、背景摘要

陳先生身體不適，懷疑有陰祟纏身，在課堂上占得此卦。

卦象顯示白無常鬼王奉旨索命，局勢凶險。筆者建議輔運化解，但陳先生遲疑未從。一週後頭部受傷，再一週後離世。

後續醫學報告顯示：陳先生罹患幻想症，因撞牆傷及頭部，最終敗血症致命。此與卦中「白虎啣刀、頭部受傷」之象相合。

# 五、鬼神憂疑占基本理論

本案沿用卦例 27 的鬼神憂疑占框架：

```text
世爻 = 自己
子孫 = 福神，制鬼、保護之神
官鬼 = 憂疑阻滯之宿
父母 = 契約、文書、鬼神與人之約定
妻財 = 功德經文、無形眾生精神糧食
```

官鬼判斷：

```text
官鬼旺相是神，休囚是鬼。
陽爻為神、男性靈體。
陰爻為鬼、女性靈體。
```

# 六、逐點解析

## 1. 子孫持世不一定保無憂

四爻申金青龍子孫持世。

傳統有「子孫持世可保無憂」之說，但本案是重要反例。

象意：

```text
當事人自恃有守護，認為外靈無法侵入。
心態上容易輕忽警訊。
子孫雖為福神，但若當事人自恃輕忽，福神亦難強制庇佑。
```

---

## 2. 伏神空亡真假判斷

二爻伏神巳火白虎父母，值辰巳空亡。

但巳火旺於庚午月。

```text
伏神巳火雖空亡，但旺於月建，並非真空。
待出空填實後，仍有力量。
```

規則：

```text
空亡若旺於月建，不可直接視為無力。
非真空者，出空後仍可發揮生剋沖合之力。
```

---

## 3. 官鬼身份：白無常鬼王

二爻寅木官鬼居二爻，持白虎，且得日辰亥水妻財生合。

象意：

```text
白虎官鬼 = 喪煞之鬼。
得日辰妻財生合 = 非尋常祟物。
```

結論：

```text
白無常鬼王奉旨索命，凶險異常。
```

---

## 4. 父母爻為契約

巳火父母伏於寅木官鬼之下。

鬼神占中，父母爻可代表契約、文書、承諾、鬼神與人之約定。

象意：

```text
白無常鬼王與陳先生曾有約定或未了心願。
伏神父母生官鬼，代表約定未履行，反而強化官鬼力量。
```

---

## 5. 二爻官鬼發動：索命之局

寅木白虎官鬼發動，具生剋沖合之力。

其作用：

```text
生伏神巳火父母，強化契約。
剋三爻辰土、五爻戌土兄弟。
衝擊世爻申金青龍子孫。
不受妻財生扶，不取錢財，也不接受經文超度交換。
```

結論：

```text
官鬼不受妻財生扶，表示錢財、經文、一般超度交換難以解決。
凶機已定，必須回到原始約定處理。
```

---

## 6. 變爻丑土：網開一面

二爻寅木官鬼動化丑土兄弟。

丑土作用：

```text
丑土生世爻申金，保住陳先生。
丑土合初爻、上爻子水妻財。
```

象意：

```text
白無常鬼王仍有網開一面之象。
若能履行約定，可用經文還願化解。
```

這是卦中僅存的轉機。

---

## 7. 沖空出空：伏神發揮力量

日辰亥水妻財沖二爻伏神巳火父母。

因巳火非真空，日辰沖空使其出空填實。

```text
沖空 = 出空有力。
伏神巳火父母開始發揮生剋沖合力量。
```

---

## 8. 日辰沖剋伏神父母：毀約背信

伏神巳火父母源自寅木官鬼所生，代表鬼王與陳先生之約。

日辰亥水妻財沖剋巳火父母。

象意：

```text
當事人得財後毀約背信。
契約受沖剋，鬼王索命之局形成。
```

---

## 9. 白虎啣刀：頭部受傷警示

五行金之墓庫在丑。

二爻變爻丑土持白虎，形成「白虎啣刀」之象。

```text
丑土為金墓，持白虎 → 凶器已備。
```

丑土與上爻子水剋合，剋三合七；上爻對應頭部。

預示：

```text
若未履行承諾還願，白無常鬼王會先傷其頭部，以輕傷示警。
```

後續陳先生頭部受傷，與卦象應驗。

# 七、一週後複占與死卦判斷

陳先生頭上貼紗布再回課堂，仍未履約還願，當場再占一卦。

複占卦象：

```text
解憂福神落入真空。
妻財官鬼群起夾擊。
爻象遍體鱗傷。
```

筆者判斷為死卦。

再一週後，陳先生離世。

---

# 八、死卦判斷標準

當一卦同時出現以下特徵，可判斷當事人生命危急或不治：

```text
1. 解憂福神子孫爻落入真空，無力制鬼。
2. 妻財、官鬼群起夾擊世爻，多面受敵。
3. 爻象遍體鱗傷，無一爻護持世爻。
```

此為本案新增的重要生死判斷規則。

---

# 九、身體部位判斷：剋三合七加爻位

某爻與另一爻形成「剋三合七」關係時，可依被合之爻位推斷受傷部位。

| 爻位 | 身體部位參考 |
|---|---|
| 上爻 | 頭部 |
| 五爻 | 肩頸 |
| 四爻 | 胸腹 |
| 三爻 | 腰腹 |
| 二爻 | 下肢 |
| 初爻 | 足部 |

本案：

```text
丑土合子水上爻，剋三合七。
上爻代表頭部。
因此預測頭部受傷，後續應驗。
```

# 十一、後續應用規則

日後遇到憂疑、鬼神、生死類卦：

```text
1. 子孫持世不可直接斷無憂，需看是否真有制鬼力。
2. 父母爻伏官鬼下，可看契約、承諾、未了約定。
3. 官鬼不受妻財生扶，表示錢財、經文、一般超度不易解決。
4. 官鬼動化之變爻若生世，表示尚有一線轉機。
5. 日辰沖剋伏神父母，可看毀約背信。
6. 白虎加金墓、丑土等象，可看刀傷凶器。
7. 剋三合七可配合爻位推身體部位。
8. 福神真空、財鬼夾世、遍體鱗傷，需高度警覺生命危急象。
```
