# 五、逐點解析

## 1. 世爻狀態：占卜者股性

五爻酉金勾陳子孫持世。酉金旺於己丑月，但在庚子日逢死絕。

象意：

```text
占卜者一直想投入股市。
但子孫休囚，表示心有餘而力不足。
想操作，但行動力、判斷力或信心不足。
```

## 2. 日辰臨妻財：市場趨勢

庚子日占卜，日辰子水為妻財。

```text
日辰持妻財 → 目前市場正值多方主流趨勢。
```

象意：

```text
股價高漲。
多方主流明顯。
市場氣勢由財爻主導。
```

## 3. 初爻子水發動：生應爻，剋伏神

初爻子水妻財發動，且與日辰同氣。

作用：

```text
子水生二爻應爻寅木玄武官鬼 → 上市公司獲利象。
子水剋伏神巳火父母 → 法令規範被壓制象。
```

因二爻伏神巳火父母逢空亡，為真空。

象意：

```text
主流趨勢超越法令限制。
股價高漲，主流股獲利象明顯。
官方法令或規範暫時無力壓制市場熱度。
```

## 4. 子孫爻力量：展望與推升力道

五爻酉金勾陳子孫為推升股價之源頭。

```text
旺於月建己丑月
休囚於日辰庚子日
```

象意：

```text
推升股價的力量存在，但不夠強。
股價指數可緩步推升，但後續上攻力道略顯疲弱。
```

# 六、實測驗證

從庚子日至封關日，共計 13 天。

筆者將期間 K 線圖對應卦象中的子孫、妻財爻及日辰地支。

結果：

```text
每日股價指數與卦象並無穩定對應關聯。
封關日指數為 33,707.83 點。
確實未達 34,000 點，但已接近。
```

結論：

```text
此卦能反映大趨勢，但無法穩定作為每日短線點位預測工具。
```

# 七、方法論邊界

## 適合用占卜輔助觀察的場景

```text
判斷整體市場多空趨勢方向。
檢視已擬定的投資思路是否符合當下氣場。
評估某公司、基金或投資標的的整體體質。
檢查自己是否適合介入，或是否心態不穩。
```

## 不適合的場景

```text
預測明天或下週具體點位。
作為期貨短線進出的訊號。
取代財經專業分析。
作為唯一投資依據。
```

核心觀念：

```text
股價是眾人投資行為的結果，屬市場集體意志。
占卜是問事者感應天地的意念顯現。
兩者屬不同維度，難以強行牽合為精準短線訊號。
```

# 九、後續應用規則

日後遇到股票、基金、指數、投資策略類六爻：

```text
1. 世爻看自己與操盤心態。
2. 應爻看投資標的或公司體質。
3. 日辰看目前市場主趨勢。
4. 妻財看股價、指數、收益象。
5. 子孫看推升股價的展望與源頭。
6. 兄弟看投資大眾，弱則散戶，強旺成群則主力群聚。
7. 官鬼、父母看官方、政策、法令與制度，居五爻更明顯。
8. 六爻可輔助檢視思路，不可作短線買賣唯一依據。
9. 財經占必須配合財經專業知識與風險控管。
```
