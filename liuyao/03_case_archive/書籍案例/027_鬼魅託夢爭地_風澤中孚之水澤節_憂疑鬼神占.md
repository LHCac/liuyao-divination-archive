# 二、背景摘要

筆者受邀鑑定張家祖墳風水，建議遷葬改向，由原本坐南朝北改為東南朝西北。

張家未回應，一週後筆者夢中見渾身泥濘鬼魅厲聲索地，驚醒後起卦占問吉凶。

後續驗證：當初造墳挖地時，曾發現一具古棺，其坐向正是東南朝西北，前任風水師將遺骨丟棄，張家祖墳實際佔用他人陰地。因此卦中「鬼魅索地」有實際來源。

後來張家依約遷葬、還地於鬼魅，並尊其為「十方尊者」。新墳定向巽山乾向兼辰戌，與古棺原向相合。後續張先生事業發展順利，累財甚豐。

# 九、逐點解析

## 1. 雙巳呈祥：先靈得道之徵

初爻青龍巳火父母、五爻白虎巳火父母，皆旺於丙午月。

```text
雙巳呈祥 → 先靈得道、位列仙班之徵
```

這表示張家祖墳相關先靈並非普通孤魂，具有較高層次或受供奉庇蔭之象。

---

## 2. 五爻巳火父母逢日沖：暗動

乙亥日沖五爻巳火父母，靜爻逢日沖為暗動。

暗動規則：

```text
暗動之爻有生剋之力
暗動之爻無沖合之力
```

五爻父母暗動，代表先靈有動作、有心意，但不是明動。

---

## 3. 仙靈心意：父母生世與兄弟

五爻巳火父母暗動，生世爻未土兄弟與三爻丑土兄弟。

鬼神占中，妻財象徵功德經文；兄弟爻剋妻財，可取爭奪擁有之慾。

此象顯示：

```text
仙靈欲藉筆者之手，引領陽世眷屬布施功德、虔誦經文，使無形眾生得以超昇。
```

---

## 4. 雙官鬼受財滋：惡鬼力旺

二爻朱雀卯木官鬼、上爻玄武卯木官鬼，同得日辰亥水妻財相生。

朱雀主咒詛口舌，玄武主陰私、不明死鬼。

```text
雙鬼並臨且受財滋 → 領有陰司符令，擅行咒詛傷人之惡鬼，勢力甚旺。
```

---

## 5. 上爻玄武官鬼發動：惡鬼圖謀

上爻玄武卯木官鬼發動，化子水妻財，形成回頭生。

上爻官鬼：

```text
生初爻、五爻巳火父母 → 侵占先靈位置
剋世爻未土與三爻丑土兄弟 → 威脅筆者與張家
```

象意：

```text
惡鬼意圖侵佔五爻白虎尊位掌權，並盤踞初爻青龍地盤為根基。
此非尋常超度可解，已構成明確陰界威脅。
```

---

## 6. 伏神子水：惡鬼暗樁

五爻伏神子水妻財，旺於日辰亥水，為可出之伏神。

此伏神正是上爻官鬼動化之變爻，形成陰陽呼應。

伏神子水作用：

```text
滋生玄武官鬼，暗助惡鬼
剋初爻、五爻巳火父母，阻斷仙靈
合三爻丑土兄弟，羈絆張家
```

象意：

```text
惡鬼已掌控樞紐，筆者、張家、先靈成敗皆受其牽制。
```

---

## 7. 福神真空：制鬼無力

三爻丑土兄弟下伏申金子孫。

申酉空亡，申金子孫為真空。

```text
子孫為制鬼福神，真空無力 → 官鬼肆虐而無制衡。
```

此象也顯示張氏一族未觸及真相核心，福蔭難以萌發。

---

## 8. 官鬼與父母重疊：陰位相侵

風澤中孚卦中，官鬼與父母有內外重疊之象。

本案重點：

```text
官鬼居上位、父母伏於其下，並非疊棺之相。
而是先靈誤佔惡鬼之位，或陰位相侵。
```

這解釋了後續實況：張家祖墳佔據古棺陰地，造成鬼魅爭地。

# 十、決策建言與實況驗證

## 建議

```text
不可只以驅鬼、超度解決。
應面對根源：還地、認錯、安置被侵占之陰靈。
```

## 後續做法

```text
事隔兩載，張家再邀筆者主庚遷葬。
依約還地於鬼魅，並尊其為十方尊者。
新墳定向：巽山乾向兼辰戌，合古棺原向。
```

## 結果

```text
自遷葬至丙午年，張先生事業勃發，累財逾十億。
認錯還地，怨懟化為庇蔭；惡緣反成福報。
```

核心啟示：

```text
心念轉化，從鬼魅到十方尊者；陰陽各安其位，則禍轉為福。
```

# 十二、後續應用規則

日後遇到憂疑、鬼神、陰宅風水類問題：

```text
1. 世爻看占卜者自己。
2. 官鬼爻看憂疑阻滯之宿。
3. 子孫爻看福神、制鬼保護之神。
4. 父母爻看先靈、文書、祖蔭、庇蔭。
5. 妻財在鬼神占中可看功德經文與供養資糧。
6. 官鬼旺相可視為有力量之神，休囚可視為弱鬼。
7. 六獸、五行、爻位可細分鬼神類型與位置。
8. 暗動只具生剋力，無沖合力。
9. 福神空亡則制鬼無力，需另尋解法。
10. 若卦象指向位置被侵占，重點不一定是驅除，而可能是歸還、安置、和解。
```
