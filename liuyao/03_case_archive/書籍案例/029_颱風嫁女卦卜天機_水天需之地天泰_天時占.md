# 二、背景摘要

庚寅年，筆者女兒出閣，精心揀選壬申日為嫁娶吉日。

氣象局突然發布凡那比颱風警報，預告婚期當日風雨直撲。筆者起卦占問後，判斷颱風會轉向，婚禮可如期舉行。

後續驗證：颱風驟然南偏，婚禮當日北部天氣轉好，婚禮圓滿完成。

# 五、核心規則：颱風卦必須指定地點

占天時，尤其是颱風、暴雨等區域性天氣，必須先指定地點。

原因：

```text
不同地點天氣不同。
卦象必須有明確空間基準。
若不指定地點，容易造成錯卦或解讀失準。
```

本案以婚禮行程與婚宴所在地作為判斷基準。

---

# 六、時間軸建立

占卜日為己巳日酉時，嫁娶佳期為壬申日。

中間經過：

```text
己巳日：占卜日
庚午日：婚禮前二日
辛未日：婚禮前一日
壬申日：婚禮當日
```

此案例使用「日辰推進法」，逐日觀察日辰地支對用神的生剋沖合，推斷天氣演變。

---

# 七、逐點解析

## 1. 颱風強弱：官鬼寅木

二爻寅木白虎官鬼代表颱風。

寅木官鬼：

```text
休囚於乙酉月。
於己巳日亦不得勢。
```

因此初判：

```text
颱風本體力量不大，或對指定地點影響有限。
```

至壬申日：

```text
申金日辰沖寅木官鬼。
```

沖代表變化、轉向。

結論：

```text
颱風會轉向，風勢轉弱。
```

---

## 2. 降雨強弱：伏神巳火父母

二爻伏神巳火父母代表雨。

己巳日占卜時，巳火即日辰，因此雨勢磅礴。

逐日變化：

```text
庚午日：午火同氣助巳火，雨勢仍強。
辛未日：未土洩火，雨勢減弱。
壬申日：申金剋巳火，雨勢微弱。
```

---

## 3. 婚禮當日天氣：子孫、妻財

壬申日，申金子孫旺相。

子孫為晴天，妻財為太陽。

象意：

```text
婚禮當日雨勢微弱，天氣可轉晴。
傍晚因火忌西方酉位，巳火受制，可能見短暫陽光。
```

# 八、日辰推進法示範

| 日期 | 日辰 | 對颱風官鬼寅木 | 對降雨伏神巳火父母 | 天氣推斷 |
|---|---|---|---|---|
| 己巳日，占卜日 | 巳火 | 巳火稍助寅木 | 巳火旺，雨勢磅礴 | 風雨俱強 |
| 庚午日 | 午火 | 午火生寅木，助颱風 | 午火同氣，雨勢仍強 | 雨勢強 |
| 辛未日 | 未土 | 未土洩寅木 | 未土洩火，雨勢減弱 | 漸轉好 |
| 壬申日，婚禮 | 申金 | 申金沖寅木，颱風轉向 | 申金剋巳火，雨勢微弱 | 天晴 |

關鍵規則：

```text
日辰沖官鬼 = 颱風變化、轉向或減弱。
```

# 九、應期逐時推算

本案例可細化至時辰層次。

壬申日婚禮當日：

| 時段 | 天氣推斷 | 技法依據 |
|---|---|---|
| 卯時出發 | 風勢猶勁 | 卯木短暫助寅木官鬼 |
| 辰時抵北 | 陰天無雨涼風 | 辰土洩火，雨勢減弱 |
| 午間宴客 | 轉晴 | 午火生申金，子孫得令 |
| 傍晚酉時 | 可見斜陽 | 酉金制火，陽光短暫顯現 |

# 十、實況驗證

實際情況：

```text
己巳日占卜，三日後壬申吉期將至。
婚禮前兩日，親家翁正式來詢。
筆者依卦象篤定回覆：婚禮照常。
```

逐時驗證：

```text
卯時：自台中出發，風勢猶勁。
辰時：抵台北，陰天無雨涼風。
午間：宴客轉晴。
傍晚：可見斜陽。
```

結果：

```text
全部應驗。
颱風驟然南偏，南部釀災，女兒婚禮圓滿。
```

# 十一、核心學習要點

| 技法 | 說明 | 重要性 |
|---|---|---|
| 天時占六親對應 | 官鬼=雷霆霧電，父母=雨，兄弟=風晴雲，子孫=晴天，妻財=太陽 | 天時占基礎框架 |
| 日辰推進法 | 從占卜日到應期日，逐日觀察日辰地支變化 | 逐日判斷技法 |
| 日辰沖官鬼 | 沖颱風用神，代表颱風轉向或減弱 | 核心技法 |
| 颱風卦指定地點 | 地點不同，卦象對應不同 | 重要前提 |
| 伏神逐日衰旺 | 伏神父母為雨，隨日辰變化推雨勢 | 進階應用 |
| 子孫旺相 | 代表晴天到來 | 基礎印證 |
| 五行忌神時位 | 火忌酉位，傍晚可能見短暫陽光 | 細緻推算 |
| 應期逐時推算 | 可細化到卯時、辰時、午時、傍晚 | 高精度應用 |

# 十二、後續應用規則

日後遇到天氣、颱風、出行、婚禮天候類六爻：

```text
1. 必須先指定地點。
2. 官鬼看烏雲、雷電、霧、颱風與暴風雨。
3. 父母看雨勢。
4. 兄弟看風、晴雲與涼爽天氣。
5. 子孫看晴天。
6. 妻財看太陽。
7. 從占卜日至應期日，可用日辰推進法逐日判斷天氣變化。
8. 日辰沖官鬼，多主颱風轉向或力量改變。
9. 伏神可隨日辰衰旺判斷雨勢強弱。
10. 應期可進一步細化到時辰，但需配合地點與實際時間軸。
```
